# 待做（很后面才做）：轮回设定 — 读档记录 + NPC 识破轮回者

> 创建日期：2026-06-02
> 状态：**仅记录构想，很后面才做**。优先级低于战斗/生存、修炼节奏等核心系统。
> 提出背景：玩家重新读档时留下"轮回"痕迹，部分 NPC 在某些条件下能看出玩家是"轮回之人"，并配套相关菜单。

## 一句话需求

玩家重新读档（回到过去）会被系统记录为一次"轮回"；世界与 NPC 对此并非全然无知——
**某些 NPC 在某些情况下能识破玩家是轮回之人**，并据此产生特殊反应/剧情/菜单。

## 设定要点（待与世界观对齐后细化）

- **轮回 = 读档**：玩家从某个存档点重新开始 / 回退进度，视为一次轮回。
- **轮回记录**：系统持久化记录玩家的轮回次数、每次轮回的关键分歧点（在哪一天/哪个事件回退）、
  上一世发生过但本世尚未发生的"未来信息"。
- **NPC 识破条件（候选，待定）**：
  - 高境界 / 特殊神通（如某些占卜、问心、天机类能力）的 NPC 才可能察觉。
  - 玩家行为与"命中注定的轨迹"偏离过大（利用未来信息走捷径）时，更容易被察觉。
  - 与玩家关系深、记忆特殊的 NPC（如宿敌、道侣）在重逢时产生"似曾相识"。
  - 轮回次数越多，痕迹越重，越容易被识破。
- **被识破后的反应（候选）**：警惕/敬畏/敌视/试探/索取未来信息/结为特殊因果等，按 NPC 性格与立场分化。
- **世界观依据**：需到 `docs/世界观参考/` 查证是否有"轮回 / 重生 / 逆天改命 / 天机 / 因果"相关设定
  （仙逆、一念永恒、遮天等可能有可借鉴的"逆天改命/被天道窥见"母题）。**确认前不得自行编造**，
  若参考资料缺失需主动告知用户。
