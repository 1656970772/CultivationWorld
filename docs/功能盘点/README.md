# 功能盘点

> 最后更新：2026-06-07

本目录用于记录“当前代码实际做了什么、哪些只是有设计或数据底座、哪些还没有落地”的项目现状盘点。

盘点默认以 `apps/game/js/`、`apps/game/data/`、`apps/editor/` 和可运行工具为主要证据；设计文档只用于对照目标，不直接算作已实现功能。

## 文档

| 文档 | 用途 |
|------|------|
| `2026-06-07-当前代码功能实现盘点.md` | 当前项目功能实现总盘点，按首个功能基本齐全版范围和代码模块分类。 |

## 维护规则

- 新增盘点文档时，文件头部必须写“最后更新”日期。
- 如果用户补充“想要的功能点”，优先在本目录新增专题盘点或补充到当前总盘点的“待用户补充区”。
- 判断状态时区分：代码已实现、数据已配置、默认启用、真实模拟已验证、仅有设计。
