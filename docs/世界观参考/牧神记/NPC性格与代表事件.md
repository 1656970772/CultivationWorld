## 性格标签体系

> 说明：下列性格标签为项目统一游戏化抽象；具体 NPC 事实、代表事件与来源需在后续采集时逐条核验。

| 标签 | 行为含义 | 游戏 AI 倾向 |
|------|----------|--------------|
| 谨慎求生 | 避免无把握冲突，重视隐藏实力和退路 | 低风险探索、优先保命、延迟复仇 |
| 争强好胜 | 在同辈竞争中主动挑战，重视名声和胜负 | 高竞争、易触发切磋/挑衅事件 |

| 重情守诺 | 对亲友、师门或承诺保持长期责任 | 高忠诚、低背叛、愿意救援 |
| 贪婪夺宝 | 对稀缺资源高度敏感，愿意冒险夺取 | 高资源觊觎、易触发怀璧其罪 |
| 偏执复仇 | 将仇恨置于长期收益之上 | 追杀链、跨区域寻仇、冲突升级 |

| 宗门责任 | 以宗门利益、传承和秩序为优先 | 护宗、招徒、惩戒、资源调度 |
