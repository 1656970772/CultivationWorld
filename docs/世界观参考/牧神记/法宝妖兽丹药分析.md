## 设定摘要

《牧神记》的物品体系强调“物品不是孤立装备”。刀法绑定师承，宝卷绑定失落功法，玄刀绑定纪元怨念，丹药绑定坐骑和补给，灵药绑定大墟生态，兽血和天工堂绑定国家工程。

## 原作事实

- 杀猪刀等早期武器承载残老村师承，武器价值不只在材质，也在招式与身份记忆。
- 金书宝卷、斩神玄刀等高阶物品带有功法、天庭、赤明时代等历史线索，可能伴随反噬或解谜条件。
- 赤火灵丹可作为龙麒麟补给，丹药生产需要药材来源和炼制过程。
- 药师的药圃展示了“大墟采集 -> 移植种植 -> 防偷防害虫”的资源链。
- 妖兽既可成为敌对生态，也可修炼成妖和尚、坐骑或特殊势力成员。

## 游戏化抽象（我的判断）

- 我的判断：物品应有 `origin_era`、`bound_technique`、`risk_effect`、`resource_chain` 字段，避免只做成攻击/防御数值。
- 我的判断：丹药不应只用于玩家回血；应支持坐骑口粮、神藏修复、元神治疗、工程材料等多用途。
- 我的判断：妖兽资源要区分“猎杀掉落、驯化伙伴、修行同类、宗教势力成员”，避免统一写成怪物掉落表。

## 待确认问题

- 斩神玄刀、无忧剑、太阳船、月亮船等高阶物品的完整属性和所有权变动仍需逐章核验。
- 丹药体系缺少统一品阶证据，当前只确认赤火灵丹、灵丹药力、灵药炼丹等事实。
- 妖兽材料是否可普遍用于炼丹/炼器，需要更多原文案例支撑。
