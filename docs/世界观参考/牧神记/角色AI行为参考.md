## 设定摘要

角色 AI 可围绕“守护原点、认知升级、变法推进、规则破译、权力异化风险”建模。秦牧不是只追求战斗胜利的角色，其决策应受残老村关系、外界认知、延康制度、天庭压力和历史身份共同影响。

- 残老村九老为秦牧的亲情和师承原点。
- 灵毓秀早期提供走出大墟、认识外界和延康新时代的引导。
- 延康变法是制度化改革，不只是个人修炼。
- 天盟和天庭线显示理想与权力会互相转化。

## AI架构映射说明

《牧神记》的核心驱动力是"变革"，秦牧作为变革者型主角，其AI行为特征与《大道争锋》的棋手型形成鲜明对比：

- **BT层**：秦牧的核心应急模式是"观察规律 → 找漏洞 → 打破规则"，这要求BT系统新增一个"规则破译"行为节点，在常规反应（逃跑/战斗）之外提供第三条路径。
- **Utility层**：秦牧的权重曲线是"理想主义驱动"，初期权重偏向"解开身世"，中期"推动变法"，后期"建立新秩序"——权重转移的触发条件不是境界突破，而是"认知升级"。
- **GOAP层**：每一步变革对应一个可拆解的子目标：分析旧规则 → 创造新功法/术法 → 联合盟友 → 推翻旧势力 → 建立新秩序。

## 核心角色分析

### 1. 秦牧

#### 性格标签
乐观开朗 / 坚韧不拔 / 智慧超群 / 敢于打破规则 / 重情重义

#### BT层应急反应
| 触发条件 | 抢占行为 | 优先级 |
|---|---|---|
| 遭遇困难/危机 | 观察规律、找漏洞 | 最高 |
| 常规方法无效 | 尝试打破规则（改良功法、创造新术法） | 次高 |
| 危机解除 | 将经验转化为新的"规则"，固化存储 | 中 |
| 亲友受威胁 | 无视常规，直接爆发式出手 | 极高（覆盖常规） |

**核心逻辑**：秦牧的BT层有一个独特的"RuleBreaker"节点——它不是简单的战斗/逃跑二选一，而是会先扫描当前环境规则的漏洞，尝试以"非典型"方式解决。

#### Utility层权重打分

| 周期 | 目标 | 权重(0-1) | 触发条件 |
|---|---|---|---|
| 短期 | 解开身世之谜，守护大墟村镇 | 0.95 | 故事初期 |
| 中期 | 推动变法，打破古神垄断 | 0.8 | 进入延康国后 |
| 长期 | 人定胜天，建立自由修行世界 | 0.6(渐进至1.0) | 认知完全觉醒后 |

**权重转换规则**：秦牧的权重转移由"认知事件"触发——每当他发现一层世界真相，Utility权重就会向更宏大的目标转移。

#### GOAP层决策链

```
[离开大墟] → [探索外界，寻找真相]
    ├── 策略：感知外界召唤，突破大墟限制
    ├── 行动：进入延康国 → 结识各方势力 → 逐步揭开世界真相
    └── 产出：获得身份认知，结识核心盟友

[推动延康变法] → [打破修行垄断，提升人族整体实力]
    ├── 策略：联合延丰帝等改革派
    ├── 行动：创造新功法/新术法 → 对抗保守派与古神势力 → 逐步推进
    └── 产出：变法基础奠定，人族力量崛起

[对抗古神与天帝] → [以创新破传统]
    ├── 策略：分析古神/天帝的力量来源（规则）
    ├── 行动：创造克制性功法 → 联合被压迫势力 → 发动战争
    └── 产出：旧秩序瓦解

[建立新天庭] → [万物平等，自由修行]
    ├── 策略：制定新规则，打破血统论
    ├── 行动：击败旧天帝 → 整合各方力量 → 建立新天庭
    └── 产出：新时代开启
```

#### 标志事件

1. **离开大墟探索外界** — 打破认知边界，从"被保护者"到"探索者"。
2. **推动延康变法** — 以非典型方式改革，创造新功法打破垄断。
3. **对抗古神天帝** — 以"创新"破"传统"，推翻旧秩序。
4. **建立新天庭** — 人定胜天，万物平等，新世界诞生。

## NPC类型化归纳

### 1. 变革者型
- **行为逻辑**：以"创新"与"打破束缚"为核心驱动力
- **BT映射**：RuleBreaker节点 → 优先扫描规则漏洞而非走常规战斗/逃跑
- **Utility映射**：理想主义权重主导，认知事件触发权重转移
- **GOAP映射**：每个子目标包含"分析旧规则 → 创造新方法 → 推广落地"三阶段
- **实例**：秦牧、延丰帝身边改革派

### 2. 守旧者型
- **行为逻辑**：维护现有秩序（古神/天帝的统治），反对变革
- **BT映射**：秩序受挑战→抢占"镇压/抹杀变革者"行为
- **Utility映射**：稳定权重极高，变革容忍度为0
- **GOAP映射**：以"清除异端"为子目标，层层升级镇压力度
- **实例**：古神阵营、天帝旧部

### 3. 谋国者型
- **行为逻辑**：以国家或阵营利益为核心，擅长权衡利弊
- **BT映射**：局势变化→抢占"重新评估立场"→切换合作/制衡/对抗
- **Utility映射**：阵营利益权重 > 个人好恶权重
- **GOAP映射**：决策链以"势力平衡"为终点
- **实例**：延丰帝、各方诸侯

### 4. 守护者型
- **行为逻辑**：行为围绕"守护特定的人/物"展开，实力恐怖但隐藏
- **BT映射**：守护对象遇险→抢占"爆发真实实力"（覆盖隐藏伪装）
- **Utility映射**：守护对象存活权重 = 1.0（绝对最高）
- **GOAP映射**：日常行为以"隐藏/观察"为主，关键时刻切换为"碾压"
- **实例**：瞎子（村长）、天师等隐藏角色

### 5. 殉道者型
- **行为逻辑**：为理想（变法/守护苍生）最终牺牲自己
- **BT映射**：理想受致命威胁→抢占"自我牺牲"（最高优先级覆盖一切）
- **Utility映射**：理想实现权重 > 自身存活权重
- **GOAP映射**：牺牲节点作为子目标链的最终出口
- **实例**：变法中牺牲的改革者、守护大墟的前辈
