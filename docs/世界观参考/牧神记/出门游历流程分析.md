## 设定摘要

游历路线从“大墟白日探索和夜晚避难”起步，进入延康后转向制度、太学、变法和势力情报；再往后通过宝卷、遗迹、纪元线索进入天庭/古神/历史断代探索。路线设计应把补给、地图、避难、同行者和情报收益作为核心变量。

## 原作事实

- 大墟游历高度依赖时间，夜晚必须寻找村落、遗迹或石像庇护。
- 地图能标记遗迹并辅助避开黑暗。
- 外界延康拥有快速变化的法术神通与新时代变法动力。
- 龙麒麟同行需要丹药补给，游历资源会影响队伍状态。

## 一、离开大墟

### 出发原因
自身身世神秘，残老村的老人们无法解答身世之谜，需前往外界了解真相，同时探索大墟之外的世界，提升自身实力。

### 目的地类型
大墟之外的延康国——外界的第一个人类国度，拥有完整的修炼体系与势力分布。

### 同行者
- 独行离开大墟
- 途中偶遇司婆婆等残老村之人暗中保护
- 后期在延康国结识江南等同伴

### 可获取物
- 对自身身世的初步认知
- 延康国的修炼体系
- 多部基础功法
- 对外界的基本认知

### 六阶段流程

| 阶段 | 内容 |
|------|------|
| **准备** | 向残老村的老人们请教基础修炼知识与外界信息，携带残老村给的防身法宝，偷偷离开大墟 |
| **出发** | 绕开大墟的禁制与守将，秘密前往延康国，途中遭遇大墟外的妖兽袭击，靠残老村教的手段化解危机 |
| **途中事件** | 进入延康国境内后，因穿着怪异、行事不懂外界规矩，多次被当成奸细盘查，与当地修士发生小冲突，逐渐了解外界规则 |
| **到达** | 进入延康国都城，初步接触延康国的核心势力 |
| **探索** | 在延康国各处游历，了解延康国的历史与现状，学习延康国的修炼体系，寻找自身身世的线索 |
| **收获** | 初步了解自己的身世背景，掌握了延康国的修炼体系，获得了多部基础功法，对外界有了基本的认知 |
| **返回** | 短暂返回大墟，将外界信息告知残老村的老人们，随后再次离开大墟，深入延康国各地游历 |

## 二、延康各地游历

### 出发原因
推动延康变法，打破旧势力的束缚，寻找变法的盟友，同时收集各方势力的情报，提升自身实力。

### 目的地类型
延康国的各个城池、势力驻地——包括世家领地、宗门驻地、蛮族部落等。

### 同行者
- 与江白圭等变法派同伴组队
- 后期结识多位志同道合的盟友，共同推进变法

### 可获取物
- 变法的支持力量
- 各方势力的完整情报
- 不同势力的修炼功法
- 推动变法的资源

### 六阶段流程

| 阶段 | 内容 |
|------|------|
| **准备** | 制定变法的初步计划，了解延康国各方势力的立场与利益，准备应对旧势力打压的手段 |
| **出发** | 与同伴组队，离开都城前往延康各地，途中遭遇旧势力的刺杀，靠自身实力与同伴配合化解危机 |
| **途中事件** | 到达各地后，与当地世家、宗门打交道，说服部分势力支持变法，与反对变法的旧势力发生争斗，多次陷入生死危机 |
| **到达** | 走遍延康国的各个区域，接触到不同阶层的民众与势力 |
| **探索** | 深入了解各地的问题与需求，收集各方的情报，寻找变法的突破口，学习不同势力的修炼功法 |
| **收获** | 获得了大量势力的支持，收集到完整的延康国各方情报，完善了变法计划，自身实力也大幅提升 |
| **返回** | 回到延康国都城，联合所有盟友正式启动变法，与旧势力展开正式争斗 |

## 三、各大圣地/秘境游历

### 出发原因
获取各大圣地的传承以提升自身实力，同时对抗古神的阴谋，寻找对抗古神的方法，收集上古情报。

### 目的地类型
延康国及周边的人族圣地、上古秘境——内有上古传承、古神的阴谋线索、高阶修炼资源。

### 同行者
- 与变法派同伴、各大圣地的盟友组队
- 共同探索圣地与秘境

### 可获取物
- 各大圣地的核心传承
- 对抗古神的方法
- 上古的情报
- 高阶修炼资源

### 六阶段流程

| 阶段 | 内容 |
|------|------|
| **准备** | 收集各大圣地与秘境的情报，准备破禁法宝、对抗古神力量的丹药，与盟友约定好行动规则 |
| **出发** | 组队前往各大圣地与秘境，途中遭遇古神势力的袭击，合力击退敌人后继续前行 |
| **途中事件** | 进入圣地与秘境后，遭遇上古禁制与机关，与圣地守护兽战斗，发现古神留下的阴谋线索，与其他争夺传承的势力发生争斗 |
| **到达** | 深入圣地与秘境的核心区域，找到传承与古神阴谋的核心线索 |
| **探索** | 破解上古禁制，获取圣地传承，梳理古神的阴谋计划，收集对抗古神所需的资源 |
| **收获** | 获得各大圣地的核心传承，摸清了古神的阴谋布局，收集到大量高阶资源，自身实力突破至更高境界 |
| **返回** | 带着传承与情报返回延康国，联合所有盟友制定对抗古神的计划，为后续对抗古神做准备 |

## 四、天庭遗迹探索

### 出发原因
寻找上古天庭的传承，获取对抗古神与旧势力的最高阶力量，同时揭开上古天庭覆灭的真相，完善自身的道途。

### 目的地类型
上古天庭的遗迹，隐藏在诸天万界的隐秘之处——内有天庭的传承、上古神器、天庭覆灭的真相线索。

### 同行者
- 与所有志同道合的盟友组队
- 包括人族、妖族、蛮族等各方势力的顶尖强者
- 共同探索天庭遗迹

### 可获取物
- 上古天庭的核心传承
- 上古神器
- 天庭覆灭的真相
- 诸天万界的最高阶修炼资源

### 六阶段流程

| 阶段 | 内容 |
|------|------|
| **准备** | 收集天庭遗迹的所有情报，准备对抗上古禁制与古神残留力量的法宝与丹药，联合所有盟友，制定详细的探索计划 |
| **出发** | 带领所有盟友前往天庭遗迹的入口，途中遭遇古神势力与旧势力的最强围杀，展开惨烈战斗，最终突破围杀抵达入口 |
| **途中事件** | 进入天庭遗迹后，遭遇天庭残留的上古禁制与守护神兽，发现古神与旧势力早已潜入遗迹争夺传承，多次陷入绝境，靠盟友配合与自身智慧化解危机 |
| **到达** | 抵达天庭遗迹的核心区域，找到天庭传承与天庭覆灭的真相记录 |
| **探索** | 破解天庭的核心禁制，获取天庭传承，解读天庭覆灭的真相，收集天庭留下的上古神器与资源，与争夺传承的敌人展开最终决战 |
| **收获** | 获得上古天庭的核心传承，收集到多件上古神器，彻底摸清了上古天庭覆灭的真相与当前诸天的最大危机，自身证得最高境界 |
| **返回** | 带着天庭传承与所有盟友返回延康国，整合诸天万界的所有力量，发起对古神与旧势力的最终决战，建立新的诸天秩序 |

## 五、游戏设计启示

### 5.1 游历系统的递进式设计

《牧神记》的出门游历呈现出清晰的**五层递进结构**：

| 层级 | 对应事件 | 核心驱动 | 设计要点 |
|------|----------|----------|----------|
| 新手启蒙 | 离开大墟 | 好奇心驱动（身世之谜） | 从封闭新手村进入开放世界，逐步解锁世界观认知 |
| 区域探索 | 延康各地游历 | 目标驱动（变法推进） | 多区域串联，每个区域有独立势力与故事线，形成网状任务结构 |
| 副本深入 | 圣地/秘境游历 | 资源驱动（传承获取） | 线性副本 + 多方势力竞争，引入PvEvP元素 |
| 终局挑战 | 天庭遗迹探索 | 终极目标驱动（真相+力量） | 最高难度团队副本，多阶段Boss战，剧情高潮收束 |

### 5.2 六阶段流程的游戏化映射

将小说中的"准备→出发→途中事件→到达→探索→收获→返回"映射为游戏机制：

| 小说阶段 | 游戏机制 | 实现建议 |
|----------|----------|----------|
| **准备** | 行前配置 | 情报收集（解锁地图迷雾）、物资采购（丹药/法宝/消耗品）、队伍编成（选择同行NPC） |
| **出发** | 出城/传送动画 | 过场演出 + 世界地图选择目的地，可加入出发仪式感（NPC送行/祝福） |
| **途中事件** | 随机遭遇系统 | 路遇妖兽战斗、商人交易、支线任务触发、天气/地形事件、NPC随机对话 |
| **到达** | 区域解锁 | 新地图点亮、区域信息面板展示（势力/资源/危险等级）、安全点激活 |
| **探索** | 开放/半开放玩法 | 副本探索（解谜+战斗）、势力交涉（选择支持/反对）、资源采集、隐藏要素发现 |
| **收获** | 结算与成长 | 功法/法宝获取、境界突破演出、情报/认知值提升解锁新线索、声望/阵营值变化 |
| **返回** | 闭环反馈 | 回归据点后的剧情推进、NPC对话更新、新区域/新任务解锁、世界状态变化（如变法推进进度条） |

### 5.3 同行者系统的设计价值

《牧神记》中同行者的演变——从独行到组建多方联军——为游戏提供了丰富的同伴系统设计参考：

- **独行阶段（离开大墟）**：适合新手引导，重点教会玩家基本操作和世界观认知
- **小团队阶段（延康各地）**：引入同伴系统，每个同伴提供不同的战斗风格和剧情分支
- **联盟阶段（圣地/秘境）**：引入阵营/声望系统，不同盟友影响可进入的副本和获得的传承
- **联军阶段（天庭遗迹）**：终局内容，此前积累的所有盟友关系在此集中体现，形成"集结"式高潮

### 5.4 关键设计原则

1. **驱动力递进**：好奇心 → 目标 → 资源 → 真相，四种驱动力在不同阶段交替主导，避免玩家动机疲劳
2. **信息不对称设计**：主角在每个阶段都面对"知与不知"的张力（身世之谜 / 古神阴谋 / 天庭真相），适合作为剧情钩子
3. **势力博弈系统**：变法派 vs 旧势力、人族 vs 古神的二元对立，为玩家提供有意义的政治/阵营选择
4. **"出—归"循环**：每个游历事件都以返回为终点，返回后世界状态发生变化（残老村得知外界消息 / 变法正式启动 / 对抗计划制定），形成强烈的成长反馈
5. **传承收集作为核心成长线**：从基础功法到圣地传承再到天庭核心传承，构建清晰的"功法收集—境界突破"成长闭环
