# 秦牧修炼历程 · 时间线事件流

> 原著：《牧神记》 | 主角：秦牧 | 总计30个事件

### 事件1: 被残老村收养，天生凡体
【时间/地点】婴儿时期 / 大墟残老村
【经过】秦牧身为弃婴被残老村的残疾人收养。天生凡体无法修炼，但村民个个是隐世绝顶高手，潜移默化受强者气息熏陶。
【得失】获残老村众人庇护，心智坚韧，对力量充满渴望。
【境界】凡体（无法聚气）

### 事件2: 偷学霸体三丹功，开启修炼之门
【时间/地点】少年时期 / 残老村附近山林
【经过】偷窥瞎子演练枪法偷学霸体三丹功前三式。以为仍是凡体时体内产生气感，凡体实为被封印的特殊体质。
【得失】踏入修炼门槛，初步掌握霸体炼体之法。
【境界】炼体一层（开窍期）

### 事件3: 离开残老村，初入大墟外界
【时间/地点】十五岁成年礼 / 大墟边缘
【经过】遵从村长嘱托离开大墟寻找天圣教。初入江湖对世事一知半解，遭遇延康国练气士追捕狼狈逃脱。
【得失】踏入修行世界，获得社会认知，接触天圣教线索。
【境界】炼体三层（罡风层）

### 事件4: 加入天圣教，觉醒霸体本相
【时间/地点】延康国都附近 / 天圣教旧址
【经过】被天圣教众人误认为教主继承人。在祭坛上霸体血脉彻底觉醒，展现惊人修炼速度，融合天圣功与霸体三丹功。
【得失】获天圣教传承和忠诚下属，霸体觉醒。
【境界】七星境界

### 事件5: 探索大墟秘密，解锁老人过往
【时间/地点】初步站稳脚跟 / 大墟深处
【经过】返回大墟探索秘密，发现老人们并非天生残疾各有惊天过往和伤势。决心帮助治愈伤势恢复修为。
【得失】了解残老村过往，获得老人们的更深信赖。
【境界】七星境界中期

### 事件6: 博采延康国神通
【时间/地点】延康国游历 / 延康国各地
【经过】四处拜师偷学各派神通，天资聪颖过目不忘掌握多种主流神通。融合霸体功法创造独特战斗风格。
【得失】博采众长神通广大，战斗能力全面提升。
【境界】七星境界后期

### 事件7: 解开自身封印，修为暴涨

【得失】修为大幅提升，对自身和世界认知加深。
