## 设定摘要

修炼主线可概括为“神藏开辟 -> 元神成形 -> 神桥接续 -> 天宫构建 -> 道境补完”。《牧神记》不把境界只写成线性经验条，而是把人体、魂魄、神祇、天象和宇宙结构互相映射。

## 原作事实

- 灵胎神藏是早期修炼核心，秦牧的元气会在灵胎神藏中呈现多属性变化。
- 灵胎与魂魄的合一关系被明确用于解释元神，说明修炼进阶包含“身体内景”和“魂魄结构”的同步变化。
- 神桥是关键断点，原文多处围绕“神桥不断/连接神桥的法决”展开。
- 天宫并非单纯宫殿意象，开辟时要调动肉身诸神，与神藏序列存在承接。
- 道境不是替代天宫，而是补足天宫体系短板的另一套大道层级。

## 游戏化抽象（我的判断）

- 我的判断：境界系统应拆成 `realm_stage`、`inner_structure`、`soul_state`、`bridge_state`、`dao_depth` 五类数据，不宜只存一个等级字段。
- 我的判断：神桥可以做成“断裂/接续/稳定/超越”状态机，影响跨域移动、飞升路线和天宫构建。
- 我的判断：道境适合作为高阶角色的“神通品质树”，让同境界角色因道境理解不同产生战斗风格差异。
