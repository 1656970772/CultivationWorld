## 世界 World

### 世界实体

#### 宇宙模型：共形循环宇宙论
- 宇宙经历"大爆炸 → 演化 → 大寂灭"的循环过程
- 前一纪元的残骸构成下一纪元的"遗迹"
- 秦牧最终在第十七纪元解决终极冷寂问题

#### 地理结构：莲花状结构
- **中心**：原界大陆，包含延康国、大墟等核心区域
- **花瓣**：环绕原界的各大世界与秘境

**主要世界实体：**

| 世界/区域 | 类型 | 核心特征 | 主要产出/资源 |
|-----------|------|----------|---------------|
| **原界大陆** | 主世界 | 故事主要发生地 | 人族文明、变法成果 |
| **延康国** | 人族变法主舞台 | 伪装成国家的武道门派，推行"延康变法" | 灵兵、变法成果、人族天才 |
| **大墟** | 禁地/试炼场 | 夜晚有"黑暗"吞噬血肉；遍布上古遗迹与宝物 | 遗迹资源、古神残骸、神兵碎片 |
| **幽都** | 冥界/亡灵归宿 | 由土伯主宰，掌管亡灵轮回，规则极强 | 亡灵、魂力、生死法则感悟 |
| **玄都** | 星空/星轨中枢 | 由造物主开辟，掌管星辰运行与天宫体系 | 星辰之力、天宫资源 |
| **归墟** | 创造与毁灭之海 | 吞噬与喷吐物质，是宇宙循环的关键节点 | 混沌之气、世界本源物质 |
| **无忧乡** | 上古避难所 | 开皇时代为躲避天庭而建的独立空间 | 上古传承、高阶技术、避难文明 |
| **祖庭** | 造物主圣地 | 造物主文明的发源地，蕴含最高传承 | 造物主遗产、太始神卵、终极神通 |
| **天庭** | 古神统治中心 | 由十天尊掌控，视众生为资源 | 神权秩序、神通体系、天材地宝 |
| **混沌海** | 终极战场 | 宇宙寂灭与重生的混沌之地 | 混沌之气、成道机缘 |

#### 纪元体系
1. **龙汉纪**：古神建立"龙汉天庭"，人族初步修炼
2. **赤明纪**：人族改良功法，证明可与半神匹敌
3. **上皇纪**：人族提出"人命大于天"等理念，革命失败
4. **开皇纪**：秦业建立开皇神朝，变法失败后率众进入"无忧乡"
5. **第十七纪元（严康纪）**：故事主舞台，延康国发起变法
6. **史前纪元**：太易、太初、太帝等史前强者活动的时代

### 世界状态

#### 纪元阶段
- 当前处于第十七纪元中期，延康变法关键时期
- 天庭统治出现裂痕，反天庭势力逐渐壮大
- 宇宙循环进入关键节点，黑暗侵蚀加剧

#### 灵气/元气浓度
- 大墟：夜晚灵气稀薄，白天灵气充沛但危险
- 延康国：通过变法提升灵气利用率
- 天庭：灵气最浓郁，但被古神垄断
- 幽都：魂力充沛，适合魂修

#### 黑暗侵蚀程度
- 大墟夜晚：黑暗完全笼罩，吞噬一切生灵
- 宇宙边缘：黑暗逐渐侵蚀，威胁世界稳定
- 归墟：黑暗与创造交织，平衡点

#### 天庭统治强度
- 核心区域：绝对统治，神权秩序森严
- 边缘世界：统治较弱，存在反抗势力
- 大墟/幽都：天庭影响力有限

#### 变法推进度
- 延康国：变法全面展开，冲击传统秩序
- 天庭内部：保守派与改革派斗争
- 其他势力：观望或抵制

### 世界行为/规则

#### 宇宙循环规则
1. 大爆炸 → 演化 → 大寂灭 → 重生
2. 前一纪元遗迹成为后一纪元资源
3. 黑暗是宇宙循环的必然产物
4. 归墟维持创造与毁灭的平衡

#### 黑暗规则
1. 夜晚降临，黑暗吞噬血肉生灵
2. 黑暗具有侵蚀性，可污染灵气
3. 某些区域永久黑暗，需特殊手段抵御
4. 黑暗与光明交替，形成世界节律

#### 神藏/天宫修炼体系规则
**神藏体系（基础）：**
1. 灵胎境：开辟灵胎，奠定修炼基础
2. 五曜境：觉醒五曜神藏（心肝脾肺肾）
3. 六合境：开辟六合神藏（上下左右前后）
4. 七星境：对应北斗七星
5. 天人境：天人合一
6. 生死境：参悟生死轮回
7. 神桥境：搭建神桥，通往彼岸

**天宫体系（进阶）：**
1. 神藏境：开辟神藏，承载大道
2. 天宫境：建造天宫，神祇居所
3. 天庭境：建立天庭，统御万神
4. 道境：参悟大道，成就道果

#### 天庭统治规则
1. 神权至上：古神统治后天生灵
2. 血统论：半神高于人族
3. 资源垄断：控制灵气、功法、传承
4. 天条约束：制定严格的行为规范
5. 轮回管理：掌控生死轮回

#### 变法规则
1. 神为人用：打破神权垄断
2. 功法普及：推广人族修炼体系
3. 技术创新：发展灵兵、丹药、阵法
4. 制度变革：建立新秩序
5. 思想解放：挑战传统观念

#### 轮回规则
1. 幽都掌管亡灵轮回
2. 土伯维持轮回秩序
3. 轮回受因果业力影响
4. 强者可干扰轮回过程
5. 特殊手段可保留记忆转世

### 世界需求

#### 制造冲突
1. 天庭与后天生灵的统治矛盾
2. 传统秩序与变法革新的冲突
3. 资源分配不均引发的争夺
4. 不同理念势力的对立
5. 个人恩怨升级为势力斗争

#### 推动纪元演化
1. 维持宇宙循环的正常进行
2. 促进文明发展和技术进步
3. 筛选适应新纪元的强者
4. 解决纪元遗留问题
5. 为下一纪元做准备

#### 维持平衡
1. 光明与黑暗的平衡
2. 创造与毁灭的平衡
3. 神权与人权的平衡
4. 秩序与变革的平衡
5. 资源消耗与再生的平衡

#### 避免死局
1. 防止宇宙陷入永久黑暗
2. 避免文明彻底毁灭
3. 防止强者垄断一切资源
4. 保持多元势力并存
5. 为变数留出空间
