## 设定摘要

世界设定可拆为五层：大墟生存规则、延康人族制度、神藏/天宫修炼宇宙、天庭/古神规则层、龙汉至开皇等纪元历史层。下方已有“世界实体/状态/行为”结构可保留为游戏化框架，但其中后期宇宙论和资源数值必须继续补证。

## 原作事实

- 大墟夜晚黑暗是明确生存规则，遗迹和石像可影响避难成功。
- 延康具有国家化修行和门派化国家的双重性质。
- 龙汉、赤明、上皇、开皇等纪元构成历史断代。
- 天公、土伯等古神具备超常尺度和规则象征。

## 世界 World

### 世界实体

#### 宇宙模型：循环/寂灭线索（待核验）
- 宇宙经历"大爆炸 → 演化 → 大寂灭"的循环过程
- 前一纪元的残骸构成下一纪元的"遗迹"
- 后期是否存在“第十七纪元/终极冷寂”主线需继续补原文锚点，本条仅作待核验线索。

#### 地理结构：莲花状结构
- **中心**：原界大陆，包含延康国、大墟等核心区域
- **花瓣**：环绕原界的各大世界与秘境

#### 纪元体系
1. **龙汉纪**：古神建立"龙汉天庭"，人族初步修炼
2. **赤明纪**：人族改良功法，证明可与半神匹敌
3. **上皇纪**：人族提出"人命大于天"等理念，革命失败
4. **开皇纪**：秦业建立开皇神朝，变法失败后率众进入"无忧乡"
5. **延康时代/第十七纪元说法（待核验）**：故事主舞台，延康国发起变法
6. **史前纪元**：太易、太初、太帝等史前强者活动的时代

### 世界状态

#### 纪元阶段
- 当前以延康变法为主舞台；“第十七纪元中期”说法需继续核验。
- 天庭统治出现裂痕，反天庭势力逐渐壮大
- 宇宙循环进入关键节点，黑暗侵蚀加剧

#### 灵气/元气浓度
- 大墟：夜晚灵气稀薄，白天灵气充沛但危险
- 延康国：通过变法提升灵气利用率
- 天庭：灵气最浓郁，但被古神垄断
- 幽都：魂力充沛，适合魂修

#### 黑暗侵蚀程度
- 大墟夜晚：黑暗完全笼罩，吞噬一切生灵
- 宇宙边缘：黑暗逐渐侵蚀，威胁世界稳定
- 归墟：黑暗与创造交织，平衡点

#### 天庭统治强度
- 核心区域：绝对统治，神权秩序森严
- 边缘世界：统治较弱，存在反抗势力
- 大墟/幽都：天庭影响力有限

#### 变法推进度
- 延康国：变法全面展开，冲击传统秩序
- 天庭内部：保守派与改革派斗争
- 其他势力：观望或抵制

### 世界行为/规则

#### 宇宙循环规则
1. 大爆炸 → 演化 → 大寂灭 → 重生
2. 前一纪元遗迹成为后一纪元资源
3. 黑暗是宇宙循环的必然产物
4. 归墟维持创造与毁灭的平衡

#### 黑暗规则
1. 夜晚降临，黑暗吞噬血肉生灵
2. 黑暗具有侵蚀性，可污染灵气
3. 某些区域永久黑暗，需特殊手段抵御
4. 黑暗与光明交替，形成世界节律

#### 神藏/天宫修炼体系规则
**神藏体系（基础）：**
1. 灵胎境：开辟灵胎，奠定修炼基础
2. 五曜境：觉醒五曜神藏（心肝脾肺肾）
3. 六合境：开辟六合神藏（上下左右前后）
4. 七星境：对应北斗七星
5. 天人境：天人合一
6. 生死境：参悟生死轮回
7. 神桥境：搭建神桥，通往彼岸

**天宫体系（进阶）：**
1. 神藏境：开辟神藏，承载大道
2. 天宫境：建造天宫，神祇居所
3. 天庭境：建立天庭，统御万神
4. 道境：参悟大道，成就道果

#### 天庭统治规则
1. 神权至上：古神统治后天生灵
2. 血统论：半神高于人族
3. 资源垄断：控制灵气、功法、传承
4. 天条约束：制定严格的行为规范
5. 轮回管理：掌控生死轮回

#### 变法规则
1. 神为人用：打破神权垄断
2. 功法普及：推广人族修炼体系
3. 技术创新：发展灵兵、丹药、阵法
4. 制度变革：建立新秩序
5. 思想解放：挑战传统观念

#### 轮回规则
1. 幽都掌管亡灵轮回
2. 土伯维持轮回秩序
3. 轮回受因果业力影响
4. 强者可干扰轮回过程
5. 特殊手段可保留记忆转世

### 世界需求

#### 制造冲突
1. 天庭与后天生灵的统治矛盾
2. 传统秩序与变法革新的冲突
3. 资源分配不均引发的争夺
4. 不同理念势力的对立
5. 个人恩怨升级为势力斗争

#### 推动纪元演化
1. 维持宇宙循环的正常进行
2. 促进文明发展和技术进步
3. 筛选适应新纪元的强者
4. 解决纪元遗留问题
5. 为下一纪元做准备

#### 维持平衡
1. 光明与黑暗的平衡
2. 创造与毁灭的平衡
3. 神权与人权的平衡
4. 秩序与变革的平衡
5. 资源消耗与再生的平衡

#### 避免死局
1. 防止宇宙陷入永久黑暗
2. 避免文明彻底毁灭
3. 防止强者垄断一切资源
4. 保持多元势力并存
5. 为变数留出空间
