# 材料分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：材料、资源与产出源调研
> 目标详细程度：L3

## 适用范围

用于整理作品中出现过的材料、资源与产出源，包括但不限于灵石、灵脉、灵药、灵果、矿石、妖兽材料、妖丹、精血、骨骼、鳞甲、灵液、异火、空间材料、星辰材料、特殊天材地宝等。

本模板只做材料分析，不分析丹药成品、法宝成品、符箓成品、功法秘籍、装备、储物袋、飞舟等完整物品。上述成品只在“用途”中作为材料的下游去向被提到。

## 前置声明

本文只整理小说中出现过，或能由小说内容明确支撑的材料、资源与产出源，不编造小说中不存在的材料。

整理时应按材料分类和适用境界尽可能覆盖完整：每一类材料、每一个境界阶段，只要小说中有对应素材，都应尽量列入表格。文档长度不设上限，宁可写长，也不要为了压缩篇幅遗漏有参考价值的材料。

如果某类材料或某个境界阶段在小说中没有可靠依据，可以留空或不写；不得为了补齐分类、境界梯度、数值体系而自行虚构。

## 必须写什么

| 字段 | 写法要求 |
|------|----------|
| 材料名称 | 写小说中的材料名、资源名或产出源名称。 |
| 稀有度 | 按作品自己的描述填写，如最低级、较为常见、稀有、非常稀有、传说级等。 |
| 效果 | 写材料本身带来的效果，如增加真气、回复气血、延寿、强化肉身、提升炼器品质、产出灵石等。 |
| 用途 | 写材料被如何使用，如修炼、炼丹、炼器、布阵、交易、宗门资源、妖兽修炼等。 |
| 来源 | 写材料从哪里来，如矿脉、山脉、秘境、妖兽体内、天地凝聚、星辰坠落、宗门药园等。 |
| 适用境界 | 按作品自己的境界说法填写，如炼气、筑基、金丹、元婴，或该作品特有境界。 |
