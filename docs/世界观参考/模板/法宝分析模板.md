# 法宝分析模板

> 最后更新：2026-06-07
> 状态：模板
> 类型：法宝、秘宝、灵宝、仙器、神器与传承宝物调研
> 目标详细程度：L3

## 适用范围

用于整理作品中出现过，或能由小说内容明确支撑的法宝、秘宝、灵宝、仙器、神器、道器、传承宝物、本命法宝、储物类法宝、阵盘阵旗、傀儡类法宝等。

普通兵器、普通护具不放入本模板；只有具备独立法宝效果、祭炼认主、本命温养、器灵、禁制、成长、秘境钥匙、储物空间等属性时，才进入法宝分析。如果某件物品主要作为武器形制和战斗招式出现，优先放入 `武器分析模板.md`。

## 前置声明

本文只整理小说中出现过，或能由小说内容明确支撑的法宝、秘宝、灵宝、仙器、神器、道器、传承宝物、本命法宝等，不编造小说中不存在的法宝。

整理时应按形态、功效类型和适用境界尽可能覆盖完整：每一种法宝形态、每一类功效、每一个境界阶段，只要小说中有对应法宝，都应尽量列入表格。文档长度不设上限，宁可写长，也不要为了压缩篇幅遗漏有参考价值的法宝。

炼制类法宝必须写清炼制材料；非炼制类法宝在“炼制材料”中写“非炼制”或“原文未说明”，并在“形成/炼制方式”中写天地孕育、先天生成、古修遗留、秘境核心凝结、仙界流传等。不得为了补齐法宝品阶、境界梯度、效果类型或数值体系而自行虚构。

如果形态、稀有度、炼制材料、形成/炼制方式、来源、绑定/认主、限制/副作用等字段在小说中没有可靠依据，可以写“原文未说明”。

## 必须写什么

| 字段 | 写法要求 |
|------|----------|
| 法宝名称 | 写小说中的法宝、秘宝、灵宝、仙器、神器、道器、传承宝物、本命法宝等名称。 |
| 形态 | 写法宝载体或外形，如鼎、塔、钟、印、镜、旗、幡、葫芦、珠、瓶、图、卷、书、舟、阵盘、阵旗、傀儡、灵虫袋、储物袋、戒指、手镯、飞剑、甲胄、盾、伞等。 |
| 稀有度 | 按作品自己的描述填写，可包含作品内品阶或层级，如普通法器、稀有秘宝、宗门重宝、传说灵宝、仙器、神器等。 |
| 功效 | 写法宝本身带来的效果，如攻杀、防御、困敌、镇压、封印、飞行、储物、隐匿、传送、破禁、保命、替死、召唤、炼丹炼器加成等。 |
| 用途 | 写法宝被如何使用，如战斗、护身、逃遁、赶路、储物、布阵、炼丹、炼器、秘境开启、宗门镇压、传承考验等。 |
| 炼制材料 | 炼制类法宝写具体材料或通用材料规则，如星辰铁、五阶妖兽骨、火属性妖丹、空间晶石、阵纹材料等；非炼制类写“非炼制”；不明时写“原文未说明”。 |
| 形成/炼制方式 | 写法宝如何产生，如炼器师炼制、本命温养、血祭重炼、宗门炼器房打造、天地孕育、先天生成、古修遗留、秘境核心凝结、仙界流传等。 |
| 来源 | 写法宝从哪里获得，如宗门赐予、拍卖、遗迹获得、杀人夺宝、秘境奖励、传承获得、坊市购买、家族流传等。 |
| 绑定/认主 | 写滴血认主、神魂绑定、本命温养、器灵择主、传承限制、不可转让、可被夺取等；没有说明时写“原文未说明”。 |
| 限制/副作用 | 写修为限制、神识消耗、法力消耗、反噬、损毁、修复条件、器灵反噬、使用次数、环境限制等；没有说明时写“原文未说明”。 |
| 适用境界 | 按作品自己的境界说法填写，如炼气、筑基、金丹、元婴，或该作品特有境界。 |
