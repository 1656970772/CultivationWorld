# 鉴宝师模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：鉴宝、估价、辨伪、拍卖与情报判断调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的鉴宝师、估价师、拍卖行鉴定、宝物辨识、材料辨识、古物溯源、真假判断和宝物情报。采集时应尽可能覆盖各修为阶段、各物品品阶、各交易场景、各鉴定手段、各误判/作伪方式、各职业身份和各情报来源出现过的鉴宝案例。它服务物品识别、交易价格、拍卖玩法、NPC 职业、任务鉴定和探索收益。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 职业类型 | 鉴宝师、估价师、拍卖师、材料鉴定师、情报商 | NPC 副业 |
| 鉴定对象 | 法宝、丹药、材料、古物、功法、残片、秘境线索 | items、线索 |
| 鉴定依据 | 眼力、传承、经验、工具、气息、铭文、来历 | 行为条件 |
| 输出结果 | 真伪、品阶、用途、价值、风险、来源、买家建议 | 交易与任务 |
| 场景 | 拍卖行、坊市、黑市、宗门库房、遗迹、战利品分配 | 经济循环 |
| 失败后果 | 误估、买假、错失宝物、泄密、引来争夺、信誉受损 | risk、事件 |

## 尽量写全的内容清单

- 鉴定对象：宝物、材料、丹药、法宝、功法、残图、古器、血脉材料、秘境钥匙。
- 鉴定流程：观察、试探、查典籍、比对气息、估价、保密、出具结论、交易撮合。
- 工具与能力：神识、瞳术、鉴宝器、拍卖行库藏、老掌柜经验、情报网络。
- 经济场景：拍卖、典当、寄售、黑市、宗门兑换、战利品估值、任务委托。
- 风险链路：伪造、钓鱼、怀璧其罪、消息泄露、争夺、信誉崩塌。
- 项目映射：物品未知状态、鉴定 Job、价格修正、拍卖事件、情报系统、NPC 信誉。

## 不要写什么

- 不把鉴宝写成万能识别；必须记录误判和信息缺口。
- 不写无依据的固定价格表。
- 不忽略保密、信誉和交易风险。
- 不把主角捡漏经验当成普通鉴宝师能力。
