# 信息传播与情报模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：神识、传闻、情报、消息可信度与信息不对称调研
> 目标详细程度：L3

## 适用范围

用于整理作品中角色如何感知世界、获取消息、传播传闻、打探情报、误判局势。采集时应尽可能覆盖各修为阶段、各感知手段、各消息渠道、各传播距离、各势力情报网络、各可信度层级、各误导/封锁/滞后后果。它服务 `info-sense.md`、`news.json`、`InfoPropagationSystem`、NPC 已知消息、玩家日志和机会点决策。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 感知方式 | 神识、目击、占卜、传音、玉简、探子 | 信息来源 |
| 传播渠道 | 口耳、宗门、坊市、商会、悬赏、情报组织 | news channels |
| 可信度 | 亲眼、可靠消息、传闻、谣言、误导 | reliability |
| 信息影响 | 赶赴秘境、抢夺、逃亡、结盟、误判 | Utility |
| 信息边界 | 保密、封锁、假消息、距离衰减、身份伪装 | 有限视角 |

## 尽量写全的内容清单

- 感知：神识范围、阵法探测、灵兽侦查、命灯魂牌、传音符、玉简、天机推演、占卜、目击、跟踪。
- 传播：宗门公告、坊市传闻、拍卖消息、商会渠道、悬赏榜、通缉令、情报贩、弟子口耳、战报。
- 可信度：亲眼确认、同门消息、商会消息、敌方假情报、谣言、夸大、延迟、失真、距离衰减。
- 信息内容：宝物出世、秘境开启、强者陨落、宗门大战、通缉、拍卖、妖兽尸骸、身家暴露。
- 行为后果：参与机会点、提前准备、逃避追杀、抢先偷袭、误判风险、引发恐慌、结盟或背叛。
- 项目映射：`WorldNews`、`spreadRadius`、`reliability`、`knownNews`、`opportunityId`、日志标签确认/消息/传闻。

## 不要写什么

- 不假设 NPC 全知。
- 不把读者知道的信息当角色知道的信息。
- 不写无来源的情报网络强度。
- 不忽略假消息和延迟。
