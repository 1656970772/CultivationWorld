# 战斗与保命机制模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：战斗方式、伤害类型、逃生保命和战斗后果调研
> 目标详细程度：L3

## 适用范围

用于整理作品中战斗如何发生、如何结算、如何逃生、什么情况下死亡或锁血保命。采集时应尽可能覆盖各修为阶段、各战斗形态、各伤害类型、各人数规模、各地形/阵法条件、各逃生保命手段和各战后后果。它服务 `combat.json`、GAS、Ability、Effect、风险结算、PvP、妖兽攻击和动态事件战斗。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 战斗类型 | 法术、法宝、肉身、神魂、阵法、毒、围杀 | combat/risk |
| 伤害来源 | 攻击、防御、控制、诅咒、环境、天劫 | DamageSpec |
| 保命手段 | 遁术、替死、护符、锁血、传送、隐匿 | Ability/Effect |
| 战斗策略 | 偷袭、试探、越级、围攻、撤退、追杀 | AI 行为 |
| 后果 | 死亡、重伤、跌境、法宝损毁、结仇、掉落 | 记忆/关系/物品 |

## 尽量写全的内容清单

- 战斗方式：法术对轰、法宝碰撞、剑修、体修、神魂攻击、毒、诅咒、阵法困杀、傀儡、灵宠、群战、偷袭、暗杀。
- 伤害类型：物理、法术、神识/魂魄、毒、火雷冰风、空间、时间、血脉、真实伤害、持续伤害、环境伤害。
- 防御方式：护盾、甲胄、护体灵光、防御法宝、阵法、肉身、灵宠护主、师长护持。
- 保命方式：遁地符、传送符、血遁、替死、分身、元婴出窍、残魂逃逸、锁血、瞬移、隐匿、假死、空间法宝。
- 战斗策略：隐藏修为、先侦查、诱敌、以弱示强、围杀、风筝、消耗、爆发、打了就跑、放弃宝物止损。
- 碾压条件：境界差、法宝差、人数差、环境克制、被偷袭、无保命物。
- 后果：死亡、重伤、跌境、失去法宝、逃亡、复仇、通缉、资源掉落、消息传播。
- 项目映射：`applyDamage`、`Trigger.LethalDamage`、`State.Dying`、`Immune.Crush`、`ability_lock_hp`、风险键、战利品。

## 不要写什么

- 不只写战斗剧情，要拆成机制和后果。
- 不把主角越级胜利当作普通概率。
- 不给特定角色开无条件免死特权。
- 不忽略逃跑和止损；修仙战斗不应只有决斗。
