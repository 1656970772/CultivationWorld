# 炼器师模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：炼器、锻造、祭炼与装备生产调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的炼器师、锻造师、铸剑师、法宝炼制、装备修复、器灵、祭炼认主和材料加工。采集时应尽可能覆盖各修为阶段、各装备/法宝品阶、各材料体系、各炼制工艺、各职业分工、各生产设施和各失败/损耗/升级场景。它服务武器/法宝物品、装备生产、强化修复、NPC 副业、宗门工坊、市场交易和任务设计。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 职业类型 | 炼器师、锻造师、铸剑师、工坊供奉 | NPC 副业、身份标签 |
| 输入资源 | 矿材、妖兽材料、灵木、魂魄、火种、图纸、旧器 | 资源点、物品经济 |
| 工具设施 | 炉火、锻台、炼器室、器炉、阵纹工具 | 建筑设施、Job/Toil |

| 失败后果 | 材料报废、器物损坏、反噬、失控、品质下降 | risk、事件 |

## 尽量写全的内容清单

- 器物类型：武器、护具、飞行器、储物器、法宝胚胎、阵器、炼丹炉、傀儡部件。
- 材料体系：矿石、灵金、妖兽骨皮、灵木、异火、精血、魂魄、旧器残片。
- 炼制流程：选材、熔炼、塑形、铭纹、祭炼、温养、认主、修复、升级。
- 品质与限制：品阶、灵性、器灵、使用门槛、消耗、维护、损毁条件。
- 社会关系：宗门炼器堂、坊市工坊、私人定制、拍卖委托、战争军备。
- 项目映射：装备数据、物品耐久、强化/修复 Job、建筑设施、资源消耗、任务链。

## 不要写什么

- 不把所有武器都当法宝；普通装备和独立法宝要区分。
- 不写无依据的品阶表或成功率。
- 不忽略器物维护、修复和失败成本。
- 不把主角专属本命器直接抽象成通用规则。
