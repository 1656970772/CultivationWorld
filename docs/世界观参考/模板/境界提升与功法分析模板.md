# 境界提升与功法分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：修炼境界、突破、功法与寿元调研
> 目标详细程度：L3

## 适用范围

用于整理作品的境界体系、寿元、突破条件、功法品级、功法效果、瓶颈、天劫/心魔和失败后果。采集时应尽可能覆盖各修为阶段、各功法路线、各资质/体质条件、各突破资源、各瓶颈类型、各劫难形式和各失败代价。它服务 `ranks.json`、`techniques.json`、修炼平衡、突破行为、丹药/资源消耗和寿元系统。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 境界层级 | 名称、顺序、小阶段、寿元或战力跨度 | `ranks.json` |
| 突破条件 | 真气/灵力、丹药、心境、功法、环境、天劫 | 修炼与突破规则 |
| 失败后果 | 受伤、跌境、死亡、夺舍、寿元损耗 | 风险、GAS、死亡机制 |
| 功法体系 | 品级、属性、限制、修炼收益、代价 | `techniques.json`、效果 |
| 特例边界 | 主角外挂、专属功法、血脉限定 | 防止普适化 |

## 尽量写全的内容清单

- 境界：大境界、小境界、圆满/巅峰/半步/伪境、凡人武道阶位、妖兽阶位、神魂/肉身并行境界。
- 寿元：基础寿元、突破增寿、寿元衰减、延寿丹药、自然死亡、夺舍/转世对寿元的影响。
- 突破条件：灵力/真气/法力积累、心境、神识、肉身、功法、丹药、法宝护持、洞府环境、护法、天劫。
- 突破风险：失败受伤、跌境、经脉损毁、心魔、天劫死亡、魂灭、法宝损毁、被敌人趁虚而入。
- 功法类型：主修功法、剑诀、体修、神魂、炼丹、炼器、阵法、御兽、魔功、邪法、佛/儒/道/妖/魔体系。
- 功法属性：五行、雷、冰、风、空间、时间、血脉、毒、魂、火、剑、肉身、法则。
- 功法资源：秘籍、玉简、传承、师授、残篇、禁术、配套法宝、配套丹药、修炼环境。
- 数据映射：`rankId`、`order`、寿元桶、修炼速度、突破概率、突破资源、功法 effect、trait tag。

## 不要写什么

- 不把未经证据支持的寿元数字写成原作事实。
- 不把不同作品境界体系强行对齐。
- 不把“高阶功法更强”停留在结论，必须说明强在哪里。
- 不把功法名录做成无机制价值的列表。

## 详细程度

- 境界表必须有排序、寿元/战力/小阶段中的至少两项。
- 每个大境界突破至少写：条件、资源、风险、成功结果、证据。
- 可转化数据需写候选字段，如 `order`、`lifespan`、`successionScore`、`effects`。
