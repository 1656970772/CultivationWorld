# 出门游历流程分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：游历、探索、准备、收益与风险链路调研
> 目标详细程度：L3

## 适用范围

用于整理角色离开常驻地点去秘境、坊市、外海、战场、异域、禁地、飞升通道等地的流程。采集时应尽可能覆盖各修为阶段、各出行目的、各路线距离、各同行关系、各准备程度、各风险等级和各类目的地的游历样本，避免只记录主角远行。它服务移动系统、旅行风险、机会点、动态事件准备/参与、Job/Toil、信息传播和收益结算。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 出发原因 | 主动求资源、被追杀、宗门任务、事件窗口 | Utility 目标 |
| 准备阶段 | 丹药、法宝、情报、队伍、身份伪装 | Job/Toil |
| 路途风险 | 妖兽、截杀、天灾、迷路、传送异常 | movement/risk |
| 探索过程 | 侦查、战斗、采集、交易、撤退 | GOAP/Execution |
| 收益转化 | 物品、境界、关系、情报、机会点 | 经济、动态事件 |

## 尽量写全的内容清单

- 出发原因：任务、避祸、追杀、复仇、寻药、猎妖、交易、拍卖、拜师、试炼、秘境开启、飞升、换地图。
- 目的地：坊市、拍卖行、秘境、遗迹、禁地、妖兽巢穴、战场、宗门驻地、异域大陆、跨界通道。
- 准备物：丹药、符箓、法宝、飞行器、地图、令牌、情报、伪装身份、队友、护法、传送费用。
- 同行关系：单人、同门队伍、临时盟友、师徒、道侣、商队、护卫、灵宠、器灵。
- 途中风险：截杀、妖兽、迷路、天灾、空间裂缝、黑店、骗局、队友背叛、身份暴露、时间窗口错过。
- 探索行为：侦查、隐藏、交易、采集、破禁、战斗、夺宝、救援、撤退、疗伤、销赃、闭关消化。
- 收益：灵石、材料、丹药、法宝、功法、情报、关系、声望、境界突破、仇怨、动态事件参与记录。
- 失败/止损：空手而归、受伤、损失装备、被追杀、被抢、同伴死亡、转入复仇或避祸目标。

## 不要写什么

- 不只写“去了哪里拿了什么”。
- 不忽略准备与撤退；当前项目需要完整行动链。
- 不把所有游历都写成高收益，必须写失败和止损。
- 不把主角路线当成所有 NPC 默认路线。

## 详细程度

- 每个游历案例必须写：原因、准备、同行者、目的地、途中事件、收获、撤退。
- 如果可转化为动态事件，必须写事件阶段：scheduled、announced、active、resolved、expired。
- 如果可转化为 Job/Toil，必须拆出步骤。
