# 物资产出与消耗模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：经济循环、资源来源、消耗去向与通胀/保值机制调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的灵石、贡献、材料、丹药、法宝、妖兽掉落、矿脉、产业、坊市、拍卖、黑市、战争消耗和突破消耗。采集时应尽可能覆盖各修为阶段、各资源品类、各产出地点、各生产职业、各势力库存、各交易渠道、各日常/突破/战争消耗和各保值/贬值机制。它服务经济平衡、物品价值、势力资源、任务奖励、机会点和长程模拟。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 产出渠道 | 矿脉、任务、猎妖、秘境、贸易、掠夺、俸禄 | reward/economy |
| 消耗去向 | 修炼、突破、疗伤、法宝、阵法、任务、战争 | economy/cultivation |
| 资源层级 | 低阶/高阶、保值/贬值、地域差 | value、grade |
| 流通机制 | 坊市、拍卖、黑市、宗门兑换、以物易物 | 市场/组织 |
| 闭环 | 资源如何从世界进入 NPC/势力，再被消耗 | 长程模拟 |

## 尽量写全的内容清单

- 产出来源：灵矿、灵脉、药园、野外采集、猎妖掉落、秘境宝箱、任务奖励、宗门俸禄、拍卖收益、战争掠夺、贡赋、商贸套利、炼丹炼器代工、传承遗产。
- 消耗去向：日常修炼、突破、疗伤、法宝购买/维修/升级、符箓消耗、阵法维护、洞府租赁、传送、情报、悬赏、战争、宗门月俸、弟子培养。
- 资源种类：灵石、贡献、丹药、灵草、矿材、妖兽材料、法宝、功法、情报、声望、人情、弟子、凡俗人口。
- 流通场景：坊市、拍卖会、宗门兑换、黑市、商会、悬赏阁、战利品分配、以物易物、高阶私下交易。
- 保值/贬值：低阶资源通胀、高阶资源保值、地域价差、境界提升后的资源替换、稀缺材料刚需。
- 风险：财富暴露、黑吃黑、假货、劫道、税/贡、宗门克扣、交易欺诈、携带成本。
- 闭环：猎妖到上交、采药到炼丹、矿材到炼器、任务到贡献、贡献到晋升/兑换、财富暴露到抢夺。
- 项目映射：`economy.json`、`reward.json`、`covet.json`、`items/`、`quests/`、`opportunities.json`、势力资源。

## 不要写什么

- 不写没有来源的价格表。
- 不把游戏数值当原作事实。
- 不只写产出，不写消耗；没有消耗就无法形成平衡。
- 不把灵石、贡献、声望混成同一种资源。

## 详细程度

- 每种重要资源至少写：来源、消耗、持有者、流通方式、保值性、证据。
- 可转化经济规则必须写输入、输出、周期、风险、项目文件。
- 价格用相对梯度或区间建议，明确标注“我的判断”。
