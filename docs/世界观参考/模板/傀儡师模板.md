# 傀儡师模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：傀儡、机关、人偶、尸傀与自动作战单位调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的傀儡师、机关术、人偶、尸傀、战斗傀儡、炼尸、自动守卫和傀儡操控。采集时应尽可能覆盖各修为阶段、各傀儡品阶、各材料来源、各操控方式、各职业/势力使用场景、各战斗与生产用途出现过的傀儡条目。它服务召唤物、守卫单位、禁忌职业、物品生产、战斗机制和秘境机关设计。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 傀儡类型 | 机关傀儡、人偶、尸傀、战斗傀儡、守卫傀儡 | 单位分类 |
| 输入资源 | 材料、核心、尸身、魂魄、灵石、图纸、禁制 | 资源消耗 |
| 制作工具 | 工坊、炼器设施、阵纹、操控令牌、养尸地 | 建筑设施 |
| 控制方式 | 神识、令牌、阵法、魂印、指令、契约 | AI 行为 |
| 用途 | 战斗、守护、搬运、侦查、试炼、陷阱 | 系统映射 |
| 失败后果 | 失控、反噬、损毁、污染、伦理冲突、暴露身份 | risk、事件 |

## 尽量写全的内容清单

- 傀儡形态：人形、兽形、机关、尸傀、魂傀、战斗傀儡、守门傀儡、采矿/搬运傀儡。
- 制作要素：材料、核心、灵石供能、阵纹、符文、炼尸、魂魄、操控器。
- 控制流程：炼制、绑定、下令、维护、修复、升级、回收、失控。
- 能力边界：智能程度、行动范围、战斗等级、能耗、耐久、弱点。
- 社会风险：正道禁忌、邪修手段、战场消耗、秘境遗留机关。
- 项目映射：召唤单位、建筑守卫、可制造物品、战斗随从、事件陷阱、禁忌行为。

## 不要写什么

- 不把傀儡和普通法宝混为一谈；傀儡重点是可行动单位。
- 不忽略控制成本、维护和失控风险。
- 不把炼尸、魂傀等禁忌手段写成普通生产。
- 不给所有傀儡默认完整智能。
