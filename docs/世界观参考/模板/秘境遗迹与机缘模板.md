# 秘境遗迹与机缘模板

> 最后更新：2026-06-07
> 状态：模板
> 类型：秘境、遗迹、洞天、传承地与机缘事件调研
> 目标详细程度：L3

## 适用范围

用于整理作品中出现过，或能由小说内容明确支撑的秘境、遗迹、洞府、洞天、小世界、传承地、试炼地、禁地、古战场、宝藏地、机缘事件、天地异象、传承考验、特殊入口和限时开放区域等。

本模板只做秘境遗迹与机缘分析，不展开材料、丹药、功法、武器、法宝、妖兽或职业流程本体。秘境中的具体产出可在“机缘/产出”中说明，并交叉参考 `材料分析模板.md`、`丹药分析模板.md`、`功法术法神通模板.md`、`法宝分析模板.md`、`妖兽分析模板.md` 等。

## 前置声明

本文只整理小说中出现过，或能由小说内容明确支撑的秘境、遗迹、传承地、机缘事件和特殊区域，不编造小说中不存在的秘境体系。

整理时应按秘境类型、开启方式、进入条件、内部规则、危险类型、机缘产出和适用境界尽可能覆盖完整：每一种秘境/遗迹/机缘类型、每一个境界阶段，只要小说中有对应内容，都应尽量写成独立卡片。文档长度不设上限，宁可写长，也不要为了压缩篇幅遗漏有参考价值的秘境、遗迹或机缘案例。

如果某类秘境、某个境界阶段，或入口位置、开启条件、内部规则、机缘产出、危险、争夺机制、关闭/刷新规则等字段在小说中没有可靠依据，可以写“原文未说明”；不得为了补齐秘境品阶、境界梯度、产出类型或数值体系而自行虚构。

## 必须写什么

- 名称：写小说中的秘境、遗迹、洞府、洞天、小世界、传承地、试炼地、禁地、机缘事件等名称。
- 类型：写秘境类型，如宗门试炼地、上古遗迹、古修洞府、洞天小世界、传承地、禁地、古战场、资源秘境、妖兽巢穴、天地异象机缘等。
- 稀有度：按作品自己的描述填写，如常见试炼地、宗门掌控、百年一开、千年一现、传说遗迹、上古禁地等。
- 入口/位置：写秘境入口、地理位置、空间入口、令牌入口、阵法传送点、异象出现地等；不明时写“原文未说明”。
- 开启条件：写开启时间、钥匙、令牌、血脉、境界、阵法、祭品、天地异象、宗门名额、特殊事件等。
- 进入限制：写境界限制、人数限制、年龄限制、身份限制、血脉限制、阵营限制、进入代价等。
- 场景环境描述：写秘境内部的整体环境气质、天地灵气、地貌气候、光线、雾气、毒瘴、灵植分布、妖兽活动、空间稳定程度等；不明时写“原文未说明”。
- 场景布局描述：写秘境内部区域结构、层级关系、核心区/外围区、传送点、山谷、洞府、宫殿、塔楼、阵眼、禁制节点、路线关系等；不明时写“原文未说明”。
- 内部规则：写时间流速、空间规则、禁飞、禁法、压制境界、随机传送、区域层级、试炼规则、禁制规则等。
- 危险：写妖兽、阵法、禁制、机关、毒雾、空间裂缝、同门争夺、杀人夺宝、守关者、失控灾变等。
- 机缘/产出：写材料、丹药、功法、神通、法宝、传承、灵脉、妖兽、灵植、道果、身份资格、境界突破机会等。
- 争夺机制：写宗门争夺、名额竞争、拍卖钥匙、团队探索、试炼排名、杀人夺宝、守门势力、消息泄露等。
- 关闭/刷新规则：写开启周期、关闭时间、传送离场、入口消失、机缘刷新、遗迹崩塌、一次性机缘等。
- 限制/失败后果：写死亡、失踪、困死、重伤、境界压制、资源损失、被追杀、传承失败、空间崩塌等。
- 适用境界：按作品自己的境界说法填写，如炼气、筑基、金丹、元婴，或该作品特有境界。
