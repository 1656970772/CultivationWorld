# 夺舍设定分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：夺舍、附身、转生、复活与身体接管调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的夺舍、附体、重生、转世、元神寄托、器灵附身、身外化身等机制。采集时应尽可能覆盖各修为阶段、各神魂状态、各肉身来源、各施术关系、各成功/失败条件、各反噬代价和各身份后果的案例。它服务死亡后果、执念、邪修行为、师徒风险、GAS 状态、NPC 身份变更和长程事件。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 机制类型 | 夺舍、附身、复活、转世、器灵 | 事件分类 |
| 前置条件 | 元神、肉身、境界差、目标资质、法宝/阵法 | GOAP、GAS |
| 行动链 | 潜伏、诱骗、削弱、接管、融合、失败 | Job/Toil |
| 失败后果 | 反噬、魂灭、记忆丢失、肉身损坏 | 风险、死亡 |
| 伦理/关系后果 | 师徒背叛、仇怨、势力追杀 | 关系、邪修 |

## 尽量写全的内容清单

- 机制类型：夺舍、附身、残魂寄宿、器灵附体、转世、重生、复活、身外化身、傀儡身体、血脉重塑。
- 发起者：高阶修士、残魂、邪修、师傅、器灵、魔头、古修、妖修、被封印存在。
- 目标条件：资质、境界差、肉身强度、灵根/血脉、神魂弱点、亲近关系、是否被削弱。
- 媒介条件：法宝、阵法、丹药、禁术、魂牌、血契、寄魂物、尸体、傀儡、秘境环境。
- 行动链：潜伏、伪装、诱骗、培养目标、削弱目标、夺取身体、融合记忆、逃避追查。
- 反抗/失败：目标神魂反击、护身宝物、外力干预、反噬、魂灭、疯癫、身体崩坏。
- 后果：身份变化、记忆继承、关系背叛、师徒破裂、通缉追杀、复仇执念、寿元重置或不变。
- 项目映射：邪修行为、师徒风险、死亡后残魂事件、Obsession、GAS Tag、动态事件。

## 不要写什么

- 不把所有“复活”都称为夺舍。
- 不把主角转生设定当作普通 NPC 可用机制。
- 不忽略目标方的反抗条件。
- 不写成纯剧情奇观；必须拆机制条件和后果。

## 详细程度

- 每种机制至少写：主体、目标、媒介、条件、成功结果、失败后果。
- 可项目化时写清是否是执念、动态事件、Ability、Effect 或关系行为。
- 不确定机制名称时，先列为待核验。
