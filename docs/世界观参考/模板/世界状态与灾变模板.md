# 世界状态与灾变模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：世界修正器、时代变局、灾害与大环境压力调研
> 目标详细程度：L3

## 适用范围

用于整理会影响整个区域、势力或时代节奏的状态：灵气复苏、魔气上涨、兽潮、旱灾、瘟疫、王朝衰败、战争乱世、秘境潮等。采集时应尽可能覆盖各修为阶段受影响方式、各区域尺度、各灾变来源、各资源变化、各势力应对、各持续时间和各恢复/恶化结果。它服务 `world/modifiers.json`、稳定度、势力 AI、资源产出和动态事件。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 状态类型 | 灾变、灵气、魔气、战争、王朝、兽潮 | modifier type |
| 产生原因 | 天象、资源、战争、修士行为、天道变化 | world rules |
| 影响范围 | 全世界、区域、势力、种族、资源点 | scope |
| 行为影响 | 修炼、扩张、防御、叛逃、猎妖、贸易 | AI modifiers |
| 消退方式 | 持续时间、解决条件、自然衰减、事件结算 | duration |

## 尽量写全的内容清单

- 灵气类：灵气复苏、末法、灵脉枯竭、天地元气变化、界面压制。
- 魔气/邪气类：魔气上涨、邪祟扩散、污染、堕魔、魔劫。
- 灾害类：旱灾、瘟疫、饥荒、天灾、空间裂缝、火灾、雷灾、寒潮。
- 生态类：兽潮、妖兽活跃、妖王陨落、妖族迁徙、灵药成熟潮。
- 政治类：王朝衰败、乱世、宗门大战、附庸叛乱、继任危机。
- 资源类：矿脉发现、秘境频出、拍卖盛会、资源枯竭、商路断绝。
- 效果：修炼加速/减速、资源产出变化、稳定度变化、NPC 风险偏好变化、势力扩张/防御倾向变化。
- 项目映射：`Modifier.type`、`intensity`、`duration`、`effects`、稳定度、dynamic event、world rule。

## 不要写什么

- 不把单个事件误写成长期世界状态。
- 不只写气氛，要写行为和资源影响。
- 不写无边界的全局灾难；要说明范围和消退。
