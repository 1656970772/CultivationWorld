# 拍卖坊市与交易模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：市场、坊市、拍卖、黑市、商会和交易规则调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的交易场所、交易制度、拍卖流程、黑市风险、商会情报和以物易物。采集时应尽可能覆盖各修为阶段、各交易场所、各物品品阶、各货币/等价物、各交易身份、各明市/黑市规则和各竞价/欺诈/追杀后果。它服务功能组织、市场资源、动态拍卖事件、物品价值、信息传播和怀璧其罪。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 交易场所 | 坊市、拍卖行、黑市、商会、宗门兑换 | 功能组织 |
| 交易品类 | 丹药、法宝、材料、功法、情报、奴仆/灵兽 | items/economy |
| 交易规则 | 保证金、竞价、手续费、匿名、鉴定、交换 | dynamic event |
| 风险 | 黑吃黑、假货、尾随、暴露财富、通缉 | covet/risk |
| 情报 | 拍卖预告、价格差、稀缺物消息 | news/opportunity |

## 尽量写全的内容清单

- 场所：固定坊市、临时交易会、拍卖会、黑市、商会、宗门兑换殿、悬赏阁、情报楼、丹阁器阁。
- 商品：灵石、丹药、法宝、功法、材料、妖兽资源、符箓、阵盘、灵兽、情报、地图、秘境钥匙、传承残篇。
- 流程：入场、鉴定、竞价、保证金、匿名竞拍、压轴品、成交、交割、离场、尾随截杀。
- 价格机制：稀缺性、品级、地域差、拍卖溢价、黑市折价/高价、以物易物、高阶资源不用低阶货币买。
- 参与者：散修、宗门、商会、拍卖师、托、劫修、情报贩、执法者。
- 风险与后果：假货、抢夺、财富暴露、结仇、通缉、交易失败、消息泄露。
- 项目映射：`org_market`、`org_auction_house`、auction dynamic event、WorldNews、item value、covet expose。

## 不要写什么

- 不写无来源的固定价目表。
- 不把所有交易都做成安全购买。
- 不忽略情报和离场风险。
- 不把宗门兑换、坊市交易、黑市交易混成同一规则。
