# 符师模板

> 最后更新：2026-06-07
> 状态：模板
> 类型：符箓、符阵、符宝与一次性术式载体调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的符师、制符、符箓、符宝、符纸、符牌、令符、传讯符、遁符、防御符、攻击符和封印符等可携带、可预制、一次性或有限次数消耗的术式载体。采集时应尽可能覆盖各修为阶段、各符箓品阶、各功效类型、各制作/触发条件、各职业使用场景、各战斗与非战斗用途出现过的符类条目。它服务消耗品、一次性能力、逃生机制、交易物品、任务奖励和战斗表现。

本模板不负责分析炼丹、炼器、阵法流程中的符文/阵纹/禁制底层应用：丹药上的丹纹和封药符纹放入 `炼丹师模板.md`，器物上的铭文、阵纹和认主禁制放入 `炼器师模板.md`，阵法结构中的符文节点、阵眼阵基和破阵解纹放入 `阵法师模板.md`。本模板侧重独立符箓、符宝、符牌、符纸等“可携带术式载体”的制作、保存、交易和使用。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 符箓类型 | 攻击、防御、遁逃、传讯、封印、辅助、储能 | 物品分类 |
| 输入资源 | 符纸、符墨、兽血、灵材、符笔、符文传承 | 资源消耗 |
| 制作流程 | 画符、注灵、封存、失败、品质区分 | Job/Toil |
| 使用规则 | 激活条件、目标、范围、持续时间、消耗 | abilities、effects |
| 职业门槛 | 符文理解、神识、法力、传承、稳定性 | NPC 能力 |
| 失败后果 | 符毁、反噬、误触、效果失控、材料浪费 | risk、事件 |

## 尽量写全的内容清单

- 符箓用途：攻击、护身、遁逃、隐匿、传讯、定位、封印、驱邪、召唤、爆破、治疗。
- 载体差异：符纸、玉符、符宝、阵符、符牌、令符、传讯符、一次性法器。
- 制作要素：材料、符笔、符墨、符文、注灵、成功率、制作者状态、保存条件。
- 使用流程：携带、激活、目标选择、效果结算、冷却或消耗、失效。
- 经济关系：低阶修士保命物、坊市流通品、宗门任务奖励、战场消耗品。
- 项目映射：`items`、`abilities`、`effects`、逃生 GA、任务奖励、战斗消耗品。

## 不要写什么

- 不把符箓和阵法完全混同；符师侧重可携带、可消耗、可预制的术式载体。
- 不把炼丹、炼器、阵法流程中的符文/阵纹/禁制底层应用重复写成本模板主体内容。
- 不写无依据的品阶、价格和成功率。
- 不忽略使用限制，例如境界、法力、目标、次数、保存时间。
- 不把一次性剧情符宝扩展成所有 NPC 都能量产。
