# 修炼流派模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：人生取向、修炼路线与行为原型调研
> 目标详细程度：L3

## 适用范围

用于整理作品中反复出现的修炼路线和人生取向，例如谨慎长生、夺宝、邪修速成、护道传承、宗门权力、散修求生、炼丹炼器、御兽、肉身、神魂等。采集时应尽可能覆盖各修为阶段、各资质/出身、各功法体系、各职业副业、各资源路径、各风险偏好和各成功/失败样本。它服务执念系统、需求权重、行为集、物品/功法分类和 NPC 原型。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 流派定义 | 目标、价值观、资源偏好 | 执念、Utility |
| 代表角色 | 哪些角色体现该路线 | NPC 原型 |
| 行为链 | 如何获取资源、规避风险、突破瓶颈 | GOAP、Job/Toil |
| 代价与边界 | 副作用、敌对关系、失败风险 | 风险、关系、GAS |
| 可复用规则 | 能否成为项目通用 NPC 类型 | Wiki/ADR |

## 尽量写全的内容清单

- 人生取向：长生苟道、夺宝冒险、传承护道、权力夺取、报仇雪恨、守护亲友、逍遥散修、宗门忠诚、养老避世。
- 修炼路线：剑修、体修、神魂、炼丹、炼器、阵法、符箓、御兽、傀儡、毒修、血修、鬼修、魔修、佛修、儒修、妖修、血脉流。
- 资源偏好：丹药、功法、法宝、灵石、灵脉、妖兽材料、传承、情报、道侣/炉鼎、弟子、势力职位。
- 行为链：闭关、游历、猎妖、夺宝、交易、拜师、收徒、入宗、叛逃、夺权、暗杀、血祭、飞升。
- 风险倾向：谨慎、赌命、激进、隐忍、抱团、单干、仗势、投机、牺牲他人、重情义。
- 可数据化输出：执念类型、goalState、motiveWeights、personality 区间、行为集、机会点偏好、风险厌恶。

## 不要写什么

- 不把“功法属性”误当成人生流派。
- 不为只出现一次的桥段创造流派。
- 不把流派写成阵营道德判断；要写行为和系统后果。
- 不把主角路线原样变成可批量生成 NPC。

## 详细程度

- 每个流派至少需要 2 个证据点或 1 个强证据角色。
- 每个可转化流派必须有目标状态和行为链。
- 适合数据化时，写建议 `obsession`、`goalState`、`motiveWeights` 或 personality 倾向。
