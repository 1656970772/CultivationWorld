# 记忆情绪与执念模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：事件记忆、情绪反应、长期执念与人生目标调研
> 目标详细程度：L3

## 适用范围

用于整理作品中角色如何因事件形成记忆、情绪、仇怨、恩义、心魔、执念和长期目标。采集时应尽可能覆盖各修为阶段、各关系类型、各事件强度、各情绪反应、各执念方向、各持续时间和各行为后果的样本。它服务 `memory.json`、`emotion.json`、`obsession.json`、关系驱动目标、复仇、传承、夺宝和动态目标权重。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 触发事件 | 被救、被辱、亲友死亡、夺宝失败、突破失败 | memory |
| 情绪反应 | 恐惧、愤怒、羞耻、感激、贪念、悔恨 | emotion |
| 长期执念 | 复仇、长生、守护、传承、夺权、复活 | obsession |
| 行为后果 | 追杀、报恩、护短、闭关、避世、堕魔 | AI |
| 衰减/强化 | 是否随时间淡化，是否被新事件加强 | decay |

## 尽量写全的内容清单

- 记忆类型：救命、赠宝、背叛、羞辱、击杀亲友、夺宝、失败、突破、传功、结盟、通缉、宗门覆灭。
- 情绪：愤怒、恐惧、贪婪、感激、羞耻、骄傲、悲痛、悔恨、嫉妒、执着、心魔。
- 执念：复仇、长生、变强、守护、传承、夺宝、夺权、复活、避世、求道、堕魔。
- 触发条件：事件强度、关系强度、角色性格、境界差、损失价值、寿元压力、世界状态。
- 行为后果：追杀、求援、护徒、报恩、闭关、夺宝、收徒、挑战、投靠、叛逃、黑化。
- 变化机制：淡化、叠加、转化、被满足、被继承、因师傅/亲友死亡而转移目标。
- 项目映射：memory event、emotion delta、obsession type、goal multiplier、relationship edge、dynamic goal。

## 不要写什么

- 不把读者感受当角色情绪。
- 不写没有行为后果的心理描写。
- 不把所有目标都收敛成“闭关变强”。
- 不用固定数值冒充原文事实。
