# 灵根体质血脉模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：资质、体质、血脉、天赋与先天标签调研
> 目标详细程度：L3

## 适用范围

用于整理作品中决定修炼速度、突破概率、寿元、战力、特殊能力和身份地位的先天/后天资质。采集时应尽可能覆盖各修为阶段、各出身族群、各灵根/体质/血脉来源、各觉醒方式、各成长上限、各副作用和各社会评价。它服务 `Trait.SpiritRoot.*`、`Trait.Physique.*`、NPC 初始化、修炼平衡、GAS Tag、Effect 和角色差异化。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 资质类型 | 灵根、体质、血脉、种族天赋、神魂天赋 | NPC 特质 |
| 获得方式 | 先天、觉醒、移植、丹药洗髓、传承改造 | 事件与物品 |
| 机制影响 | 修炼速度、瓶颈、战力、寿元、技能、代价 | balance/GAS |
| 社会影响 | 收徒、联姻、追杀、夺舍、保护、歧视 | 关系与势力 |
| 边界 | 主角专属、种族限定、不可复制、会反噬 | 数据约束 |

## 尽量写全的内容清单

- 灵根：五行灵根、单灵根、双灵根、三灵根、伪灵根、天灵根、异灵根、变异灵根、废灵根、隐灵根。
- 体质：先天道体、圣体、魔体、妖体、神体、毒体、寒体、火体、剑体、肉身体质、炉鼎体质、长生体质。
- 血脉：妖兽血脉、神兽血脉、王族血脉、古族血脉、魔族血脉、真灵血脉、可觉醒血脉。
- 神魂/感知天赋：神识强大、悟性、天机感应、阵法/炼丹/御兽天赋。
- 获得/改变方式：出生、觉醒、洗髓、吞服灵物、换血、夺舍、传承、功法改造、血脉融合、诅咒。
- 正面效果：修炼加速、突破加成、属性亲和、特殊能力、战力增幅、寿元增加、社会地位提高。
- 负面效果：被觊觎、被夺舍、修炼限制、反噬、寿元损耗、心魔、血脉冲突、被势力控制。
- 项目字段：trait tag、initial roll、magnitude、修炼速度倍率、突破 bonus、Ability、Effect、关系/觊觎触发。

## 不要写什么

- 不把“天才”写成单一标签，必须拆成资质来源和机制影响。
- 不把主角唯一体质直接做成可批量生成特质。
- 不写无证据倍率；倍率属于“我的判断”。
- 不把血脉、体质、功法效果混为同一类。
