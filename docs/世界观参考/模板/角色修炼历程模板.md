# 角色修炼历程模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：角色成长、突破、资源消耗与关键转折调研
> 目标详细程度：L2；主角或标杆角色要求 L3

## 适用范围

用于整理关键角色从低阶到高阶的修炼路径：境界变化、资源来源、突破条件、失败风险、功法更换、机缘事件和行为转折。采集时应尽可能覆盖各修为阶段、各资质类型、各出身背景、各资源获取方式、各突破路径、各失败/重修后果和各成长速度的角色样本。它服务修炼节奏、突破概率、寿元压力、动态事件收益和 NPC 成长模型。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 时间阶段 | 早期、中期、后期或明确章节阶段 | 时间线、成长节奏 |
| 境界变化 | 从什么阶段到什么阶段 | `ranks.json`、突破规则 |
| 资源消耗 | 丹药、灵石、功法、洞府、机缘 | 经济、物品、任务 |
| 行为选择 | 闭关、游历、猎妖、交易、夺宝、拜师 | AI 行为分布 |
| 瓶颈与风险 | 失败、重伤、心魔、天劫、追杀 | 风险、动态事件 |

## 尽量写全的内容清单

- 角色范围：主角、主要同伴、主要反派、师傅、宗门天才、散修标杆、邪修标杆、妖修/异族标杆。
- 成长阶段：入门、拜师、初次突破、瓶颈、转修功法、获得核心资源、重大失败、被迫逃亡、飞升/换地图、最终阶段。
- 修炼资源：丹药、灵石、灵脉洞府、功法、法宝、妖兽材料、秘境机缘、师傅指点、血脉体质、神识/肉身资源。
- 行动方式：闭关、任务、游历、猎妖、交易、抢夺、拜师、传承、参战、拍卖、探索秘境、寻仇。
- 风险/代价：寿元压力、突破失败、伤势、心魔、天劫、追杀、法宝损毁、关系破裂、势力覆灭。
- 时间线信息：章节阶段、相对先后、年龄/年数、地图迁移、势力归属变化、关键关系变化。
- 项目启示：修炼节奏、资源消耗曲线、NPC 成长路径、失败后恢复路径、长期目标切换。

## 不要写什么

- 不写全书流水账。
- 不把章节顺序不明的事件硬排成精确年表。
- 不把主角专属机缘变成普通 NPC 必经路径。
- 不用“修为提升很快”替代资源和行为链说明。

## 详细程度

- 每个阶段至少写：起点、目标、资源、行动、结果、证据。
- 关键突破必须写成功条件和失败风险。
- 可用“相对阶段”代替精确日期，但必须明确先后顺序的证据强度。
