# 炼丹师模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：炼丹、医术、毒术与灵植调研
> 目标详细程度：L3

## 适用范围

用于整理作品中的炼丹师，以及与丹药体系高度相关的医师、毒师、灵植夫、药园管理、药材辨识和解毒救治内容。采集时应尽可能覆盖各修为阶段、各丹药品阶、各炼制流派、各药材来源、各职业分工、各治病/毒伤/突破场景和各失败代价。它服务丹药物品、药材资源、治疗/毒伤效果、NPC 职业、宗门设施、坊市交易和任务设计。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 职业类型 | 炼丹师、医师、毒师、灵植夫、采药人 | NPC 副业、身份标签 |
| 输入资源 | 丹方、药材、灵草、灵果、妖兽材料、火种、炉鼎 | 物品经济、资源点 |
| 工具设施 | 丹炉、药园、火室、医馆、毒室、灵田 | 建筑设施、Job/Toil |
| 产出 | 丹药、药液、解药、毒药、治疗方案、药材培育成果 | items、effects |
| 技艺门槛 | 传承、境界、神识、火候、药性辨识、经验 | 行为条件、NPC 能力 |
| 失败后果 | 炸炉、药性冲突、丹毒、误诊、中毒、药材损毁 | risk、事件 |

## 尽量写全的内容清单

- 丹药类型：疗伤、恢复法力/真气、突破、延寿、解毒、淬体、养魂、隐匿、燃命、控制、毒丹。
- 医毒体系：医术诊断、针药救治、解毒流程、毒物来源、毒性发作、抗性、反噬。
- 灵植体系：药园、灵田、采药、移植、培育、成熟周期、采摘限制、药性保存。
- 炼制流程：辨药、配伍、控火、凝丹、成丹、分品、失败、试药、出售或自用。
- 社会地位：供奉丹师、宗门丹房、坊市丹铺、代炼、悬赏求医、毒师禁忌。
- 项目映射：`items`、`effects`、`abilities`、NPC trait、Job/Toil、建筑、任务、经济循环。

## 不要写什么

- 不把丹药数值写成原作事实，除非原文明示。
- 不把医师、毒师、灵植内容混成炼丹的附属备注；有独立流程时要单独成卡。
- 不把通用修仙丹药名套入某作品事实。
- 不只列丹药名称，要写输入、流程、产出、限制和失败。
