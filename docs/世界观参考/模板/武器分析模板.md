# 武器分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：武器、兵器、护具与战斗装备调研
> 目标详细程度：L3

## 适用范围

用于分析单部作品中的武器、兵器、护具、战斗装备和与招式/职业绑定的器具。重点不是挑代表武器，而是按作品原文尽可能多收集各修为阶段、各势力、各职业、各战斗形态出现过的武器条目，并记录形制、材料、使用者、获取方式、战斗用途、损耗和升级路线。

如果某件武器具备独立法宝效果、器灵、祭炼认主、本命温养、禁制、秘境钥匙等属性，应同时在 `法宝分析模板.md` 中收录；本模板侧重“武器形制、战斗方式和装备生产”。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 武器阶段体系 | 凡兵、利器、灵兵、法器武器、神兵等作品内分层或替代体系 | 装备分层、战斗资产 |
| 武器条目清单 | 尽可能多收集各阶段武器，写清名称、类型、使用者、获取方式和用途 | `items/artifact.json`、角色装备 |
| 形制与战法 | 剑、刀、枪、弓、锤、斧、鞭、盾、甲、傀儡兵器、暗器、远程器械等 | 动作、技能、AI 战斗偏好 |
| 材料与生产 | 金属、骨、角、鳞、木、矿、妖兽材料、炼制/锻造/修复方式 | `items/material.json`、生产任务 |
| 使用限制 | 修为、职业、功法、血脉、体魄、神识、传承、消耗和损坏 | 装备条件、能力门槛 |
| 交易与普及 | 普通装备、宗门制式、稀有武器、战利品、拍卖和夺取风险 | 经济、掉落、势力军备 |

## 尽量写全的内容清单

- 武器形制：剑、刀、枪、矛、戟、斧、锤、棍、鞭、弓、弩、针、环、爪、盾、甲、舟船武装、傀儡武器、机关器械、暗器。
- 装备层级：凡兵、精铁兵器、制式兵器、灵兵、法器武器、宝兵、神兵、仙兵、传承武器、职业专用武器。
- 关联战法：剑阵、刀法、枪术、体修近战、远程狙杀、护身甲盾、破甲、破禁、群攻、偷袭、逃生。
- 来源：锻造、炼制、宗门配发、战利品、拍卖、遗迹、家族传承、妖兽材料改造。
- 损耗：磨损、断裂、修复、升级、重铸、吞噬材料、被夺取、被克制。
- 项目映射：`items/artifact.json`、`items/material.json`、`definitions/techniques.json`、`abilities/*.json`、drops。

## 不要写什么

- 不只写主角武器；要按阶段尽可能多收集普通、制式、稀有、顶阶和特殊武器。
- 不把所有武器都当成法宝；只有具备独立法宝机制的才进入法宝分析。
- 不把武器伤害数值写成原作事实；数值化必须标注为“我的判断”。
- 不忽略生产与普及度；普通武器和制式装备也可能影响势力军备和经济。

## 详细程度

- 不设固定上限；目标是覆盖凡俗/低阶、中阶、高阶、顶阶、特殊/传承级等阶段。
- 每个阶段必须尽量收集条目，并说明该阶段武器的普及度、材料、战斗用途。
- 每个入表武器必须写“阶段/品阶、形制、核心用途、使用限制”和“来源说明”。
- 如果某阶段材料不足，明确写“材料不足”，并列出已检索关键词、已查路径和后续检索方向。
