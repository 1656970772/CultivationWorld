# 武器分析模板

> 最后更新：2026-06-07
> 状态：模板
> 类型：武器、兵器、护具与战斗装备调研
> 目标详细程度：L3

## 适用范围

用于整理作品中出现过，或能由小说内容明确支撑的武器、兵器、护具、战斗装备和与招式/职业绑定的器具等。

普通武器、兵器、护具、制式装备、职业器具放入本模板。如果某件武器具备独立法宝效果、器灵、祭炼认主、本命温养、禁制、秘境钥匙、储物空间等属性，也应同时进入 `法宝分析模板.md`；本模板侧重武器形制、战斗方式、制作材料、使用者和装备生产。

## 前置声明

本文只整理小说中出现过，或能由小说内容明确支撑的武器、兵器、护具、战斗装备和与招式/职业绑定的器具等，不编造小说中不存在的武器。

整理时应按武器形制、战斗用途和适用境界尽可能覆盖完整：每一种形制、每一类用途、每一个境界阶段，只要小说中有对应武器，都应尽量列入表格。文档长度不设上限，宁可写长，也不要为了压缩篇幅遗漏有参考价值的武器。

如果某类武器、某个境界阶段，或制作材料、制作/获得方式、使用者/势力、限制/损耗等字段在小说中没有可靠依据，可以写“原文未说明”；不得为了补齐武器品阶、境界梯度、形制类型或数值体系而自行虚构。

## 必须写什么

| 字段 | 写法要求 |
|------|----------|
| 武器名称 | 写小说中的武器、兵器、护具、战斗装备、制式装备或职业器具名称。 |
| 形制 | 写武器或装备形制，如剑、刀、枪、矛、戟、斧、锤、棍、鞭、弓、弩、针、环、爪、盾、甲、暗器、机关器械、傀儡武器、舟船武装等。 |
| 稀有度 | 按作品自己的描述填写，可包含作品内品阶或层级，如凡兵、制式兵器、灵兵、法器武器、神兵、传承武器、职业专用武器等。 |
| 功效 | 写武器本身带来的效果，如攻杀、防御、破甲、远程狙杀、群攻、破禁、护身、增幅招式、辅助职业能力等。 |
| 用途 | 写武器被如何使用，如近战、远程、护身、军阵、偷袭、破甲、破禁、逃生、职业作业、宗门制式装备等。 |
| 制作材料 | 写具体材料或通用材料规则，如精铁、玄铁、妖兽骨、妖兽角、鳞甲、灵木、矿石、星辰铁等；不明时写“原文未说明”。 |
| 制作/获得方式 | 写武器如何制作或获得，如锻造、炼制、宗门配发、战利品、拍卖、遗迹获得、家族传承、妖兽材料改造；不明时写“原文未说明”。 |
| 使用者/势力 | 写使用者、职业、势力或适用人群，如散修、宗门弟子、体修、剑修、军队、某家族、某角色；不明时写“原文未说明”。 |
| 限制/损耗 | 写修为限制、职业限制、体魄要求、功法配套、磨损、断裂、修复、升级、重铸、被夺取、被克制等；没有说明时写“原文未说明”。 |
| 适用境界 | 按作品自己的境界说法填写，如凡俗、炼气、筑基、金丹、元婴，或该作品特有境界。 |
