# 妖兽分析模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：妖兽、灵兽、异兽、灵虫与妖兽资源调研
> 目标详细程度：L3

## 适用范围

用于分析单部作品中的妖兽、灵兽、异兽、凶兽、真灵、灵虫、妖族个体和可产出资源的非人生命。重点不是挑代表妖兽，而是按作品原文尽可能多收集各等级、各区域、各修为阶段出现过的妖兽/灵虫条目，并记录生态、行为、战斗能力、掉落资源、契约驯养、化形和与修士的关系。

如果重点是“妖兽与修士之间的长期社会关系、冲突模式、驯化制度”，可同时参考 `妖兽与修士关系分析模板.md`；本模板侧重条目收集、阶段覆盖和资源/战斗映射。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 妖兽品阶体系 | 妖兽等级、对应修士阶段、该阶段已收集条目、化形/真灵/异种层级 | `definitions/monsters.json`、战斗强度 |
| 妖兽/灵虫条目 | 尽可能多收集各等级、各区域、各阶段出现的妖兽/灵虫 | 怪物库、生态事件、任务目标 |
| 行为与生态 | 栖息地、群居/独居、守护资源、捕食、迁徙、进食周期、领地 | 地图生态、动态事件 |
| 战斗能力 | 属性、技能、毒、飞行、肉身、防御、神魂攻击、群体战术 | AI、Ability、战斗平衡 |
| 资源产出 | 妖丹、妖核、骨、皮、鳞、甲、精血、毒囊、翅翼、角爪、虫卵 | `items/material.json`、drops |
| 驯养与关系 | 契约、坐骑、灵宠、护山兽、血脉进化、化形、妖族势力 | 伙伴、势力、任务 |

## 尽量写全的内容清单

- 类型：妖兽、灵兽、凶兽、异兽、荒兽、真灵、神兽、妖族、灵虫、虫群、海兽、飞禽、蛇蛟龙类、魂兽、毒兽。
- 阶段：低阶、中阶、高阶、化形、真灵/神兽、传说级、界外/上古级。
- 资源：妖丹、妖核、内丹、兽骨、兽皮、鳞甲、精血、毒囊、翅翼、角爪、宝骨、虫卵、兽魂、尸骸机会点。
- 行为：守护灵药/矿脉/秘境、群居虫海、吞食金属或灵气、周期性进食、领地巡游、受修士驱使、被丹药/血脉吸引。
- 用途：炼丹、炼器、淬体、补灵根/血脉、坐骑、灵宠、任务目标、灾害源、秘境守卫。
- 项目映射：`definitions/monsters.json`、`items/material.json`、`behavior-trees/monster-*.json`、`balance/monster-spawn.json`、drops、GE/GA。

## 不要写什么

- 不只写强大妖兽；低阶可狩猎、可掉落、可驯养妖兽同样需要收集。
- 不把妖兽当成单纯战斗单位，必须写资源产出和生态位置。
- 不把其他作品的妖兽等级或化形规则套入本作品事实。
- 不虚构掉落、等级或战斗数值；没有原文依据时标注“我的判断”。

## 详细程度

- 不设固定上限；目标是覆盖低阶、中阶、高阶、化形、真灵/神兽、特殊/传说级等阶段。
- 每个阶段必须尽量收集条目，并说明该阶段妖兽的生态位置、战斗特点和资源价值。
- 每个入表妖兽/灵虫必须写“等级/阶段、出现区域/场景、特点、用途、掉落/资源”和“来源说明”。
- 如果某阶段材料不足，明确写“材料不足”，并列出已检索关键词、已查路径和后续检索方向。
