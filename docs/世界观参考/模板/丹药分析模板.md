# 丹药分析模板

> 最后更新：2026-06-07
> 状态：模板
> 类型：丹药、药剂、药散、毒丹、禁丹与药液调研
> 目标详细程度：L3

## 适用范围

用于整理作品中出现过，或能由小说内容明确支撑的丹药、药剂、药散、毒丹、禁丹、药液，以及可直接服用并产生药效的灵液等。

本模板只做丹药分析，不展开材料本体分析。药材、灵草、妖兽材料、矿材、灵液等材料只在“丹方”中作为投入材料或配方规则被提到；如果需要分析材料本身，放入 `材料分析模板.md`。

## 前置声明

本文只整理小说中出现过，或能由小说内容明确支撑的丹药、药剂、药散、毒丹、禁丹、药液等，不编造小说中不存在的丹药。

整理时应按功效类型和适用境界尽可能覆盖完整：每一类功效、每一个境界阶段，只要小说中有对应丹药，都应尽量列入表格。文档长度不设上限，宁可写长，也不要为了压缩篇幅遗漏有参考价值的丹药。

如果某类丹药、某个境界阶段，或丹方、炼制方式、来源、限制/副作用等字段在小说中没有可靠依据，可以写“原文未说明”；不得为了补齐丹药品阶、境界梯度、效果类型或数值体系而自行虚构。

## 必须写什么

| 字段 | 写法要求 |
|------|----------|
| 丹药名称 | 写小说中的丹药、药剂、药散、毒丹、禁丹、药液等名称。 |
| 稀有度 | 按作品自己的描述填写，如常见、稀有、宗门垄断、传说级、仙界罕见等。 |
| 功效 | 写服用或使用后的效果，如增加真气、辅助突破、疗伤、解毒、延寿、短期爆发、洗髓、补魂、毒杀等。 |
| 用途 | 写丹药被如何使用，如修炼、突破、疗伤、战斗、解毒、延寿、交易、喂养灵兽、任务奖励等。 |
| 丹方 | 写具体材料或通用材料规则，如百年人参、青灵草、火属性三阶妖兽内丹、冰属性灵草、数种珍稀灵药；不明时写“原文未说明”。 |
| 炼制方式 | 写丹药如何产生，如炼丹师炼制、天地生成、妖兽体内凝结、古修遗留、仙界流传；不明时写“原文未说明”。 |
| 来源 | 写丹药从哪里获得，如宗门兑换、坊市购买、炼丹师出售、秘境获得、拍卖、遗迹丹瓶、妖族传承、任务奖励、杀人夺宝等。 |
| 限制/副作用 | 写丹毒、耐药、境界限制、服用间隔、透支潜力、毒性、体质要求等；没有说明时写“原文未说明”。 |
| 适用境界 | 按作品自己的境界说法填写，如炼气、筑基、金丹、元婴，或该作品特有境界。 |
