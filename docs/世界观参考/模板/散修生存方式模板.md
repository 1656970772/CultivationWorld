# 散修生存方式模板

> 最后更新：2026-06-06
> 状态：模板
> 类型：无宗门或弱组织角色的生存、资源和风险调研
> 目标详细程度：L3

## 适用范围

用于整理散修、游侠、小家族修士、临时队伍、底层修士如何获取资源、规避风险、加入/离开势力、参与坊市与悬赏。采集时应尽可能覆盖各修为阶段、各资源贫富状态、各临时组织形态、各谋生职业、各风险来源、各加入/脱离势力路径和各失败结局。它服务 wanderer NPC、悬赏任务、坊市、游历、机会点、怀璧其罪和阵营切换。

## 必须写什么

| 模块 | 必写内容 | 项目用途 |
|------|----------|----------|
| 资源来源 | 采集、猎妖、悬赏、交易、夺宝、炼制、打工 | 经济、任务 |
| 风险来源 | 劫杀、信息不对称、无靠山、伤病、黑市 | 风险、怀璧 |
| 社交策略 | 临时组队、依附、结盟、伪装、逃亡 | 关系、AI |
| 上升路径 | 拜入宗门、得到传承、积累财富、突破 | NPC 成长 |
| 失败路径 | 死亡、被夺、投靠、堕为邪修、沦为炮灰 | 动态事件 |

## 尽量写全的内容清单

- 散修类型：低阶散修、老散修、隐居高人、游商、炼丹/炼器散修、情报贩、猎妖人、赏金猎人、临时队伍、散修联盟成员。
- 收入来源：采药、挖矿、猎妖、护送、悬赏、代工炼丹炼器、坊市倒卖、拍卖套利、秘境探险、杀人夺宝、投靠宗门。
- 生存成本：丹药、符箓、法宝维修、洞府租金、传送费、情报费、疗伤费、保护费、贡奉。
- 风险：被大宗门压榨、被黑市坑杀、被强者夺宝、任务死亡、队友背叛、无护法突破、无资源疗伤。
- 社交策略：伪装身份、抱团、依附、结盟、交易人情、拜入宗门、加入商会/悬赏组织、远走避祸。
- 上升路径：得到传承、拜师、积累贡献、参加试炼、进入大势力、成为客卿、创立小势力。
- 失败路径：死亡、被夺舍、堕为邪修、负债、重伤退隐、被迫效忠、投敌、转为凡俗。
- 项目映射：wanderer 行为集、机会点偏好、悬赏任务、坊市交易、动态目标、怀璧其罪。

## 不要写什么

- 不把散修写成“自由职业者”美化版，必须写生存压力。
- 不只写主角的成功路线。
- 不把所有散修都写成孤狼；临时合作和依附也要记录。
- 不忽略资源暴露后的风险。

## 详细程度

- 每条生存方式写：收入、成本、风险、适用境界、证据。
- 每个可映射行为写清 SimpleAction/JobAction/Quest/Opportunity 的落点。
- 至少写 3 类失败后果，支持长程模拟的死亡和关系链。
