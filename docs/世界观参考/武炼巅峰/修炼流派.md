## 流派卡片

### 基础武道/元气流

| 项 | 内容 |
|---|---|
| 定义 | 以境界积累、元气/真元/源力等阶段资源为核心的通用路线。 |
| 代表角色/事件 | 凌霄阁弟子、早期杨开从淬体到开元。 |
| 触发条件 | 角色拥有基础功法、修炼环境和可突破境界。 |
| 长期目标 | 稳定进入下一大境界，获得师承、资源和身份提升。 |
| 典型行动链 | 日常修炼 -> 比斗/任务 -> 资源积累 -> 心性/瓶颈检查 -> 突破。 |
| 常用资源/物品/功法 | 基础功法、丹药、师承、洞府、贡献点。 |
| 代价与失败后果 | 境界停滞、宗门地位下降；气动阶段还可能心性失衡。 |
| 可映射系统 | `rankProgress`、`breakthrough_gate`、宗门身份。 |
| 不可照搬点 | 杨开早期奇遇不是普通 NPC 默认模板。 |
| 证据 | W08-SCH-01、W08-CUL-09 |

## 待核验问题

- 炼器、阵道已补入基础证据；符箓、御兽、傀儡体系仍缺少独立证据表。
- 空间神通成长阶段、空间法阵材料、失败后果需要继续抽样。
- 星主/星辰本源路线需补全星主身份、星域本源排斥和传承规则。
- 噬天战法的无垢金莲适配、乌邝化解弊端方式需继续核验。
- 小乾坤圈养生灵、世界树子树、天地泉之间的效果差异需要拆成数据规则。
- 龙凤传承、圣灵本源与后期龙族/凤族正式体系之间的连续性仍需深挖。
- 墨化伪装与净化战线涉及大量后期开天战争内容，当前已补核心侵蚀证据，战舰/关隘/驱墨丹仍待核验。
