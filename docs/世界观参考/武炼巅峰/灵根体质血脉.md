## 游戏化抽象（我的判断）

| 抽象特质 | 依据 | 建议实现 | 边界 |
|---|---|---|---|
| 资质不等于命运锁死 | 杨开早期低资质但靠努力、奇遇、功法改变成长。 | 初始 `aptitude` 影响速度，但可被机缘/功法/资源改写。 | 不把主角奇遇作为普通低资质补偿。 |
| 特殊体质稀有池 | 原文称特殊体质极稀少且领域各异。 | NPC 初始化小概率抽 `Trait.Physique.*`。 | 未见可靠概率，不写具体万分比。 |
| 神魂至宝作为后天 trait | 温神莲长期改变神魂成长和防御。 | 物品绑定角色后转为 trait + GE。 | 至宝唯一性/稀缺性需要限制。 |
| 血脉带来能力也带来觊觎 | 圣灵幼体、龙威、圣灵材料均引发争夺或克制。 | `covetScore` 与 `bloodlinePower` 同步提高。 | 不把圣灵血脉当免费战力。 |
| 小乾坤根基特质只给开天后 | 天地泉、小乾坤稳定是开天后机制。 | `innerWorldTraits` 独立于普通体质。 | 低阶角色不可使用。 |
| 洗髓资源可改变资质但不绝对 | 化龙池、万药灵液都有改善资质/排杂质证据，且化龙池检验被质疑。 | `aptitude` 可被 `Opportunity` 修正，效果受个体和资源品质影响。 | 不把“泡池时间”当绝对资质真值。 |
| 家族血脉可做门禁/追踪 | 杨家血脉触发鹰追踪、玉佩、四圣兽雕像。 | `bloodlineKey`、`bloodScentTracking`。 | 不扩展到所有家族，除非有证据。 |
| 墨之力诱惑不是资质正向 trait | 墨徒声称有晋升好处，但代价是唯墨至上且真实性待核验。 | 做成 `corruptionTemptation` 而不是 `aptitudeBoost` 常规正面词条。 | 只能作为人物说法/诱惑机制。 |

## 待核验问题

- 当前未见《武炼巅峰》存在传统“五行灵根/天灵根/伪灵根”体系的可靠证据；暂记为“未见可靠证据”。
- 药灵圣体的完整来源、是否可遗传、是否会被夺舍/抢夺体质，待核验。
