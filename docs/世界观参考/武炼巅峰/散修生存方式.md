## 行为卡片

### 宗门试炼/杂役

| 项 | 内容 |
|----|------|
| 触发条件 | 原作事实：低阶弟子或杂役需要贡献点、基础资源和宗门身份维持日常修炼。 |
| 行动链 | 接宗门任务/日常劳作 -> 接受挑战或主动挑战 -> 上缴寻宝、采集所得 -> 暗堂或宗门机构记录贡献 -> 到后勤/贡献渠道换资源。 |
| 产出 | 贡献点、宗门基础供给、战绩记录、晋升机会。 |
| 消耗 | 时间、体力、伤势恢复、被挑战造成的资源损耗。 |
| 风险 | 失败被降格，弱者被周期性挑战压榨；资源不足会迫使继续外出采集。 |
| 成功后续 | 晋升普通弟子、座下弟子，或积累名声后脱离底层身份。 |
| 失败后续 | 降格、负伤、离宗流浪、转向更高风险采集或市场服务。 |
| 可映射文件 | `jobs/sect_labor`、`actions/sect_challenge`、`factions/rank_progression` |
| 证据 | WLF-ROGUE-02 |

### 采药与低阶猎妖

| 项 | 内容 |
|----|------|
| 触发条件 | NPC 缺贡献点、丹药、低阶材料，且附近存在山林、矿点或妖兽活动区。 |
| 行动链 | 打听地点 -> 准备疗伤药/武器 -> 进入外围 -> 避开高阶妖兽 -> 采集/战斗 -> 返回交付或出售。 |
| 产出 | 草药、低阶材料、贡献点、少量声望。 |
| 消耗 | 时间、伤势、装备耐久、丹药。 |
| 风险 | 低阶妖兽也可能致伤；尸体价值可能低；深入后遇高阶妖兽或追杀者。 |
| 成功后续 | 换取修炼资源，触发“发现秘境/特殊草药”机会点。 |
| 失败后续 | 受伤、失踪、死亡、被救援、背包掉落。 |
| 可映射文件 | `actions/gathering`、`jobs/forage`、`world/opportunities.json` |
| 证据 | WLF-ROGUE-01、WLF-ROGUE-02 |

### 坊市委托炼丹

| 项 | 内容 |
|----|------|
| 触发条件 | NPC 有材料但缺炼丹能力，或炼丹师缺练手机会。 |
| 行动链 | 进入坊市 -> 比较摊位/口碑 -> 交材料 -> 等待炼制 -> 成功收丹或失败索赔。 |
| 产出 | 丹药、炼丹熟练度、人情、交易记录。 |
| 消耗 | 原材料、时间、信息费。 |
| 风险 | 十炉八炉坏的失败率；炼丹师稀少导致议价弱；贵重材料暴露。 |
| 成功后续 | 成为固定客户，或被炼丹师/商会招揽。 |
| 失败后续 | 材料损失、冲突、转向黑市或抢夺。 |
| 可映射文件 | `jobs/craft_commission`、`market/services` |
| 证据 | WLF-ROGUE-03 |

## 游戏化抽象（我的判断）

| 散修原型 | 依据 | 建议机制 | 边界 |
|----------|------|----------|------|
| 宗门边缘弟子 | 贡献点与挑战制度同时提供资源与压迫。 | NPC 可在“修炼、打工、挑战、采集”之间权衡行动点。 | 不把杨开的主角机缘当普通弟子常态。 |
| 小家族修士 | 幽暗星小家族受大势力庇护和盘剥。 | 加入 `protectorFaction`、`tribute`、`exploitationRisk`。 | 具体税率/盘剥概率未见可靠证据。 |
| 高阶游离强者 | 鬼祖和华勇案例显示强者也会互相招揽。 | 招揽不只看境界，还看炼丹、护法、目的地、救人目标。 | 誓约细则需另查。 |
| 战时普通将士 | 战功兑换让底层有上升通道。 | 前线事件可短期提高突破机会，但显著增加死亡风险。 | 不等同和平期宗门福利。 |
| 技能型弱者 | 药王谷和帝辰星都说明炼丹/炼器能让弱者跨势力议价。 | 丹师、器师、阵法师应拥有更高 `service_value` 与 `asylum_weight`。 | 职业等级和真实产能需由物品/职业文档细化。 |
| 安全区逃生者 | 药王谷城内和乾坤殿内禁斗，但外部恩怨仍存在。 | 安全区只暂停冲突，离开后恢复 `enmity_pending`。 | 禁斗执法强度和例外待逐地图确认。 |
| 危机抱团者 | 太墟境秩序崩坏后，抱团比单人搜刮更稳定。 | 动态危机中允许 NPC 自动创建/加入临时组织。 | 帝天内部规则和分赃比例待核验。 |
| 战功经济参与者 | 墨族战争阶段，战功承担资源、功法、委托服务多重职能。 | 战争地图中用 `merit` 替代部分贡献点/圣晶结算。 | 各关隘兑换目录待查。 |
