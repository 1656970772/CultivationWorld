## 丹药/医毒/灵植卡片

### 炼丹师

| 项 | 内容 |
|----|------|
| 职业/技艺 | 炼丹师 |
| 输入资源 | 丹方、灵草、灵果（如七灵果）、妖兽妖丹、火种 |
| 工具/设施 | 丹炉（含灵阵）、丹房、火室、药园 |
| 产出 | 修炼丹、疗伤丹、突破丹（开天丹）、大道丹、驱墨丹、龙血丹 |
| 技艺门槛 | 神识（药性辨识）、火候（火系灵阵/火焰控制）、丹炉操控、丹方传承 |
| 失败后果 | 炸炉（炉体和人员损害）、丹毁（药材损失）、药效失败/丹方残缺 |
| 社会地位 | 宗门丹房供奉、乾坤之外硬通货生产源头（开天丹）、丹堂组织 |
| 项目映射 | `craft_pill_job`、`pill_quality_gate`、`cauldron_pillar_slot` |
| 证据 | WLF-PROD-01、WLF-FBD-14~17、WLF-FBD-21、WLF-FBD-24 |

## 炼丹流程

| 步骤 | 内容 | 关键风险 | 证据 |
|------|------|----------|------|
| 辨药 | 识别药材品阶、药性、配伍禁忌 | 误判导致药效失败 | WLF-PROD-01 |
| 配伍 | 按丹方比例配药 | 丹方残缺导致失败 | WLF-FBD-24 |
| 炼化 | 丹炉中熔炼药材，精确控制火候 | 火候不稳导致炸炉 | WLF-PROD-01 |
| 凝丹 | 提纯药效，凝聚成丹 | 纯度不够则品阶低 | WLF-PROD-01 |
| 开炉 | 开炉取丹，检验品相 | 杂质丹存在 | 间接推断 |
