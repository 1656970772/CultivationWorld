## 行为卡片

### 邪魔之气修炼/污染：吸收邪魔之气恢复神魂

| 项 | 内容 |
|---|---|
| 发起者 | 地魔等魔修/残魂 |
| 目标 | 邪魔之气源头、困龙涧、污染区域 |
| 前置条件 | 自身能承受/利用邪气；最好有控制或契约约束。 |
| 收益 | 恢复神魂、增强神魂战力、修炼相关神魂技。 |
| 代价/风险 | 控制失效、心性污染、神魂失控；普通人会被吞噬心神。 |
| 被发现后果 | 被正派/宗门警惕，可能遭封印或驱逐。 |
| 可映射系统 | `evilSource.DemonicQi`、`ge_soul_recover_by_demonic_qi`、`MindCorruption`。 |
| 证据 | W08-EVIL-01、W08-EVIL-02、W08-EVIL-03 |

|---|---|

## 事实边界

- 魔族不等于邪修：魔族是种族/阵营，个体行为和修炼方式需另判。
- 邪魔之气不等于墨之力：前者更多表现为邪气污染、神魂/心智影响；后者是墨族外源侵蚀和阵营覆盖。
- 噬天战法不等于所有吞噬天赋：石傀天赋吞噬矿物、归墟/小黑狗等吞噬天赋需另证。
- 夺舍不等于法身：法身是分神注入载体，详见 `夺舍设定分析.md`。
- 正常双修不等于采补邪功：原文明说采阴补阳、采阳补阴才是邪功。
- 毒寡妇、古魔血脉属于体质/血脉风险，不应自动贴“邪修”标签；只有主动伤人、采补、污染、夺舍等行为才进入邪修事件。
- 墨之力“打破极限”的说法不是已证实修炼事实，应按诱惑/谣言/敌方宣传处理。

## 游戏化抽象（我的判断）

| 规则 | 依据 | 建议实现 | 边界 |
|---|---|---|---|
| 邪源拆分 | 原文中邪气、邪功、恶行、墨化来源不同。 | `evilSource: DemonicQi / DevouringArt / Crime / InkCorruption / Possession`。 | 不用单一 `isEvil=true` 覆盖全部。 |
| 邪气污染有环境层 | 云霞右半岛、困龙涧是持续污染区。 | 地图格加 `hazard.demonicQi`，长期叠 `MindCorruption`。 | 有适配者可收益，但普通人高风险。 |
| 掠夺成长必须有代价 | 噬天战法收益极高但弊端/暴露风险明确。 | 成长加速 + `reputationRisk` + `backlashRisk`。 | 不做无成本高收益。 |
| 墨化是阵营事件 | 墨徒化改变忠诚和阵营。 | `InkCorruption` 到阈值触发 faction override。 | 净化能力稀有，不能随意清除。 |
| 灰黑势力触发任务 | 云霞宗/雷光神教式黑幕可成为调查任务。 | `dark_sect_scheme`、`rescue_mortals`、`revenge_event`。 | 不把所有宗门都默认黑化。 |
| 正邪判定分“来源”和“行为” | 魔族/体质不是邪修，采补/夺舍/污染/剥削才是行为证据。 | `sourceTag` 与 `crimeTag` 分列。 | 不用阵营名自动定罪。 |
| 墨化诱惑可做叛变钩子 | 墨徒对开天极限的宣传有诱惑力。 | 高瓶颈角色触发 `temptedByInk` 事件。 | 真实性待核验，不能当稳定正向成长线。 |
| 正道处置有误判 | 杨开被误认邪功，八大家施压。 | 添加 `falseAccusation` 和 `orthodoxPressure` 事件。 | 需由可见邪气/行为触发，不随机污名化。 |

## 待核验问题

- 雷光神教“见不得人的勾当”具体事件链需继续逐章补证。
- 噬天战法的完整弊端、无垢金莲化解机制、乌邝与法身差异待核验。
- 魔族内部的正统魔功、魔神秘典、古魔一族与普通邪修的边界待整理。
- 墨虫、墨烟、墨之力转化已补核心证据；墨巢、驱墨丹、驱墨舰、净化之光来源和限制待继续核验。
- 采补边界已有“双修不等于邪功、采阴补阳/采阳补阴才是邪功”证据；具体炉鼎组织、功法条目仍待核验。
- 血修、鬼修、尸修、毒修仅有零散风险线索，尚未形成可靠类型表；毒寡妇目前按体质风险而非邪修处理。
- 古魔一族/魔族正统魔功需要另开种族与功法线，不能混入普通邪修表。
