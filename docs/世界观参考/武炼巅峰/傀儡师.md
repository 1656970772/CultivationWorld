## 傀儡/技艺卡片

### 傀儡师

| 项 | 内容 |
|----|------|
| 职业/技艺 | 傀儡师 |
| 输入资源 | 材料（矿材、妖兽骨皮、灵材）、核心（能量源）、灵石、图纸、禁制 |
| 工具/设施 | 工坊、炼器设施、阵纹工具、操控令牌 |
| 产出 | 战斗傀儡（人形/兽形/机关）、采矿搬运傀儡、守门傀儡 |
| 控制方式 | 神识操控、令牌指令、阵法自动触发、魂印绑定 |
| 用途 | 战斗辅助、秘境守卫、采矿搬运、侦查试炼、陷阱 |
| 失败后果 | 失控（敌我不分）、反噬（操控反伤）、核心损毁、伦理争议 |
| 项目映射 | `craft_puppet_job`、`puppet_control_ai`、`puppet_maintenance` |
| 证据 | WLF-PROD-01 |

### 石傀（特殊傀儡/随从）

| 项 | 内容 |
|----|------|
| 类型 | 活体矿脉生物，非人造傀儡但可作为开采/战斗辅助"单位" |
| 来源 | 矿脉中寻得，数量稀少 |
| 核心能力 | 感知珍稀矿物位置；可战斗（炼化血精石等精华力量） |
| 控制方式 | 命令式指令（智力有限但执行力强） |
| 用途 | 探矿定位、矿物鉴别、战斗辅助 |
| 特殊产出 | 血精石、矿物精华 → 丹材/炼器添加物 |
| 风险 | 稀有、易被觊觎抢夺 |
| 项目映射 | `minion_stone_puppet`、`mineral_sense_ability` |
| 证据 | WLF-FBD-11 |
