## 阵法/技艺卡片

### 乾坤大阵（驱墨舰）

| 项 | 内容 |
|----|------|
| 阵法类型 | 净化/防御大阵 |
| 输入资源 | 阵旗、阵基、控制阵旗、黄晶蓝晶（净化之光）、驱墨舰平台 |
| 工具/设施 | 驱墨舰、阵眼、控制阵旗 |
| 操作者 | 杨开（布置）、四军人员（操作） |
| 运行规则 | 封存净化之光于驱墨舰中，战场救治墨之力侵蚀者 |
| 覆盖范围 | 四军后方区域 |
| 失败后果 | 阵旗损坏需炼器大师修复；净化之光不足则效果下降 |
| 项目映射 | `war_facility`、`formation_anti_ink` |
| 证据 | WLF-FBD-10、WLF-FBD-20 |

### 阵法师

| 项 | 内容 |
|----|------|
| 职业/技艺 | 阵法师 |
| 输入资源 | 阵图、阵旗、阵基、灵石、地脉、空间法则认知 |
| 工具/设施 | 阵眼、阵台、宗门山门、传送点 |
| 产出 | 防御阵法、攻击阵法、困阵、传送阵、聚灵阵 |
| 技艺门槛 | 阵图传承、神识、空间法则理解（传送阵）、阵纹铭刻 |
| 失败后果 | 阵旗损坏、阵眼反噬、误传送、阵法失控 |
| 社会用途 | 宗门防御、秘境入口、战争攻防、洞府保护、坊市安全 |
| 项目映射 | `formation_deployment`、`formation_maintenance_job` |
| 证据 | WLF-PROD-01、WLF-FBD-10 |

## 布阵与维护流程

| 步骤 | 内容 | 关键风险 | 证据 |
|------|------|----------|------|
| 阵图设计 | 根据地形/用途选择合适的阵图传承 | 阵图不符则阵法失效 | WLF-PROD-01 |
| 布设阵旗 | 安放阵基、阵旗于阵眼位置 | 阵眼暴露则大阵易破 | WLF-FBD-10 |
| 启动供能 | 注入灵石/连接地脉 | 供能不足则覆盖差 | WLF-PROD-01 |
| 维护检修 | 定期检修替换阵旗 | 老化断裂则阵法衰减 | WLF-FBD-12 |
| 破阵 | 攻击阵眼或力大破之（非专业方式） | 破阵反噬 | WLF-PROD-01 |
