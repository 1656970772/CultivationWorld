## 游戏化抽象（我的判断）

| 规则 | 依据 | 建议实现 | 边界 |
|---|---|---|---|
| 夺舍必须检查适配度 | 地魔说明身体像特定容器，别的神魂难存活。 | `bodyCompatibility`、`soulPowerContest`、`hostResistance`。 | 不允许任何残魂随便换身体。 |
| 夺舍失败也要有后果 | 地魔被烙印反控，玄冰被吞噬，神魂可能消耗或魂灭。 | 失败分支：反噬、奴役、魂灭、身体腐败。 | 不做无风险重试。 |
| 神魂保护资源极关键 | 温神莲可滋养/修复神魂。 | `soulTreasureBuff`、`Resistance.Possession`。 | 温神莲是稀有至宝，不可批量生成。 |
| 法身不等于夺舍 | 法身是分神注入载体，与侵占目标身体不同。 | 独立 `avatar` 系统。 | 不把法身事件记为邪修行为。 |
| 墨化是阵营覆盖 | 墨徒化来自墨之力侵蚀，不是某个神魂接管身体。 | `InkCorruption`、`FactionOverride`、`Purification`。 | 不与夺舍混用，净化规则另算。 |
| 高阶分神要留下后续因果 | 蒙戈分神被灭会记住气息，大帝分神牵涉唤醒暗棋。 | `fragmentOwnerId`、`memoryReturn`、`revengeFlag`。 | 分神不等于普通召唤物。 |
| 烙印控制独立于夺舍 | 钧天引控制神魂烙印，可掌控生死但不占身体。 | 独立 `soulContract` 系统。 | 不把所有控制都归入“夺舍”。 |
| 抗夺舍看神魂防线 | 温神莲、识海、神魂强度会决定夺舍成败。 | `soulDefenseScore` 加至宝/识海/境界修正。 | 不给主角式绝对免疫。 |

## 待核验问题

- 地魔、疯鳄妖王、尸灵教教主、玄冰/北璃陌等案例需要逐章补完整上下文和结果。
- 连续夺舍的消耗、冷却、身体限制、记忆继承机制未见统一规则。
- “转世/重生/复活”当前未见可靠证据，不写成原作常规机制。
- 器灵附体、傀儡身体、身外化身与法身之间的边界还需从法宝/法身线补证。
- 温神莲导致死后识海保留，是否可形成可交互遗迹/残魂事件，需进一步核验。
- 大帝分神/法身/本体唤醒与真正夺舍之间的界线需要继续核验；当前只采信阳炎线已有表述。
- 神魂烙印、魂契、奴印、钧天引的施法流程、解除条件和跨距离限制需补证。
- 魔将分神是否能把记忆完整回传本尊、回传条件如何，当前仅见威胁和“记住气息”线索。
