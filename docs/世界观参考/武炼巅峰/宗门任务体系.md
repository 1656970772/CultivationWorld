## 任务卡片

### 试炼期考核任务

| 项 | 内容 |
|----|------|
| 任务目标 | 在限定试炼期内突破或证明留宗潜力，避免被淘汰或长期停留底层身份。 |
| 发布者 | 宗门入门/试炼制度 |
| 接取条件 | 新弟子、试炼弟子或低阶杂役身份。 |
| 推荐参与者 | 凌霄阁早期低阶弟子。 |
| 行动链 | 入门 -> 试炼期修炼/劳役 -> 考核或期限判断 -> 正式留宗、贬为试炼弟子或离开 |
| 成功奖励 | 正式身份、继续留宗机会和后续挑战/贡献体系资格。 |
| 失败后果 | 淘汰、贬为试炼弟子、地位下降。 |
| 对宗门的回流 | 筛选弟子潜力，维持底层劳役与正式弟子分层。 |
| 可映射文件 | `quests/sect_trial_assessment`、`factions/member_status` |
| 证据 | WLF-SECT-01 |

### 挑战切磋任务

| 项 | 内容 |
|----|------|
| 任务目标 | 挑战同门并取胜，或应对被挑战 |
| 发布者 | 宗规自动触发 / 弟子主动发起 |
| 接取条件 | 每日一次挑战机会；每五日一次被挑战机会；双方实力差距不得超过三个层次 |
| 推荐参与者 | 凌霄阁弟子、试炼弟子 |
| 行动链 | 发起挑战 -> 暗堂记录 -> 胜负结算 -> 月度贡献汇总 |
| 成功奖励 | 按对手弟子等级获得贡献点 |
| 失败后果 | 按自身弟子等级扣贡献点；违抗宗规可逐出门墙 |
| 对宗门的回流 | 维持弟子竞争、筛选实力、制造贡献点流动 |
| 可映射文件 | `quests/sect_challenge.json`、`actions/challenge_peer` |
| 证据 | WLF-SECT-02、WLF-SECT-03 |

### 贡献兑换任务

| 项 | 内容 |
|----|------|
| 任务目标 | 用贡献点兑换丹药、秘籍、兵器、宝甲等修炼物品。 |
| 发布者 | 后勤处 / 贡献堂 |
| 接取条件 | 拥有贡献点，具备宗门弟子身份。 |
| 推荐参与者 | 需要成长资源的低阶弟子。 |
| 行动链 | 获得贡献点 -> 到贡献堂/后勤处 -> 选择物品 -> 扣除贡献并领取资源 |
| 成功奖励 | 修炼物品、战斗装备或金银等兑换结果。 |
| 失败后果 | 贡献不足、库存或资格限制导致无法兑换。 |
| 对宗门的回流 | 回收贡献点，控制资源发放，鼓励弟子参与挑战和采集。 |
| 可映射文件 | `economy/contribution_exchange`、`buildings/contribution_hall` |
| 证据 | WLF-SECT-02、WLF-SECT-03 |

### 日常劳役任务

| 项 | 内容 |
|----|------|
| 任务目标 | 完成扫地等杂务 |
| 发布者 | 宗门杂事体系 |
| 接取条件 | 底层身份，如扫地小厮或试炼弟子 |
| 推荐参与者 | 低阶弟子、杂役 |
| 行动链 | 接受区域 -> 执行劳役 -> 结束后修炼或外出 |
| 成功奖励 | 原文明确维持身份和生活；是否给贡献点待核验 |
| 失败后果 | 未见可靠证据；可推测影响宗门评价，但不能列为原作事实 |
| 对宗门的回流 | 维持日常运转，消耗低阶弟子时间 |
| 可映射文件 | `jobs/daily_labor`、`toils/sweep_area` |
| 证据 | WLF-SECT-01、WLF-SECT-04 |

### 采集上缴任务

| 项 | 内容 |
|----|------|
| 任务目标 | 外出采草药并上缴 |
| 发布者 | 宗门贡献体系 / NPC 自主 |
| 接取条件 | 需要外出许可或向杂事处告假；具备野外行动能力 |
| 推荐参与者 | 低阶弟子、缺贡献点者 |
| 行动链 | 请假 -> 出门采集 -> 返回 -> 上缴换贡献 |
| 成功奖励 | 贡献点 |
| 失败后果 | 耽搁修炼、野外风险、任务收益低 |
| 对宗门的回流 | 补充草药库存，给低阶弟子贡献来源 |
| 可映射文件 | `quests/gather_herbs`、`jobs/forage` |
| 证据 | WLF-SECT-04、WLF-SECT-05 |

| 对宗门的回流 | 获取传承和资源，提高宗门实力 |

## 制度闭环（我的判断）

| 闭环 | 原作依据 | 项目映射 | 边界 |
|------|----------|----------|------|
| 低阶宗门成长闭环 | 挑战/采集/劳役 -> 贡献点 -> 兑换修炼物品 -> 提升实力 | 新手宗门、贡献经济、日常 Job | 原文未明确所有劳役都给贡献，不能强写 |
| 秘境机会闭环 | 名额/优先权 -> 探索 -> 传承/资源 -> 势力实力提升 | 秘境动态事件、队伍任务 | 名额规则因势力而异，不统一套模板 |
