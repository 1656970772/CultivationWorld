## 原作事实

| 事实点 | 说明 | 证据 |
|---|---|---|
| 资源获取总伴随机会成本 | 低阶采草能换贡献点，但会耽搁修炼；坊市炼丹能赚钱，但会招来同行和派系利益冲突。 | WLF-ECON-02、07、09 |
| 货币随世界阶段替换 | 晶石、圣晶、源晶、开天丹分别覆盖不同地区和境界阶段。 | WLF-ECON-03、05、08、15 |
| 生产职业参与经济闭环 | 炼丹师通过代工赚源晶，炼丹炼器材料在战争中涨价，职业者因此被争抢。 | WLF-ECON-07、09、11 |
| 阵法和战舰是高消耗系统 | 阵旗阵基可损坏修复，战舰需要圣晶和动力室维持，维护成本能判断势力强弱。 | WLF-ECON-06、10 |
| 战争会创造独立货币 | 碧落关以功曹殿清算战功，战功可在战备殿兑换修行资源。 | WLF-ECON-14 |

## 经济抽象（我的判断）

| 规则 | 依据 | 建议落点 | 边界 |
