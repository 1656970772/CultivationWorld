## 原作事实：战斗后果

| 后果 | 触发 | 机制意义 | 证据 |
|---|---|---|---|
| 秘宝破损 | 防御秘宝承受超限攻击 | 装备应有耐久和破坏状态 | W08-CBT-01 |
| 元气枯竭 | 长时间催动护盾/秘宝 | 防御不是无成本锁血 | W08-CBT-02 |
| 神魂受创 | 神识碰撞、舍魂刺、识海战 | 需要单独 `soulIntegrity`，不只扣 HP | W08-CBT-05、W08-CBT-12 |
| 空间逃生失败 | 空间封锁、虚空甬道危险 | 逃跑行动可失败，并导致更重惩罚 | W08-CBT-03、W08-CBT-04 |
| 阵营覆盖 | 墨之力转化为墨徒 | 战败不一定死亡，也可能身份/忠诚改变 | W08-CBT-11 |
| 根基受损 | 小乾坤侵蚀/割舍/动荡 | 开天后战斗应伤及长期修行能力 | W08-CBT-07 |
| 消息传播/觊觎 | 圣灵现身、至宝暴露 | 高价值战斗事件应触发情报传播 | W08-CBT-09 |
| 秘境限制 | 禁飞、禁星梭、定时送出 | 环境规则会改写战斗和逃生选择 | W08-CBT-16 |
| 物资暴露 | 空间储物、黑书空间、圣灵秘宝暴露 | 资源越强，越容易引起觊觎和追杀 | W08-CBT-17、W08-CBT-18 |
| 烙印受制 | 神魂烙印落入敌手 | 不是单纯伤害，而是生死和行动控制 | W08-CBT-21 |

## 游戏化抽象（我的判断）

| 规则 | 依据 | 建议实现 | 边界 |
|---|---|---|---|
| 战斗要有“逃跑层” | 原文大量遁走、撕裂空间、星梭、拖延待援。 | `CombatIntent.Escape` 与 `Chase` 独立结算。 | 主角空间逃生不能变成普通 NPC 免费技能。 |
| 防御消耗耐久和资源 | 防御秘宝会破、天级秘宝耗元气。 | 护盾有 `durability`、`energyCost`、`breakEvent`。 | 不做永久无敌护盾。 |
| 神魂是独立血条 | 温神莲、灭世魔眼、舍魂刺都围绕神魂。 | 增 `soulIntegrity`、`SoulDamageSpec`。 | 神魂归零后果需区分昏迷、夺舍、魂灭。 |
| 墨化不是普通伤害 | 墨徒化改变阵营和忠诚。 | `InkCorruption` 叠层到阈值触发 `FactionOverride`。 | 净化条件需严控，避免无成本救回。 |
| 开天后伤根基 | 小乾坤稳定直接影响战力和长期修炼。 | 开天角色额外结算 `innerWorldStability`。 | 低阶角色不使用此系统。 |
| 高价值能力带来觊觎 | 圣灵幼体、天地泉、温神莲、空间法则都会引来争夺。 | 战斗暴露后触发 `intelSpread`、`covetQuest`。 | 不把所有 NPC 都变成无脑追杀。 |
| 环境规则应压过通用技能 | 流炎沙地禁飞禁星梭，说明秘境可重写常规逃跑方式。 | 地图/秘境加 `forbidTags`、`autoExitTimer`、`hazardTicks`。 | 禁制必须有来源，不可随意禁玩家能力。 |
| 物资续航是保命机制 | 乾坤袋、空间戒、黑书空间让角色携带丹药和资源。 | `inventoryCapacity` 与逃亡/战斗续航挂钩。 | 黑书空间是主角特例。 |
| 烙印控制要算社会后果 | 钧天引、神魂烙印可掌控生死和情绪。 | `soulContract` 影响忠诚、恐惧、背叛概率。 | 与自愿契约/夺舍分开。 |

## 待核验问题

- 替死、假死、复活、传送符等保命手段当前未见可靠证据，暂写“未见可靠证据/待核验”。
- 法宝品级体系、耐久、损毁与修复规则需从法宝文档联动补全。
- 空间法则具体技能列表、空间封锁条件和反制手段仍需逐章整理。
- 舍魂刺数量、炼化流程、使用上限和恢复周期需要补证。
- 墨之力侵蚀与天地泉镇压/净化之光之间的精确关系需要继续核验。
- 大规模战斗、战舰、行宫秘宝、法阵齐射等开天后战斗形态尚未完整整理。
- 流炎沙地以外的秘境环境规则、自动送出/封锁条件需继续抽样。
- 乾坤袋、空间戒、玄界珠、黑书空间、行宫秘宝之间的容量、活物限制和安全性需要拆分。
- 神魂烙印、钧天引、魂契、奴印等控制类机制需与夺舍文档联动。
