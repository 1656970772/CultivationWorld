## 器物/技艺卡片

### 炼器师（秘宝炼制）

| 项 | 内容 |
|----|------|
| 职业/技艺 | 炼器师 |
| 输入资源 | 矿材（矿石、灵金）、妖兽材料（骨、皮、鳞、甲、妖丹）、精血、真灵、龙骨龙珠 |
| 工具/设施 | 炼器炉、阵纹工具、锻台、火种 |
| 产出 | 攻击秘宝、防御秘宝、飞行秘宝（星梭）、丹炉、阵旗阵基、傀儡部件 |
| 技艺门槛 | 传承（门派/家族）、神识（材料辨识）、控火（火系力量）、阵纹知识 |
| 失败后果 | 材料报废、器物品质下降、反噬、器物损坏 |
| 特殊工艺 | 真灵炼入秘宝（提升档次）；龙骨龙珠炼化为成长型攻击秘宝 |
| 项目映射 | `craft_artifact_job`、`artifact_upgrade_chain`、`material_expertise_gate` |
| 证据 | WLF-PROD-01、WLF-FBD-04~12、WLF-FBD-23 |

### 锻造师

| 项 | 内容 |
|----|------|
| 职业/技艺 | 锻造师（基础装备生产） |
| 输入资源 | 矿石、金属 |
| 工具/设施 | 锻台、炉火、冷却槽 |
| 产出 | 武器坯体、甲片、器械部件 |
| 技艺门槛 | 锻造经验（低于炼器师） |
| 与炼器师区别 | 锻造师生产基础坯体和材料，炼器师在此基础上铭纹、祭炼、融合妖兽材料 |
| 项目映射 | `forge_job` → `craft_artifact_job` 的上下级关系 |
| 证据 | 间接推断（产业分工逻辑） |

## 炼制流程

| 步骤 | 内容 | 关键风险 | 证据 |
|------|------|----------|------|
| 选材 | 辨识矿材、妖兽材料品阶和适配性 | 材料不符则产出品质低 | WLF-PROD-01 |
| 熔炼/锻造 | 在炼器炉中熔炼矿材，或锻台打制坯体 | 火候不稳或力度不足致坯体不均 | WLF-PROD-01 |
| 铭纹/祭炼 | 篆刻阵纹、灵阵，融合妖兽材料/精血 | 铭纹错误致器物报废 | WLF-PROD-01 |
| 认主/绑定 | 滴血或神魂绑定 | 绑定后不可轻易转手 | WLF-FBD-05 |
| 修复/升级 | 破损器物送回炼器炉修复或升级 | 修复失败或品质下降 | WLF-FBD-12 |
