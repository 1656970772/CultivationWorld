## 鉴定卡片

### 鉴宝师

| 项 | 内容 |
|----|------|
| 职业/技艺 | 鉴宝师、估价师 |
| 鉴定对象 | 法宝、丹药、材料、古物、功法残片、秘境线索 |
| 鉴定依据 | 眼力（经验）、神识（感应品阶）、传承（宗门/家族鉴别知识）、工具（鉴宝器） |
| 输出结果 | 真伪判断、品阶定位、功效分析、市场估价、风险提示、买家建议 |
| 场景 | 拍卖行（聚宝楼）、坊市、黑市、宗门库房盘存、遗迹勘探、战利品分配 |
| 失败后果 | 误估（高估/低估损失）、买假（假货损失）、错失宝物、泄密引来觊觎、信誉受损 |
| 项目映射 | `identify_item_job`、`appraisal_skill`、`price_uncertainty_system` |
| 证据 | WLF-PROD-01 |
