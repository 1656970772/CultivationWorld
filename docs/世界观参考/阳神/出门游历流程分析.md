# 《阳神》出门游历流程分析

> 基于《阳神》原著搜索数据整理，为修仙模拟游戏设计提供参考。

---

## 一、游历事件概览

| 序号 | 事件名称 | 游历阶段 | 核心驱动 |
|------|---------|---------|---------|
| 1 | 进京赶考 / 初入玉京 | 前期 | 世俗晋升 + 修炼启蒙 |
| 2 | 海外游历 / 收服禅银纱 | 中期 | 势力扩张 + 盟友招募 |
| 3 | 各大遗迹 / 诸子百家道场 | 中后期 | 传承收集 + 功法补全 |
| 4 | 对抗造化之神的终极游历 | 终局 | 终极决战 + 世界存亡 |

---

## 二、事件详细分析

### 事件1：进京赶考 / 初入玉京

#### 出发原因

- **表层动机**：参加科举考试，获取功名，洗刷母亲被家族羞辱的耻辱。
- **深层动机**：接触修炼界上层圈子，获取修炼资源，打探《过去经》的下落。
- **游戏映射**：新手村毕业后的第一个主城任务，触发主线剧情的起点。

#### 目的地类型

- 世俗王朝核心都城（玉京城），兼具世俗权力中心与修炼势力交汇地的双重属性。
- 城市结构分层：科举考场区 → 权贵府邸区 → 修炼者聚集区 → 皇室禁区。

#### 同行者

- **独自前行**：初期无同伴，仅携带少量随身行李与防身武艺。
- 到达后结识的角色可转化为后续游历的同伴或盟友。

#### 可获取物

| 类别 | 具体内容 | 稀有度 |
|------|---------|--------|
| 世俗身份 | 科举功名、官职 | 普通 |
| 人脉资源 | 洪玄机、梦神机等修炼界高层的关系 | 稀有 |
| 功法传承 | 《过去经》初步线索 | 传说 |
| 修炼资源 | 灵石、丹药等基础修炼物资 | 普通 |

#### 六阶段流程

| 阶段 | 内容 | 关键节点 |
|------|------|---------|
| **准备** | 整理赶考文书，携带少量盘缠，打听玉京城修炼势力分布，备好防身武艺 | 情报收集 |
| **出发** | 从家乡步行/乘马车沿官道向玉京城行进 | 大地图移动 |
| **途中** | 遭遇山匪劫道（战斗教学），顺手斩杀匪首立威；偶遇同去赶考的书生，听闻玉京城异动传闻 | 遭遇战 + 情报获取 |
| **到达** | 入城后安顿在客栈，报名科举，暗中打探《过去经》消息 | 主城解锁 |
| **探索** | 参加科举考试 → 出入权贵宴席、修炼者集会 → 接触大乾皇室、道门高层 | 多线任务并行 |
| **收获** | 获得科举功名，结识修炼界高层，得到《过去经》初步线索，获取第一批修炼资源 | 主线推进 |
| **返回** | 暂留玉京城修炼，后续因机缘需要离开 → 引出海外游历 | 任务链衔接 |

---

### 事件2：海外游历 / 收服禅银纱

#### 出发原因

- **表层动机**：冲击更高修炼境界，寻找海外散落的修炼资源。
- **深层动机**：扩大自身势力，收服强力盟友，为后续对抗造化道积累资本。
- **游戏映射**：玩家实力达到一定阈值后解锁的海外地图扩展。

#### 目的地类型

- 海外群岛、云蒙国边境的修炼势力聚集地。
- 地图特征：海洋航路（有随机遭遇）→ 岛屿据点 → 上古遗迹。

#### 同行者

- **出发时独行**，途中逐步收服部下（狐仙、龟将等）。
- 收服后的部下可编入玩家麾下，形成随行队伍。

#### 可获取物

| 类别 | 具体内容 | 稀有度 |
|------|---------|--------|
| 强力盟友 | 禅银纱（核心战力） | 传说 |
| 势力地盘 | 海外部分岛屿的控制权 | 稀有 |
| 功法资源 | 海外修炼功法、上古法宝碎片 | 稀有 |
| 材料 | 海兽素材（用于炼器/炼丹） | 普通 |

#### 六阶段流程

| 阶段 | 内容 | 关键节点 |
|------|------|---------|
| **准备** | 备好渡海船只，携带干粮与防身法宝，了解海外势力分布与危险区域 | 出海准备任务链 |
| **出发** | 从大乾沿海港口乘船出海，向海外群岛航行 | 海域地图解锁 |
| **途中** | 遭遇海盗与海兽袭击（海战），斩杀海兽获取材料；偶遇被追杀的禅银纱 | 遭遇战 + 关键NPC登场 |
| **到达** | 抵达海外核心岛屿，进入云蒙国边境修炼势力范围 | 新据点解锁 |
| **探索** | 介入海外势力纷争，与禅银纱联手对抗追杀者，探索上古遗迹 | 阵营选择 + 遗迹副本 |
| **收获** | 成功收服禅银纱为盟友，获得海外岛屿控制权，拿到上古功法残卷与法宝 | 势力扩张 |
| **返回** | 整合海外势力后返回中土，为后续对抗大乾朝廷与造化道做准备 | 剧情衔接 |

### 事件3：各大遗迹 / 诸子百家道场

#### 出发原因

- **表层动机**：获取上古诸子百家的传承功法，补全自身修炼体系。
- **深层动机**：积累对抗造化之神的底牌与底蕴。
- **游戏映射**：中期核心玩法——遗迹探索与传承收集，类似魂系列的大BOSS房。

#### 目的地类型

- 上古诸子百家遗留的道场、荒古遗迹、秘境。
- 地图特征：多个独立副本（儒家道场、道家道场、佛家道场等），每个副本有独立的机制与首领。

#### 同行者

- 麾下收服的修炼者部下。
- 临时结交的同道修炼者（可能产生背叛/支援的随机事件）。

#### 可获取物

| 类别 | 具体内容 | 稀有度 |
|------|---------|--------|
| 核心功法 | 《过去弥陀经》《现在如来经》《未来无定经》三经完整传承 | 传说（套装） |
| 法宝 | 多件上古法宝（武器/防具/饰品） | 传说 |
| 修炼心得 | 诸子百家的修炼心得（技能点/属性加成） | 稀有 |

#### 六阶段流程

| 阶段 | 内容 | 关键节点 |
|------|------|---------|
| **准备** | 根据古籍整理诸子道场位置，备好破解遗迹禁制的法宝与丹药 | 情报收集 + 道具准备 |
| **出发** | 按图索骥，逐个前往散落在中土、海外、蛮荒的诸子道场 | 多目标大地图移动 |
| **途中** | 遭遇其他争夺传承的修炼者势力，多次发生争斗，获取更多道场位置线索 | PvP/势力冲突 |
| **到达** | 依次抵达诸子道场入口，破解外围禁制进入 | 副本入口解锁 |
| **探索** | 闯过层层考验与禁制，与其他进入者争夺传承，挑战道场守护者（BOSS战） | 副本机制 |
| **收获** | 获得三经完整传承，拿到多件上古法宝，补全功法缺陷 | 核心战力质变 |
| **返回** | 集齐传承后返回据点闭关消化，为最终对抗造化之神做准备 | 养成阶段 |

### 事件4：对抗造化之神的终极游历

#### 出发原因

- **表层动机**：造化之神即将复苏毁灭世界，需要集结所有力量对抗。
- **深层动机**：寻找彻底消灭造化之神的方法，终结神魔乱局。
- **游戏映射**：最终章——所有前置任务完成后的终局之战。

#### 目的地类型

- 造化之神的封印地、上古神魔战场、所有与造化相关的秘地。
- 地图特征：线性推进的最终关卡，途中有关卡BOSS，最终为大BOSS战。

#### 同行者

- **全员集结**：所有结盟的修炼势力、收服的诸天神魔部下。
- 可设计为多队协同作战（玩家操控主角 + AI控制盟友）。

#### 可获取物

| 类别 | 具体内容 | 稀有度 |
|------|---------|--------|
| 终极力量 | 彻底掌控诸天生死轮盘 | 神话 |
| 境界突破 | 超越造化境界的力量 | 神话 |
| 结局 | 终结神魔乱局的契机 | 剧情 |

#### 六阶段流程

| 阶段 | 内容 | 关键节点 |
|------|------|---------|
| **准备** | 整合所有盟友势力，备好全部收集到的上古法宝与传承，制定作战计划 | 战前总动员 |
| **出发** | 带领大军前往封印地，分兵探索所有造化相关秘地 | 大军出征 |
| **途中** | 与造化之神手下多次交战，逐步破解诸天杀局 | 连续BOSS战 |
| **到达** | 抵达封印核心区域，进入上古神魔战场最终决战地 | 最终迷宫 |
| **探索** | 深入封印地寻找造化之神的弱点，探索神魔战场获取上古神魔遗留力量 | 最终探索 |
| **收获** | 彻底掌控诸天生死轮盘，突破超越造化境界，获得消灭造化之神的完整方法 | 终极质变 |
| **返回** | 消灭造化之神后重整世间秩序，游历以终结乱局、平定天下告终 | 结局 |

## 三、游戏设计启示

### 3.1 游历结构设计

《阳神》的游历呈现出清晰的**四阶递进结构**，可作为修仙模拟游戏游历系统的骨架：

```
前期（世俗阶段） → 中期（势力扩张） → 中后期（传承收集） → 终局（终极决战）
```

每个阶段的游历规模、同行者规模、可获取物质量都呈指数级增长，符合玩家的成长曲线。

### 3.2 六阶段流程模板化

四个事件均可用统一的七步流程概括，适合设计为可配置的游历任务模板：

```
准备 → 出发 → 途中 → 到达 → 探索 → 收获 → 返回
```

其中「途中」和「探索」是两个核心玩法承载阶段：
- **途中**：适合放置随机遭遇、战斗、情报收集等动态内容。
- **探索**：适合放置副本机制、解谜、BOSS战等深度内容。

### 3.3 同行者系统

《阳神》中同行者从独行→收服部下→带领大军，呈现出明显的规模递增。建议设计：
- **独行阶段**：玩家自行应对所有战斗，强化个人操作感。
- **小队阶段**：可配置同伴，触发协同技能与组合技。
- **大军阶段**：转为策略指挥模式，玩家调配多支部队。

### 3.4 可获取物分层

按事件对应不同的稀有度层级：

| 游历阶段 | 主要获取物 | 稀有度 | 作用 |
|---------|-----------|--------|------|
| 进京赶考 | 功名、基础资源、人脉 | 普通/稀有 | 前期立足 |
| 海外游历 | 盟友、地盘、中级功法 | 稀有/传说 | 势力扩张 |
| 遗迹探索 | 三经传承、上古法宝 | 传说（套装）| 战力质变 |
| 终极游历 | 终极力量、境界突破 | 神话 | 游戏通关 |

### 3.5 任务链衔接设计

每个事件的「返回」阶段并非真正结束，而是埋下下一个游历的引子：
- 进京赶考 → 引出海外游历（需要更多资源）
- 海外游历 → 引出遗迹探索（需要更强功法）
- 遗迹探索 → 引出终极游历（需要最终对抗）

这种叙事衔接保证了游历之间的连贯性，避免玩家产生「割裂感」。

### 3.6 世界观与游历融合

《阳神》的游历不脱离世界观主线——每次游历的出发原因都与核心冲突（对抗造化道/造化之神）紧密相关。建议在设计游历系统时，始终让每个游历服务于主线的推进，而非孤立的支线任务。
