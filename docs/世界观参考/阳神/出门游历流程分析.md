## 设定摘要

《阳神》的游历不是从宗门任务板出发，而是从读书人的现实路径扩展：侯府/科举提供社会入口，西山狐谷打开修行世界，荒郊日光和海域妖仙提供修炼与资源冲突，军政与王府任务提供制度压力，后期才进入太古龙之墓地、九渊神域、虚空乱流等高危遗迹。

## 原作事实

- 早期游历紧邻现实世界：侯府、玉京、西山、荒郊、弓箭铺、科举场景与修行线交错。
- 妖族接触不是野外打怪，而是狐族读书、请教、藏书和大禅寺遗产线。
- 学习游历可通过功法交换、抄录、长老鉴定产生奖励。
- 海域游历牵涉元牝天珠、银鲨王禅银纱、神魂入海与妖仙冲突。
- 长程道术游历受肉身限制，神魂离体时间和护法/安置是硬约束。
- 后期虚空遗迹有高风险入口、空间迷宫和中千世界层级，不适合早期开放。

## 游戏化抽象（我的判断）

- 我的判断：游历流程可拆为 `现实起点 -> 近郊异遇 -> 修炼试炼 -> 功法交换 -> 海外资源 -> 军政任务 -> 虚空遗迹 -> 彼岸法器`。
- 我的判断：每次出行应带 `identity`、`body_status`、`route_info`、`companion`、`legal_risk`、`artifact_storage` 六类状态。
- 我的判断：高阶游历应要求护法、空间能力和情报，而不是只要求等级。
- 我的判断：游历奖励不只掉落，还包括见闻、功法抄本、盟友、官声、民心、法宝线索。
