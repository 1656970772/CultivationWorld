## NPC

### NPC 静态数据

#### 角色类型
1. **武道强者**：专修武道，锤炼肉身
2. **道术修士**：专修道术，锤炼神魂
3. **儒道圣人**：以儒入道，读书明理
4. **佛门高僧**：修习佛法，参禅悟道
5. **魔道巨擘**：修炼魔功，不择手段
6. **神道神灵**：香火成神，依赖信仰

### NPC 实体

#### 主角：洪易
- **身份**：大乾王朝庶子，周易书院创始人
- **核心功法**：融合《易经》与佛门三经，创出"周易道统"
- **境界**：最终超越阳神与粉碎真空，接近"彼岸"
- **核心理念**：以"人人如龙"为目标，试图带领众生超脱苦海
- **成长轨迹**：从科举士子起步，历经武道、道术、神魂修行

#### 主要角色
1. **洪玄机**
   - 身份：大乾太师，洪易之父
   - 类型：武道强者/儒道圣人
   - 特点：理学与武道大家，代表旧秩序
   - 结局：最终被洪易炼化

2. **梦神机**
   - 身份：太上道创始人
   - 类型：道术修士
   - 特点：追求"太上忘情"的极致无情大道
   - 地位：洪易的终极对手

3. **禅银纱**
   - 身份：主角道侣，银鲨王
   - 类型：妖族强者
   - 特点：出身妖族，战力强大
   - 作用：与洪易并肩作战

4. **元妃**
   - 身份：狐族强者
   - 类型：妖族/导师
   - 特点：洪易的"道理之师"，助其领悟"道理最大"

5. **白子岳**
   - 身份：真罡门掌门，八大妖仙之一
   - 类型：武道强者
   - 特点：后投靠洪易，成为其重要助力

### NPC 状态

#### 状态维度
1. **存活**：是/否
2. **年龄**：实际年龄（岁）
3. **寿元**：剩余寿命（年）
4. **伤势**：健康/轻伤/重伤/濒死
5. **道基**：修炼基础稳固程度（1-10）
6. **心境**：心理状态稳定程度（1-10）
7. **修炼进度**：当前境界完成度（%）
8. **因果缠结**：与他人因果联系强度（1-10）
9. **气运**：命运眷顾程度（1-10）

### NPC 需求

#### 需求类型
1. **生存需求**：维持生命，避免死亡
2. **修炼需求**：提升境界，增强实力
3. **突破需求**：突破瓶颈，进入新境界
4. **传承需求**：传承道统，培养弟子
5. **复仇需求**：报复仇敌，了结因果
6. **保护需求**：保护亲友，守护重要之人
7. **彼岸需求**：冲击彼岸，追求永恒

#### GOAP行为链（目标导向行动规划）
```
目标：突破到阳神境界
└── 前提：达到雷劫九重
    └── 前提：渡过雷劫八重
        └── 前提：准备渡劫法宝
            ├── 行动：炼制法宝（需要材料）
            ├── 行动：购买材料（需要灵石）
            └── 行动：获取灵石（需要完成任务/探索秘境）
```

### NPC 行为

1. **修炼**：日常修炼，提升实力
2. **炼丹**：炼制丹药，辅助修炼
3. **炼器**：炼制法宝，增强战力
4. **布阵**：布置阵法，防御或攻击
5. **征战**：参与战斗，争夺资源
6. **探索秘境**：进入秘境，寻找机缘
7. **建立神国**：开辟独立空间
8. **冲击彼岸**：尝试超脱轮回
