## 设定摘要

角色 AI 不应只按“战斗/非战斗”拆分。《阳神》人物的行为动机与身份绑定：洪易读书、辩难、修炼、著经和制度破局联动；洪玄机维护旧秩序、朝堂和家族权威；杨盘用皇权与未来推演维护统治；苏沐代表太上道试探与干预；禅银纱等妖仙强者在结盟时会考虑肉身、海域、道术风险；梦神机则偏向长线超脱与大势干预。

## 原作事实

- 洪易会在弱势时通过读书、科举和身份合法性积累资源；面对对手时能用文字和义理破道心。
- 洪玄机拥有朝堂权威和旧秩序位置，AI 不宜只表现为武力追杀。
- 杨盘的行为核心是皇权、继承和未来掌控，后期未来之主可作为高阶推演能力来源。
- 苏沐是太上道圣女，行动通常携带圣地立场和政治判断。
- 禅银纱是银鲨王，不是随从单位；她的 AI 需要保留妖仙强者的独立判断。
- 梦神机的行动频率应低，但介入强度高，关注超脱、太上道和天下大势。

## 游戏化抽象（我的判断）

### Utility 权重建议

| NPC | 高权重目标 | 中权重目标 | 低权重/抑制 |
|---|---|---|---|
| 洪易 | 道理/义理胜利、修炼突破、著经立道、保护盟友 | 科举/官声、资源获取、军事掌控 | 无证据滥杀、无目的夺宝 |
| 洪玄机 | 朝堂秩序、家族权威、旧礼法、皇权同盟 | 武道突破、压制洪易 | 公开破坏法统名分 |
| 杨盘 | 皇权稳定、未来掌控、资源集中 | 夺嫡平衡、神庙/外敌应对 | 放任不确定强者坐大 |
| 苏沐 | 太上道利益、圣地判断、试探强者 | 夺嫡布局、道术斗法 | 无立场个人冒险 |
| 禅银纱 | 自身安全、海域/妖仙利益、盟友协作 | 道术交流、资源交换 | 无视肉身限制的冒进 |
| 梦神机 | 超脱契机、太上道安全、雷劫/阳神目标 | 大势观察、关键棋局 | 日常小冲突 |

### GOAP 行为链建议

- 洪易：发现压迫或义理破绽 -> 收集典籍/证据 -> 以文章或言语破局 -> 获取合法身份或道统声望 -> 必要时转入道术/武道对抗。
- 洪玄机：检测家族/朝堂秩序受损 -> 评估名分与舆论 -> 通过朝堂、家法或武力施压 -> 避免公开破坏法统。
- 杨盘：检测未来不确定性 -> 调用情报/未来推演 -> 平衡夺嫡与朝局 -> 集中皇权资源压制变量。
- 苏沐：评估太上道布局 -> 试探目标道术与道心 -> 以圣地身份提出条件或压迫 -> 若受阻则上升为道统冲突。
- 禅银纱：评估肉身/神魂风险 -> 判断盟友可信度 -> 提供海域、妖仙、道术支援 -> 高风险时要求护法或后勤。
