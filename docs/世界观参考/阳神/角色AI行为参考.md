# 阳神 — 角色AI行为参考

## AI架构映射说明

《阳神》的主角洪易是"书生型变革者"的典范，其AI行为模式的核心特征是"借势"与"以理服人/必要时碾压"的二元切换：

- **BT层**：洪易的应急体系包含两条并行分支——评估对手实力后的"镇压/拖延"分支，以及"过去弥陀经保命"的安全网分支。这种"实力判定 → 行为分流"的模式可作为所有"策略型NPC"的BT模板。
- **Utility层**：洪易的目标权重从"洗刷母辱（情感驱动）"逐步转向"创《易经》/整合道统（理念驱动）"再到"建立理想国（制度驱动）"，是一个从个人情感向宏大理念的渐进升级。
- **GOAP层**：每一步都体现"借力打力"——考功名借朝廷之势，收服禅银纱借鬼仙之势，对抗皇室借云蒙之势。

## 核心角色分析

### 1. 洪易

#### 性格标签
书生出身（知书达理/初始弱势） / 理想主义+现实主义（儒学为理想，武道/权力为手段） / 善于借势 / 以理服人但必要时杀伐果断

#### BT层应急反应
| 触发条件 | 抢占行为 | 优先级 |
|---|---|---|
| 遭遇生死危机 | 使用"过去弥陀经"或"未来无生经"保命 | 最高 |
| 对手实力评估(不如己) | 直接镇压/碾压 | 次高 |
| 对手实力评估(过强) | 启动口才拖延 / 利用地形/阵法脱身 | 次高 |
| 遭遇羞辱/挑衅(涉及母辱) | 愤怒阈值骤降 → 立刻出手 | 极高(覆盖常规评估) |

**核心逻辑**：洪易的BT层有一个"实力判定"前置节点，根据对手实力分流至"镇压"或"智取"两条路径。唯一的覆盖条件是对"母辱"的敏感——触发时绕过一切理性评估。

#### Utility层权重打分

| 周期 | 目标 | 权重(0-1) | 触发条件 |
|---|---|---|---|
| 短期 | 洗刷母辱，考取功名 | 0.95 | 故事初期 |
| 中期 | 创《易经》，打破百家垄断，整合道统 | 0.8 | 获得过去经后 |
| 长期 | 建立"丰饶之国"（理想国），对抗造化之神 | 0.5(渐进至1.0) | 整合道统后 |

**权重转换规则**：洪易的权重转移由"道统整合度"驱动——每整合一方道统，Utility视野就向更宏大的目标扩展一层。

#### GOAP层决策链

```
[离家考功名] → [获取生存资源与正统身份]
    ├── 策略：被家族排挤→唯有权力和实力才能立足
    ├── 行动：进京赶考 + 寻找《过去经》契机
    └── 产出：功名身份，修道起点

[收服禅银纱] → [获取鬼仙战力与同盟]
    ├── 策略：分析对方弱点（心魔）
    ├── 行动：使用《过去经》度化/结盟 → 转化为己方战力
    └── 产出：核心盟友与战力

[对抗大乾皇室] → [借力打力，积累势力]
    ├── 策略：面对洪玄机压迫→不盲目硬拼
    ├── 行动：游走于大罗派与云蒙国之间 → 建立"易"字号势力
    └── 产出：独立势力与政治资本

[最终证道] → [创立新秩序]
    ├── 策略：集齐过去、现在、未来三经
    ├── 行动：创立新道德体系 → 玉京城与诸神大战
    └── 产出：新秩序建立，理想国落地
```

#### 标志事件

1. **离家考功名** — 从"被排挤的庶子"到"主动求变"，确立借势策略。
2. **收服禅银纱** — 以理服人的典范操作，度化强敌转化为盟友。
3. **对抗大乾皇室** — 借力打力的高峰期，合纵连横建立独立势力。
4. **集齐三经创立新秩序** — 从个人修行到制度建设的终极跨越。

## 其他角色

### 梦神机
**性格标签**：隐忍 / 大宗师气度 / 追求绝对自由 / 亦正亦邪

- **BT应急**：面对洪易崛起→初期观察→中期利用棋局博弈
- **Utility**：破碎虚空，追求超越阳神的境界
- **决策特征**：布下天下棋局，扶持或打压各方势力以寻求自己突破契机，是洪易最复杂的对手

### 禅银纱
**性格标签**：敢爱敢恨 / 妖娆 / 追求力量 / 从属转伴侣

- **BT应急**：遭遇强弱→利用魅惑或元素力量周旋
- **Utility**：摆脱天道束缚，成就鬼仙/阳神
- **决策特征**：被洪易度化后成为最坚定的盟友，多次在关键时刻提供情报与战力支持

## NPC类型化归纳

### 1. 世家子弟型
- **行为逻辑**：傲慢、重视血统、初期压迫主角
- **BT映射**：遭遇主角反抗→抢占"压制/羞辱"行为
- **Utility映射**：家族荣誉权重 > 个人道德权重
- **GOAP映射**：挑衅 → 被打败 → 怀恨在心或认怂
- **实例**：洪易的家族对手

### 2. 宗门长老型
- **行为逻辑**：固执、维护道统、因循守旧
- **BT映射**：道统受挑战→抢占"镇压异端"行为
- **Utility映射**：道统纯正权重极高
- **GOAP映射**：视主角为异端 → 派出弟子试探 → 最终被超越
- **实例**：诸子百家保守派

### 3. 皇朝权谋型
- **行为逻辑**：多疑、务实、手握重兵，擅长合纵连横
- **BT映射**：势力格局变化→抢占"重新站队"评估
- **Utility映射**：皇权稳固权重 > 一切
- **GOAP映射**：观察 → 试探 → 结盟或对抗 → 主角势大后归顺
- **实例**：洪玄机、乾帝等

### 4. 方外高人型
- **行为逻辑**：随性、追求大道、不问世事（涉及机缘则例外）
- **BT映射**：机缘出现→抢占"干预"行为（覆盖隐居状态）
- **Utility映射**：大道追求权重极高，世俗事务权重极低
- **GOAP映射**：隐居 → 观察主角 → 关键节点点拨或考验
- **实例**：部分隐修大能
