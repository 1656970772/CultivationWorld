## 世界 World

### 世界实体

#### 宇宙层级
《阳神》的宇宙结构遵循"小千 → 中千 → 大千"的嵌套模式，并存在更高维度的领域：

1. **小千世界**：一个稳固的独立空间，相当于一个小型宇宙或位面
2. **中千世界**：由无量小千世界构成，拥有日月星辰、山河大地，如同一个完整的"星球世界"
3. **大千世界**：由无量中千世界构成，是故事的主要舞台，如"中央世界"
4. **天外天**：位于大千世界之上的更高维度空间，是顶尖强者的居所和战场
5. **太古九渊神域**：太古时代遗留的九处深渊秘境，封印着远古强者、神魔与宝物，连通冥古与今古
6. **光阴长河**：贯穿过去、现在、未来的时间之河，诸子、圣皇等强者的"道"与"影子"存留其中
7. **起源之地**：宇宙开辟之初的核心战场，最终成为洪易融合诸道、冲击彼岸的终极之地

#### 历史纪元
宇宙以"一元"（129,600年）为周期轮回，太古阳神亦难超脱一元寿命：

1. **冥古宙**：宇宙初开，混沌未分
2. **寒武纪元**：生命萌芽，万物初生
3. **太古**：人、神、仙、妖、魔共存，盘皇崛起，为人族争得主导权
4. **上古**：圣皇立下人道规则
5. **中古**：诸子削皇帝号，以道德文章治天下
6. **今古**：王朝更迭频繁，如大周、大乾等

#### 地理区域
1. **中央世界**：大千世界核心
   - 中土天州：文明中心，大乾王朝所在地
   - 西域：异域风情，多沙漠绿洲
   - 四海：环绕大陆的广阔海域
   - 莽荒：未开化的原始地带，妖兽横行
2. **大乾王朝**：中央世界最强王朝，是故事前期的主要政治舞台
3. **海外诸国**：如云蒙帝国、柔然帝国、火罗国等，各具特色

### 世界状态

1. **纪元阶段**：今古纪元末期，一元轮回即将结束
2. **灵气/元气浓度**：随纪元更迭波动，今古纪元灵气相对稀薄
3. **雷劫活跃度**：每129,600年一次大轮回，雷劫频率与强度周期性变化
4. **香火愿力分布**：集中在大乾王朝、精元神庙等神道势力控制区
5. **龙脉状态**：中央世界龙脉被大乾王朝控制，海外诸国争夺剩余龙脉
6. **神国强度**：太上道、造化道等顶尖势力已建立神国雏形

### 世界行为/规则

1. **宇宙轮回规则**：一元129,600年，宇宙经历生、长、衰、灭的完整周期
2. **雷劫规则**：九重天劫，每重对应不同境界，失败则形神俱灭
3. **香火愿力规则**：信仰之力可壮大神魂，但依赖信徒，自身无法修炼
4. **龙脉规则**：大山的灵魂，地气汇聚之所，影响国家气运和修士修炼
5. **神国规则**：强者可开辟独立神国，庇护信徒，抵御外敌
6. **彼岸规则**：超脱一元轮回，达到永恒不灭的境界

### 世界需求

1. **制造冲突**：通过资源争夺、境界突破、势力竞争推动剧情发展
2. **推动纪元演化**：一元轮回机制强制世界不断更新换代
3. **维持平衡**：武道与道术、王朝与宗门、神道与魔道之间的动态平衡
4. **避免死局**：通过秘境探索、功法创新、跨界交流提供变数

### 消耗/产出关系

| 消耗项 | 产出项 | 转化率 |
|--------|--------|--------|
| 时间（修炼/突破） | 境界提升/寿元增加 | 非线性，越后期效率越低 |
| 资源（灵石/天材地宝） | 实力增长/法宝炼制 | 1:1基础转化，高级资源有特殊加成 |
| 香火愿力 | 神道力量/神国建设 | 信徒数量×信仰纯度 |
| 龙脉地气 | 国家气运/修炼加速 | 龙脉等级×控制程度 |
| 雷劫风险 | 神魂淬炼/境界突破 | 成功率随境界提升而降低 |
