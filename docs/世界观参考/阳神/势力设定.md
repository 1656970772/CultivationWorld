## 势力 Faction

### 势力静态数据

#### 类型枚举
1. **王朝**：以国家政权形式存在的势力
2. **道统**：以修炼传承为核心的宗门
3. **宗门**：综合性修炼组织
4. **神道**：以香火愿力为基础的信仰体系
5. **魔道**：追求力量与欲望，不择手段的势力
6. **世家**：以家族血脉为纽带的修行势力
7. **圣地**：传承久远、底蕴深厚的特殊地域

### 势力实体

#### 主要王朝
1. **大乾王朝**
   - 类型：王朝
   - 领袖：乾帝
   - 核心区域：中土天州
   - 制度：文官科举与武官军功并行
   - 实力：中央世界霸主，控制主要龙脉
   - 特点：皇权与顶尖强者相互制衡

2. **云蒙帝国**
   - 类型：王朝
   - 特点：以骑兵和刀法著称，精于炼体
   - 实力：海外强国，与大乾多次交锋
   - 资源：战马、矿产丰富

3. **柔然帝国、火罗国等**
   - 类型：王朝
   - 特点：各具特色的邻国
   - 作用：主角对外战争和势力扩张的主要对象

#### 其他势力
1. **巫鬼道、闻香教**：以鬼魂、怨念、香火为食的邪道
2. **梵州梵家**：以家族为核心的修行势力，掌握独特传承
3. **祖神山**：上古传承的圣地，与太古神王、盘皇等关系密切

### 势力状态

1. **稳定度**：大乾王朝（高）、太上道（中）、大禅寺（低，已被灭）
2. **资源压力**：随修士数量增加而增大，海外势力资源相对匮乏
3. **高阶修士数量**：太上道（阳神1+人仙若干）、大乾王朝（人仙若干）
4. **香火愿力储备**：精元神庙（极高）、大乾王朝（高）
5. **龙脉控制度**：大乾王朝（中央世界主要龙脉）、海外诸国（分散控制）
6. **跨界影响力**：太上道（跨越多重世界）、大乾王朝（中央世界主导）

### 势力需求

#### 需求类型
1. **生存需求**：维持势力存在的基本条件
2. **扩张需求**：扩大势力范围和控制区域
3. **资源需求**：获取修炼资源和天材地宝
4. **传承需求**：确保道统延续和功法传承
5. **争夺龙脉**：控制地气汇聚点，增强气运
6. **建立神国**：开辟独立空间，庇护信徒
7. **冲击彼岸**：追求永恒不灭的终极目标

#### 优先级计算逻辑
```
优先级 = 基础权重 × 紧急系数 × 资源系数

基础权重：
- 生存需求：100
- 传承需求：80
- 扩张需求：60
- 资源需求：40
- 其他需求：20

紧急系数（1-3）：
- 立即威胁：3
- 短期风险：2
- 长期规划：1

资源系数（0.5-2）：
- 资源充足：2
- 资源一般：1
- 资源匮乏：0.5
```

### 势力行为

1. **开国**：建立新的王朝或国家
2. **变法**：改革制度以适应新时代
3. **传法**：传播道统，招收弟子
4. **争夺资源**：抢夺灵石、天材地宝等
5. **布设大阵**：建立防御或攻击阵法
6. **建立神国**：开辟独立空间
7. **冲击彼岸**：尝试超脱一元轮回
