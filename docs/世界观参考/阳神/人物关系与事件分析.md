## 设定摘要

《阳神》关系网的核心不是“主角宗门人际圈”，而是侯府父子名分、朝廷法统、皇权继承、圣地干预、妖仙盟友、民间信仰和彼岸道统之间的拉扯。洪易的人际关系不断从私人屈辱扩展到制度冲突：父子冲突牵连家法与国法，朝堂冲突牵连科举与夺嫡，太上道与精元神庙冲突牵连圣地与宗教战争，最终落到《易经》道统与“人人如龙”的秩序重写。

## 原作事实

- 洪易/洪玄机：原文明确父子/侯府阶层基础，后续附体洪康事件把私人恩怨、朝廷律法、舆论和父子名分交织在一起。
- 洪易/杨盘：以朝堂、皇权、未来之主、造化之舟等资源线形成对抗；杨盘不只是君主 NPC，而是皇权修行者。
- 洪易/太上道/苏沐：苏沐作为太上道圣女，因太上道支持冠军侯夺位与洪易冲突。
- 洪易/妖仙：元妃、白子岳、禅银纱等将狐族、转世、海域妖仙和道侣/盟友线接入主线。
- 大乾/精元神庙：精元神庙发动圣战，体现政教合一与帝国军事的关系。
- 洪易/天下众生：人人如龙宏愿把关系网推向道统传播和秩序改革。

## 游戏化抽象（我的判断）

- 我的判断：关系类型应分为 `family_legitimacy`、`court_law`、`succession_politics`、`sacred_site_intervention`、`demon_ally`、`religion_war`、`dao_ideal`。
- 我的判断：关系值不应只用好感度；父子关系还需名分压力，朝廷关系需要法统/官声，圣地关系需要道统利益。
- 我的判断：事件升级可以从私人冲突走向制度冲突，例如“侯府受辱 -> 科举入世 -> 朝堂弹劾 -> 道统公开对抗”。
- 我的判断：妖仙关系应支持从抢夺资源、误会、斗法、盟约到道侣/并肩作战的多阶段变化。
