## 设定摘要

《阳神》的冲突常从个人事件开始，迅速升级为制度、法统、道统或世界终局层面的冲突。洪易与洪玄机不是简单复仇；父子名分、朝廷律法和士大夫物议共同压迫。太上道和精元神庙说明修行势力能直接影响皇权和战争。造化之舟、未来之主、众圣殿等法宝把冲突推向彼岸与世界秩序。

## 原作事实

- 家族冲突：洪易的低位与洪玄机的权威形成早期压迫背景。
- 法统冲突：附体洪康事件显示道术行为会触犯朝廷律法和父子名分。
- 军政冲突：军功、太子、玉亲王和地方权贵互相牵连。
- 圣地冲突：太上道通过苏沐支持冠军侯夺位，介入皇权。
- 宗教战争：精元神庙发动西域圣战，对大乾形成外部压力。
- 资源冲突：空间法宝影响后勤，彼岸法器影响终局格局。
- 理念冲突：人人如龙与永久统御形成洪易/杨盘/洪玄机的核心理念对照。

## 游戏化抽象（我的判断）

- 我的判断：冲突系统应按“升级链”设计：私人矛盾 -> 法统舆论 -> 朝堂军事 -> 圣地介入 -> 世界法器 -> 道统终局。
- 我的判断：每次冲突应记录 `public_legitimacy`、`court_risk`、`dao_reputation`、`artifact_stakes`、`faith_pressure`。
- 我的判断：玩家选择不应只影响胜负，还应影响名分、舆论、民心、圣地敌对和后续资源开放。
- 我的判断：宗教战争类事件需要战略资源和地图压力，不宜做成单次斗法。
