## 物品 Item

### 物品静态数据

#### 基础属性
1. **名称**：物品的正式名称
2. **类型**：资源/功法/法宝/丹药等
3. **品质**：凡品/灵品/仙品/神品
4. **等级**：1-10级
5. **重量**：物理重量（斤）
6. **体积**：占用空间（立方尺）
7. **耐久**：使用次数或耐久度
8. **价值**：市场价值（灵石）

### 物品分类体系

#### 资源类
1. **灵石**：修炼通用货币，蕴含灵气
   - 下品灵石：基础修炼资源
   - 中品灵石：中等修炼资源
   - 上品灵石：高级修炼资源
   - 极品灵石：顶级修炼资源

2. **元气**：天地灵气，修炼基础
   - 普通元气：无处不在
   - 精纯元气：秘境特有
   - 先天元气：开天辟地时遗留

3. **香火愿力**：信仰之力，神道基础
   - 普通愿力：凡人信仰
   - 精纯愿力：虔诚信徒
   - 神圣愿力：圣徒奉献

4. **龙脉**：地气汇聚，影响气运
   - 小型龙脉：影响方圆百里
   - 中型龙脉：影响方圆千里
   - 大型龙脉：影响一国气运
   - 祖龙脉：影响整个世界

5. **天材地宝**：天然生成的珍贵材料
   - 药材类：炼丹原料
   - 矿石类：炼器原料
   - 灵物类：特殊功效

6. **丹药**：炼制而成的药物
   - 修炼类：辅助修炼
   - 疗伤类：治疗伤势
   - 突破类：帮助突破瓶颈

### 物品状态

1. **完好度**：全新/轻微磨损/中度磨损/严重损坏/破碎
2. **能量储备**：满/高/中/低/空
3. **绑定状态**：无主/认主/灵魂绑定
4. **使用冷却**：可用/冷却中（剩余时间）
5. **特殊状态**：封印中/激活中/进化中

### 物品需求

1. **认主需求**：需要特定条件或实力才能认主
2. **能量需求**：需要特定能量驱动或充能
3. **使用需求**：需要特定境界或技能才能使用
4. **维护需求**：需要定期维护保养
5. **进化需求**：需要特定材料或条件才能进化

### 物品行为

1. **采集**：从自然界获取资源
2. **炼制**：将原材料加工成成品
3. **消耗**：使用物品，消耗其能量或耐久
4. **交易**：通过市场交换物品
5. **掠夺**：从他人手中强行获取
6. **祭炼**：通过特殊仪式强化物品
7. **开辟**：开辟新的空间或世界
