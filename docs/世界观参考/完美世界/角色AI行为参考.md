# 《完美世界》角色AI行为参考

## 一、AI架构映射说明
- **BT层**：情绪反应 / 应急抢占
- **Utility层**：人生目标选择（权重打分）
- **GOAP层**：目标分解为步骤链
- **Action Queue**：具体动作序列

## 二、核心角色分析

### 石昊
**性格标签**：霸气、重情义、杀伐果断、护短、吃货属性、乐天派、野性直觉

**BT层 - 应急反应**：
- 触发：被欺负 / 同伴受威胁 / 遇到强敌
- 行为：直接干架 → 越级挑战(若打不过临时突破) → 以战养战(吸收对方宝术/精血) → 追杀到底(绝不留下祸患)

**Utility层 - 长期目标权重**：
- 亲友安全（∞，高于自身安全）
- 变强（950）
- 寻找父母（900）
- 守护石村/补天阁（850）
- 宝物/美食（750，遇到吃的权重飙升）
- 平定异域动乱（700，中后期）

**GOAP层 - 标志事件决策链**：
1. 补天阁覆灭：拼命修炼 → 联合少年战友 → 潜入敌后 → 强势斩杀仇敌 → 自损也要重创对方传承
2. 七神下界：明知必死也迎战 → 借用法宝/禁区力量 → 每战濒临死亡极限突破 → 最终惨胜
3. 异域征战：伪装身份混入 → 监狱救人或切磋夺宝 → 暴露身份大杀四方 → 举世皆敌也边战边退

**与韩立对比**：
- 石昊：直率/正面硬刚，拳头即真理，兄弟遍地，情感外露
- 韩立：谨慎/隐忍布局，利益即真理，独狼，情感内敛

### 柳神（导师型NPC）
**行为模式**：高维观察者，平时休眠/低活跃度，主角遇"灭世级"危机时苏醒释放大招或关键指点
**AI参数**：极低频率但极高权重的干预

### 云曦/火灵儿（情感NPC）
**行为模式**：守护与等待，高剧情推动权重
**AI参数**：遇到石昊→支持/辅助/或被抓走作为主角行动理由

### 石毅/重瞳者（对手型NPC）
**行为模式**：天才的傲慢与执念，抢夺气运/主角光环
**AI参数**：不定期出现骚扰主角，在主角突破关键时刻前来挑战

## 三、NPC类型化归纳

### 1. 天才/天骄型 (Prodigy)
- 示例：少年石昊、秦昊、十冠王
- BT层：高自信，主动寻找强敌切磋
- Utility：宝物敏感度极高，有专属宝术(Skill Set)
- GOAP：感知强敌 → 挑战切磋 → 掠夺宝术

### 2. 纯血生物/太古凶兽型 (Beast)
- 示例：朱厌、黄金狮子
- BT层：野性思维，领地意识强，肉搏倾向
- Utility：领地(800) > 食物(700) > 变强(600)
- 条件：好感度/实力达标时可驯服

### 3. 不朽之王/幕后黑手型 (God Level / Manipulator)
- 示例：安澜、俞陀
- BT层：轻易不出手，出手必碾压
- Utility：收割下界/炼化世界(1000)，布局百万年
- GOAP：布局百万年 → 收割 → 规则级攻击

### 4. 吃瓜群众/八域修士型 (Civilian)
- BT层：趋炎附势，遇战斗围观或逃跑
- Utility：崇拜强者(500) > 传播情报(300) > 自保(900)
