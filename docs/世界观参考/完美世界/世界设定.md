## 一、世界 World

### 1.1 世界实体

《完美世界》的宇宙由多层嵌套的位面构成，核心结构为"下界八域 → 上界三千州 → 九天十地 → 异域/仙域/葬土 → 界海 → 上苍之上"的垂直层级体系。

#### 1.1.1 下界八域

- **名称**：天域、地域、玄域、黄域、宇域、宙域、洪域、荒域（共八块大陆）
- **定位**：故事起点，石昊的故乡荒域所在；被上界视为"牢笼"和"人体药田"
- **本质**：灵气稀薄的底层位面，是上界大教的资源产地与试炼场
- **面积**：每一域都极为辽阔，普通修士穷尽一生难以跨越
- **隐秘**：八域人族实为上古"边荒七王"后裔，曾为守护九天十地浴血奋战，后被上界污蔑为"罪血后代"，血脉中被刻入诅咒烙印
- **域使**：各域之间存在域使，负责维护下界秩序和通道

#### 1.1.2 上界三千州

- **来源**：九天十地中"十地"被打碎后残存的碎片
- **规模**：每州面积远超下界八域总和，修士终其一生难以跨越一州
- **格局**：大教林立、天骄争霸、遍布禁区和秘境
- **势力**：补天教、截天教、西方教、不老山、仙殿、天国、冥土等顶级势力盘踞
- **特殊**：三千州每隔一段时间会举办"三千州大战"，选拔最杰出的天骄

#### 1.1.3 九天十地

九天十地是上古时期完整大界的主体，在仙古纪元因与异域大战而崩碎。

- **历史**：仙古纪元，九天十地是一个完整大界，因抵抗异域入侵和黑暗侵蚀而支离破碎
- **边荒帝关**：九天十地与异域的边境要塞，由孟天正等至尊镇守，是抵御异域入侵的第一道防线
- **帝城**：边荒帝关的核心，由历代至尊加固

#### 1.1.4 异域

- **本质**：黑暗四帝（苍帝、鸿帝、羽帝、灭世老人）的前哨站与实验场
- **统治**：由不朽之王统治，安澜、俞陀等为核心战力
- **手段**：通过"黑血"污染并同化其他世界，将被污染者转化为黑暗生灵
- **目标**：持续扩张，侵蚀诸天万界，收割世界本源
- **帝族**：异域存在不朽帝族（如安澜族、俞陀族），血脉高贵，战力恐怖

#### 1.1.5 仙域

- **起源**：仙古纪元修士为对抗黑暗而建立的理想国度
- **特点**：拥有完整的大道规则，仙气充沛，是修士梦寐以求的终极归宿
- **封闭**：仙古纪元后期，因恐惧黑暗侵蚀而封闭界门，拒绝九天十地生灵进入
- **打通**：后被石昊以无上伟力打通界门，成为对抗黑暗的重要后方基地
- **层次**：仙域亦分层次，存在普通仙域与核心仙域之别

#### 1.1.6 葬土

- **创立**：由葬士始祖创建，以特殊方式避开黑暗清算
- **规则**：修炼体系与生死规则迥异于外界，以"葬"为修行核心
- **生灵**：葬士以休眠为主要生存方式，苏醒后战力惊人
- **关联**：与冥土有着千丝万缕的联系，但本质不同

#### 1.1.7 界海

- **位置**：位于各大世界之外的混乱之地，是破碎古界的汇聚之所
- **危险性**：连仙王进入都要小心翼翼，是通往更高层次（上苍之上）的试炼场
- **特征**：充斥着残破古界、黑色古船、挣扎的残魂和未知的恐怖存在
- **浪涛**：界海中的浪涛足以拍碎普通世界，黑暗气息浓郁
- **彼岸**：界海对岸是终极真相所在——尸骸仙帝的所在

#### 1.1.8 上苍之上

- **定位**：界海之外更高层次的位面，超越仙帝级别的存在栖居之所
- **信息**：石昊独断万古之后前往的终极之地，设定极为神秘
- **关联**：与黑暗源头、祭道之上的秘密密切相关

#### 1.1.9 虚神界

- **创造者**：上古神明联手构建的精神世界
- **功能**：磨砺修士精神力、提供安全对战环境
- **规则**：肉身无法进入，只有精神体可降临；死亡则精神受损而非真实陨落
- **资源**：存在各种记录碑、藏经阁、秘境通道，连接下界与上界
- **争夺**：各方势力在此争夺资源、探查情报，是跨位面的信息战场

#### 1.1.10 秘境

- **百断山**：下界著名秘境，石昊在此获得大量机缘
- **仙古遗地**：仙古纪元的遗迹，产出仙种、圣药、古宝等顶级资源
- **原始之门**：连接不同位面的古老通道
- **禁区**：各处存在生命禁区，有古代至尊或恐怖存在蛰伏

### 1.2 世界状态

#### 1.2.1 纪元体系

| 纪元 | 状态 | 特征 |
|------|------|------|
| 帝落时代 | 已终结 | 最古老的纪元，准仙帝陨落的时代，留下脚印帝的传说 |
| 仙古纪元 | 已覆灭 | 高度发达的修炼文明，因异域入侵和黑暗侵蚀而终结 |
| 乱古纪元 | 进行中 | 主角石昊所在纪元，黑暗动乱频发，仙路断绝 |
| 末法时代 | 将到来 | 大道衰落、灵气枯竭的终极时代 |

#### 1.2.2 灵气强度

- **下界八域**：灵气稀薄（1级），仅够修炼到尊者境
- **上界三千州**：灵气中等（3级），可支撑至尊境修炼
- **九天十地**：灵气充沛（5级），可诞生真仙
- **仙域**：灵气充裕（8级），完整大道，可修炼至仙王
- **界海**：灵气混乱（波动1-10级），不稳定但蕴含顶级资源
- **虚神界**：无灵气，纯精神修炼场所

#### 1.2.3 黑暗侵蚀程度

- **起源**：尸骸仙帝被黑暗污染后，黑暗物质扩散至诸天
- **传播**：通过黑血污染修士、侵蚀世界本源
- **下界八域**：侵蚀度低（10%），偏远地带
- **异域**：侵蚀度高（80%），黑暗前哨
- **九天十地**：侵蚀度中（40%），处于对抗前线
- **仙域**：侵蚀度低（15%），但恐惧蔓延
- **界海**：侵蚀度极高（90%+），黑暗的源头区域

#### 1.2.4 界海波涛

- **周期性**：界海浪涛有周期性涨落，涨潮时黑暗气息外溢
- **影响**：界海波涛汹涌时，连仙王都不敢轻易横渡
- **关联**：与黑暗动乱的周期性爆发直接相关

#### 1.2.5 秘境活跃度

- **开启周期**：秘境有固定开启周期（百年、千年、万年不等）
- **活跃度波动**：仙古遗地等高级秘境在纪元末期更为活跃
- **关联**：秘境活跃度与天地大道变化、灵气潮汐相关

### 1.3 世界行为/规则

#### 1.3.1 黑暗动乱规则

- **触发条件**：黑暗侵蚀积累到临界值 → 黑暗至尊苏醒 → 黑暗动乱爆发
- **表现形式**：黑暗生灵大规模入侵，收割生灵，吞噬世界本源
- **周期性**：约每隔数十万年至百万年爆发一次，与界海潮汐同步
- **终结条件**：黑暗至尊被击杀/封印，或世界本源被收割殆尽
- **终极根源**：尸骸仙帝被黑暗污染，黑暗物质从源头扩散

#### 1.3.2 成仙规则

- **仙古法**：以仙种为根基，融合天地大道，最终成仙
- **今世法**：以身为种，将自身开辟为世界，石昊开创
- **红尘仙**：在红尘中历经万劫而不灭，最终成仙（如曹雨生/段德）
- **条件**：至尊大圆满 → 渡过仙劫 → 凝聚仙道法则 → 真仙
- **仙域接引**：仙古纪元修士可被仙域接引，乱古纪元此路已断

#### 1.3.3 界海规则

- **进入条件**：至少需要仙王级战力方可尝试横渡
- **风险**：迷失方向、被黑暗侵蚀、遭遇界海中的恐怖存在
- **指引**：传说存在横渡界海的古路，但早已失传
- **彼岸**：界海彼岸是真相所在，但也是最危险之地

#### 1.3.4 虚神界规则

- **进入**：精神体降临，肉身为锚点留在外界
- **死亡**：精神体被击杀则精神受损，不会真实死亡，但严重者可导致精神崩溃
- **等级**：分区域，修为越高可进入越深层
- **资源**：记录碑记载各种功法、宝术；存在连通各处秘境的通道
- **限制**：无法带入或带出实物，但获得的知识和精神体提升可带回

#### 1.3.5 纪元更替规则

- **小纪元**：约十二万九千六百年（一元），天地大道会发生小幅度变化
- **大纪元**：当黑暗动乱积累至不可调和，或天地本源枯竭，触发大纪元更替
- **末法**：纪元末期，大道开始衰败，灵气逐渐枯竭
- **新生**：旧纪元终结后，若世界未被完全毁灭，新的纪元将在残骸上重生

#### 1.3.6 血脉传承规则

- **血脉觉醒**：后裔可能觉醒祖先的神通和天赋
- **血脉诅咒**：罪血后人被刻入诅咒烙印，修炼受限
- **血脉压制**：高级血脉对低级血脉存在天然压制

### 1.4 世界需求

| 需求 | 说明 | 优先级 |
|------|------|--------|
| 制造冲突 | 维持世界的历史张力，驱动各方势力行动 | 高 |
| 维持平衡 | 防止任一势力/位面彻底碾压其他，保持世界多元 | 高 |
| 推动纪元演化 | 随剧情推进，纪元应有生灭更替 | 中 |
| 避免死局 | 确保世界不会被黑暗彻底吞噬，保留希望火种 | 最高 |
| 资源循环 | 灵气、天材地宝的再生与消耗需保持可持续性 | 中 |
| 天地反噬 | 对过于逆天的行为（如批量屠杀低阶生灵）产生反噬 | 低 |

### 1.5 世界消耗/产出关系

```
灵气来源：天地大道 → 灵脉/仙脉 → 弥漫至各层位面
          ↑                    ↓
     仙域供给         修士吸收修炼 → 消耗灵气
     
黑暗侵蚀：尸骸仙帝 → 黑暗物质 → 污染世界本源 → 灵气枯竭
          ↓
     黑暗至尊吞噬世界本源 → 减少世界总体灵气量

纪元产出：每纪元的终结 → 新的仙种/圣药/古宝诞生 → 下一纪元修士获取机缘
         ↓
    大破灭 → 残破古界碎片 → 汇入界海 → 成为新资源
```
