# 《完美世界》法宝武器·妖兽·丹药 补充分析

## 一、法宝武器

### 1.1 品阶体系

| 品阶 | 代表 | 特点 |
|:---|:---|:---|
| 普通法器/宝具 | 部落祭灵遗宝、上古法器 | 部落家族传承 |
| 重宝 | 五行环、翻天印 | 影响一方战局 |
| 至宝 | 小塔、六道轮回盘 | 接近仙道层次，仙王/巨头持有 |
| 无上至宝/混沌至宝 | 三世铜棺、法则池、仙帝剑 | 宇宙级，开辟世界、逆转时空 |
| 准仙帝器/仙帝器 | 荒剑 | 仙帝级战力标配 |

### 1.2 石昊核心法宝

| 法宝 | 品阶 | 核心属性与用途 |
|:---|:---|:---|
| 三世铜棺 | 混沌至宝 | 九龙拉棺，内含小世界和符文仙火，贯穿三部曲核心谜团 |
| 大罗剑胎 | 帝剑（准仙帝级） | 遇强则强，吸收仙金。尸骸仙帝用三世铜棺边角料炼制，后熔炼入荒剑 |
| 原始真解 | 无上法器 | 万灵图等篇章，演化万灵印记，辅助修行与布阵 |
| 小塔 | 无上至宝 | 空间至宝，内含小世界，可镇压敌人。帝落时代被打断 |
| 法则池 | 仙帝器 | 主防御、囚禁、炼化敌人道则 |
| 仙帝剑/荒剑 | 仙帝器 | 主杀伐，融合无数至宝，石昊亲手炼制，最终一剑劈开尸骸仙帝 |

### 1.3 其他重要法宝

| 法宝 | 品阶 | 核心能力 |
|:---|:---|:---|
| 仙王五禽扇 | 仙王级 | 自动运转，释放仙王级毁灭力量 |
| 混元葫芦 | 仙王级 | 收摄万物，镇压真仙，攻防一体 |
| 五行旗 | 无上秘宝 | 摇动演化一界，仙王难攻，天庭终极防御 |
| 八凶剑 | 上古法器 | 八种神禽凶兽原始符骨祭炼的剑阵 |

### 1.4 炼制方式要点

- **材料层级**：普通灵材 → 仙金 → 世界石/混沌土 → 万灵血金
- **石昊自创手法**：融合+重炼，将大罗剑胎、紫电锤等熔炼为法则池与仙帝剑
- **修为门槛**：仙王级法宝需仙王修为；仙帝器必需仙帝级祭炼

## 二、妖兽/凶兽

### 2.1 太古十凶

| 十凶 | 核心特性 | 宝术与能力 | 结局/传承 |
|:---|:---|:---|:---|
| 真龙 | 十凶之首，每头最终成仙王 | 真龙宝术：龙道法则极致 | 几灭绝，遗子赤龙葛沽被荒培养成仙王 |
| 鲲鹏 | 可鱼可鸟，太阴太阳，吞噬星河 | 鲲鹏宝术：太阳拳、太阴掌 | 女鲲鹏陨落，鲲鹏子号"不灭生灵" |
| 凤凰 | 极致火道，祥瑞与毁灭 | 凤凰宝术：焚尽诸天 | 堕落血凰被蛄祖斩杀，银凰雪凰为后裔 |
| 雷帝 | 雷灵族极致，天劫演化 | 雷帝宝术：借天劫之力 | 终极古地被黑暗利用 |
| 天角蚁 | 力之极致，可狂化 | 天角蚁宝术：肉身力量极大增强 | 小天角蚁被荒培养成十凶层次 |
| 麒麟 | 祥瑞神兽，蓄力后爆发惊人 | 麒麟宝术：踏碎星辰 | 荒结识一只纯血白麒麟 |
| 蛄 | 时空大道，虚空穿梭封锁 | 蛄族宝术：时空涟漪 | 蛄祖潜伏异域，最终为救石昊而死 |
| 九幽獓 | 头似狼，九尾各末端龙头 | 九幽獓宝术：杀伐诅咒 | 始祖灭世老人被荒镇压 |
| 九叶剑草 | 九片剑叶，剑意冲天 | 草字剑诀：剑意剑阵 | 被四大不朽王围攻后消失 |
| 打神石 | 坚不可摧，无视部分防御 | 打神石宝术：绝对防御 | 石昊亲密伙伴 |

### 2.2 其他重要凶兽/神兽

| 名称 | 特点 |
|:---|:---|
| 混世魔猿 | 朱厌同祖，八九天功（七十二变） |
| 金乌族 | 太阳神鸟后裔，仙域顶尖强族 |
| 四凶后裔 | 穷奇、梼杌等，多效忠异域或堕落势力 |
| 异域帝族 | 安澜、无殇、赤王（时间之兽）等 |

### 2.3 妖兽对修士的用途

| 用途 | 说明 |
|:---|:---|
| 宝术来源 | 十凶宝术是修士梦寐以求的传承，极大提升战力 |
| 炼器材料 | 骨骼、精血、鳞片是顶级炼制材料 |
| 精血淬体 | 凶兽精血洗礼可强化肉身、突破境界 |
| 坐骑/战宠 | 鲲鹏、白麒麟等是强者标配 |

## 三、丹药/宝药

### 3.1 品阶体系

| 品阶 | 说明 | 代表 |
|:---|:---|:---|
| 普通灵药 | 基础炼丹材料 | 紫丹参、九香虫 |
| 宝药 | 对境界有显著帮助 | 蟠桃果、天仙子 |
| 神药 | 改变体质、延寿、辅助突破 | 真龙神药、麒麟神药 |
| 仙丹 | 仙王级炼丹师炼制 | 混元仙王炉所炼丹药 |

### 3.2 代表性神药

| 神药 | 功效 | 主材 |
|:---|:---|:---|
| 真龙神药 | 激发真龙血脉，强化肉身 | 真龙血肉、龙骨 |
| 麒麟神药 | 激发麒麟血脉，增强防御与祥瑞之气 | 麒麟骨、麒麟血 |
| 鲲鹏神药 | 激发鲲鹏血脉，提升速度与吞噬力 | 鲲鹏精血、鲲鹏骨 |
| 九叶剑草神药 | 强化剑意与肉身 | 九叶剑草本体的剑叶 |
| 打神石神药 | 强化防御与破防 | 打神石碎片或精血 |

### 3.3 石昊使用的代表宝药

- 真龙神药：仙古中服用，强化真龙血脉
- 鲲鹏神药：领悟鲲鹏宝术，提升速度
- 九叶剑草神药：修行草字剑诀
- 仙王级仙丹：乱古纪元后期修复道伤、稳固仙王境界
