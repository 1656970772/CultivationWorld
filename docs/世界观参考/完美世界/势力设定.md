## 二、势力 Faction

### 2.1 势力静态数据 — 类型枚举

| 类型 | 说明 | 典型实例 |
|------|------|----------|
| 国家 | 世俗修炼国家，以皇族为统治核心 | 石国、火国 |
| 宗门 | 以师承为核心的修炼组织 | 补天阁、逐鹿书院 |
| 教派 | 以信仰/教义凝聚的修炼组织 | 补天教、截天教、西方教 |
| 家族/世家 | 以血脉为纽带的修炼家族 | 雨族、武王府、风族 |
| 古皇族 | 拥有古皇血脉的顶级家族 | 石皇族 |
| 长生世家 | 拥有真仙/仙王祖先的世家 | 风族、秦族（不老山） |
| 禁区 | 古代至尊或恐怖存在蛰伏的生命禁区 | 各生命禁区 |
| 帝族（异域） | 异域不朽之王的后裔族群 | 安澜族、俞陀族 |
| 书院 | 培养修炼人才的学院 | 天神书院、仙院、圣院 |
| 杀手/情报组织 | 雇佣性质的特殊势力 | 天国 |
| 黑暗势力 | 与黑暗相关的特殊势力 | 冥土 |
| 祭灵/荒兽势力 | 非人修炼势力 | 太古神山、西陵兽山 |
| 特殊族群 | 被诅咒或特殊修炼体系的族群 | 八臂魂族、葬士 |

### 2.3 势力状态

| 状态维度 | 说明 | 取值范围 |
|----------|------|----------|
| 稳定度 | 内部是否团结 | 1-10（10为完全稳定） |
| 资源压力 | 修炼资源是否充沛 | 1-10（10为极度匮乏） |
| 军事压力 | 外部军事威胁程度 | 1-10（10为围城危机） |
| 黑暗侵蚀 | 势力被黑暗污染程度 | 0%-100% |
| 继任风险 | 核心人物陨落后是否有接班人 | 1-10（10为后继无人） |
| 声望 | 在诸天万界的声誉 | -10 至 10 |

### 2.4 势力需求

| 需求类型 | 说明 | 优先级计算公式 |
|----------|------|----------------|
| 生存 | 势力面临覆灭威胁时的首要需求 | 威胁程度 × 2 |
| 扩张 | 势力壮大后的自然需求 | 实力 ÷ 领地饱和度 |
| 资源 | 修炼资源、天材地宝的获取 | 资源压力 × 势力规模 |
| 安全 | 防御外部威胁 | 军事压力 × 1.5 |
| 复仇 | 对敌对势力的报复 | 仇恨值 × 实力差距倒数 |
| 结盟 | 寻找盟友以增强实力 | (军事压力 + 资源压力) ÷ 现有盟友数 |
| 继任 | 培养下一代核心人物 | 继任风险 × 0.8 |
| 对抗黑暗 | 面对黑暗动乱时的优先需求 | 黑暗侵蚀程度 × 3 |
| 争渡仙路 | 势力内顶级强者的成仙需求 | 至尊数量 × （1 - 成仙成功率） |

### 2.5 势力行为

#### 发展类
- 培养天才弟子：选拔有天赋的年轻修士重点培养
- 扩张领地：派遣修士占领新的灵脉和资源点
- 建设宗门：升级护山大阵、建造修炼设施
- 研发传承：改良现有功法和宝术

#### 征伐类
- 发起宗门战争：对敌对势力发动全面攻击
- 派遣暗杀队：由天国等杀手组织执行定点清除
- 围攻山门：集结多个势力围攻单一目标
- 远征秘境：组织队伍探索高危秘境获取资源

#### 外交类
- 结盟：与实力匹配的势力缔结盟约
- 贸易：进行资源换购（灵药换宝具等）
- 联姻：通过婚姻巩固联盟关系
- 臣服：弱小势力向强大势力进贡以换取保护

#### 对抗类
- 守边关：派遣战力镇守边荒帝关等前线
- 抵抗黑暗：黑暗动乱爆发时组织抵抗
- 清除内奸：排查被黑暗侵蚀的成员

#### 争渡类
- 开启仙路：组织至尊冲击仙路
- 争夺仙种：在仙古遗地等秘境争夺仙种
- 三千州大战：参与跨势力选拔，争夺顶级天骄名额

### 2.6 势力消耗/产出关系

```
势力资源输入：
  领地灵脉产出 → 灵石/灵气
  秘境探索产出 → 宝术/宝具/仙药/仙种
  贸易获得 → 资源互补
  战争掠夺 → 战利品
  天才弟子招募 → 人才资源

势力资源输出：
  弟子修炼消耗 → 灵石/灵药
  战争消耗 → 宝具/丹药/修士伤亡
  维持大阵 → 灵石持续消耗
  朝贡/上交 → 向更高层势力进贡

势力战力输入：
  弟子突破境界 → 增加高阶战力
  获得新宝术/宝具 → 战力提升
  结盟 → 外部战力支援

势力战力输出：
  修士陨落 → 战力永久损失
  宝具损毁 → 战力下降
  叛逃/分裂 → 内部战力流失
```
