## 二、散修地位与生存策略

### 2.1 地位概述

| 区域 | 散修地位 | 特征 |
|------|---------|------|
| 大教核心区 | 低下 | 被视作"可欺之人"，常遭大教弟子欺凌 |
| 边荒 / 混乱之地 | 中等 | 实力强者获得尊重，弱肉强食 |
| 古国疆域 | 低～中 | 受限于古国法规，需缴纳贡赋 |
| 下界八域 | 浮动 | 无大教压制，散修群体庞大 |

### 2.2 五大生存策略

#### 策略一：低调隐忍

- **核心原则**：隐藏真实修为和宝物，不招惹大势力
- **适用人群**：所有散修，尤其是身怀重宝者
- **技巧**：
  - 使用隐匿气息的宝具或骨文
  - 不在公开场合展示高阶宝术
  - 交易时伪装身份
- **风险**：长期隐忍可能导致心魔，突破困难

#### 策略二：抱团取暖

- **核心原则**：组建散修盟或依附小势力换取庇护
- **形式**：
  - 散修盟：松散联盟，信息共享，互助御敌
  - 雇佣兵团：以战力换取报酬和保护
  - 依附小宗门：以劳务换取修炼资源
- **优势**：降低个人风险，扩大信息渠道
- **劣势**：分享收益，可能被利用当炮灰

#### 策略三：专精一道

- **核心原则**：在特定领域做到极致，以技艺立足
- **典型方向**：
  - 炼丹师：炼制丹药换取资源和庇护
  - 阵法师：布阵、破解禁制获得报酬
  - 炼器师：炼制宝具出售
  - 符师：制作符文、骨文卷轴
- **优势**：各大势力都需要，地位较高
- **劣势**：技艺提升需要大量资源和传承

#### 策略四：游走边缘

- **核心原则**：在各大势力夹缝中生存，不轻易站队
- **技巧**：
  - 熟悉各方势力范围和禁忌
  - 利用势力矛盾谋取利益
  - 随时准备更换活动区域
- **优势**：高度灵活，不受约束
- **劣势**：没有稳定后方，容易被各方敌视

#### 策略五：以战养战

- **核心原则**：通过战斗获取资源，在生死边缘突破
- **方式**：猎杀敌对修士、参与势力战争、挑战强者
- **优势**：成长速度极快，实战经验丰富
- **劣势**：死亡率极高，需要过人的天赋和运气

## 三、部落修士与散修关系

### 3.1 关系模式

| 关系 | 特征 | 典型场景 |
|------|------|---------|
| 庇护与依附 | 部落提供庇护和资源，散修提供战力和劳力 | 边荒部落招收散修做护卫 |
| 交易与合作 | 部落需要外界丹药/法器，散修需要部落情报 | 以物易物，信息交换 |
| 冲突与戒备 | 部落担心散修窃取机缘，散修警惕被吞并 | 禁止散修进入部落禁地 |
| 联姻与融合 | 散修通过婚姻融入部落，获得正式身份 | 散修强者入赘/娶部落女子 |

### 3.2 部落对散修的评估标准

| 评估维度 | 权重 | 说明 |
|---------|:---:|------|
| 当前战力 | 40% | 能提供的即时战斗力 |
| 潜力 | 25% | 未来的成长空间 |
| 忠诚度 | 20% | 是否有背叛记录 |
| 技艺特长 | 15% | 炼丹/炼器/阵法等实用技能 |

## 七、坊市交易系统

### 7.1 坊市类型

| 类型 | 规模 | 流通物品 | 安全保障 | 入场条件 |
|------|:---:|---------|:---:|---------|
| 野市 / 黑市 | 小型 | 来路不明物品、禁品 | 低 | 无 |
| 散修集市 | 中型 | 常规修炼物资 | 中 | 无 |
| 宗门坊市 | 大型 | 丹药、宝具、功法 | 高 | 无（需守规矩） |
| 商会拍卖行 | 大型 | 稀有物品、高阶传承 | 极高 | 邀请或担保 |
| 跨域交易会 | 超大型 | 天材地宝、镇教级物品 | 最高 | 高等修士 |

| 大类 | 子类 | 流通性 | 价格波动 |

### 7.3 货币体系

| 货币 | 价值（以灵石为基准） | 使用场景 |
|------|:---:|------|
| 下品灵石 | 1 | 日常小额交易 |
| 中品灵石 | 100 | 中等规模交易 |
| 上品灵石 | 10000 | 高阶物品交易 |
| 极品灵石 | 1000000 | 天材地宝级交易 |
| 天材地宝 | 不定 | 以物易物，顶级交易 |

### 7.4 坊市生存法则

1. **货比三家**：同一物品在不同坊市价格差异大
2. **隐藏身份**：出售贵重物品时伪装气息和外貌
3. **分批出售**：大量物资分批出售，避免引起注意
4. **远离黑市陷阱**：黑市虽有好东西，但被坑概率极高
5. **关注拍卖会**：拍卖会是获取稀有物品的最佳渠道
6. **建立商会关系**：长期交易积累信誉，获得优先权和折扣

## 八、散修设计建议（沙盒项目）

1. **资源随机性**：散修的资源获取高度依赖随机事件和探索
2. **信息不对称**：散修获取秘境/传承信息比宗门弟子晚 1～3 轮
3. **自由度高**：不受宗门任务约束，可自由选择修炼方向
4. **风险收益曲线**：散修的收益波动远大于宗门修士
5. **转正机制**：散修可通过完成特定条件加入宗门，改变身份
6. **隐藏传承**：为散修设计专属的隐藏传承，仅散修身份可触发
