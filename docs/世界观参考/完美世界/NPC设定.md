### 3.1 NPC 静态数据 — 角色类型

| 类型 | 说明 | 典型实例 |

### 3.4 NPC 需求

#### 3.4.1 需求类型

| 需求类型 | 说明 | GOAP 行为链示例 |
|----------|------|----------------|
| 生存 | 面对致命威胁时的首要需求 | 被追杀 → 隐匿行踪 → 寻找庇护所 → 恢复伤势 → 反杀敌人 |
| 修炼 | 提升境界的持续需求 | 灵石不足 → 探索秘境 → 获取资源 → 闭关修炼 → 突破瓶颈 |
| 获取宝术 | 提升战力的关键需求 | 得知宝术所在 → 前往争夺 → 击败守护者 → 获取宝术 → 参悟修炼 |
| 提升境界 | 修炼的根本目标 | 积累修为 → 寻找突破机缘 → 渡劫 → 晋升新境界 |
| 保护族人 | 守护所在势力的需求 | 族人受威胁 → 返回支援 → 击退敌人 → 加强防御 |
| 复仇 | 对仇敌的报复需求 | 确认仇敌位置 → 准备战斗 → 前往复仇 → 杀死仇敌 |
| 对抗黑暗 | 面对黑暗动乱的需求 | 黑暗动乱爆发 → 加入抵抗 → 击杀黑暗生灵 → 保护世界 |
| 争渡仙路 | 成仙的终极追求 | 达到至尊大圆满 → 寻找仙路入口 → 渡仙劫 → 成仙 |
| 寻找机缘 | 提升实力的捷径 | 秘境开启 → 前往探索 → 获取机缘 → 实力提升 |

#### 3.4.2 GOAP 行为链计算

GOAP 通过**目标 → 子目标分解 → 动作序列**的方式驱动 NPC 行为：

```
目标：突破到尊者境
├── 子目标1：积累足够修为
│   ├── 动作：日常修炼
│   ├── 动作：炼化灵石
│   └── 动作：服用丹药
├── 子目标2：获取突破契机
│   ├── 动作：探索秘境
│   ├── 动作：观摩强者突破
│   └── 动作：寻找护道者
├── 子目标3：渡劫
│   ├── 动作：准备渡劫资源（丹药/宝具）
│   ├── 动作：选择渡劫地点
│   └── 动作：承受雷劫
└── 子目标4：巩固境界
    ├── 动作：闭关稳固
    └── 动作：验证战力
```

### 3.5 NPC 行为

#### 修炼类
- 闭关修炼：消耗灵石/丹药，提升修为
- 参悟宝术：消耗时间，获得/强化宝术
- 渡劫：承受天劫，成功则突破境界
- 观摩悟道：观察强者突破或天地异象获得感悟

#### 争渡类
- 争夺仙种：在仙古遗地等秘境抢夺仙种
- 冲击仙路：至尊大圆满后发起成仙尝试
- 闯仙关：突破仙域与凡界的关口

#### 夺宝类
- 探索秘境：进入百断山、仙古遗地等获取机缘
- 抢夺宝具：与其他修士争夺顶级宝具
- 拍卖竞购：在拍卖会中竞买稀有资源

#### 征战类
- 参与战争：响应势力号召参与宗门战争/位面战争
- 单挑对决：与其他天骄一对一决斗
- 围攻强者：集结多人围杀高阶敌人

#### 守护类
- 守护族人/宗门：在外敌入侵时保卫家园
- 镇守边关：在边荒帝关等前线长期驻守

#### 叛逃/背叛
- 叛变投敌：被黑暗侵蚀后背叛原势力
- 临阵脱逃：在必败之战中选择逃跑

#### 挑战类
- 挑战天骄榜：争夺战力排名
- 挑战禁地：闯入生命禁区寻求机缘

#### 献祭类
- 燃烧精血：以自身精血为代价释放禁术
- 献祭生命：以生命为代价守护重要目标
- 自爆杀敌：在绝境中引爆自身

### 3.6 NPC 消耗/产出关系

```
NPC 成长消耗：
  灵石/灵药 → 日常修炼 → 修为增长
  宝术传承 → 参悟学习 → 获得新技能
  战斗消耗 → 丹药/宝具耐久 → 击败敌人/获取战利品
  渡劫消耗 → 大量资源 → 突破境界

NPC 产出：
  击杀敌人 → 战利品（宝具/资源/功法）
  探索秘境 → 机缘（仙种/圣药/宝术）
  守护势力 → 势力稳定度提升
  培养弟子 → 势力未来战力
  陨落 → 传承遗留（至尊骨/骨书/宝具）
```
