### 4.6 物品行为

#### 生产
- 种植灵药：在灵田中培养药材
- 炼器：使用仙金/神铁铸造宝具
- 炼丹：使用药材炼制丹药
- 刻阵：布置和制作阵法物品
- 铭文骨书：将宝术铭刻于骨片上传承

#### 采集
- 采集灵药：在野生灵脉中采集药材
- 开采灵石：在灵石矿脉中开采灵石
- 提炼仙金：从矿石中提炼仙金

#### 消耗
- 修炼消耗：吸收灵石/丹药中的灵气
- 战斗消耗：宝具耐久度减少、丹药使用
- 渡劫消耗：渡劫时大量资源被消耗
- 献祭消耗：以物品为代价释放禁术
- 维持阵法：阵法持续消耗灵石

#### 交易
- 拍卖：在拍卖会中竞价交易
- 以物易物：资源/宝具之间的直接交换
- 购买：用灵石购买物品

#### 掠夺
- 击杀掉落：击杀敌人后获取其物品
- 秘境抢夺：在秘境中争夺无主之物
- 战争缴获：战争胜利后获取敌方势力资源

#### 传承
- 师承传授：师父将宝术/功法传授给弟子
- 骨书领悟：修士通过骨书学习宝术
- 血脉传承：后裔觉醒祖先的神通

#### 装备
- 炼化宝具：将宝具炼化为本命武器
- 绑定物品：将物品与自身绑定防止被盗
- 替换装备：更换更高品质的宝具

#### 炼制
- 合成丹药：多种药材合成为丹药
- 强化宝具：为宝具附加新能力
- 修复宝具：消耗材料修复受损宝具

### 4.7 物品消耗/产出关系

```
资源获取：
  野生灵脉 → 开采 → 灵石
  森林/秘境 → 采集 → 灵药/圣药
  矿脉 → 提炼 → 仙金/神铁
  猎杀荒兽 → 材料/兽骨

物品制造：
  灵药 + 丹方 → 炼丹 → 丹药
  仙金 + 炼器术 → 铸造 → 宝具
  灵石 + 阵图 → 刻阵 → 阵法
  宝术 + 骨片 → 铭刻 → 骨书

物品流转：
  拥有者 → 交易/传承/掉落 → 新拥有者

物品消耗：
  灵石 → 修炼/维持阵法 → 能量耗尽
  丹药 → 服用 → 药效发挥
  宝具 → 战斗 → 耐久降低 → 修复或报废
  骨书 → 领悟 → 知识传承（骨书本体留存）

特殊循环：
  至尊骨 → 被挖走 → 移植他人 → 产生新宝术
  仙种 → 融合修士 → 随修士成长 → 成仙后化为仙道根基
  黑暗物质 → 污染物品 → 物品黑暗化 → 成为黑暗阵营资源
```
