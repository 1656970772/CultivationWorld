# 完美世界 — 宗门任务体系

> 适用沙盒项目：修仙动态世界 · 宗门模块
> 基于辰东《完美世界》原著设定，整合为可复用的任务模板体系。

---

## 一、任务类型分类（9大类别）

| 序号 | 任务类型 | 说明 | 典型发布者 | 风险等级 |

| 4 | 护送任务 | 护送重要人物（圣女、商队、传承者）穿越险地 | 宗门 / 商会委托 | 中 |

## 三、任务接取条件

### 3.1 条件维度

| 条件类型 | 说明 | 示例 |
|---------|------|------|
| 境界门槛 | 修士必须达到指定境界才能接取 | D级：无限制 / C级：筑基 / B级：金丹 / A级：元婴 |
| 声望等级 | 在势力中的声望必须达到指定级别 | 中立 → 友善 → 尊敬 → 崇敬 → 崇拜 |
| 贡献点门槛 | 需要积累足够的贡献点证明资历 | D级：0 / C级：500 / B级：5000 / A级：20000+ |
| 宗门地位 | 仅特定身份弟子可接取 | 外门 → 内门 → 核心 → 亲传 → 长老 |
| 前置任务 | 必须先完成前置任务链 | 探索秘境前需完成情报收集任务 |
| 战力证明 | 通过战力测试或持有特定信物 | 击败同级傀儡 / 持长老手令 |

### 3.2 接取条件矩阵

| 难度 | 境界最低要求 | 声望最低 | 贡献最低 | 地位要求 | 前置任务 |
|:---:|:-----------:|:------:|:------:|:-------:|:-------:|
| D | 搬血 | 中立 | 0 | 外门 | 无 |
| C | 洞天 | 友善 | 500 | 内门 | 可选 |
| B | 化灵 | 尊敬 | 5000 | 内门+ | 1～2个 |
| A | 铭纹 | 崇敬 | 20000 | 核心+ | 任务链 |

## 四、任务奖励机制

### 4.1 奖励类别

| 奖励类型 | 具体内容 | 获取难度 | 稀有度 |
|---------|---------|:---:|:---:|
| 宝术 / 功法 | 宗门传承宝术、秘技、敌人掉落功法玉简 | 中～高 | ★★～★★★★★ |
| 宝具 / 装备 | 宗门炼制法器、护甲、秘境掉落宝具碎片 | 低～高 | ★～★★★★★ |
| 丹药 / 灵药 | 恢复类、疗伤类、辅助修炼类、突破境界类 | 低～中 | ★～★★★ |
| 骨文 / 符文 | 记载古法的骨书、炼器/布阵符文石 | 高 | ★★★～★★★★ |
| 修炼秘境资格 | 进入宗门秘境的权限，修炼速度倍增 | 中～高 | ★★★～★★★★★ |
| 长老指点 | 高阶修士一对一指点，甚至传授完整宝术 | 极高 | ★★★★★ |
| 贡献点 | 通用宗门货币 | 所有任务 | — |
| 声望值 | 提升在宗门和势力中的声望等级 | 所有任务 | — |

### 4.2 奖励分配规则

- **单人任务**：奖励全归个人
- **组队任务**：按贡献比例分配，队长额外获得 10%
- **宗门战争**：按战功榜分配，首功额外奖励
- **秘境探索**：谁发现归谁，宗门抽取 20% 稀有物品

### 5.2 使用途径

| 用途 | 消耗 | 说明 |
|------|:---:|------|

## 七、任务系统设计建议

1. **动态刷新**：日常任务每日刷新，讨伐/秘境任务每周或不定期刷新
2. **任务链**：同类型任务可串联为任务链，完成整条链获得额外奖励
3. **隐藏任务**：满足特殊条件（特定时间/天气/携带某物品）触发隐藏任务
4. **随机事件**：任务途中可能触发随机事件（天降宝物/遭遇仇敌/天气突变）
5. **声望联动**：任务完成情况影响势力声望，进而解锁更多任务类型
