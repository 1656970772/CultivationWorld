## 设定摘要

本文件把《神墓》核心 NPC 的可观察行为转化为项目可用的性格标签。原作事实只记录人物在具体事件中的行为；“谨慎求生、隐忍反击、贪婪夺宝、顽皮惹事、逆天执念”等标签均为我的判断，用于后续 AI 权重、事件触发器和关系变化设计。

## 性格标签体系（我的判断）

| 标签 | 行为含义 | 游戏 AI 倾向 |
|---|---|---|
| 谨慎求生 | 避免无把握冲突，重视隐藏实力和退路 | 低风险探索、优先保命、延迟复仇 |
| 隐忍反击 | 被追杀时积累线索，寻找反击窗口 | 追踪敌人、伏击、反袭杀 |
| 争强好胜 | 在同辈竞争中主动挑战，重视名声和胜负 | 高竞争、易触发切磋/挑衅事件 |
| 权谋算计 | 通过联盟、交易、诱导、借势达成目标 | 高策略、低信任、偏好借刀杀人 |
| 重情守诺 | 对亲友、师门或承诺保持长期责任 | 高忠诚、低背叛、愿意救援 |
| 贪婪夺宝 | 对稀缺资源高度敏感，愿意冒险夺取 | 高资源觊觎、易触发怀璧其罪 |
| 偏执复仇 | 将仇恨置于长期收益之上 | 追杀链、跨区域寻仇、冲突升级 |
| 顽皮惹事 | 好奇、嘴硬、爱闹，常引发非计划冲突 | 高随机事件、低纪律、伙伴喜剧风险 |
| 宗门责任 | 以宗门利益、传承和秩序为优先 | 护宗、招徒、惩戒、资源调度 |
| 逆天执念 | 以对抗天道或终局使命压过常规利益 | 高牺牲、高号召、世界事件参与 |

## 可转化为数据或事件的条目

| 条目 | 类型 | 触发条件 | 可能影响 |
|---|---|---|---|
| `trait_cautious_survivor` | 性格标签 | NPC 被追杀或处于弱势 | 降低正面硬拼，增加撤退/伏击 |
| `trait_vindictive_planner` | 性格标签 | 仇怨值高且有目标线索 | 触发追杀、揭露、借势复仇 |
| `trait_treasure_greedy_dragon` | 性格标签 | 稀有法宝出现 | 提高夺宝、讨价还价、擅自行动概率 |
| `event_companion_causes_trouble` | 伙伴事件 | 顽皮惹事标签 + 圣地/玄界环境 | 降低当地声望，开启补救任务 |
| `event_hidden_counterattack` | 冲突事件 | 连续追杀 + 线索充分 | 由逃亡转为反袭杀 |

## 待确认问题

- 性格标签仍是“我的判断”，后续应为更多人物补充至少 2 条代表事件后再定权重。
- 小公主、纳兰若水、澹台璇、南宫仙儿、东方凤凰等人物未在本轮事件卡中充分展开。
- 个别证据片段来自长章节中的局部行为，不能直接推断人物全书稳定性格。
