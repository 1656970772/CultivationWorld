## 设定摘要

《神墓》物品可先按“神器/神兵、圣地法宝、镇派兵器、奇书功法、灵物资源、禁忌圣宝”分类。物品价值往往来自传承、势力归属、太古来源或复活能力，而不是单纯数值。

## 游戏化抽象（我的判断）

| 可转化机制 | 我的判断 | 依据 |
|---|---|---|
| 物品来源决定冲突 | 物品归属皇室、圣地、邪道或太古阵营时，应附带追索和声望后果。 | SM-ITEM-001 至 SM-ITEM-004。 |
| 奇书有代价 | 功法书应配置修炼代价、心性风险和剧情锁，而不是普通技能书。 | SM-ITEM-006、SM-ITEM-007。 |
| 禁忌圣宝唯一性 | 生命源泉这类物品应作为世界级资源，不宜大量掉落。 | SM-ITEM-008。 |
| 神兵可承载关系 | 后羿弓这类物品应绑定人物记忆和传承事件。 | SM-ITEM-001。 |
