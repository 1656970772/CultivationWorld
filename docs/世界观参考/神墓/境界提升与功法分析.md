## 设定摘要

《神墓》的境界体系呈“前期阶位 -> 中期六阶/破碎虚空 -> 天界神王/主神 -> 天阶 -> 逆天级”的叙事升级。原文并非每次都用固定数值等级解释强弱，而是通过进入天界、神王感应、天阶高手、世界种子、逆天级巅峰等事件标记阶段跃迁。
