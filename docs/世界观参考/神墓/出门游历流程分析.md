## 设定摘要

《神墓》的游历流程不是单纯跑图，而是“身份风险 + 地点门槛 + 伙伴风险 + 跨界通道 + 世界事件”的组合。辰南常因追查秘辛、逃避追杀、获取资源、救人或受托进入高危地点；随着世界阶段提升，游历从楚国和罪恶之城扩展到西大陆、玄界、天界和终局多界。
