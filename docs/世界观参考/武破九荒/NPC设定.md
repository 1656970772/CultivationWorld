## NPC

### NPC静态数据

#### 角色类型枚举
1. **武者**：后天境至帝武境的修炼者
2. **圣阶**：化灵境至圣阶巅峰的强者
3. **神阶**：神、真神、天神、道神、涅神境
4. **终极境**：超越涅神境的至高境界
5. **混沌生灵**：诞生于混沌的特殊存在
6. **妖兽**：拥有灵智的兽类修炼者
7. **凡人**：未修炼或低阶修炼者
8. **特殊存在**：器灵、阵灵、残魂等

#### 基础属性维度
- **境界**：当前修炼境界
- **年龄**：实际年龄
- **寿命**：当前境界下的剩余寿命
- **资质**：修炼天赋 (1-10)
- **气运**：气运值 (1-100)
- **血脉**：血脉纯度 (0-100%，非古族为0)
- **阵营**：所属势力或阵营
- **关系网**：亲友、师徒、敌对关系

### NPC实体

#### 萧叶（主角）
- **角色类型**：武者→圣阶→神阶→终极境
- **境界轨迹**：炼体境 → 后天境 → 先天境 → 玄武境 → 虚武境 → 王武境 → 皇武境 → 帝武境 → 半帝 → 大帝 → 圣阶 → 神阶 → 终极境
- **出身**：黑龙国萧家村
- **特质**：理性型战神，思维先行，意志坚定
- **核心传承**：
  - 时间之塔（核心外挂）
  - 霸体传承（炼体极致）
  - 各大帝经（融合创新）
  - 古族血脉（后期获得）
- **专属能力**：
  - 时间加速修炼
  - 霸体无双（肉身极致）
  - 多系法则融合
  - 因果神通（后期）
- **关系网**：
  - 道侣：冰雅
  - 灵兽：小白
  - 弟弟：萧凡
  - 挚友：南宫星宇
  - 师尊：待补全
  - 势力：萧盟创始人
- **成长特点**：稳扎稳打，越级战斗，多次绝境突破

#### 冰雅
- **角色类型**：圣阶
- **身份**：冰雪宫圣女
- **境界**：圣阶（具体待补全）
- **特质**：独立强大，外冷内热，为萧叶付出巨大
- **核心传承**：冰雪系功法、圣女本源
- **专属能力**：冰雪法则、寒冰领域、圣女秘法
- **与萧叶关系**：道侣，并肩作战
- **成长轨迹**：与萧叶共同成长，后期成为重要助力

#### 小白
- **角色类型**：混沌生灵→灵兽
- **身份**：混沌本源异化体，萧叶的灵兽伙伴
- **特质**：随萧叶成长而进化，忠诚可靠
- **核心能力**：吞噬进化、混沌神通、空间能力
- **成长方式**：吞噬天材地宝、神灵残躯
- **最终形态**：待补全（可能进化为混沌神兽）

#### 南宫星宇
- **角色类型**：圣阶
- **身份**：铁血帝域弃子，萧叶的"镜像对照组"
- **特质**：因功法缺陷停滞，衬托主角的奇遇，重情重义
- **核心传承**：星光大道、帝域功法（有缺陷）
- **专属能力**：星辰法则、星辉剑法
- **成长限制**：功法缺陷导致瓶颈，需特殊机缘突破
- **与萧叶关系**：挚友，互相扶持

#### 萧凡
- **角色类型**：武者→圣阶
- **身份**：萧叶之弟，代表"凡人视角"
- **特质**：依靠自身努力成长，坚韧不拔
- **核心传承**：宗门基础功法、自创武技
- **成长方式**：宗门积分积累、傀儡塔历练
- **代表意义**：普通武者的成长路径，与萧叶形成对比

#### 其他重要NPC（待补全详细属性）
- **重阳门师尊**：萧叶的启蒙导师
- **太一圣宫宫主**：人族领袖之一
- **四大帝域帝君**：各帝域统治者
- **古族族长**：古族领袖
- **混沌阵营首领**：最终反派
- **各大天骄**：与萧叶竞争的同代强者

### NPC状态

#### 状态维度
1. **存活状态**：存活、重伤、濒死、死亡、转世
2. **年龄**：实际年龄/生理年龄
3. **伤势**：健康、轻伤、重伤、道伤、本源伤
4. **忠诚度** (0-100)：对势力/领袖的忠诚
5. **忙碌度** (0-100)：当前事务繁忙程度
6. **心境**：平和、愤怒、悲伤、顿悟、心魔等
7. **修炼进度**：当前境界进度百分比
8. **时间之塔进度**：塔内修炼进度（如有权限）
9. **因果纠缠度**：因果线密集程度
10. **气运值** (1-100)：当前气运强度

#### 状态影响
- **伤势**：影响战斗力、修炼速度
- **心境**：影响突破成功率、可能产生心魔
- **忙碌度**：影响可分配时间
- **因果纠缠度**：影响命运走向，过高可能引劫

#### 状态恢复机制
- **伤势恢复**：丹药治疗、功法疗伤、时间自愈
- **心境调整**：静修、历练、顿悟、亲友开导
- **忙碌度降低**：完成事务、分配任务
- **因果清理**：了结因果、斩断因果线

### NPC需求

#### 需求层次（马斯洛武道版）
1. **生存需求**：保命、疗伤、避免死亡
2. **安全需求**：避免威胁、保障亲友安全
3. **归属需求**：势力归属、亲情友情爱情
4. **尊重需求**：声望、地位、被认可
5. **自我实现**：突破境界、达成武道巅峰
6. **超我实现**：悟道、创道、影响世界

#### GOAP（目标导向行动规划）行为链

**目标：突破到皇武境**

```
当前状态：王武境巅峰，资源不足，有敌对势力威胁

行动链：
1. 收集突破资源（子目标）
   - 探索秘境获取天材地宝
   - 完成任务获取宗门贡献
   - 交易获取所需资源
   
2. 确保安全环境（子目标）
   - 解决或避开敌对势力威胁
   - 寻找安全闭关地点
   - 布置防御阵法
   
3. 调整状态（子目标）
   - 疗愈暗伤
   - 调整心境至最佳
   - 参悟皇武境法则
   
4. 执行突破
   - 服用突破丹药
   - 运转功法冲击瓶颈
   - 渡天劫（如有）
   
5. 巩固境界
   - 稳定新境界修为
   - 适应新增力量
   - 完善皇武境神通
```

#### 需求优先级计算
```
优先级 = 基础需求值 × 紧急系数 × 个人性格系数

基础需求值：
- 生存需求：100
- 安全需求：80
- 归属需求：60
- 尊重需求：40
- 自我实现：30
- 超我实现：20

紧急系数 (1.0-5.0)：
- 立即威胁：5.0
- 短期威胁：4.0
- 中期需求：3.0
- 长期目标：2.0
- 理想追求：1.0

个人性格系数 (0.5-2.0)：
- 谨慎型：1.5（生存安全权重高）
- 野心型：0.8（自我实现权重高）
- 重情型：1.2（归属需求权重高）
- 求道型：0.6（超我实现权重高）
```

### NPC行为

#### 行为类型
1. **修炼**：日常修炼、闭关突破
2. **征战**：参与战斗、征伐敌对
3. **探索**：探索秘境、遗迹、未知区域
4. **社交**：与亲友交流、势力事务
5. **交易**：买卖资源、以物易物
6. **生产**：炼丹、炼器、制符、布阵
7. **传授**：教导弟子、传授经验
8. **研究**：研究功法、推演神通
9. **游历**：游历天下、增长见闻
10. **潜伏**：潜伏敌对势力、收集情报

#### 行为决策模型
```
行为选择 = argmax( 预期收益 - 预期成本 - 风险损失 )

预期收益 = Σ(收益项 × 收益权重)
预期成本 = Σ(成本项 × 成本系数)
风险损失 = 失败概率 × 失败损失 × 风险厌恶系数

收益项：修为提升、资源获取、声望增加、关系改善等
成本项：时间消耗、资源消耗、机会成本等
```

#### 行为时间分配
- **修炼**：30-70%时间（核心行为）
- **征战/探索**：10-40%时间（资源获取）
- **社交/交易**：5-20%时间（关系维护）
- **生产/研究**：5-30%时间（技能提升）
- **休息/游历**：5-15%时间（状态恢复）
