# 《武破九荒》—— 夺舍设定参考文档

**小说背景**：寒香寂寞著，典型的强者为尊、魂穿夺舍流玄幻小说，设定较为传统但逻辑自洽。

### 一、夺舍的基本规则
1. **核心定义**：指灵魂占据他人肉体，通常伴随着原主人的死亡或奴役。
2. **前提条件**：
   * **灵魂强度**：夺舍者的灵魂力量必须远超被夺舍者的灵魂防御。
   * **肉体契合度**：通常要求性别、年龄（尽量年轻）、血脉属性相近，否则会产生极强的排斥反应。
   * **无主状态**：最佳目标是刚死、濒死或灵魂受重创的躯体。
3. **实施方式**：
   * 强行吞噬对方魂魄。
   * 利用特殊法宝（如魂旗、寄生种子）保护灵魂进行转移。

### 二、夺舍的代价与风险
1. **修为尽失**：夺舍成功后，新肉体通常无法继承前世的修为，需要从头修炼（除非有特殊的传承秘法）。
2. **根基受损**：夺舍来的身体，其经脉、丹田可能已有暗伤，或者与原魂适配度低，导致修炼速度变慢。
3. **心魔与因果**：夺舍属于逆天行事，容易引来心魔劫。被夺舍者若怨气不散，会形成"怨魂反噬"。
4. **天地规则压制**：高阶修士夺舍低阶修士，若肉身过于弱小，会受限于肉体容纳上限。

### 三、典型案例分析
1. **主角（通常为魂穿）**：
   * **表现**：主角因意外灵魂穿越，附着在废柴或死者身上。
   * **特殊性**：往往带有前世记忆和功法，属于"优势夺舍"。
2. **反派夺舍**：
   * **表现**：老怪物濒死时，强行夺取天才子弟的身体。
   * **冲突点**：通常会引发该势力内部的追杀，因为"身份"虽然对了，但"灵魂"已经变了。

### 四、夺舍相关的特殊设定
1. **夺舍与"借尸还魂"**：在小说中常混用，但严格来说，借尸还魂可能指的是灵魂融入无意识的尸体，而夺舍特指有意识的抢夺。
2. **禁忌与追杀**：正道宗门通常严禁夺舍，一旦发现"魂不符体"，会视为邪修就地格杀。

### 五、游戏设计启示
1. **肉身天赋随机**：玩家夺舍不同NPC时，应随机生成不同的"肉身天赋"和"隐藏血脉"。
2. **原主残魂**：可以设计"原主残魂未清"的支线任务，玩家需要帮助身体原主完成遗愿，否则会被反向吞噬。
3. **夺舍冷却**：设定夺舍有极长的冷却时间或极高的资源消耗，防止玩家频繁换号。
