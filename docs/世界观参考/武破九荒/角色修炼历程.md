## 设定摘要

可核时间线为：重阳门内门崛起 -> 死亡沼泽/女帝相关历练 -> 散修求生 -> 真灵大陆/星陨王国/天才营 -> 中州/四大帝域/太一圣宫 -> 三千大界 -> 太初古经与圣阶道路。

## 原作事实

- 萧叶在重阳门阶段通过傀儡塔、内门挑战、亲传线等建立早期成长。
- 死亡沼泽与女帝线推动四鼎天功、肉身突破和更高目标。
- 逃离重阳门后，萧叶进入散修身份，依靠疗伤、隐匿、商会交易等继续生存。
- 抵达真灵大陆后，资源密度与对手层级升级，天才营/贡献点/任务成为重要成长机制。
- 中州阶段接入四大帝域、太一圣宫、冰雅、南宫星宇等主线。
- 三千大界后，太初古经和灵阶/圣阶体系成为新成长轴。

## 游戏化抽象（我的判断）

- 我的判断：角色历程可拆为 `phase_id`，每个阶段绑定地图、势力、主资源、主敌人和境界上限。
- 我的判断：转阶段事件应强制刷新任务池和经济池，如“进入真灵大陆”“进入三千大界”。
- 我的判断：不要用固定 18 事件线模拟萧叶，应按证据锚点逐步扩展。
