## 势力 Faction

### 势力静态数据

#### 类型枚举
1. **宗门**：武道传承组织，如重阳门、太一圣宫
2. **帝国**：世俗王朝，如黑龙国及各大皇朝
3. **古族**：古老血脉传承种族，底蕴深厚
4. **帝域**：帝境强者统御的顶级势力，如四大帝域
5. **商会**：商业组织，如青冥商会
6. **混沌阵营**：混沌世界中的势力，意图毁灭或重塑秩序
7. **联盟**：多势力联合体，如萧盟
8. **交易平台**：如万器阁

#### 势力属性维度
- **规模**：小型、中型、大型、超大型
- **影响力**：区域、大陆、大界、跨位面
- **传承年限**：新生、百年、千年、万年、远古
- **核心传承**：功法、血脉、技艺、宝物

### 势力实体

#### 太一圣宫
- **类型**：宗门
- **定位**：中州顶级势力，人族重要圣地
- **规模**：超大型
- **影响力**：大陆级
- **传承年限**：万年以上
- **核心传承**：太一帝经、圣宫秘法
- **资源产出**：高阶功法、帝经碎片、圣器碎片
- **特殊属性**：
  - 圣子/圣女培养体系
  - 护宗大阵：太一星辰阵
  - 圣地秘境：太一秘境（每百年开启一次）

#### 四大帝域
- **类型**：帝域
- **组成**：铁血帝域、飘雪帝域、雷霆帝域、幽冥帝域（待补全具体名称）
- **定位**：人族最强势力集团，各由一位帝境强者统御
- **规模**：每个均为超大型
- **影响力**：大界级
- **核心传承**：各帝域专属帝经、帝兵
- **资源产出**：顶级资源、帝兵碎片、帝经传承
- **相互关系**：既合作又竞争，维持微妙平衡

#### 古族
- **类型**：古族
- **代表**：具体古族名称待补全
- **特征**：拥有古老血脉传承，血脉浓度决定潜力
- **规模**：中型至大型
- **影响力**：大陆至大界级
- **核心传承**：血脉秘法、古族圣器、祖地传承
- **资源产出**：血脉秘药、古族圣器、祖地宝物
- **与人族关系**：既有合作也有冲突

#### 萧盟
- **类型**：联盟
- **创立者**：萧叶
- **发展**：从家族联盟发展为跨域联盟
- **规模**：从小型发展至大型
- **核心成员**：萧家族人、盟友势力
- **组织架构**：
  - 盟主：萧叶
  - 长老会
  - 战堂、丹堂、器堂、情报堂
- **资源产出**：联盟积分、战功、同盟专属资源
- **特殊机制**：贡献度系统，按贡献分配资源

#### 青冥商会
- **类型**：商会
- **定位**：跨域商业组织
- **规模**：大型
- **业务范围**：丹药、法宝、功法、情报交易
- **核心资源**：商业网络、拍卖行、客卿体系
- **资源产出**：贡献点、各类商品、情报服务
- **特殊服务**：客卿制度，强者可为客卿获取资源

#### 万器阁
- **类型**：交易平台
- **定位**：大型交易平台
- **功能**：功法、武技、宝物兑换
- **货币体系**：积分系统
- **资源产出**：积分、各类商品
- **兑换规则**：按品阶和稀有度设定积分价格

#### 重阳门
- **类型**：宗门
- **定位**：萧叶的宗门起点，位于黑龙国
- **规模**：小型（后衰落）
- **核心传承**：基础重阳功法
- **资源产出**：基础功法、武技、宗门贡献点
- **现状**：已衰落，但保留部分传承

#### 混沌阵营
- **类型**：混沌阵营
- **定位**：后期出现的敌对阵营
- **目标**：意图毁灭或重塑现有秩序
- **规模**：待补全
- **核心力量**：混沌生灵、堕入混沌的强者
- **资源产出**：禁忌传承、混沌至宝
- **威胁等级**：灭世级

### 势力状态

#### 状态维度
1. **稳定度** (0-100)：内部团结程度，受内斗、叛变等因素影响
2. **资源压力** (0-100)：资源需求与供给的平衡程度
3. **军事压力** (0-100)：外部军事威胁程度
4. **血脉浓度** (0-100)：古族专属，血脉纯度
5. **继任风险** (0-100)：领袖更替风险
6. **声望** (0-100)：在武道世界的声誉
7. **发展度** (0-100)：综合发展水平
8. **危机值** (0-100)：面临危机的严重程度

### 势力需求

#### 需求类型
1. **生存需求**：维持势力存在的基本需求
2. **扩张需求**：扩大势力范围和影响力
3. **资源需求**：获取修炼资源、天材地宝
4. **对抗需求**：对抗敌对势力
5. **复兴需求**：衰落势力的复兴需求
6. **传承需求**：确保传承不断绝
7. **稳定需求**：维持内部稳定
8. **声望需求**：提升势力声望

#### 优先级计算逻辑
```
优先级 = 基础权重 × 紧急系数 × 战略价值

基础权重：
- 生存需求：1.0
- 扩张需求：0.8
- 资源需求：0.7
- 对抗需求：0.9
- 复兴需求：0.6
- 传承需求：0.5
- 稳定需求：0.4
- 声望需求：0.3

紧急系数 (1.0-3.0)：
- 危机临近：2.5-3.0
- 短期威胁：2.0-2.5
- 中期需求：1.5-2.0
- 长期规划：1.0-1.5

战略价值 (1.0-2.0)：
- 核心战略：2.0
- 重要战略：1.5
- 一般战略：1.2
- 次要战略：1.0
```

#### 需求满足机制
- **资源型需求**：通过探索、交易、掠夺满足
- **安全型需求**：通过结盟、增强防御、先发制人满足
- **发展型需求**：通过培养弟子、研究创新、扩张满足
- **传承型需求**：通过保护传承者、完善传承体系满足

### 势力行为

#### 行为类型
1. **争战**：主动攻击敌对势力
2. **防御**：抵御外来攻击
3. **结盟**：与其他势力结盟
4. **交易**：资源、情报交易
5. **培养弟子**：培养新生代强者
6. **探索秘境**：探索远古战场、秘境
7. **研究创新**：研究新功法、丹药、阵法
8. **情报收集**：收集其他势力情报
9. **内部整顿**：整顿内部秩序
10. **祭祀仪式**：古族、宗门特有仪式

#### 行为决策逻辑
```
决策得分 = 预期收益 - 预期成本 - 风险值 × 风险厌恶系数

预期收益：资源获取、势力增强、威胁消除等量化收益
预期成本：资源消耗、人员伤亡、时间成本等
风险值：行动失败概率 × 失败后果严重度
风险厌恶系数：势力性格参数 (0.5-2.0)
```

#### 行为链示例：探索秘境
1. 情报收集 → 2. 队伍组建 → 3. 资源准备 → 4. 秘境探索 → 5. 收获分配 → 6. 成果消化
