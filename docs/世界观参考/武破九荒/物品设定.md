## 物品 Item

### 物品静态数据

#### 物品基础属性
- **名称**：物品名称
- **类型**：资源、功法、丹药、法宝、阵法等
- **品阶**：凡阶、灵阶、王阶、皇阶、帝阶、圣阶、神阶、混沌阶
- **稀有度**：普通、稀有、罕见、传说、神话
- **重量**：物理重量
- **体积**：占用空间
- **耐久**：使用次数/耐久度
- **绑定状态**：已绑定、未绑定
- **来源**：制作、掉落、任务、交易等

#### 物品品质等级
1. **凡品**：凡人使用，无特殊效果
2. **灵品**：蕴含灵气，对修炼有帮助
3. **王品**：王武境适用，有特殊效果
4. **皇品**：皇武境适用，效果显著
5. **帝品**：帝武境适用，蕴含帝威
6. **圣品**：圣阶适用，蕴含圣道法则
7. **神品**：神阶适用，蕴含神性
8. **混沌品**：混沌级，蕴含混沌法则

### 物品需求

#### 需求产生场景
1. **修炼需求**：需要功法、丹药、灵石等辅助修炼
2. **战斗需求**：需要法宝、阵法、符箓等增强战力
3. **生产需求**：需要材料、工具等进行生产
4. **交易需求**：需要货币、等价物进行交易
5. **任务需求**：需要特定物品完成任务
6. **收藏需求**：收集稀有物品
7. **传承需求**：需要传承物品延续道统
8. **仪式需求**：需要特定物品进行仪式

#### 需求强度计算
```
需求强度 = 基础需求值 × 紧急系数 × 替代难度

基础需求值：
- 生存相关：100（疗伤丹药、保命法宝）
- 突破相关：80（破境丹药、辅助功法）
- 战力相关：60（攻击法宝、战斗丹药）
- 生产相关：40（炼器材料、丹方）
- 其他需求：20

紧急系数 (1.0-5.0)：
- 立即需要：5.0（战斗中）
- 短期需要：4.0（准备突破）
- 中期需要：3.0（计划中）
- 长期需要：2.0（未来规划）
- 可有可无：1.0

替代难度 (1.0-3.0)：
- 无可替代：3.0（唯一物品）
- 难以替代：2.5（稀有物品）
- 有替代品：2.0（有类似功能物品）
- 易替代：1.5（多种选择）
- 极易替代：1.0（常见物品）
```

### 物品行为

#### 行为类型（物品相关行为）
1. **采集**：采集天材地宝、矿石等
2. **炼制**：炼丹、炼器、制符、布阵
3. **消耗**：使用丹药、消耗灵石等
4. **交易**：买卖、以物易物
5. **掠夺**：抢夺他人物品
6. **装备**：装备法宝、穿戴护甲
7. **研究**：研究功法、参悟法宝
8. **修复**：修复损坏物品
9. **强化**：强化物品效果
10. **传承**：传承功法、法宝给后人

#### 行为效果
- **采集**：获取原材料，成功率受技能、工具影响
- **炼制**：将原材料转化为成品，成功率受技能、配方影响
- **消耗**：直接使用物品获得效果
- **交易**：交换物品，价值受市场、谈判影响
- **掠夺**：高风险高收益，可能引发仇恨
- **装备**：获得物品加成，可能有装备要求
- **研究**：可能领悟新知识、改进配方
- **修复**：恢复物品完好度，可能降低最大耐久
- **强化**：提升物品效果，有失败风险
- **传承**：确保物品传承，可能要求特定条件

#### 行为链示例：炼制皇品丹药
1. 收集丹方和材料 → 2. 准备丹炉和地火 → 3. 处理材料 → 4. 按顺序投料 → 5. 控制火候 → 6. 凝丹 → 7. 收丹 → 8. 品质鉴定
