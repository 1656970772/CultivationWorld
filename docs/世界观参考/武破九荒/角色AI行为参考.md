# 武破九荒 — 角色AI行为参考

## AI架构映射说明

《武破九荒》的叶秋是"标准打怪升级模板"的典型代表，其AI行为模式对构建修仙沙盒中的"玩家型NPC"或"成长线敌人"具有直接参考价值：

- **BT层**：叶秋的应急体系是两个分支的循环——"血脉/神器爆发越级"和"嗑药突破返场复仇"。这种"爆发-撤退-突破-返场"的循环是修仙战斗AI的经典模式。
- **Utility层**：叶秋的目标权重随"境界"线性迁移——宗门 → 大陆 → 位面 → 武道极致，是最清晰的"地图扩展驱动"型权重曲线。
- **GOAP层**：每一步都是"奇遇 → 突破 → 展示 → 危机 → 再突破"的经典循环。

## 核心角色分析

### 1. 叶秋

#### 性格标签
坚韧 / 机遇主义者 / 杀伐果断 / 专注修炼 / 对敌人冷酷

#### BT层应急反应
| 触发条件 | 抢占行为 | 优先级 |
|---|---|---|
| 遭遇强敌(实力相近) | 开启隐藏血脉/神器 → 越级爆发 | 最高 |
| 遭遇强敌(实力碾压) | 服用丹药/临时突破 → 力争脱身 | 次高 |
| 脱身后 | 隐藏修炼 → 突破境界 → 返场复仇 | 中 |
| 宗门/亲友受威胁 | 无视个人风险 → 立刻回援 | 极高(覆盖撤退逻辑) |

**核心逻辑**：叶秋的BT层是最纯粹的"实力判定 → 战斗/撤退"二元结构。没有棋手型的后手评估、没有变革者型的规则破译——就是简单直接的实力对比 + 底牌判断。

#### Utility层权重打分

| 周期 | 目标 | 权重(0-1) | 触发条件 |
|---|---|---|---|
| 短期 | 变强，获取修炼资源 | 0.95 | 始终 |
| 中期 | 保卫宗门/家族，打入中央大陆 | 0.75 | 宗门大比后 |
| 长期 | 打入更高位面，追求武道极致 | 0.5(渐进至1.0) | 飞升/进入中央大陆 |

**权重转换规则**：叶秋的权重转移是最简单的"地图解锁"驱动——每进入一个新地图，Utility权重向上层目标转移。

#### GOAP层决策链

```
[获得奇遇] → [修为暴涨，获得核心竞争力]
    ├── 策略：坠崖/入秘境→发现上古传承/神秘小塔
    ├── 行动：经历考验 → 获得传承 → 修为突破
    └── 产出：核心功法和外挂

[宗门大比] → [展示实力，确立地位]
    ├── 策略：被羞辱→隐忍发育→大比中一鸣惊人
    ├── 行动：击杀外门仇敌 → 获得宗门认可
    └── 产出：宗门地位提升，资源获取权

[对抗外敌/叛徒] → [临危受命，守护阵营]
    ├── 策略：宗门危机→借助外力（上古英灵等）
    ├── 行动：临危受命 → 召唤/借助外挂 → 扭转乾坤
    └── 产出：成为宗门英雄，获取更高权限

[进入中央大陆] → [扩展地图，追求更高境界]
    ├── 策略：当前位面资源耗尽→向更高位面进军
    ├── 行动：飞升/跨越 → 从底层重新攀升
    └── 产出：新地图、新敌人、新一轮循环
```

#### 标志事件

1. **坠崖/入秘境得奇遇** — 经典"废柴翻身"起点，获得核心竞争力。
2. **宗门大比一鸣惊人** — 展示实力，从底层跃升为宗门焦点。
3. **宗门危机中临危受命** — 借助外挂扭转乾坤，确立核心地位。
4. **打入中央大陆/更高位面** — 地图扩展，开启新一轮循环。

## NPC类型化归纳

### 1. 天才傲娇型（主角踏脚石）
- **行为逻辑**：天赋极高但性格傲慢，天生看主角不顺眼
- **BT映射**：遭遇主角→抢占"打压/羞辱"行为
- **Utility映射**：优越感权重极高，无法接受被超越
- **GOAP映射**：嘲笑主角 → 被越级反杀 → 退出舞台（或沦为小弟）
- **实例**：各阶段的"天才师兄/少主"

### 2. 慈祥长辈/废柴父亲型
- **行为逻辑**：保护主角、提供初期资源，后期因主角太强退居二线
- **BT映射**：主角遇险→抢占"舍命保护"（初期）/ 被超越后→抢占"默默守望"
- **Utility映射**：主角成长权重 = 1.0
- **GOAP映射**：培养 → 保护 → 目送 → 退居幕后
- **实例**：宗门长老、家族长辈

### 3. 反派高层型
- **行为逻辑**：为夺取主角宝物或历史恩怨，持续派出杀手
- **BT映射**：主角存活→抢占"加大力度/亲自出手"
- **Utility映射**：消除威胁权重 > 一切
- **GOAP映射**：降下必杀令 → 派出一波波更强的杀手 → 最终亲自出马 → 被反杀
- **实例**：各大势力的敌对高层
