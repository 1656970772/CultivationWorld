## NPC

### NPC 静态数据

| 类型 | 特征 | 行为模式 |

| 武者 | 人类修炼者，有完整境界体系 | 修炼→战斗→突破→传承 |
| 魔兽 | 非人生物，天生拥有元力 | 吞噬→进化→领地意识 |

| 丹师/阵师/器师 | 拥有专业技能的非主战型NPC | 炼制→交易→传授技艺 |

### NPC 需求

#### GOAP（Goal-Oriented Action Planning）行为链模型（推演）

NPC的行为由其需求驱动，通过GOAP模型计算出最优行为序列：

**楚南的核心需求优先级**：

| 优先级 | 需求 | 触发条件 | 典型行为链 |
|--------|------|----------|-----------|
| 1 | 生存 | 生命值<30% | 寻找安全地点 → 服用疗伤丹药 → 恢复战力 |
| 2 | 保护亲友 | 亲友受到威胁 | 感知威胁 → 赶往现场 → 战斗介入 |
| 3 | 突破境界 | 达到当前境界瓶颈 | 收集突破材料 → 寻找突破地点 → 渡劫突破 |
| 4 | 修炼 | 无紧急任务 | 打坐修炼 → 服用辅助丹药 → 提升修为 |
| 5 | 复仇 | 遭遇灭门/朋友被杀 | 追查敌人 → 修炼/提升 → 击杀仇敌 |
| 6 | 探索 | 发现上古遗迹情报 | 收集线索 → 准备物资 → 进入遗迹 |

**配偶需求（推演）**：

| 角色 | 核心需求 | 优先级逻辑 |
|------|----------|-----------|
| 南宫灵芸 | 家族存续 > 辅助楚南 > 自身修炼 | 家族稳定度<30时，家族需求优先级最高 |
| 紫梦儿 | 商业扩张 > 楚南辅助 > 自身修炼 | 商会利润下降时触发扩张需求 |
| 天然 | 自然平衡 > 陪伴楚南 > 自身修炼 | 感知天地异动时触发平衡需求 |

**魔道子的师徒需求（生前/传承）**：
- 寻找继承者 → 考验心性 → 传授功法 → 暗中守护 → 陨落托付

### NPC 行为

| 行为大类 | 具体行为 | 消耗 | 产出 |
|----------|----------|------|------|
| 修炼 | 打坐、吞服丹药、与强者切磋 | 时间 + 元石/丹药 | 修为提升 |
| 征战 | 挑战强敌、参与宗门战争 | 生命力 + 法宝耐久 + 丹药 | 战斗经验 + 战利品 |
| 炼丹 | 收集材料 → 开炉炼丹 | 材料 + 时间 + 元力 | 丹药 |
| 炼器 | 收集材料 → 锻造 | 材料 + 时间 + 元力 | 法宝 |
| 探索 | 进入遗迹/禁区 | 时间 + 消耗品 | 奇遇/宝物/功法残篇 |
| 交易 | 前往商会/拍卖行 | 元石/物品 | 所需物品/元石 |
| 建立势力 | 招募弟子→建立据点→扩张 | 资源 + 时间 | 势力影响力 + 资源产出 |

### 消耗/产出关系

| NPC类型 | 日常消耗（推演） | 境界突破额外消耗 | 产出 |
|---------|-----------------|------------------|------|
| 普通武者 | 每日3-5枚下品元石 | 突破大境界需100-1000倍日常消耗 + 特殊材料 | 战力 |
| 天才/特殊体质 | 每日10-30枚下品元石 | 同普通武者，但成功率高3-5倍 | 高倍战力 |
| 楚南 | 每日50+枚下品元石（五行之体消耗五倍） | 需五行平衡材料+大量元石 | 同阶无敌战力 |
| 魔兽 | 血肉/魔核 + 领地元气 | 吞噬同阶魔兽魔核或天材地宝 | 魔核/材料 |
