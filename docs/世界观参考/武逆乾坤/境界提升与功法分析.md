## 设定摘要

本作基础境界链明确列至武神；楚南的特殊性不在于“境界名新奇”，而在于他没有常规经脉，却通过肉身、五行元力、自造通道与功法兼容打破常规修炼路径。后期“武神是否终点、破碎虚空之后如何分层”有原文讨论，但本轮不硬接成完整后续境界表。

## 原作事实

- 基础境界：武士 -> 武师 -> 大武师 -> 武将 -> 武君 -> 武王 -> 武皇 -> 武帝 -> 武尊 -> 武圣 -> 武神。
- 武诀/武技品级：天、地、玄、黄，每级上中下三品。
- 《逆乾坤》不是常规经脉功法，而是让血肉、骨骼、细胞等身体部位储存和调集元力。
- 修炼《逆乾坤》必须承受痛苦；魔道子明确指出死亡概率大、功法未完成。
- 自由镇五行淬体后，楚南仍经历“能感受元力但不能使用”的瓶颈，后来借五行相克、重剑与意志突破。
- 自造经脉是楚南解决元力输出不稳定的关键方向。

## 游戏化抽象（我的判断）

- 境界系统应分成“常规境界等级”和“特殊修炼结构”两层；楚南这类角色即使境界低，也可因肉身、元力密度、特殊通道获得越级战力。
- 《逆乾坤》可抽象为高风险成长树：每次引入新属性/新通道都需要承受伤害、饥饿、失控、昏迷或死亡风险。
- 功法兼容机制可做成“主功法吸收副功法原理”，但必须消耗修炼时间和试错代价。
