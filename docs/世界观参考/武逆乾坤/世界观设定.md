## 设定摘要

《武逆乾坤》的原作事实核心不是“苏牧/楚枫式通用逆袭模板”，而是楚南这名天生经脉尽断、被判定难以练武的少年，在天武大陆从白家村出发，经龙角山/十万大山奇遇、魔道子传承、自由镇五行淬体、云罗门测试、神器派修行等节点，逐步形成以《逆乾坤》为中心的特殊武道路径。

## 原作事实

- 主角为楚南；原文前置说明和第001章均直接支持这一点。
- 初始舞台为天武大陆，早期生活点为白家村，周边出现大庆国、北齐国、十万大山、自由镇等地名。
- 基础武者等级链为：武士、武师、大武师、武将、武君、武王、武皇、武帝、武尊、武圣、武神。
- 武诀与武技分为天、地、玄、黄四级，每级上中下三品；《焰火诀》是黄级下品，《乾坤转》被魔道子记为地级上品。
- 《逆乾坤》是魔道子自创、面向经脉尽毁者的特殊武诀，需要强横肉身、坚韧意志，并经受金木水火土五类痛苦。
- 天一宗、乾坤宗、圣火门、万毒门、云罗门、神器派、南宫家/南宫世家等均有原文支撑；“玄天宗、逆天宗、乾坤殿、逆乾坤塔、破禁之眼”等未在原文命中。

## 游戏化抽象（我的判断）

- 可以把本作抽象成“异常体质 + 极端痛苦修炼 + 多势力追索/庇护 + 复仇与报恩并行”的武道沙盒参考。
- 楚南的主角路径适合做成“无经脉角色”的特殊成长分支，而不是普通宗门天才起手。
- 世界规则应突出“资源、势力、地理危险、武道心性”共同塑造角色命运，而不是只做境界数值堆叠。
