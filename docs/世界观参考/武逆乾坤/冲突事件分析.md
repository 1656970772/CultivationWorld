## 设定摘要

本作冲突类型可确认为：尊严欺凌、师仇复仇、救援追捕、资源/据点毁灭、宗门体制误判、同门权势冲突、魔兽威胁等。旧文档中的“秘境探索、正邪对抗、上位面降维打击”可保留为后期待补方向，但不能作为本轮事实主轴。

## 原作事实

- 白家村冲突是楚南成长的第一个意志试炼。
- 魔道子传承把楚南与天一宗、玄无奇、青凤的仇怨绑定。
- 万毒门追捕南宫灵芸引出楚南救援、毒火淬体、南宫家与万毒门仇怨。
- 自由镇毁灭把楚南个人修炼转为报恩复仇。
- 云罗门测试失败展示宗门体制无法识别特殊体质。
- 神器派同门冲突围绕紫梦儿、凌霄、大长老、楚南身份与战力展开。

## 游戏化抽象（我的判断）

- 冲突升级应由“羞辱 -> 伤害 -> 杀意 -> 势力介入 -> 长期仇恨”逐层推进。
- 特殊角色可把敌方攻击转化为成长资源，如毒火淬体，但需保留失控/死亡风险。
- 宗门内部冲突不仅是战斗，也包括身份、婚约、长老权威和门派声望。
