# 武逆乾坤 — 角色AI行为参考

## AI架构映射说明

《武逆乾坤》的楚枫是"逆天改命型主角"的代表，其AI行为模式与《武破九荒》的叶秋形成对照：

- **BT层**：楚枫的应急体系以"情感触发"为最大特征——亲友受辱时愤怒激增，无视等级差距强行出手。这种"情感覆盖理性"的BT逻辑是修仙AI中重要的"不稳定性因素"，适合作为沙盒中的"复仇者NPC"或"玩家情感羁绊触发点"。
- **Utility层**：楚枫的权重曲线是"执念驱动"——解密封印 → 寻亲 → 推翻压迫者，权重转移由"真相解锁度"触发，而非境界突破。
- **GOAP层**：每一步都包含"觉醒 → 禁忌冒险 → 清算血仇"的高风险循环。

## 核心角色分析

### 1. 楚枫

#### 性格标签
重情重义 / 睚眦必报 / 执着 / 拥有特殊血脉/破禁之眼 / 不向命运低头

#### BT层应急反应
| 触发条件 | 抢占行为 | 优先级 |
|---|---|---|
| 亲友受辱/被杀 | 愤怒激增 → 无视等级差距强行出手 | 最高(覆盖一切) |
| 遭遇常规强敌 | 评估 → 发挥特殊能力(破禁之眼等) → 寻找弱点 | 次高 |
| 为复活亲友/解除封印 | 制定高风险计划 → 夺取天材地宝 | 极高 |
| 遭受不公/压迫 | 绝不屈服 → 寻找规则漏洞 → 暴力反抗 | 中 |

**核心逻辑**：楚枫的BT层有一个"情感阈值"机制。正常情况下走常规战力评估，但一旦涉及"亲友受辱"标签，愤怒值直接溢出，绕过一切理性判断强制出手。这使他与张衍形成极端对比——一个是"永不冲动"，一个是"为情必冲"。

#### Utility层权重打分

| 周期 | 目标 | 权重(0-1) | 触发条件 |
|---|---|---|---|
| 短期 | 解开封印/觉醒血脉，证明自己 | 0.95 | 故事初期 |
| 中期 | 寻找失踪的亲人/爱人 | 0.9 | 真相逐步揭开 |
| 长期 | 推翻压迫众生的"天"或古老家族，逆天改命 | 0.6(渐进至1.0) | 真相完全解锁 |

**权重转换规则**：楚枫的权重转移由"真相解锁度"驱动——每揭开一层身世/世界真相，Utility权重向更根本的目标转移。这与叶秋的"地图驱动"形成对比。

#### GOAP层决策链

```
[觉醒之日] → [从废人到觉醒者]
    ├── 策略：长期被视废人→契机（祖先残魂唤醒）→开启特殊能力
    ├── 行动：开启破禁之眼等能力 → 开始逆袭
    └── 产出：核心竞争力，身份转变

[勇闯禁地] → [获取禁忌力量，突破极限]
    ├── 策略：为救人/提升 → 无视宗门禁令进入死地
    ├── 行动：破解上古阵法 → 获得禁忌力量
    └── 产出：超越常规的实力跃升

[清算血仇] → [查明真相，复仇灭门之敌]
    ├── 策略：查清家族灭门真相
    ├── 行动：苦练神功 → 在上位面找到仇人 → 发动宗门大战
    └── 产出：血仇得报，执念释放

[逆天改命] → [推翻压迫者，建立新规则]
    ├── 策略：联合被压迫势力
    ├── 行动：对抗"天"或古老家族 → 打破宿命
    └── 产出：命运自主，新秩序
```

#### 标志事件

1. **觉醒之日** — 废人标签撕下，特殊能力觉醒，逆袭起点。
2. **勇闯禁地** — 跨越生死线获取禁忌力量，体现"不向命运低头"。
3. **清算血仇** — 查明灭门真相，发动宗门大战，情感驱动的巅峰。
4. **逆天改命** — 对抗终极压迫者，打破宿命建立新规则。

## NPC类型化归纳

### 1. 神秘老爷爷型（戒指残魂）
- **行为逻辑**：居住在主角戒指/脑海中的残魂，提供关键指引
- **BT映射**：主角遇绝境→抢占"关键时刻指点/借力"
- **Utility映射**：主角存活与成长权重 = 1.0
- **GOAP映射**：隐匿观察 → 关键时刻出现 → 提供丹方/功法/指点 → 最终消散或复活
- **实例**：楚枫的戒指残魂导师

### 2. 霸道宗门型
- **行为逻辑**：讲究资源垄断、弱肉强食、血统至上
- **BT映射**：主角出身低微→抢占"轻视/打压"
- **Utility映射**：资源控制权重 = 1.0
- **GOAP映射**：轻视主角出身 → 打压 → 被主角连根拔起
- **实例**：压迫楚枫的各大宗门势力

### 3. 红颜知己型（要挟触发器）
- **行为逻辑**：背景深厚但喜欢主角，愿意牺牲自己
- **BT映射**：被抓作为要挟→激发主角"情感覆盖"→主角爆发
- **Utility映射**：主角存活权重 > 自身存活权重
- **GOAP映射**：陪伴 → 被抓 → 成为主角爆发的导火索 → 获救
- **实例**：楚枫身边的红颜
