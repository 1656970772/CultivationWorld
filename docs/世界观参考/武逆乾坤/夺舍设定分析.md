# 《武逆乾坤》—— 夺舍设定参考文档

**小说背景**：未最后通牒（注：市面上存在多部同名小说，此处根据常见玄幻设定逻辑推演，若为特定作者请告知）。通常这类小说涉及武道巅峰、转世重生。

### 一、夺舍的基本规则
1. **武道体系下的灵魂寄居**：
   * 在武道世界中，灵魂被视为精神力或神识。夺舍即"神识转移"。
   * **限制**：肉身强度必须足以承载强大的神识。如果肉身太弱，强者的神识会直接撑爆肉体。
2. **天地规则**：
   * 夺舍通常被视为违背轮回之道。天道会降下"魂罚"。
   * 某些世界设定中，每个人出生时都有"命星"或"命格"，夺舍者无法继承原主的"气运"。

### 二、夺舍的代价与风险
1. **气血冲突**：武道修炼讲究气血。夺舍者的灵魂属性（阴/寒）与被夺舍者的肉身属性（阳/热）若冲突，会导致走火入魔。
2. **记忆融合混乱**：夺取身体后，原主的记忆碎片可能会干扰夺舍者的判断，导致人格分裂。
3. **瓶颈**：使用夺舍来的身体，在冲击大境界（如从先天到宗师）时，成功率会降低，因为这不是"本命之身"。

### 三、典型案例分析
1. **废柴流主角**：
   * **模式**：少年被退婚/羞辱 → 神秘强者灵魂（或主角前世）觉醒 → 占据主导权。
   * **分析**：这属于**"觉醒式夺舍"**，即灵魂原本就属于这具身体，只是以前沉睡了。
2. **家族内斗**：
   * **模式**：长老夺舍族长之子，以控制家族资源。

### 四、夺舍相关的特殊设定
1. **肉身印记**：高阶武者会在细胞深处留下灵魂烙印。即使夺舍成功，若遇到原主的长辈或同源功法，可能被识破。
2. **夺舍与炼尸**：将尸体炼制成傀儡（无灵魂），然后注入自己的灵魂，这叫"借尸"，比夺舍活人更无道德负担，但尸体没有生长潜力。

### 五、游戏设计启示
1. **气运值系统**：夺舍后，玩家的"气运"可能清零或变为负值，需要通过特定事件（如改变原主命运）来重新积累。
2. **肉体爆裂机制**：如果玩家试图用低阶肉身承载高阶灵魂，应设计"肉体崩溃"即死机制。
3. **身份伪装玩法**：夺舍后如何瞒过原主的父母、宗门是关键玩法，可以通过"模仿记忆"或"系统掩盖"来实现。
