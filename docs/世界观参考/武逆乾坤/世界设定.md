## 设定摘要

本作早期世界以天武大陆为基准，地方层级从白家村、龙角山、十万大山、自由镇、大庆国、北齐国逐步展开；中期以北齐国宗门格局和神器派山门为重点；后期才出现诸天大陆、诸天殿、虚空小道等更高层空间。旧文档中“神武大陆、圣武大陆、九重天”等未在本次原文检索中形成稳定证据，不能作为事实区设定。

## 原作事实

- 白家村是楚南早期生活地，龙角山是他遭遇玄火血蟒并进入十万大山奇遇链的触发点。
- 大庆国、北齐国、蛮越国等国名与自由镇相关；自由镇处于三国夹缝，靠近十万大山，因魔兽交易、魔兽核、皮肉、丹药、武器而繁荣。
- 北齐国势力格局明确有一宗二派三大门，其中天一宗在北齐国门派中居首。
- 神器派以炼器著名，山门按层级修炼，灵气浓度和修炼资格与实力层级相关。
- 后期出现诸天大陆、诸天殿、虚空、天武阵等更大尺度空间，但本轮只确认其存在，不展开完整宇宙层级。

## 游戏化抽象（我的判断）

- 世界地图适合按“村落起点 -> 危险山地 -> 边境交易镇 -> 北齐宗门圈 -> 上层大陆/虚空”分层，而不是一开始铺满多大陆。
- 自由镇可做成资源型灰色城镇：魔兽资源回流带来商队、武器、丹药、客栈，魔兽消失则经济衰退。
- 神器派山门可抽象为按境界开放的分层修炼区，兼具炼器、切磋、藏书阁、特殊修炼地。
