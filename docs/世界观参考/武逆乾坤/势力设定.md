## 势力 Faction

### 势力静态数据

#### 类型枚举（推演）

| 类型 | 特征 | 核心利益 |
|------|------|----------|
| 宗门 | 师徒传承制，以武道追求为核心，有完整功法和培养体系 | 武道传承、天才弟子、修炼资源 |
| 家族 | 血脉传承，以家族血脉和血脉武技为核心竞争力 | 血脉延续、家族荣耀、领地与资源 |

| 散修联盟 | 松散组织，没有师承的武者抱团取暖 | 生存、资源共享、对抗宗门压迫 |
| 炼丹师协会/阵法师公会 | 专业技术组织，掌握核心技艺传承 | 技艺垄断、材料优先获取权 |

### 势力状态

每个势力的状态可使用以下维度量化（推演）：

| 状态维度 | 说明 | 示例（天道宗） |
|----------|------|----------------|
| 稳定度 | 内部团结程度（0-100） | 85（正道领袖，内部纪律严明） |
| 资源压力 | 对修炼资源的需求紧迫度 | 中（拥有多处元石矿脉） |
| 军事压力 | 外部威胁程度 | 中（魔渊殿持续威胁） |
| 继任风险 | 掌门/族长换代的不确定性 | 低（长老会制度成熟） |
| 声望 | 在天武大陆的名望 | 95（正道第一宗门） |

### 势力需求

| 需求类型 | 优先级计算逻辑（推演） | 示例触发条件 |
|----------|------------------------|-------------|
| 生存需求 | 当稳定度<30 或军事压力>80 时，优先级最高 | 魔渊殿大举入侵 |
| 扩张需求 | 当资源压力>60 且军事压力<50 时触发 | 发现新元石矿脉 |
| 资源需求 | 当资源压力>70 时触发，驱动交易或掠夺 | 矿脉枯竭 |
| 对抗需求 | 当敌对势力声望增长过快时触发 | 对手宗门诞生天才弟子 |
| 培养需求 | 日常需求，优先级中低 | 持续招收新弟子 |

### 势力行为（推演）

| 行为 | 描述 | 触发条件 |
|------|------|----------|
| 争战 | 发动宗门战争，抢夺资源或消灭对手 | 资源压力高 + 军事压力低 |
| 结盟 | 与其他势力建立同盟 | 军事压力高 + 有共同敌人 |
| 交易 | 通过商会进行资源交换 | 有剩余资源 + 有特定需求 |
| 培养弟子 | 招收并培养新弟子 | 日常运转需求 |
| 情报收集 | 派出探子收集敌对势力情报 | 军事压力>40 |
| 遗迹探索 | 组织弟子探索上古遗迹 | 发现遗迹情报 |

### 消耗/产出关系

```
┌─────────────────────────────────────────────────┐
│                势力资源循环                       │
│                                                  │
│  元石矿脉 → 开采 → 元石储量                       │
│                  ↓                                │
│  元石储量 → 弟子修炼 → 弟子实力提升 → 势力战力     │
│                  ↓                                │
│  势力战力 → 争战/交易 → 抢夺/换取更多资源            │
│                  ↓                                │
│  丹药材料 → 炼丹师炼制 → 丹药储备 → 突破/疗伤       │
│                  ↓                                │
│  魔兽材料 → 炼器师炼制 → 法宝储备 → 战力提升         │
└─────────────────────────────────────────────────┘
```

| 消耗 | 产出 | 效率系数（推演） |
|------|------|------------------|
| 100枚下品元石 + 1名弟子修炼1月 | 弟子修为提升约1小阶 | 取决于弟子资质（1-5倍） |
| 宗门招收100名弟子 + 培养1年 | 约5-10人达武师境 | 取决于宗门培养力 |
| 发动1次宗门战争 | +资源或-全灭 | 取决于战力差与策略 |
