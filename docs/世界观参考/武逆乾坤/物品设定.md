## 物品 Item

### 物品静态数据

所有物品的核心属性：

| 属性 | 说明 |
|------|------|
| 名称 | 物品名 |
| 类型 | 资源/法宝/功法/武技/丹药/阵法/魔兽相关 |
| 品阶 | 具体等级（取决于类型分类体系） |
| 稀有度 | 普通/稀有/史诗/传说/神话（推演） |
| 来源 | 采集/炼制/掉落/任务/交易 |
| 可交易性 | 是否可交易 |

### 物品分类体系

#### 资源类

**元石**（已确认硬通货地位）：

| 品阶 | 兑换比例（推演） | 用途 |
|------|-----------------|------|
| 下品元石 | 基准单位 | 日常修炼、普通交易 |
| 中品元石 | 1中品 = 100下品 | 武师以上修炼、中档交易 |
| 上品元石 | 1上品 = 100中品 | 武君以上修炼、大额交易 |
| 极品元石 | 1极品 = 100上品 | 武皇突破、顶级拍卖会 |
| 元晶 | 1元晶 = 100极品 | 武帝以上修炼、圣地运转 |

**元力/玄气**：天地之间的基础能量，武者修炼与战斗的底层资源。元力偏阳性，玄气偏阴性。修至高层可融合为更高阶能量（推演）。

**五行元力**（已确认）：更高阶的能量形态。分别是金行元力、木行元力、水行元力、火行元力、土行元力。楚南的五行之体可兼容转化五行之力。

**魔兽材料**（推演）：

| 类型 | 来源 | 用途 |
|------|------|------|
| 魔核/兽核 | 魔兽体内核心 | 炼丹主材、炼器能量核心、修炼吸收 |
| 兽皮/鳞甲 | 高阶魔兽躯体 | 炼制护甲/法宝 |
| 兽骨/角 | 高阶魔兽躯体 | 炼制攻击法宝/阵旗 |
| 兽血 | 高阶魔兽血液 | 炼丹/炼体/血脉淬炼 |

**天材地宝**（推演）：

| 类型 | 来源 | 用途 |
|------|------|------|
| 灵药/灵草 | 灵脉附近采集 | 炼丹主材 |
| 灵石矿/金属矿 | 矿脉开采 | 炼器材料 |
| 天地灵物 | 特殊环境诞生 | 特殊体质淬炼、顶级法宝材料 |
| 混沌材料 | 混沌世界产出 | 古宝/祖宝级神兵材料 |

#### 道具-法宝/神兵

**品阶体系**（已确认）：

> 真器 → 法器 → 灵器 → 宗器 → 圣器 → 神器 → 祖宝 → 古宝

每阶分5个品级：伪 → 下 → 中 → 上 → 极（推演）

| 品阶 | 对应境界（推演） | 炼制难度 | 价值参考 |
|------|-----------------|----------|----------|
| 真器 | 武士~武师 | 低 | 10-100下品元石 |
| 法器 | 大武师~武君 | 中低 | 100-1000下品元石 |
| 灵器 | 武王~武皇 | 中 | 1-100中品元石 |
| 宗器 | 武帝~武尊 | 中高 | 1-100上品元石 |
| 圣器 | 武圣 | 高 | 1-100极品元石 |
| 神器 | 武神 | 极高 | 有价无市 |
| 祖宝 | 武祖 | 近乎不可能 | 非卖品 |
| 古宝 | 古之境以上 | 需要混沌之气 | 仅传承 |

### 物品行为

| 行为 | 描述 | 消耗 | 产出 |
|------|------|------|------|
| 采集 | 矿脉开采元石、野外采摘灵草 | 时间 + 人力 | 原材料 |
| 炼制 | 将材料炼制成成品 | 材料 + 元力 + 时间 + 成功率折损 | 丹药/法宝/阵法 |
| 消耗 | 使用物品（修炼/战斗/恢复） | 物品本身 | 修为/战力/伤口恢复 |
| 交易 | 通过商会或拍卖行交换 | 物品 + 手续费 | 元石/其他物品 |
| 掠夺 | 击杀敌人或攻破宗门夺取 | 战力消耗 + 风险 | 敌人掉落 + 宗门库存 |
| 装备 | 佩戴法宝、携带丹药等 | — | 战力/生存能力提升 |

### 消耗/产出关系

#### 核心炼制链（推演）

```
┌─────────────────────────────────────────────────────┐
│               炼制产业链                              │
│                                                      │
│  采矿 → 矿石 → 提炼 → 金属锭 → 锻造 → 法宝            │
│  采药 → 灵草 → 搭配 → 开炉 → 丹药                     │
│  猎兽 → 尸骸 → 分解 → 魔核+材料 → 丹/器              │
│  布阵 → 阵旗 + 元石 → 阵法激活                        │
└─────────────────────────────────────────────────────┘
```
