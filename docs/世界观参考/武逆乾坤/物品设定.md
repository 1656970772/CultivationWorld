## 物品 Item

### 物品静态数据

所有物品的核心属性：

| 属性 | 说明 |
|------|------|
| 名称 | 物品名 |
| 类型 | 资源/法宝/功法/武技/丹药/阵法/魔兽相关 |
| 品阶 | 具体等级（取决于类型分类体系） |
| 稀有度 | 普通/稀有/史诗/传说/神话（推演） |
| 来源 | 采集/炼制/掉落/任务/交易 |
| 可交易性 | 是否可交易 |

### 物品分类体系

#### 资源类

**元石**（已确认硬通货地位）：

| 品阶 | 兑换比例（推演） | 用途 |
|------|-----------------|------|
| 下品元石 | 基准单位 | 日常修炼、普通交易 |
| 中品元石 | 1中品 = 100下品 | 武师以上修炼、中档交易 |
| 上品元石 | 1上品 = 100中品 | 武君以上修炼、大额交易 |
| 极品元石 | 1极品 = 100上品 | 武皇突破、顶级拍卖会 |
| 元晶 | 1元晶 = 100极品 | 武帝以上修炼、圣地运转 |

**元力/玄气**：天地之间的基础能量，武者修炼与战斗的底层资源。元力偏阳性，玄气偏阴性。修至高层可融合为更高阶能量（推演）。

**五行元力**（已确认）：更高阶的能量形态。分别是金行元力、木行元力、水行元力、火行元力、土行元力。楚南的五行之体可兼容转化五行之力。

**魔兽材料**（推演）：

| 类型 | 来源 | 用途 |
|------|------|------|
| 魔核/兽核 | 魔兽体内核心 | 炼丹主材、炼器能量核心、修炼吸收 |
| 兽皮/鳞甲 | 高阶魔兽躯体 | 炼制护甲/法宝 |
| 兽骨/角 | 高阶魔兽躯体 | 炼制攻击法宝/阵旗 |
| 兽血 | 高阶魔兽血液 | 炼丹/炼体/血脉淬炼 |

**天材地宝**（推演）：

| 类型 | 来源 | 用途 |
|------|------|------|
| 灵药/灵草 | 灵脉附近采集 | 炼丹主材 |
| 灵石矿/金属矿 | 矿脉开采 | 炼器材料 |
| 天地灵物 | 特殊环境诞生 | 特殊体质淬炼、顶级法宝材料 |
| 混沌材料 | 混沌世界产出 | 古宝/祖宝级神兵材料 |

| 行为 | 描述 | 消耗 | 产出 |
