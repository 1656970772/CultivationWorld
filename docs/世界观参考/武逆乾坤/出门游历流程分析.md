## 设定摘要

楚南的出门游历流程应按“被迫/主动离开 -> 禁地奇遇 -> 山野遭遇 -> 灰色据点修炼 -> 城镇打听与宗门测试 -> 强者收徒/宗门落脚 -> 后期虚空与秘境”建模。旧版“上位面复仇游历”可作为后期方向，但需逐章抽证。

## 原作事实

- 初次危险出行来自龙角山与十万大山，目标并非寻宝而是抓小蟒蛇/进入禁忌地带，结果触发血蟒奇遇。
- 山野游历会遭遇魔兽、毒门追捕、陌生女子求救等复合事件。
- 自由镇是游历后的灰色落脚点，提供训练、资源和情报。
- 离开自由镇后，楚南通过雄罗城、陈晓峰等获得云罗门与北齐势力信息。
- 宗门测试失败后，吕阳明的介入使楚南进入神器派。
- 后期存在雾禁海、锁海秘境、虚空小道等高阶路线，但本轮只确认名称与大致功能。

## 游戏化抽象（我的判断）

- 游历流程应包含“目的偏差”：玩家/角色原目的很小，实际触发大奇遇。
- 每次出门可生成三类节点：危险地形、遭遇 NPC、资源/势力入口。
- 宗门准入不应保证成功，失败后可通过强者赏识、救援事件、特殊能力暴露进入下一阶段。
