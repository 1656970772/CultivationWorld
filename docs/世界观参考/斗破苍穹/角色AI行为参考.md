# 《斗破苍穹》角色AI行为参考

## AI架构映射说明

本游戏的NPC AI采用四层混合架构：

| 层级 | 职责 | 时间粒度 | 斗破苍穹世界观映射 |
|------|------|----------|-------------------|
| **BT（行为树）** | 情绪反应与应急抢占 | 即时 | 被羞辱→隐忍/爆发；同伴遇险→底牌介入；感知杀意→先手反击 |
| **Utility（效用系统）** | 中长期目标选择，含权重打分 | 数天至数年 | 修炼方向选择、炼药/战斗优先级、势力归属决策 |
| **GOAP（目标导向规划）** | 目标分解为步骤链 | 数小时至数月 | 从"我要变强"到具体的异火获取、丹药炼制、斗技修炼序列 |
| **Action Queue（动作队列）** | 执行具体战斗/炼药/移动动作 | 逐帧/逐回合 | 施展斗技、炼制丹药、吸收异火、移动至目标区域 |

### 斗破苍穹特色机制：等级碾压与底牌系统

斗破苍穹的战斗体系高度依赖**斗气等级压制**，AI设计中必须体现：

- **等级差 >= 一个大境界** → 低等级方常规手段几乎无效，必须动用"底牌"
- **底牌**：异火、天阶斗技、灵魂力量、丹药爆发等，每张底牌有使用次数限制和代价
- **"三年之约"模式**：被羞辱后隐忍，标记复仇目标，疯狂修炼直至超越

## 一、萧炎（复仇型主角 NPC）

### 性格标签

```
坚韧不拔 | 恩怨分明 | 极度隐忍→爆发式复仇 | 护短 | 炼药狂热
```

- **隐忍阈值**：极高。能在三年内忍受全城嘲讽而不冲动。
- **爆发条件**：标记的复仇目标实力被超越 / 同伴受到直接生命威胁。
- **冷却期**：爆发后进入"冷静反思期"，复盘战斗并继续修炼。

### BT 应急反应层

| 触发条件 | 响应行为 | 抢占优先级 |
|----------|----------|------------|
| 同伴遇险（生命威胁） | **爆发模式**：无视等级差距，利用异火/天阶斗技/药老灵魂之力强行介入 | **100（最高）** |
| 被公开羞辱/嘲讽 | **隐忍模式**：不立刻发作，内心标记"必杀/必超越"，转为疯狂修炼驱动力 | 20（低抢占） |
| 标记的复仇目标出现在可战斗范围 | **立刻动手**：不再隐忍，"今天的我已不是昨天的我" | 90 |
| 感知药老灵魂虚弱 | **紧急救援**：暂停一切当前任务，寻找补充灵魂力量的药材 | 95 |
| 发现异火气息 | **高度兴奋**：评估获取难度，值得冒高风险 | 70 |
| 遭偷袭/暗杀 | 瞬间展开灵魂感知 → 判断敌人实力 → 能打则异火招呼，不能打则雷动三千遁走 | 85 |
| 美色诱惑/陷阱 | 保持理智，不轻易入局（比叶凡更谨慎） | 40 |

> **BT设计要点**：萧炎的BT树有两个"模式状态机"——**隐忍态**与**爆发态**。隐忍态下，被羞辱的响应是"记录仇恨值+修炼驱动力+1"而非战斗。爆发态切换条件：仇恨目标实力判定 ≤ 自身实力 × 1.1。

### Utility 长期目标权重

| 阶段 | 核心目标 | 权重 | 权重变化条件 |
|------|----------|------|-------------|
| 第一阶段 | 恢复天才名誉 | 90% | 在家族中重新展现天赋后降为 20% |
| 第一阶段 | 探索斗气流失之谜 | 70% | 药老揭示真相后降为 0% |
| 第二阶段 | 三年之约（击败纳兰嫣然） | **100%** | 完成后锁定为 0% |
| 第二阶段 | 修炼焚诀、吞噬异火 | 80% | 每获得一种异火 -20%，直至饱和 |
| 第三阶段 | 拯救药老 | **100%** | 药老被救出后降为 0% |
| 第三阶段 | 建立炎盟/对抗魂殿 | 75% | 随魂殿被削弱而降低 |
| 第四阶段 | 对抗魂族、阻止魂天帝 | 90% | 魂天帝被击败后降为 0% |
| 终极阶段 | 成为斗帝 | **100%** | 终极目标 |

> **Utility 设计要点**：与叶凡类似，萧炎的目标存在严格的"阶段锁定"——三年之约完成前，拯救药老虽然权重高但不会被激活。设计中使用"阶段解锁"条件实现。此外，萧炎的"复仇"类目标权重会随仇恨值积累而增加。

### GOAP 标志事件决策链

#### 事件一：退婚与三年之约

```
目标：洗刷耻辱，击败纳兰嫣然

GOAP 步骤链：
  1. [事件] 纳兰嫣然登门退婚，萧炎受辱
  2. [决策] 签下三年之约 → 定下复仇目标
  3. [前置] 发现戒指中的药老 → 获得导师
  4. [修炼] 恢复斗气 → 重新修炼焚诀
  5. [修炼] 前往魔兽山脉实战磨砺
  6. [分支] 探索异火线索 → 放弃（当前阶段未准备好）
  7. [修炼] 炼化伴生紫晶源 → 突破大斗师
  8. [修炼] 前往塔戈尔大沙漠 → 获得青莲地心火
  9. [备战] 炼制三纹青灵丹等丹药辅助
  10. [决战] 登上云岚宗 → 击败纳兰嫣然

前置条件：斗气等级 >= 大斗师巅峰, 至少掌握一种玄阶高级斗技
关键决策点：是否在沙漠中冒险炼化异火（成功率 60%）
```

#### 事件二：吞噬异火（青莲地心火 → 陨落心炎 → 虚无吞炎）

```
目标：吞噬异火，进化焚诀

GOAP 链（以陨落心炎为例）：
  1. [世界状态] 得知迦南学院天焚炼气塔下有陨落心炎
  2. [前置] 炼药术等级 >= 五品, 焚诀进化至玄阶以上
  3. [前置] 准备辅助丹药：护脉丹、冰心丹、生骨丹
  4. [行动] 进入天焚炼气塔底层
  5. [行动] 引诱陨落心炎现身
  6. [危机] 陨落心炎反噬，灵魂和肉体双重灼烧
  7. [底牌] 药老灵魂之力护住心神
  8. [底牌] 坠龙焰/骨灵冷火作为引线，包裹陨落心炎
  9. [吞噬] 焚诀运转，将异火纳入体内
  10. [炼化] 闭关炼化，期间极度脆弱
  11. [成果] 焚诀进化，获得陨落心炎

前置条件：灵魂境界 >= 凡境大圆满, 身体强度 >= 斗王级别
失败后果：轻则重伤修为倒退，重则被异火反噬焚身
```

#### 事件三：救药老——闯圣域/魂殿

```
目标：从魂殿手中救出药老

GOAP 链：
  1. [世界状态] 药老被魂殿抓走，囚禁于亡魂山脉
  2. [前置] 炼药术提升至八品以上（圣品门槛）
  3. [前置] 组建联盟：炎盟 + 丹塔 + 其他反魂殿势力
  4. [情报] 通过丹塔情报网获取魂殿分殿位置
  5. [渗透] 逐一拔除魂殿外围分殿，获取药老囚禁地点
  6. [决战] 集结联盟力量攻入亡魂山脉
  7. [关键] 在魂殿护法中杀出血路
  8. [灵魂战斗] 对抗魂殿殿主的灵魂攻击
  9. [剥离] 强行剥离药老魂魄与魂殿锁链的连接
  10. [复活] 收集天材地宝，为药老炼制躯体

前置条件：斗气等级 >= 斗尊, 联盟势力 >= 魂殿分殿战力
关键决策：是否等待实力达到斗圣再行动（更安全但药老可能被转移/处决）
```

#### 事件四：双帝之战

```
目标：击败魂天帝，拯救斗气大陆

GOAP 链：
  1. [世界状态] 魂天帝收集齐陀舍古帝玉，即将晋升斗帝
  2. [前置] 自身达到九星斗圣巅峰
  3. [探索] 寻找陀舍古帝洞府
  4. [机遇] 获得帝品雏丹 → 晋升斗帝
  5. [集结] 号召斗气大陆所有势力联合
  6. [决战] 与魂天帝展开斗帝之战
  7. [底牌] 异火广场——将万火归于己用
  8. [终结] 以自身为代价，封印魂天帝

前置条件：斗气等级 = 斗帝, 异火数量 >= 8种
最终代价：失去斗帝修为 / 部分异火
```

## 二、药老（导师型 NPC / 戒指老爷爷模式）

### 性格标签

```
护短 | 老谋深算 | 嗜酒 | 落魄天才 | 亦师亦父
```

### 导师型 NPC 专属机制

药老是修仙模拟游戏中"导师型 NPC"的经典模板：

```
戒指老爷爷模式：
  状态：灵魂体（需寄生在戒指中）
  能力输出限制：消耗自身灵魂本源，使用次数有限
  与宿主关系：师徒 → 如父子 → 相互依赖
```

### BT 应急反应

| 触发条件 | 响应行为 | 优先级 |
|----------|----------|--------|
| 萧炎生命濒危 | **消耗灵魂本源**，强行接管身体或增幅萧炎 | 100 |
| 遇到魂殿中人 | 仇恨值 MAX，理智下降，可能强行出手 | 95 |
| 发现珍稀药材 | 职业病发作，催促萧炎收集 | 60 |
| 萧炎修炼偷懒 | 严厉训斥 | 40 |
| 萧炎陷入情感纠葛 | 旁观不语，偶尔调笑 | 10 |

### Utility 长期目标权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 保护萧炎周全 | 100% | 最高优先级 |
| 复活肉身 | 80% → 60% | 初期强烈，后期萧炎安全权重更高 |
| 向魂殿复仇 | 70% | 贯穿始终 |
| 培养萧炎成为顶尖炼药师 | 60% | 传承执念 |
| 品尝美酒 | 30% | 个人爱好 |

### 消耗灵魂本源的决策模型

```
触发条件：萧炎 HP < 10% 且敌人实力 > 萧炎当前实力 × 2

决策步骤：
  1. 评估自身灵魂剩余量
     ├── >= 30% → 直接出手增幅萧炎
     ├── 10% - 30% → 犹豫 1 回合后出手
     └── < 10% → 只能提供建议，无法实质出手
  
  2. 出手方式：
     ├── 短期增幅 → 消耗 5% 灵魂本源，萧炎战力 × 2（持续 3 回合）
     ├── 接管身体 → 消耗 15% 灵魂本源，以药老斗技经验战斗
     └── 绝命一击 → 消耗全部剩余灵魂，发出相当于斗尊全力一击
```

## 五、魂天帝（反派 Boss 型 NPC）

### 性格标签

```
野心勃勃 | 冷酷无情 | 隐忍万年 | 战略大师 | 为达目的不择手段
```

### 反派 Boss 专属模式：千年布局型

魂天帝不是一时冲动的反派，他是一个万年级别的战略规划者：

```
核心驱动：成为斗帝，统治大陆

布局时间线：
  千年前 → 策划萧族覆灭（分割龙脉，削弱萧族气运）
  数百年间 → 暗中收集陀舍古帝玉碎片
  近百年 → 建立魂殿，四处收集灵魂本源
  当代 → 挑起大陆纷争，坐收渔利
  终局 → 收集全部古玉 → 开启帝品雏丹 → 献祭 → 晋升斗帝
```

### BT 应急反应

| 触发条件 | 响应行为 |
|----------|----------|
| 计划被干扰 | 冷静调整策略而非冲动反击 |
| 萧炎出现且实力飙升 | 开始重视，派出更强战力 |
| 古帝玉碎片暴露 | 不计代价抢夺 |
| 自身受威胁 | 果断牺牲族人保全自己 |

### Utility 权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 收集陀舍古帝玉 | 100% | 核心目标 |
| 成为斗帝 | 100% | 终极目标 |
| 削弱其他远古种族 | 80% | 减少竞争对手 |
| 吸收强大灵魂本源 | 70% | 提升自身实力 |
| 培养魂族年轻一代 | 30% | 工具性质 |

### GOAP 标志事件链

```
事件：千年布局终局——献祭苍生成帝

GOAP 链：
  1. [世界状态] 古帝玉已集齐，帝品雏丹位置确定
  2. [前置] 魂族实力达到可以对抗全大陆的程度
  3. [行动] 开启陀舍古帝洞府
  4. [夺取] 吞噬帝品雏丹
  5. [献祭] 以整个中州生灵为代价，完成最终突破
  6. [成就] 晋升斗帝
  7. [统治] 碾压所有反对势力

关键特征：每一步都是上一步的必然结果，不存在"冲动决策"
AI实现建议：魂天帝的GOAP规划不需要频繁重新计算，他的目标是固定的，只有执行路径随世界状态微调
```

## 六、NPC 类型化归纳

### 类型一：炼药师型

```
代表角色：丹塔三巨头、药老（生前）、小医仙（后期）

核心性格：高傲 | 资源至上 | 沉迷炼丹 | 社交网络庞大

Utility 权重配置：
  - 炼制药效最强的丹药：70%
  - 收集珍稀药材：80%
  - 维护炼药师声誉：60%
  - 战斗：30%（不主动参与，除非被威胁）
  - 结交强者（让他们欠人情）：50%

BT 应急：
  ├── 药材被抢 → 发布悬赏任务，动用人脉而非亲自出手
  ├── 被人挑战炼药术 → 必须应战（声誉很重要）
  ├── 自身受到生命威胁 → 使用保命丹药/召唤欠人情的高手
  └── 有求于己者带珍稀药材前来 → 心情愉悦，优先接待

GOAP 链（炼制八品丹药）：
  1. 获取丹方/自行研究丹方
  2. 列出所需药材清单
  3. 通过人脉网络收集药材
  4. 选择合适炼丹场所（通常有专属丹房）
  5. 调整状态至最佳（精气神巅峰）
  6. 起炉炼丹（可能持续数天至数月）
  7. 应对丹雷/丹劫
  8. 出炉 → 丹药品级判定

AI 参数：
  - 战斗倾向：3/10
  - 社交倾向：8/10
  - 资源敏感度：9/10
  - 闭关频率：高
```

### 类型二：宗门天才/少主型

```
代表角色：纳兰嫣然（云岚宗）、林修崖（迦南学院）、云凌（云岚宗）

核心性格：宗门荣誉感 | 资源掠夺 | 顺我者昌逆我者亡 | 天赋与实际能力可能不匹配

Utility 权重配置：
  - 维护宗门/家族声望：90%
  - 争夺修炼资源：75%
  - 碾压/收服同辈竞争者：60%
  - 探索秘境：50%
  - 与更强势力联姻/结交：40%

BT 应急：
  ├── 被外宗人挑衅 → 直接动手压制
  ├── 发现稀有资源（矿脉/药园） → 驱逐原有势力，打上宗门标记
  ├── 被散修击败 → 耻辱 + 召唤宗门长辈
  └── 宗门受外敌入侵 → 出战迎敌

GOAP 链（资源争夺）：
  1. 宗门探子发现某处有珍稀资源
  2. 评估资源价值与可获取性
  3. 派遣先遣队探查
  4. 若已有占据者 → 评估对方实力
     ├── 弱于己方 → 直接武力驱逐
     └── 强于己方 → 上报宗门，请求增援
  5. 占领后设置阵法/留下看守
  6. 定期收取资源

AI 参数：
  - 战斗倾向：7/10
  - 傲慢程度：8/10（对散修）
  - 宗门忠诚度：8/10
  - 成长上限：受天赋限制
```

```
  8. 返回交任务 → 分账
```
