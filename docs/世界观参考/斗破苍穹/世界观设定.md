## 设定摘要

《斗破苍穹》的世界核心不是灵根、筑基、金丹、元婴或修仙飞升链，而是“斗气大陆 + 斗气修炼 + 功法/斗技/炼药/异火/血脉势力”的体系。斗气从民间生活到顶级势力都具有基础地位；人物成长通常由实力、功法、斗技、炼药资源、异火机缘和势力关系共同驱动。

## 原作事实

- 斗气大陆明确以斗气为主调，功法以天、地、玄、黄四阶十二级区分；同境界下，功法高低会影响战力表现。
- 斗者是能量术语的分水岭：斗者前为“斗之气”，凝聚斗之气旋后才进入真正“斗气”阶段。
- 炼药师是斗气大陆极稀少且地位显赫的职业，其价值来自可炼制提升实力的丹药，但原作要求火焰斗气、火木属性倾向、灵魂感知、材料与火种条件，不是通用炼丹职业模板。
- 后期世界层级从加玛帝国扩展至黑角域、迦南学院、中州和远古八族；斗帝血脉是远古种族叙事的重要资源。

## 游戏化抽象（我的判断）

- 可将世界主键设为 `douqi_world`，核心资源设为 `douqi`、`dou_zhi_qi`、`flame_seed`、`pill_material`、`magic_core`、`bloodline_power`。
- 地图层级宜按“乌坦城/萧家 -> 加玛帝国/云岚宗/魔兽山脉 -> 塔戈尔沙漠 -> 黑角域/迦南学院 -> 中州 -> 远古八族空间”递进。
- “大千世界、灵力、飞升通道”等若用于项目扩展，应标为跨作品或后续衍生设定，不宜写入本目录的原作事实。
