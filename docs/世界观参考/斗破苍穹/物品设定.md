### 资源类

#### 斗气

- **描述**：天地间自然存在的能量，是一切修炼的根基
- **获取方式**：功法吸收转化、天焚炼气塔加速淬炼、丹药直接补充（回气丹等）
- **稀有度**：普通（西北地域稀薄，中州/远古空间浓郁）
- **消耗场景**：修炼突破、战斗消耗、炼丹、维持大阵

#### 药材

- **描述**：炼制丹药的天然原材料
- **分类**：按药性（寒/热/温/凉/毒/补）、按年份（十年/百年/千年/万年）、按品阶（低级/中级/高级/顶级）
- **稀有度**：普通→唯一（如万年魂婴果仅存在于特定秘境）
- **获取方式**：野外采集、魔兽山脉探险、势力药圃培育、拍卖会竞拍

#### 魔核

- **描述**：魔兽体内能量核心结晶，同阶魔兽的魔核价值远高于其肉身
- **分类**：按魔兽等阶（一阶至九阶魔核，九阶魔核极稀有）
- **用途**：炼丹辅料、炼制法器、直接吸收（低阶）、镶嵌武器
- **稀有度**：普通→传说（七阶以上在西北地域极罕见）

#### 灵魂

- **描述**：生灵的精神本源，魂族/魂殿体系的核心资源
- **来源**：斩杀敌人后提取、魂殿捕捉（以锁魂链禁锢）、天墓中先贤灵魂印记
- **灵魂强度**：凡境→灵境→天境→帝境
- **用途**：炼制魂丹（魂殿专用）、增幅自身灵魂境界、作为魂族功法的修炼材料
- **稀有度**：天境灵魂极其珍贵，帝境灵魂大陆历史不超过五人

#### 源气

- **描述**：突破斗帝的必需能量，远古时期充沛，陀舍古帝时代开始枯竭
- **来源**：远古时期天地自然产出、斗帝开辟空间时残留、陀舍古帝洞府中的帝品雏丹
- **稀有度**：唯一（萧炎时代几乎绝迹）
- **获取途径**：
  1. 集齐陀舍古帝玉 → 开启古帝洞府 → 获取帝品雏丹
  2. 血祭大陆亿万生灵 → 以庞大生命能量还原源气（魂天帝方案）
  3. 融合全部23种异火 → 以异火聚合能量替代源气（萧炎方案）

### 道具-功法

功法分天地玄黄四阶，每阶又分初/中/高级。同一境界下，高阶功法修炼速度远超低阶，且能承载更强的斗技。

| 品阶 | 代表功法 | 特性 |
| :--- | :--- | :--- |
| **黄阶** | 基础炼气诀（各帝国通用） | 入门级，仅能修炼至大斗师 |
| **玄阶** | 紫极斗气（萧炎前期功法） | 可支撑至斗灵/斗王 |
| **地阶** | 青天化龙诀（太虚古龙族）、雷族天雷诀 | 远古种族核心功法，可支撑至斗圣 |
| **天阶** | 焚诀（唯一可变阶功法） | 大陆最特殊功法 |

#### 焚诀（核心功法）

- **初始品阶**：黄阶低级
- **特殊机制**：可通过吞噬异火进化，每吞噬一种异火可提升品阶，同时提升修炼者的境界潜力
- **理论上限**：吞噬23种异火后可达天阶巅峰，承载斗帝境界
- **获取方式**：药尘从远古遗迹中获得，传给萧炎
- **稀有度**：唯一
- **风险**：吞噬异火过程极为凶险，需要血莲丹等丹药护体，且肉身必须足够强大

#### 天火三玄变

- **品阶**：地阶高级斗技（焚炎谷秘传）
- **作用**：
  - 第一变：激发天火之力，短时间增幅战力约30%-50%
  - 第二变：战力翻倍，但持续时间短，对肉身负荷大
  - 第三变：战力暴增数倍，为拼死一搏的底牌
- **来源**：焚炎谷，以天火精血为修炼原料
- **稀有度**：史诗（仅焚炎谷核心弟子可修炼）
- **限制**：需要火属性体质和天火精血

### 道具-斗技

斗技同样分天地玄黄四阶，是战斗力的直接体现。同一品阶的斗技因属性和用途不同，实战差距可能极大。

#### 黄阶斗技
- 八极崩、吸掌、吹火掌——萧炎早期使用的近身格斗斗技，以巧劲伤敌筋骨

#### 玄阶斗技
- 焰分噬浪尺——玄阶中级，萧炎以玄重尺施展，气势霸道刚猛
- 狮虎碎金吟——声波斗技，以斗气压缩声浪攻击，破防效果显著

#### 地阶斗技
- **黄泉掌**（黄泉阁镇派斗技）：阴寒霸道，掌劲带有黄泉死气，中者浑身僵硬如坠冰窟
- **风雷掌**（风雷阁镇派斗技）：迅捷凌厉，藏风雷之力于掌中，快如闪电
- **万剑诀**（万剑阁镇派斗技）：剑气化万，铺天盖地，群攻斗技天花板之一
- **帝印诀**（古族传承斗技）：五印齐出，拥有崩碎虚空之威
- **大天造化掌**（远古斗技）：一掌造化，一掌毁灭，半圣以上方可修炼

#### 天阶斗技
- **佛怒火莲**（萧炎自创）：融合多种异火于一体，形成毁灭性的火焰之莲。异火种类越多威力越大。最终版融合23种异火，一击击杀魂天帝
- **金刚琉璃身**（远古炼体斗技）：肉身坚如金刚琉璃，斗圣级攻击难以破防

### 道具-丹药

丹药分一至九品，九品又分宝丹、玄丹、金丹、帝品雏丹四个层级。品阶越高，成丹率越低，对炼药师的要求越苛刻。

#### 一品丹药
- **回气丹**：恢复少量斗气，修炼/战斗必需品，大陆最常见丹药
- **疗伤丹**：治疗轻度外伤，止血化瘀

#### 二品丹药
- **筑基灵液**（萧炎炼制的第一种二品丹药）：改善斗之气修炼者体质，加速气旋凝聚
- **复伤丹**：治疗中度内伤，修复经脉

#### 三品丹药
- **养气散**：培元固本，辅助斗者/斗师突破
- **解毒丹**：化解常见毒素

#### 四品丹药
- **三纹青灵丹**：萧炎成名丹药之一，可帮助大斗师稳固甚至突破一个小境界。品相以丹纹数量定：一纹普通、二纹优良、三纹极品
- **斗灵丹**：斗王跨入斗皇的辅助丹药，药力刚猛，需特殊体质承受

#### 五品丹药
- **龙力丹**：以龙族精血炼制的肉体强化丹药，短期大幅提升肉身强度和抗打击力
- **复魂丹**：修复灵魂损伤的珍贵丹药，灵魂受创时救命之物

#### 六品丹药
- **破宗丹**：斗皇巅峰冲击斗宗时的辅助丹药，西北地域斗皇强者愿意倾尽家产换取一枚
- **皇极丹**：进一步提升斗皇实力的丹药，药效与炼药师的熟练度直接相关

#### 七品丹药
- **阴阳玄龙丹**：药尘为萧炎炼制的七品巅峰丹药，蕴含阴阳二气与龙魂之力，可大幅强化肉身根基
- **化形丹**：助魔兽凝聚人形的关键丹药，对魔兽族群有战略意义

#### 八品丹药
- **菩提大还丹**：以菩提子为主料，可帮助斗尊冲击半圣
- **生骨融血丹**：八品顶级丹药，用于重塑肉身。药尘恢复肉身时使用

#### 九品丹药
- **九品宝丹**：药尘晋升九品炼药师的标志，一般用于斗圣的日常修炼
- **九品玄丹**：丹塔最核心的库存丹药，一枚可令低阶斗圣晋升一阶
- **九品金丹**：萧炎巅峰期炼制的顶级丹药，有起死回生之效
- **帝品雏丹**：陀舍古帝洞府中的丹药，蕴含一丝源气，是后世冲击斗帝的最大机缘，大陆仅此一枚

#### 异火核心机制

- **异火压制**：高排名异火对低排名有天然压制力，吞噬时压制力越强炼化越难
- **异火共鸣**：修炼焚诀后，萧炎体内诸异火产生共鸣，互相促进进化
- **异火融合**：多种异火可融合为新形态（如佛怒火莲），威力呈几何级增长，但难度极大
- **异火化形**：排名前3的异火拥有化形为独立个体的能力，拥有自我意志

### 物品行为

| 行为 | 说明 | 适用物品 |
| :--- | :--- | :--- |
| **采集** | 从自然界获取原材料 | 药材、魔核 |
| **炼制** | 通过炼药术将原材料转化为丹药 | 药材 → 丹药 |
| **吞噬** | 将异火纳入体内与功法融合 | 异火 |
| **交易** | 在拍卖会、丹塔等平台购买或出售 | 丹药、药材、功法、斗技 |
| **掠夺** | 击杀持有者后夺取 | 异火、丹药、功法（魂殿常用手段） |
| **装备** | 将物品绑定用于战斗增益 | 武器、防御法器 |
| **封印/解封** | 对异火或远古秘境施加/解除封印 | 异火（陨落心炎被封于天焚炼气塔）、秘境 |

### 消耗/产出关系

#### 炼药术消耗/产出链

```
① 寻找/采集药材 → 消耗时间、探索风险
② 筛选药材 + 确定丹方 → 消耗知识储备
③ 以异火驱动鼎炉 → 消耗异火能量 + 灵魂力量
④ 炼制 → 消耗斗气 + 精神力 + 时间
⑤ 产出丹药 → 丹药可用于修炼、交易、疗伤、复活
```

#### 异火生态循环

```
异火（不可再生自然资源）
    ↓ 被吞噬
持有者实力提升（焚诀进化 / 境界突破 / 获得新型斗技）
    ↓ 持有者死亡或主动传承
异火可重新被他人吞噬（异火榜单更新，持有者更迭）
    ↓ 最终
全部23种异火融合为帝炎（萧炎），异火循环终结，炎帝时代开启
```

#### 势力资源循环（以丹塔为例）

```
大陆各地药材汇集丹塔 → 丹塔炼药师炼制 → 丹药流回大陆
    ↑                                             ↓
  采集者/雇佣兵/宗门                              拍卖会/交易/势力供奉
    ↑                                             ↓
  野外探险/药圃种植                            修炼者消耗 → 突破 → 产生新药材需求
```

## 设定摘要

物品体系应以斗破原作资源为准：丹药、药材、魔核/魔晶、功法、斗技、异火、药鼎、纳戒、重尺、傀儡、古玉、残图、火晶卡/火能等。项目若沿用“法宝”分类，应改名为“器物/异宝/装备”，并标明这是游戏化归类。

## 原作事实

- 聚气散用于九段斗之气凝聚斗之气旋，价值来自突破斗者门槛。
- 魔核是魔兽体内狂暴能量晶核，原文明确斗王也不敢强行直接吸收，适合做炼药/交易材料而非通用灵石。
- 异火是炼药、战斗和焚诀成长的核心稀缺资源。
- 陀舍古帝玉属于远古传承线索，不是普通可交易道具。
- 高阶丹药如九品玄丹具有强烈风险，吞服需要辅助炼化，不能只做无副作用属性药。
