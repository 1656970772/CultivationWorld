## 势力 Faction

### 势力静态数据

#### 类型枚举

| 类型 | 说明 | 代表势力 |
| :--- | :--- | :--- |
| **远古种族** | 远古斗帝后裔，拥有专属独立空间和血脉传承 | 魂族、古族、萧族、炎族、雷族、药族、石族、灵族 |
| **中州宗门** | 中州明面上的大型宗门，实力覆盖面广 | 花宗、天冥宗 |
| **隐世谷地** | 中州隐居型势力，传承独特功法或异火 | 焚炎谷、冰河谷、音谷 |
| **四方阁** | 中州四大标志性阁派势力 | 星陨阁、风雷阁、万剑阁、黄泉阁 |
| **学院** | 培养修炼者的教育型势力 | 迦南学院 |
| **帝国** | 世俗政权，以皇室和贵族家族为核心 | 加玛帝国、出云帝国 |
| **魔兽族群** | 以魔兽为核心的种族型势力 | 太虚古龙族、天妖凰族、九幽地冥蟒族 |

### 势力实体

#### 远古八族

##### 魂族（已覆灭）

- **族长**：魂天帝（九星斗圣后期，后晋升斗帝）
- **核心战将**：虚无吞炎（九星斗圣，异火化形）、魂灭生（五星斗圣，魂殿殿主）
- **核心资源**：魂界、魂殿网络、虚无吞炎、千年积累的灵魂体
- **势力目标**：集齐陀舍古帝玉，打开古帝洞府，以血祭大陆为代价令魂天帝成帝
- **结局**：魂天帝被萧炎以佛怒火莲击杀，魂族覆灭

##### 古族

- **族长**：古元（九星斗圣巅峰）
- **核心后裔**：萧薰儿（九星斗圣，神品血脉）
- **核心资源**：古界、金帝焚天炎（异火榜第4）、天墓、完整的斗帝血脉传承
- **势力目标**：守护斗帝血脉，联合萧炎对抗魂族
- **结局**：战后以萧薰儿为核心，与萧炎联姻巩固了炎帝时代的格局

##### 萧族（已衰落后由萧炎复兴）

- **末代族长**：萧玄（九星斗圣巅峰，冲击斗帝失败陨落）
- **复兴者**：萧炎（斗帝·炎帝）
- **核心资源**：萧玄传承（天墓灵魂印记）、萧炎本人
- **势力目标**：从边陲小镇复兴为大陆最强族
- **结局**：萧炎成帝后，萧族以炎帝之名重回巅峰，萧界重建

##### 炎族

- **族长**：炎烬（八星斗圣）
- **核心资源**：八荒破灭焱、红莲业火等多种异火
- **势力目标**：在远古八族中保持中立偏古族，对抗魂族
- **消耗/产出**：多名族人参悟异火，产出强横火属性斗技，但异火天然争夺压力极大

##### 雷族

- **族长**：雷赢（八星斗圣）
- **特点**：尚武好战，雷属性斗气极为刚猛
- **势力目标**：对抗魂族，维持雷族尊严与领地
- **消耗/产出**：战斗损耗大，但战斗意志和战力长期维持高位

##### 药族

- **族长**：药丹（七星斗圣）
- **特点**：远古八族中炼药术传承最完整的一族，拥有龟灵地火、九幽风炎两种异火
- **核心活动**：药典大会，每届评选族内炼药术最强弟子
- **势力目标**：以炼药术立足，通过丹药贸易保持中立
- **结局**：药典期间遭魂族突袭，药族几乎灭族，仅少数族人幸存

##### 石族

- **特点**：以肉身为核心优势，防御力八族之首
- **状态**：故事开始时已被魂族悄然覆灭，仅极少数族人存活于外
- **势力目标**：生存（已失败）

##### 灵族

- **特点**：灵魂感知力卓绝，精神力八族最强
- **状态**：与石族同期被魂族覆灭
- **势力目标**：生存（已失败）

#### 魂殿

- **殿主**：魂灭生（五星斗圣）
- **定位**：魂族在明面上的代理人，大陆第一反派组织
- **核心手段**：捕捉灵魂体、炼制魂丹、吞噬强者灵魂
- **网络覆盖**：天罡殿、地煞殿遍布大陆各地，与众多二流势力有暗中勾结
- **资源流向**：捕捉的灵魂体 → 魂殿强者增幅 → 多余部分供给魂族核心层
- **结局**：天府联盟攻破魂殿总部，魂灭生陨落

#### 丹塔

- **塔主**：玄空子（高级半圣）
- **核心长老**：玄衣、天雷子
- **定位**：斗气大陆炼药师最高殿堂
- **核心资源**：三千焱炎火（异火榜第9）、丹会（炼药师认证与评级体系）、丹界
- **丹会大比**：大陆炼药师最高赛事，每届冠军可获收服三千焱炎火的机会
- **炼药师等级**：一至九品，九品炼药师又分宝丹宗师、玄丹宗师、金丹宗师
- **产出**：大陆八成以上的高阶丹药经丹塔之手或丹塔体系认证
- **消耗**：巨量药材、炼药师时间与精力、异火维持

#### 云岚宗

- **宗主**：云山（三星斗宗，后被魂殿控制）、云韵（斗皇，云山之徒）
- **定位**：加玛帝国第一宗门，西北地域霸主之一
- **核心事件**：纳兰嫣然退婚 → 萧炎三年之约 → 云山败亡 → 云岚宗解散
- **产出/消耗**：加玛帝国半数修炼天才，消耗帝国供奉与药材

#### 迦南学院

- **院长**：邙天尺（雷族族人，高级半圣）
- **内院大长老**：苏千（斗宗）
- **定位**：大陆明面上最著名的学府，黑角域中唯一秩序之地
- **核心设施**：天焚炼气塔（封印陨落心炎，异火榜第19）
- **核心功能**：以陨落心炎的淬炼之力帮学员加速修炼，斗气修炼速度可达外界数倍
- **产出**：持续为大陆输送斗王至斗宗级别的精英修炼者
- **消耗**：学员修炼消耗的斗气和药材；陨落心炎长期被封印状态消耗学院高层的压制力量

#### 炎盟

- **盟主**：萧炎
- **核心成员**：美杜莎（彩鳞）、加玛帝国皇室、三大家族残余力量
- **定位**：萧炎整合加玛帝国及周边资源后建立的势力
- **功能**：统一西北地域力量对抗外部威胁，为萧炎后续中州争霸提供后方基地
- **产出**：稳定的税收、情报网络与后备兵力
- **消耗**：管理成本、驻军、炼丹原材料

#### 天府联盟

- **核心成员**：星陨阁（萧炎）、丹塔、焚炎谷、花宗、古族（后加入）
- **定位**：联合对抗魂殿与魂族的正道联盟
- **核心战役**：魂殿总部歼灭战、双帝山决战
- **产出**：整合正道势力资源，最终以联盟形式击溃魂族
- **消耗**：战损、丹药、联盟内部分歧的调和成本

#### 魔兽三大族

##### 太虚古龙族

- **龙皇**：烛坤（九星斗圣巅峰，被困古帝洞府数千年）
- **少主**：紫妍（龙凰血脉，后晋升斗圣，代龙皇）
- **特征**：虚空穿梭天赋，肉身魔兽中最强
- **消耗/产出**：依赖东龙岛的空间裂缝汲取虚空能量，产出顶层魔兽战力

##### 天妖凰族

- **特征**：飞行速度魔兽第一，高傲排他
- **与太虚古龙族**：世仇，常年边境冲突
- **产出**：天妖凰精血是炼制飞行类高阶丹药和锻造飞行斗技的顶级材料

##### 九幽地冥蟒族

- **领地**：九幽黄泉，阴寒属性
- **特征**：数量庞大，单体战力弱于前两者，但以数量弥补
- **产出**：蛇类魔核、九幽寒泉等阴寒类资源

### 势力需求

#### 需求分类与优先级计算逻辑

六大核心需求，优先级从高到低：

1. **生存**（优先级权重：1.0）：势力面临覆灭威胁时触发，如药族被魂族偷袭时全部资源转向自保
2. **对抗**（权重：0.9）：有明确的敌对势力且已进入战争状态，如天府联盟 vs 魂族
3. **成帝**（权重：0.85）：远古种族高层专用，源气枯竭时代优先级降低，直到获取成帝契机后飙升为绝对第一
4. **扩张**（权重：0.7）：当前资源不足以支撑发展时触发，如萧炎建立炎盟整合西北
5. **资源**（权重：0.6）：日常运营需求，保持势力运转的最低消耗
6. **复兴**（权重：0.5）：衰落势力的长期目标，优先级低于生存和资源

**计算公式**：`需求优先级 = 需求权重 × 紧迫程度系数 × 可实现系数`

### 势力行为

| 行为 | 适用势力类型 | 示例 |
| :--- | :--- | :--- |
| **征伐** | 帝国、宗门、远古种族 | 魂族覆灭灵族、石族；炎盟统一加玛帝国周边 |
| **结盟** | 全部类型 | 天府联盟的成立 |
| **培养弟子** | 学院、宗门、远古种族 | 迦南学院、古族神品血脉培养 |
| **炼丹** | 炼药师组织、势力附属炼药师 | 丹塔、药族药典 |
| **吞噬** | 反派组织 | 魂殿捕猎灵魂；魂族吞噬他族血脉 |
| **掠夺灵魂** | 魂殿、魂族 | 魂殿关押大陆强者的灵魂用于炼制魂丹 |
| **探索秘境** | 全部类型（有实力门槛） | 古族天墓试炼；多方联合进入妖火空间 |
| **守护传承** | 远古种族、隐世谷地 | 焚炎谷守护九龙雷罡火 |
| **贸易** | 丹塔、帝国 | 丹药成品 vs 药材原材料的跨地域流动 |
| **血脉维护** | 远古种族 | 古族严格控制与外族通婚，以维持血脉纯度 |

### 消耗/产出关系

| 势力 | 消耗 | 产出 | 净收益评估 |
| :--- | :--- | :--- | :--- |
| **魂族** | 大量生灵灵魂、时间（千年布局） | 斗圣级战力、魂天帝成帝 | 短期负收益，长期以成帝一局定乾坤 |
| **古族** | 结界维护斗气、血脉稀释风险 | 神品血脉天才、大陆最高声望 | 净正收益，但增长缓慢 |
| **丹塔** | 巨量药材、异火维持 | 高阶丹药（大陆80%+）、炼药师认证 | 净正收益 |
| **迦南学院** | 陨落心炎压制力量、药材 | 精英修炼者（持续输出） | 净正收益（长期投资） |
| **魂殿** | 灵魂狩猎人力、分殿维护 | 灵魂体、魂丹、情报网络 | 短期正收益（对魂族而言） |
| **炎盟** | 帝国税收、兵员、物资 | 统一政权、安全屏障、情报网络 | 净正收益（整合效应） |
