## 设定摘要

人物关系的核心不是泛化“道侣/师门/仙缘”，而是尊严冲突、师徒互利、家族婚约、隐藏血脉背景、异火共同风险、宗门/族群责任。三年之约把萧炎的个人尊严、纳兰嫣然的宗门身份和萧家/纳兰家的家族面子绑定为主线冲突。

## 原作事实

- 纳兰嫣然退婚带来公开羞辱，萧炎拒绝用聚气散或云岚宗修习资格交换婚约解除。
- 药老与萧炎关系以“收徒、传炼药、传功法”为核心，同时药老自身也需要萧炎未来帮助其脱离灵魂状态。
- 萧薰儿在萧炎低谷时仍保持支持，并隐藏更高层级势力背景。
- 美杜莎/彩鳞线与青莲地心火、蛇人族进化和七彩吞天蟒相关，关系变化应跟事件推进绑定。

## 游戏化抽象（我的判断）

- 关系系统可用 `humiliation_debt`、`mentor_contract`、`hidden_background`、`shared_fire_risk`、`clan_pressure` 表示，而不是只做好感度。
- 三年之约适合做“公开承诺 + 倒计时 + 修炼目标 + 势力声望”复合事件。
- 药老关系可加入 `soul_power_cost` 与 `revival_goal`，让导师帮助不是无代价外挂。
