## 设定摘要

斗破口径中，“法宝”应拆成武器、斗技、功法、药鼎、纳戒、傀儡、古玉、残图等；“妖兽”应以原作“魔兽”称呼为主；丹药和魔核构成低中阶资源经济，异火构成高阶稀缺成长资源。

## 原作事实

- 炼药师和丹药在社会中地位极高，聚气散能帮助成为斗者。
- 魔兽产出魔核/魔晶，但魔核能量狂暴，不宜直接吞吸。
- 魔兽山脉是加玛帝国周边重要危险资源区。
- 异火不仅是战斗资源，也是焚诀成长和炼药效率的核心。

## 游戏化抽象（我的判断）

- 原“法宝”表应拆为 `equipment`、`method`、`technique`、`artifact_token`、`puppet`。
- 魔兽资源循环可配置 `hunt -> magic_core/material -> alchemy/auction`。
- 丹药设计必须加入副作用和适用境界，避免无成本堆数值。
