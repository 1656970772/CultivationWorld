### NPC 实体

#### 萧炎

- **身份**：主角，最后一位斗帝（炎帝），萧族复兴者，天府联盟实际领袖
- **修炼路径**：
  - 乌坦城时期：斗之气三段（天才陨落）→ 三年蛰伏后以焚诀重新崛起
  - 加玛帝国时期：斗者 → 斗师 → 大斗师 → 斗灵 → 斗王 → 斗皇 → 斗宗（三年之约前后）
  - 黑角域 & 中州前期：斗宗 → 斗尊（吞噬陨落心炎后飞跃）
  - 中州争霸期：斗尊 → 斗尊巅峰（九转）→ 半圣 → 斗圣（天墓中萧玄传承）
  - 最终决战：斗圣 → 九星斗圣巅峰 → 斗帝（以23种异火融合成帝）
- **核心能力**：焚诀、佛怒火莲（自创斗技）、天火三玄变、炼药术（九品金丹宗师）
- **异火持有**：青莲地心火(14)、陨落心炎(19)、骨灵冷火(11)、三千焱炎火(9)、净莲妖火(3)、金帝焚天炎(4)、海心焰(15)、八荒破灭焱(12)等，最终融合全部23种异火为帝炎
- **性格特征**：坚韧不拔、重情重义、杀伐果断但不滥杀

#### 药尘（药老）

- **身份**：萧炎之师，药尊者，大陆第一炼药师，星陨阁前阁主
- **修为**：巅峰期高级半圣 → 魂殿偷袭陨落 → 灵魂体寄居戒指 → 萧炎帮助恢复肉身后达一星斗圣
- **炼药术**：九品宝丹宗师，大陆炼药术天花板
- **异火**：骨灵冷火（异火榜第11，后传给萧炎）
- **核心贡献**：传授焚诀、引导萧炎走上异火吞噬之路、解析古帝洞府秘密

#### 萧薰儿（古薰儿）

- **身份**：古族族长古元之女，神品血脉，萧炎青梅竹马与妻子
- **修为**：故事开始时斗者 → 古族修炼数年后九星斗圣
- **异火**：金帝焚天炎（异火榜第4，古族传承异火）
- **角色定位**：萧炎的最强后盾与情感支柱，代表古族与炎帝的时代联姻

#### 美杜莎（彩鳞）

- **身份**：蛇人族女王 → 萧炎之妻 → 炎盟副盟主
- **修为**：斗皇巅峰（初见）→ 吞噬青莲地心火进化后斗宗巅峰 → 最终斗圣
- **特殊**：以七彩吞天蟒血脉为基础，借助异火淬炼进化为人形，肉身极为强悍
- **角色定位**：萧炎在西北时期的守护者和事业伙伴，性格冷傲刚烈

#### 小医仙

- **身份**：厄难毒体拥有者，毒宗宗主
- **修为**：毒功大成后斗尊巅峰
- **特殊体质**：厄难毒体——天生凝聚天地间毒素能量，可无限吞噬毒物增强自身。此体质危险至极，需要在修炼同时服用特定丹药压制毒性反噬，否则会毒发身亡。
- **角色定位**：萧炎在黑角域结识的挚友，毒宗创始人，代表"毁灭性力量也可以守护"的主题

#### 魂天帝

- **身份**：魂族族长，最终反派
- **修为**：九星斗圣后期 → 血祭大陆后突破斗帝
- **布局**：千年时间，暗中覆灭灵族、石族、药族，与虚无吞炎结盟，派遣魂殿捕猎灵魂
- **性格特征**：极度理性、冷血、以天下为棋盘，不择手段追求成帝。不嗜杀但认为杀戮是实现目标的必要手段
- **结局**：被萧炎以佛怒火莲（融合23种异火）摧毁肉身与灵魂，形神俱灭

#### 陀舍古帝

- **身份**：最后一位远古斗帝，异火之祖
- **本体**：帝炎（异火榜第1），由全部23种异火融合而成
- **核心遗产**：陀舍古帝洞府（异火广场、帝品雏丹）、陀舍古帝玉（8块碎片）
- **历史地位**：万年前最后一个成帝者，以异火入道的绝世天才，消失后源气开始枯竭

### NPC 状态

NPC 状态模型：

| 状态维度 | 取值范围 | 说明 |
| :--- | :--- | :--- |
| **存活** | 存活 / 重伤濒死 / 死亡 | 基础生命状态 |
| **年龄** | 实际年龄 + 外表年龄 | 高阶修炼者外表年龄远小于实际 |
| **伤势** | 无 / 轻伤（战斗可继续）/ 重伤（需休养）/ 濒死（无外力必死） | 通过丹药或疗伤斗技可以改变 |
| **忠诚** | 绝对忠诚 / 利益绑定 / 摇摆 / 敌对 | 对所属势力的态度 |
| **忙碌** | 空闲 / 修炼中 / 炼丹中 / 战斗 / 重伤休养 | 决定是否可响应事件 |
| **心境** | 坚定 / 动摇 / 愤怒 / 绝望 / 顿悟 | 影响修炼效率和战斗决策 |
| **修炼进度** | 当前境界 + 当前星级 + 距离突破的百分比 | 修炼的根本状态 |
| **灵魂状态** | 完整 / 受创 / 离体 / 被封印 / 被吞噬 | 魂族相关事件的核心触发条件 |

### NPC 需求

#### GOAP（目标导向行动规划）行为链

以萧炎为例的需求优先级与行为链：

| 需求优先级 | 需求 | GOAP 行为链 |
| :--- | :--- | :--- |
| 1 | 守护亲友 | 检测亲友是否受威胁 → 受威胁则中断当前行动 → 战斗/救援 → 确认安全后恢复 |
| 2 | 提升实力 | 判断当前瓶颈 → 有无异火可吞噬 → 有则寻找异火 → 无则选择丹药/秘境/战斗突破 |
| 3 | 获取异火 | 确定异火位置 → 准备血莲丹等丹药 → 炼化异火 → 融合入焚诀 |
| 4 | 复仇 | 评估敌我实力 → 准备底牌 → 寻找时机 → 正面挑战 |
| 5 | 建立势力 | 整合资源 → 招揽人才 → 确立制度 → 扩张巩固 |
| 6 | 炼药 | 确定丹方 → 收集药材 → 筛选鼎炉与异火 → 炼制 → 丹药分配 |

### NPC 行为

| 行为 | 触发条件 | 资源消耗 | 产出 |
| :--- | :--- | :--- | :--- |
| **修炼** | 空闲 + 有足够斗气/资源 | 时间、药材、斗气 | 境界提升 |
| **炼丹** | 有丹方 + 有药材 + 有异火/鼎炉 | 药材、灵魂力量 | 丹药 |
| **吞噬异火** | 发现可吞噬异火 + 实力足够 | 异火（不可再生）、肉身承受 | 焚诀进化、境界突破 |
| **征战** | 势力冲突 / 个人恩怨 | 斗气、伤势风险 | 势力扩张/仇恨了结 |
| **建立势力** | 资源积累到一定程度 | 资源、精力、时间 | 势力实体 |
| **探索秘境** | 秘境开启 + 实力达标 | 探索风险、时间 | 传承、异火、丹药等稀有资源 |
| **传授弟子** | 有可造之材 + 自身有余力 | 时间、精力 | 弟子成长 |
| **结盟/背叛** | 利益变化 | 声望、信任 | 势力关系重组 |
