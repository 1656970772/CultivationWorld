## 设定摘要

《斗破苍穹》的境界提升核心是斗之气到斗气、斗者星级和斗气大境界的递进；功法按天/地/玄/黄而非灵根/筑基/金丹/元婴体系。焚诀是特例功法：初始低阶但可通过吞噬异火进化，因此它是成长曲线和异火收集系统的关键。

## 原作事实

- 斗气大陆功法分四阶十二级，功法高低会影响同境界战斗优势。
- 斗者前修炼斗之气，成为斗者后才吸收真正斗气。
- 聚气散的原作作用是帮助九段斗之气凝聚斗之气旋，不是通用“筑基丹”。
- 焚诀的风险来自吞噬异火，本质是高风险高潜力功法。

## 游戏化抽象（我的判断）

- 境界数据可用 `realm` + `star` + `energyState` 表示，斗者前 `energyState=dou_zhi_qi`，斗者后 `energyState=douqi`。
- 功法可配置 `rank=tian/di/xuan/huang`、`subRank=low/mid/high`、`element`、`growthMode`。
- 焚诀应配置为 `growthMode=devour_fire`，每次进化触发高风险检定、异火占用和身体/灵魂损耗。
