## 物品 Item

### 物品静态数据

物品系统是遮天世界观中承载战力、资源和经济体系的核心。所有物品按来源和用途分为资源类、帝兵类、极道武器类、功法经文类、秘术类、丹药类、阵法类和特殊物品类。

---

### 物品分类体系

#### 一、资源类

##### 源石 / 神源

- **源石**：天地灵气凝结而成的晶石，是修炼的基础资源。普通修士通过炼化源石加速修炼。分为下品、中品、上品、极品。
- **神源**：源石中的极品，灵气纯度和密度远超普通源石。可封存上古生物和神物，是源天师的主要工作对象。大帝古皇常用神源封印自己或重要物品。
- **异种源**：具有属性的特殊源石（如火源、冰源），稀有度更高。
- **产出方式**：开采矿区（太初古矿最著名）、交易、掠夺。
- **消耗方式**：修炼吸收、布阵消耗、炼制帝兵材料、交易货币。

##### 灵药 / 神药 / 不死药

| 类型 | 药龄 | 作用 | 稀有度 | 代表 |
|------|------|------|--------|------|
| **灵药** | 百年 ~ 数千年 | 疗伤、增进修为 | 常见 | — |
| **药王** | 数万年 | 延寿数百年、生死人肉白骨 | 稀少 | — |
| **神药** | 十万年+ | 延寿数千年、治道伤 | 极其稀有 | 蟠桃等 |
| **不死药** | 天地奇珍 | 让大帝再活一世，活出第二世 | 仅有数株 | 九妙不死药、蟠桃不死药、菩提树、真龙不死药、玄武不死药、麒麟不死药、白虎不死药、人参果树 |

- **产出方式**：天地孕育（多藏于禁区）、大帝培育、上古遗迹。
- **消耗方式**：直接服食（疗伤/延寿）、炼丹原料、大帝活出第二世。
- **不死药循环**：大帝服食不死药活出第二世后，不死药会"遁走"回归天地，进入循环再生。

##### 混沌石 / 仙金

- **混沌石**：蕴含混沌气的神物，可用于炼制帝兵。
- **仙金**：大帝炼兵的核心材料，如龙纹黑金、仙泪绿金、凰血赤金、神痕紫金、永恒蓝金、道劫黄金、羽化青金。
- **产出方式**：上古遗迹、混沌中采集、禁区寻找。
- **消耗方式**：炼制帝兵。

#### 三、道具 — 极道武器

极道武器仅次于帝兵，多为古皇或大教炼制，战力远超普通法宝，但无法与完整帝兵抗衡。

- **古皇兵**：太古皇者炼制的兵器，如紫龙枪（万龙皇）、麒麟杖（麒麟古皇）、黄金锏（黄金古皇）。
- **皇道极兵**：中州四大皇朝等势力的镇教之宝。
- **产出方式**：古皇/准帝以上炼制。
- **消耗方式**：战斗消耗（需大量源石催动）、传承。

- **消耗方式**：修炼参悟、刻印传承。

#### 五、道具 — 秘术 / 九秘

九秘是九位天尊开创的九种无上秘术，各自代表一条大道的极致，合一可达仙道领域。

| 九秘 | 开创者 | 作用 | 获得方式 |
|------|--------|------|----------|
| **者字秘** | 长生天尊 | 疗伤复生，修炼到极致可滴血重生 | 叶凡机缘所得 |
| **行字秘** | 逍遥天尊 | 速度极致，快过时间 | 叶凡机缘所得 |
| **斗字秘** | — | 演化万法，可模拟任何攻击 | 叶凡从姜太虚处获得 |
| **兵字秘** | — | 操控天下兵器 | 叶凡机缘所得 |
| **数字秘** | — | 推演天机、预知未来 | — |
| **组字秘** | — | 阵纹之道极致 | — |
| **前字秘** | — | 修炼元神 | — |
| **皆字秘** | — | 十倍战力暴发 | 叶凡机缘所得 |
| **临字秘** | — | 防御无双 | — |

- **产出方式**：上古秘境、天尊传承、天劫中领悟。
- **消耗方式**：战斗中释放（消耗法力/神力）、参悟融合。

##### 其他重要秘术

- **斗战圣法**：姜太虚传授叶凡的战斗秘术，可演化万般攻伐。
- **一气化三清**：老子开创，可化出三道分身，战力翻倍。
- **六道轮回拳**：无上拳法，叶凡标志性攻击手段之一。
- **天帝拳**：叶凡自创拳法，证道后成天帝专属。

### 物品实体

物品实体在沙盒中作为可交互对象，具有以下属性：

- **物品 ID**：唯一标识
- **物品名称**：如"万物母气鼎"
- **物品类型**：资源 / 帝兵 / 极道武器 / 功法 / 秘术 / 丹药 / 阵法 / 特殊
- **品阶**：凡品 / 灵品 / 圣品 / 帝品 / 仙品
- **持有者**：当前归属的势力或 NPC
- **位置**：当前所处的物理位置

### 物品需求

| 需求类型 | 说明 |
|----------|------|
| **帝兵复苏** | 需要海量神源或大帝血脉催动，完全复苏可匹敌大帝一击 |
| **功法匹配** | 大帝古经需对应体质或血脉才能发挥最大威力 |

| **阵纹维持** | 需持续消耗源石/灵气维持运行 |
| **帝兵炼制** | 需仙金/混沌石等顶级材料 + 大帝道则烙印 |

| **复苏** | 帝兵从沉睡状态逐渐恢复力量，消耗大量能量 |

### 消耗/产出关系（物品层面）

```
神源/源石矿脉
    → 开采（消耗矿脉、矿工）→ 产出源石/神源
        → 修士修炼消耗 → 产出境界提升
        → 帝兵复苏消耗 → 产出大帝级战力
        → 布阵消耗 → 产出防御/阵法战力
        → 交易消耗 → 产出经济流通

仙金/混沌石
    → 秘境/禁区采集 → 产出炼器材料
        → 大帝炼制 → 产出帝兵
            → 战斗消耗（磨损、能量）→ 产出战力
            → 传承 → 产出势力威慑力

灵药/神药/不死药（天地自然产出）
    → 采集/移植 → 炼丹 → 产出丹药
        → 修士消耗 → 产出疗伤/突破/延寿
    → 大帝直接服食 → 活出第二世 → 产出额外大帝寿元
    → 不死药遁走 → 回归天地 → 进入下一轮再生循环

大帝古经
    → 大帝毕生感悟自然产出
        → 修士参悟 → 战力/境界提升
        → 天劫投影 → 渡劫者获得 → 传承不灭
        → 失传 → 文明损失

九秘
    → 天尊开创 → 分散于各秘境
        → 修士机缘获得 → 独一传承或共享
        → 战斗中消耗 → 产出战术胜利/保命

帝兵
    → 大帝炼制（一次性投入大帝级精力+顶级材料）
        → 守护势力（产出百年万年安全）
        → 战损消耗 → 消耗大量神源修复或永久损毁
        → 被夺取 → 一方势力增强/一方失去根基
```
