## 世界 World

### 世界实体（宇宙星域结构）

遮天的世界观构建在一个极其宏大的宇宙体系之上，包含多个星域、生命星球、古路和终极仙界。

#### 北斗星域 / 葬帝星

故事的主舞台，因古往今来多位大帝在此葬身而得名"葬帝星"。分为五大域：

- **东荒**：故事的核心区域，修炼文明最为繁盛。下分南域（摇光圣地所在、燕国六大洞天）、北域（太初古矿所在的荒凉矿区、源矿丰富）、中部等地。荒古禁地即位于东荒境内。
- **南岭**：南方蛮荒之地，多原始森林与妖兽，修炼文明较为分散，有蛮族部落传承。
- **西漠**：佛教净土，瑶池圣地所在，主张和平。佛门修士众多，有须弥山等佛教圣地。
- **北原**：北方冰雪荒原，黄金家族、王家等世家盘踞，民风彪悍。
- **中州**：人杰地灵之地，四大不朽皇朝（大夏、九黎、古华、神州）并立，诸子百家争鸣，奇士府坐落于此。

#### 紫微星域

人族祖地之一，划分多洲大洋，宗门林立。有太阳神教、太阴神教、人王殿、天神宫等古老传承。八景宫（老子传承）与紫微教也在该星域。妖族势力强大，如金乌族、天妖宫等。

#### 地球

现代科技文明星球，灵气枯竭，修真文明隐匿于幕后。因九龙拉棺事件与北斗产生联系，是叶凡的故乡。地球隐藏着昆仑仙脉、函谷关等上古遗迹，老子、释迦牟尼等先贤曾在此留下传承。

#### 永恒国度

科技与修行结合的高等文明星域，掌握机甲技术、进化液等独特体系。曾以"青金圆盘"镇压诸天，有梵族等掌握机甲与圣箭的势力。

#### 试炼古路 / 星空古路

连接各星域的星空之路，又称"人族古路"、"妖族古路"等，遍布古老遗迹、险地秘境和异族威胁。是各族天骄的终极试炼场，从各星域出发，沿途经过不同关隘（如帝关），最终汇聚到终极试炼之地。

#### 飞仙星

成仙路的重要节点之一，多位大帝和至尊在此争渡。在叶凡时代，成仙路在飞仙星开启，引发万族争渡的浩大场面。

#### 仙域

修士的终极目标，完整的大道世界，法则完整、灵气充沛。历代大帝与至强者都渴望进入仙域以求长生。仙域与凡间世界有壁垒阻隔，仅在成仙路开启时才能进入。最终由叶凡、狠人大帝、无始大帝等红尘仙联手打通仙域之门。

#### 生命禁区

禁区是古代大帝/至尊自斩一刀后沉睡之地，全区共七个主要禁区：

| 禁区 | 位置 | 主宰 | 特征 |
|------|------|------|------|
| **荒古禁地** | 东荒 | 狠人大帝布局 | 叶凡的起点，内有九妙不死药、青铜仙殿等 |
| **太初古矿** | 东荒北域 | 多位太古至尊 | 最神秘的禁区，源矿丰富，至尊蛰伏之地 |
| **不死山** | — | 石皇 | 掌控不死神药的禁区，后被叶凡覆灭 |
| **仙陵** | — | 长生天尊 | 长生天尊盘踞，与叶凡多次交锋 |
| **神墟** | 东荒西部 | — | 青帝坟场所在地，内有南天门、神泉等至宝 |
| **轮回海** | — | 轮回之主 | 与轮回、时间长河力量相关的禁区 |
| **上苍** | — | — | 最为神秘的禁区之一，与仙域有联系 |

### 世界状态

#### 当前时代

故事主线跨越从叶凡初入修行（后荒古时代末期）到其证道成帝（天帝时代），再到率众打入仙域的数万年时间跨度。后荒古时代灵气逐渐稀薄，天地法则压制变大，修行越发艰难，证道成帝几乎不可能——这正是叶凡崛起的背景。他打破荒古圣体不能修炼的诅咒，开创了一个全新的黄金大世。

#### 灵气强度

- **后荒古时代**：灵气日渐稀薄，天地压制严重，证道难度极大。
- **黄金大世**：叶凡打破格局后，天地规则松动，各族天骄井喷式涌现，灵气有所回升。
- **各星域差异**：北斗天地灵气最为浓郁（葬帝星特性），地球灵气枯竭几近于无，紫微星域中等偏强，永恒国度依赖科技辅助。

#### 黑暗动乱周期

黑暗动乱是禁区至尊为延续寿命而发动的周期性收割。当禁区至尊寿元将尽时，便会苏醒并发动黑暗动乱，收割天下生灵的生命精华来续命。周期不固定，但通常每隔数万年至十余万年发生一次。每一次黑暗动乱都导致亿万万生灵殒命，是人族和万族的最大噩梦。

#### 成仙路状态

成仙路是连接凡间与仙域的通道，在特定时间节点开启。开启规律极为复杂，与天地格局、星位运转、大帝证道等因素相关。成仙路开启时，禁区至尊齐出、大帝级战力爆发，万族争渡，是全书最核心的大事件之一。成仙路最终由叶凡、狠人、无始联手打通。

#### 禁区活跃度

- **荒古禁地**：相对活跃，内有狠人大帝持续布局。
- **太初古矿**：高度活跃，是禁区至尊主要蛰伏地。
- **其他禁区**：在黑暗动乱和成仙路事件期间活跃度骤增，平时处于沉寂状态。

### 世界行为/规则

#### 成仙路开启规则

1. 成仙路在特定星域、特定时间节点开启，以星位运转和天地变化为信号。
2. 开启地点不固定，可能在北斗、飞仙星、帝关等地。
3. 进入成仙路需要至少准帝级战力，大帝方可站稳脚跟。
4. 成仙路危险之极，通道内蕴含无尽杀机，能活着进入仙域者凤毛麟角。
5. 红尘仙（叶凡、狠人、无始）方可真正打通并稳固成仙路。

#### 黑暗动乱规则

1. 禁区至尊自斩后，以沉睡延缓衰老。寿元将尽时苏醒，需吸收大量生命精华续命。
2. 黑暗动乱通常由多个禁区同时发动，规模浩大，波及整个北斗乃至多个星域。
3. 人族大帝往往以命相搏，以身镇守，才能暂时平息动乱。虚空大帝即因此战死。
4. 平定黑暗动乱需具有大帝级战力与相应帝兵。

#### 禁区规则

1. 禁区由自斩的大帝/古皇占据。自斩者跌落大帝境界，但可恢复短暂的大帝级战力。
2. 禁区之内自成空间，法则与外界不同，至尊在此能最大限度延缓寿元流逝。
3. 禁区与外界有结界隔绝，非特殊时期不轻易开启。
4. 禁区之间既有合作（发动黑暗动乱时联手），也有竞争（争夺不死药、成仙路名额）。

#### 大帝证道规则

1. 一个时代通常只允许一位大帝证道（天道压制）。以自身道则覆盖天地，后来者难以再证。
2. 证道需将五大秘境（轮海→道宫→四极→化龙→仙台）修炼圆满，凝聚道果。
3. 证道过程需渡过天劫（大帝劫），以自身道则压服天地。
4. 特殊体质（如荒古圣体、先天圣体道胎、混沌体等）证道难度更大，但成就也更强。
5. 大帝寿元约两万年左右，可通过不死药再活一世，手段尽出的极限约为三世。

#### 源术规则

1. 源术是探索源矿、寻找神源的核心技艺，分为天、地、人三阶，此外还有源天师一脉。
2. 源天师可窥探源中奥秘，甚至能从源中封存的上古生物和神物中获取信息。
3. 源术可用于布源阵、斗法（源术对决），高阶源术师战力不弱于同阶修士。
4. 源天师晚年会遭遇"源天师诅咒"（不详之祸），晚年不详、化为红毛怪物。
5. 叶凡修成源天师，但以天帝手段压制并化解了诅咒。

### 世界需求（制造冲突、推动时代演化、维持平衡、避免死局）

| 需求维度 | 具体内容 |
|----------|----------|
| **制造冲突** | 禁区至尊寿元危机 → 黑暗动乱；成仙路开启 → 万族争渡；太古万族苏醒 → 人妖古三族矛盾；体质歧视（荒古圣体不能修炼）→ 主角成长线冲突 |
| **推动时代演化** | 后荒古时代资源枯竭 → 修士寻求突破（新功法、新路线）；黄金大世 → 天骄井喷、格局重塑；成仙路打通 → 新时代开启；天庭建立 → 万族秩序重组 |
| **维持平衡** | 大帝镇压天地 → 一个时代仅一帝（防止绝对混乱）；禁区与外界有结界 → 限制至尊随意出没；天道压制 → 防止过多人证道导致天地崩溃 |
| **避免死局** | 不死药的存在 → 给最强者续命机会；成仙的希望 → 激励修士不断攀登；禁区至尊收割并非完全灭绝 → 保留文明火种 |

### 消耗/产出关系（世界层面）

```
源矿开采（消耗矿区资源）
    → 产出源石/神源
        → 修士修炼消耗（提升境界）
        → 布阵消耗（防御/攻伐/渡劫）
        → 交易流通（经济体系基础）

天地灵气（自然产出，受时代影响波动）
    → 修士吞吐炼化 → 提升修为（产出战力）
    → 灵药/神药生长 → 炼丹/疗伤/延寿

禁区至尊生命收割（消耗亿万万生灵）
    → 产出至尊续命寿命
    → 引动人族大帝出手（大帝寿元消耗）
        → 平息动乱（安全产出）
        → 大帝陨落（最坏产出，进入新一轮循环）

成仙路开启（消耗大帝/至尊级战力争渡）
    → 成功入仙域（仙道产出）
    → 失败陨落（战力损失）
    → 通道打穿（为后来者留下成仙希望）

不死药（天地奇珍自然产出，极其稀有）
    → 大帝续命（消耗，再活一世）
    → 多次使用后遁走/回归天地 → 循环再生
```
