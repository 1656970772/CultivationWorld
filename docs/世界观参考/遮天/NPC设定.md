## NPC

### NPC 静态数据（角色类型）

| 角色类型 | 特征描述 |
|----------|----------|
| **圣体** | 上古最强战体，荒古后遭天地诅咒不可修炼。叶凡即荒古圣体，打破诅咒后战力无双 |
| **大帝** | 一个时代的最强者，证道压服天地，寿元约两万年，可借不死药活出多世 |
| **至尊** | 自斩一刀的大帝/古皇，蛰伏于禁区，以沉睡和收割生命延缓寿命 |
| **禁区之主** | 至尊中的最强几位，掌控禁区运转，是黑暗动乱的真正发动者 |
| **天骄** | 各势力的核心传人，拥有大帝之姿，是黄金大世的核心战力 |
| **古皇** | 太古时代证道的皇者，与大帝同级别，但时代更久远 |
| **神王** | 圣人之上的王者，可称神王，战力远超同阶。姜太虚即神王 |
| **杀手** | 地狱/人世间/古天庭培养的职业杀人者，以杀证道 |
| **红尘仙** | 超越大帝的存在，在红尘中磨砺成仙，战力远超普通大帝。叶凡、狠人、无始 |
| **源天师** | 源术巅峰，可探寻源中奥秘，但晚年有不详诅咒 |

### NPC 实体

#### 主角阵营

**叶凡（叶天帝）**
- 身份：荒古圣体、天庭之主、叶天帝、红尘仙
- 来历：地球普通大学生，被九龙拉棺带入北斗星域
- 成就：打破荒古圣体诅咒，创立天庭，平定所有黑暗动乱，击杀多位禁区至尊，最终以红尘仙之姿带领天庭举教飞升仙域
- 核心能力：万物母气鼎（帝兵）、源天师传承、九秘（齐集）、《道经》《西皇经》《恒宇经》等多部古经

**庞博**
- 身份：叶凡挚友、妖帝传人
- 特点：与叶凡一同被九龙拉棺带入北斗，后获得青帝传承，成为妖族重要战力
- 能力：妖帝古经、妖族圣术后加入天庭

**姬紫月**
- 身份：姬家小姐、叶凡妻子、元灵体、后天帝后
- 特点：天真烂漫，多次在叶凡危机时施以援手
- 能力：元灵体（可与天地共鸣加速修炼），后随叶凡成帝

**黑皇**
- 身份：无始大帝的宠物（一只大黑狗）
- 特点：贪财好宝、极度护短、精通阵纹。与叶凡亦师亦友亦敌（经常坑叶凡的源石）
- 能力：无始大帝阵纹传承、布置杀阵

**段德（曹雨生 / 渡劫天尊）**
- 身份：轮回转世者、渡劫天尊、叶凡损友
- 特点：无良道士，以挖坟盗墓为乐，口口声声"无量天尊，贫道段德"
- 能力：渡劫天尊的完整道果（觉醒后），轮回印，最终觉醒为顶级战力

**姜太虚**
- 身份：姜家神王、叶凡的引路人和恩人
- 特点：被困紫山中四千年不死，一出世便展现绝世战力
- 能力：《恒宇经》、斗战圣法、神王体

#### 古代大帝与至尊

**狠人大帝**
- 身份：史上最强大帝之一、红尘仙
- 特点：以凡体逆天证道的绝代女帝。其兄被杀后，她发誓逆天而行。创《吞天魔功》和《不灭天功》，吞噬万族本源以养己身
- 成就：红尘仙，与叶凡、无始联手打通仙域
- 标志：青铜面具、鬼脸印记、荒古禁地是她为兄长设下的局
- 名言："不为成仙，只为在红尘中等你归来"

**无始大帝**
- 身份：荒古最后一位人族大帝、先天圣体道胎
- 特点：仙路尽头谁为峰，一见无始道成空。以无始钟镇压不死山
- 帝兵：无始钟
- 结局：冲击仙域未果，与叶凡、狠人汇合后共同打通仙域

**虚空大帝**
- 身份：姬家始祖、为人族战死的大帝
- 特点：一生与禁区至尊血战，以命换命，最终战死虚空。是全书中最令人敬仰的大帝之一
- 帝兵：虚空镜，传承《虚空经》
- 评价："虚空一生，不弱于人"

**恒宇大帝**
- 身份：姜家始祖
- 特点：开创姜家，帝兵恒宇炉焚天煮海
- 帝兵：恒宇炉，传承《恒宇经》

**青帝**
- 身份：万古青天一株莲，妖帝
- 特点：本体为混沌青莲，荒古后唯一证道的妖族大帝
- 帝兵：混沌青莲
- 传承：颜如玉获其部分传承，庞博得妖帝古经

**西皇母**
- 身份：瑶池圣地开创者、无始大帝的母亲
- 特点：女帝之姿，创《西皇经》
- 帝兵：仙泪绿金塔

### NPC 需求

#### NP 核心需求层级

| 层级 | 需求 | NPC 示例 |
|------|------|----------|
| **生存** | 活下去、躲避追杀、寻求庇护 | 初入北斗的叶凡、被追杀的天璇老疯子 |
| **修炼** | 获取功法、资源、名师指点，提升境界 | 叶凡拜入灵墟洞天时期 |
| **证道成帝** | 打破天地压制，证得大帝道果 | 准帝时期的叶凡、盖九幽 |
| **平定黑暗** | 击杀禁区至尊，保护族人 | 虚空大帝、叶凡、无始大帝 |
| **成仙** | 进入仙域，获得永生 | 所有大帝和至尊的终极追求 |
| **复仇** | 为亲友报仇，清算旧敌 | 狠人大帝（为兄复仇）、叶凡（为姜太虚复仇） |
| **保护族人** | 维护家族/圣地的传承 | 姬家圣主、姜家圣主 |

#### GOAP 行为链示例：叶凡"证道成帝"

```
目标: 证道成帝
    ↓
前提条件: 五大秘境圆满 + 渡过准帝劫 + 凝聚完整道果
    ↓
子目标1: 修炼轮海秘境 → 获取《道经》→ 开辟苦海 → 到达彼岸
    ↓
子目标2: 修炼道宫秘境 → 获取后续经文 → 五藏化神
    ↓
子目标3: 修炼四极秘境 → 打破圣体诅咒（需神药+大帝级手段）
    ↓
子目标4: 修炼化龙秘境 → 九变化龙 → 脊柱如龙
    ↓
子目标5: 攀登仙台 → 半步大能 → 大能 → 王者 → 半圣 → 圣人 → 圣人王 → 大圣 → 准帝
    ↓
子目标6: 渡准帝劫 → 对抗天劫 + 禁区插手
    ↓
子目标7: 证道大帝 → 渡大帝劫 → 以自身道则压服天地
    ↓
结果: 证道成功，成为叶天帝
```

### NPC 行为

| 行为类型 | 具体表现 |
|----------|----------|
| **修炼** | 吞吐灵气、参悟经文、炼化源石 |
| **悟道** | 观想天地、参悟大帝道痕、领悟法则 |
| **征战** | 与仇敌/禁区至尊战斗、参与黑暗动乱平叛 |
| **探索秘境** | 进入太初古矿、青铜仙殿、荒古禁地等秘境获取机缘 |
| **对抗至尊** | 正面击杀禁区至尊，或以大阵压制限制 |
| **布阵** | 布置源阵、杀阵、防御阵、渡劫阵纹 |
| **炼丹** | 炼制疗伤丹、破境丹、延寿丹 |
| **冲击境界** | 渡劫突破，由天劫洗礼 |
| **收徒/传法** | 大帝晚年收徒传道，将传承延续下去 |
| **沉睡** | 禁区至尊沉睡以延缓寿元消耗，大帝受重伤后封印自身 |

### 消耗/产出关系（NPC 层面）

```
叶凡（圣体）
    消耗: 大量源石/神源（修炼资源）、不死药（突破诅咒）、战斗损伤（道伤）
    产出: 平定黑暗动乱、建立天庭、打通仙域、万物母气鼎（帝兵）

狠人大帝（凡体→红尘仙）
    消耗: 万族本源（吞噬）、兄长的死（执念动力）、逆天而行的代价
    产出: 《吞天魔功》《不灭天功》、荒古禁地布局、红尘仙战力

禁区至尊（自斩大帝）
    消耗: 沉睡时间、发动黑暗动乱时短暂的大帝战力恢复
    产出: 大量生命收割（续命寿命）、对世间的恐惧压迫
    最终消耗: 被叶凡等人击杀（产出天地安定）

普通修士
    消耗: 源石/神源、经文参悟时间、渡劫风险、道伤/寿命
    产出: 战力提升、境界突破、可交易的法宝/丹药/阵法
```
