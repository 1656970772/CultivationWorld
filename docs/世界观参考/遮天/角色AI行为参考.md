# 《遮天》角色AI行为参考

## AI架构映射说明

本游戏的NPC AI采用四层混合架构，将《遮天》角色的行为模式映射到对应层级：

| 层级 | 职责 | 时间粒度 | 遮天世界观映射 |
|------|------|----------|---------------|
| **BT（行为树）** | 情绪反应与应急抢占，毫秒级决策 | 即时 | 遭遇偷袭/亲友遇险/感知杀意时的条件反射 |
| **Utility（效用系统）** | 中长期目标选择，含权重打分与动态调整 | 数天至数年 | 选择修炼方向、加入势力、争夺机缘的优先级 |
| **GOAP（目标导向规划）** | 将目标分解为可执行步骤链 | 数小时至数月 | 从"我需要成帝"到具体的秘境探索、功法获取序列 |
| **Action Queue（动作队列）** | 执行具体战斗/移动/交互动作 | 逐帧/逐回合 | 施展斗技、移动至目标点、使用道具 |

## 一、叶凡（主角模板 NPC）

### 性格标签

```
冷静理智 | 杀伐果断 | 重情重义 | 领袖气质 | 极度危机嗅觉
```

- **冷却值**：高。叶凡极少被情绪完全支配，战斗中以理智判断为主。
- **情绪阈值**：亲友受威胁时突破理智上限，触发"暴走"模式。

### BT 应急反应层

| 触发条件 | 响应行为 | 抢占优先级 |
|----------|----------|------------|
| 遭遇强者挑衅 / 生死危机 | 瞬间评估敌我差距 → 差距不大则雷霆反击；差距过大则利用万物母气鼎/行字秘拖延并伺机逃脱 | 80 |
| 亲友受直接威胁 | 情绪暴走，无视逻辑代价，触发"疯狂复仇"状态 | **100（最高抢占）** |
| 感知未知禁地/神秘区域 | 极度谨慎，先观察他人/放探路手段，自身保持全盛状态 | 70 |
| 发现机缘（古经/圣果/九秘线索） | 果断出手但保留撤退后手 | 60 |
| 被轻视/嘲讽（圣体诅咒期） | 冷眼相对，内心标记"必超越"，转化为修炼驱动力 | 30 |

> **BT设计要点**：叶凡的BT树根节点是"评估当前环境威胁等级"，分支后进入"战斗/谈判/撤退/伪装"四级决策。亲友保护节点位于行为树最左侧（最高优先级）。

### Utility 长期目标权重

| 阶段 | 目标 | 权重 | 权重衰减/增长条件 |
|------|------|------|-------------------|
| 序盘（荒古禁地→离开北斗） | 活下去 | 90% | 每获得一种保命底牌 -5% |
| 序盘 | 救回庞博 | 100% | 庞博被救出后降为 0% |
| 序盘 | 打破圣体诅咒 | 70% | 获得圣果后降为 10% |
| 中盘（北斗星域争锋） | 登临大帝 | 80% | 每突破一个大境界 -15% |
| 中盘 | 铲平禁区、平定黑暗动乱 | 100% | 每个禁区被铲平 -25% |
| 终盘（成帝后） | 飞升成仙 | 100% | 终极目标，不可衰减 |

> **Utility 设计要点**：叶凡的目标权重存在强"阶段性锁定"——救庞博完成前，其他目标即使权重高也不会被选中。设计中用"前置条件锁"实现。

### GOAP 标志事件决策链

#### 事件一：荒古禁地夺圣果

```
目标：解除圣体诅咒 / 变强

GOAP 步骤链：
  1. [世界状态] 确认九妙圣果成熟周期
  2. [前置] 积累底牌 —— 源术基础、九秘残片、破空符
  3. [行动] 混入进荒古禁地的朝圣队伍
  4. [行动] 在外围区域按兵不动，观察禁地规律
  5. [行动] 趁禁地生物活动间隙深入核心区
  6. [行动] 采摘九妙圣果（九种各采一枚）
  7. [逃生] 利用虚空法器 / 破空符逃离
  8. [后置] 寻找安全地点炼化圣果

前置条件：源术等级 >= 3, 拥有至少一种虚空逃生手段
失败回退：若被禁地生物发现 → 立即使用逃生手段，标记此地为"暂不可入"
```

#### 事件二：天璇圣地遗址悟道

```
目标：领悟属于自己的"道"

GOAP 步骤链：
  1. [世界状态] 得知天璇遗址中有西皇经线索
  2. [前置] 与黑皇建立合作关系 / 获得进入资格
  3. [行动] 抵达北域天璇遗址
  4. [行动] 破解外围阵法（依赖黑皇的阵法知识）
  5. [行动] 进入遗址核心——古棋盘
  6. [行动] 在棋盘上推演天机，领悟空间大道
  7. [危机] 推演中触发古老残魂苏醒
  8. [战斗] 以新领悟的空间之道反制残魂
  9. [成果] 将感悟沉淀为自身道基

前置条件：阵法认知 >= 2（或队伍中有阵法专精者）, 空间感知 >= 5
关键决策点：推演深度 —— 浅层推演（安全但收益低） vs 深层推演（危险但可能领悟完整大道）
```

#### 事件三：紫山解救与获取经书

```
目标：进入紫山获取《西皇经》/ 解救被困者

GOAP 步骤链：
  1. [世界状态] 得知紫山中有《西皇经》和无始大帝传承
  2. [前置] 加入探索紫山的势力队伍（或自行组队）
  3. [前置] 准备克制紫山内圣灵生物的阳雷手段
  4. [行动] 破解紫山入口的无始杀阵
  5. [行动] 深入紫山内部，躲避圣灵生物巡逻
  6. [分支-机遇] 发现被困的圣地修士 → 评估救援代价
  7. [分支-核心] 找到经文存放处 → 快速抄录/拓印
  8. [危机] 被紫山深处恐怖存在察觉
  9. [逃生] 利用黑皇的传送阵纹/自身步法脱离

前置条件：阳雷属性道具 >= 1, 至少掌握一种阵法基础
分支决策权重：
  - 救援被困者 → 收益：获得人脉/情报（权重 60%）
  - 直接获取经书 → 收益：核心目标完成（权重 100%）
  - 两者兼顾 → 风险大幅增加（叶凡倾向此选项，权重 80%）
```

## 二、狠人大帝（传奇 Boss NPC）

### 性格标签

```
极度偏执 | 为情所困 | 天赋绝世 | 神秘莫测 | 万古第一
```

### 执念驱动的极端行为模式

狠人大帝是整个遮天世界中**最不适合用传统 Utility 系统**建模的角色——她的行为模式是单目标驱动：

```
核心执念：等待哥哥归来（成仙后再相见）

行为模型（执念驱动型）：
  Utility 永远锁定在单一目标："为哥哥归来铺路"
  所有子目标仅为此服务：
    ├── 自创《吞天魔功》→ 凡体证道（为了有能力等待）
    ├── 斩出过去身、未来身 → 留在不同时空寻人
    ├── 建立荒古禁地 → 作为等待的据点
    ├── 留下一口青铜棺 → 镇压万古、等待轮回
    └── 留下小囡囡（道果化身） → 在红尘中继续寻找
```

### BT 应急反应

| 触发条件 | 响应 | 说明 |
|----------|------|------|
| 任何人打扰她"葬下过去" | 直接抹杀 | 无情绪波动，纯机械式清除 |
| 感知到与哥哥相关的气息 | 停下一切动作，全力追踪 | 唯一能让她产生"反应"的刺激 |
| 黑暗动乱爆发 | 出手镇压 | 守护这片可能有哥哥转世的世界 |

### GOAP 标志事件链

```
事件：以凡体证道

GOAP 链：
  1. [世界状态] 荒古时代，凡体被视为"废体"
  2. [核心] 自创《吞天魔功》—— 吞噬万般体质为己用
  3. [行动] 不断吞噬特殊体质 → 积累 → 蜕变
  4. [行动] 在无数次生死战中磨砺
  5. [成就] 以最弱的凡体登临最强的大帝位
  6. [后续] 自斩一刀，等待轮回中的哥哥

事件：斩出过去未来身

GOAP 链：
  1. [世界状态] 已成就大帝，但未感应到哥哥的存在
  2. [目标] 跨越时间维度寻找
  3. [行动] 将自身斩为三段 —— 过去身、现在身、未来身
  4. [行动] 三段身分别镇守不同时空节点
  5. [代价] 自身陷入半沉睡状态，实力分散
```

> **AI设计建议**：狠人大帝适合设计为**环境型 Boss**——玩家无法与之正常交互，仅能在特定条件下触发其反应。她的 AI 本质是一个"永远在等待"的状态机。

## 三、黑皇（非人狡诈型 NPC）

### 性格标签

```
贪婪 | 腹黑 | 嘴炮 | 欺软怕硬 | 阵法宗师 | 极度记仇
```

### 非人 NPC 专属行为模式

黑皇代表着修仙模拟游戏中"非人类队友"的经典模型：**利益绑定 > 情感绑定**。

### BT 应急反应

| 触发条件 | 响应行为 | 抢占优先级 |
|----------|----------|------------|
| 敌方实力远超自己 | 立刻跑路 / 丢下队友 / 大叫"叶凡救我" | 100 |
| 发现珍稀神源/宝物 | 眼睛发光，立刻扑上去抢夺 | 90 |
| 被嘲讽/被揭短 | 记入黑名单，嘴上反击但不动手 | 50 |
| 遇到阵法/秘术 | 职业病发作，必须先研究再前进 | 70 |
| 叶凡陷入真正绝境 | 权衡利弊后出手（前提：叶凡死了自己也没好处） | 60 |

### Utility 长期目标权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 寻找神源/修炼资源 | 80% | 核心驱动力 |
| 坑人（报复/取乐） | 40% | 性格使然 |
| 维护无始大帝传承 | 60% | 仅当传承受到威胁时触发 |
| 帮助叶凡（有限度） | 30% → 50% | 随叶凡实力增长而增长 |
| 重建西皇母道统 | 20% | 嘴上说着重要但实际行动优先级低 |

### GOAP 典型决策链

```
场景：发现一处上古遗迹

GOAP 链（黑皇模式）：
  1. [评估] 从外围判断遗迹中是否有神源/宝物
     ├── 有 → 继续
     └── 没有 → 立刻失去兴趣，催促离开
  2. [策略] 如果阵纹危险 → 推叶凡或其他队友先进去探路
  3. [行动] 破解阵纹（自己主力时小心谨慎）
  4. [搜刮] 率先抢夺最值钱的物品
  5. [分赃] "这个我先保管，以后给你"（几乎不会兑现）
```

## 四、其他关键角色

### 庞博

```
性格标签：豪爽 | 重义气 | 大智若愚 | 妖族血脉觉醒后战力暴涨

BT 应急：
  - 叶凡有难 → 不顾一切冲上去并肩作战（优先级 100）
  - 被妖族血脉影响心智 → 需要意志判定，失败则短暂失控
  - 被轻视 → 用拳头说话，不废话

特色机制：妖族血脉觉醒后，战斗力临时翻倍但理智值下降
```

### 姬紫月

```
性格标签：活泼 | 狡黠 | 对叶凡近乎盲目的信任 | 姬家小月亮

BT 应急：
  - 叶凡遇到危险 → 不顾家族立场强行介入
  - 被家族长辈施压 → 表面顺从，暗中帮助叶凡
  - 被其他女性接近叶凡 → 醋意爆发但不直接冲突

Utility 权重：
  - 叶凡安全：90%
  - 家族利益：50%
  - 自身修炼：40%
```

### 安妙依

```
性格标签：清冷 | 理智 | 将情感藏在心底 | 妙欲庵传人

BT 应急：
  - 叶凡有难 → 理性评估后暗中相助（不正面冲突）
  - 门派利益与叶凡冲突 → 痛苦抉择，倾向牺牲自己
  - 被叶凡疏远 → 不纠缠，独自承受

特色机制：情感隐藏 —— 对外表现为冷漠无情，实际内心权重与姬紫月相当
```

## 五、NPC 类型化归纳

### 类型一：圣地天骄型

```
代表角色：摇光圣子、姬皓月、金翅小鹏王

性格标签：傲慢 | 资源垄断 | 家族/宗门荣誉至上 | 视散修为蝼蚁

Utility 权重配置：
  - 维护宗门声望：80%
  - 打压潜在威胁者（如叶凡）：70%
  - 追求更强功法/法宝：60%
  - 与同级别天骄结交：50%
  - 入禁区探险：20%（高出身惜命）

BT 应急：
  ├── 被散修挑衅 → 立刻出手镇压
  ├── 发现散修实力超预期 → 上报宗门请求支援
  └── 宗门危难 → 放下个人恩怨一致对外

GOAP 链（打压威胁者）：
  1. 察觉某人展现超过自身的天赋
  2. 调查其背景 —— 是否有势力支撑
  3. 若无势力 → 亲自出手 / 联合其他天骄围杀
  4. 若有势力 → 通过宗门层面施压
  5. 若围杀失败 → 上报宗门长老，将对方列为"夺运者"
```

### 类型二：禁区至尊/古皇型

```
代表角色：不死山至尊、轮回海之主、太初古矿古皇

性格标签：苟活万古 | 极度自私 | 不安全感爆棚 | 视众生为食物

Utility 权重配置：
  - 延续生命：100%（核心驱动）
  - 等待成仙路开启：80%
  - 吞噬生灵补充寿元：60%（由"延续生命"触发）
  - 清除威胁自身的存在：70%
  - 收集续命神药：50%

BT 应急：
  ├── 被低境界者打扰 → 随手抹杀
  ├── 遭遇同级别至尊 → 评估胜负代价，代价过高则退让
  ├── 寿元将尽 → 发动黑暗动乱
  └── 成仙路开启 → 不惜一切代价冲入

GOAP 链（发动黑暗动乱）：
  1. [状态] 寿元 <= 1000年
  2. [评估] 是否有其他续命手段 → 无
  3. [决策] 与禁区其他至尊达成默契
  4. [行动] 同时出禁区，分片收割生灵
  5. [手段] 大范围吞噬术，亿万人瞬间化为精气
  6. [结局] 被当世大帝或狠人虚影镇压
```

### 类型三：散修求道型

```
代表角色：段德（早期）、路人散修、小门派弟子

性格标签：机缘驱动 | 极度谨慎 | 趋利避害 | 能屈能伸

Utility 权重配置：
  - 获取修炼资源：70%
  - 安全存活：80%
  - 结交强者/抱大腿：50%
  - 探险秘境：30%（高风险时降为 5%）
  - 加入势力：仅在获得实质好处时考虑

BT 应急：
  ├── 遇到强者 → 恭敬退让，不争不抢
  ├── 发现机缘 → 先观察是否有陷阱和竞争者
  ├── 被卷入大势力争斗 → 立刻遁走
  └── 有把握捡漏 → 趁强者互相消耗时出手

GOAP 链（探索古墓遗迹）：
  1. 得知某地出现古墓/遗迹
  2. 评估消息可靠性
  3. 远远观望，等待大势力入场
  4. 大势力破开禁制后，在外围寻找遗漏
  5. 若内部出现恐怖存在 → 立刻远遁
  6. 若安全 → 捡漏后找地方消化
```

### 类型四：人族大帝型

```
代表角色：虚空大帝、恒宇大帝、无始大帝

性格标签：守护苍生 | 镇压禁区 | 威严不可侵犯 | 高处不胜寒

Utility 权重配置：
  - 镇压黑暗动乱：100%（在位时不可动摇）
  - 培养继承者：50%
  - 延寿/飞升：30%（远低于禁区至尊）
  - 探索宇宙奥秘：40%

BT 应急：
  ├── 黑暗动乱爆发 → 无论代价多大，出手镇压
  ├── 禁区至尊挑衅 → 强势回应，杀鸡儆猴
  ├── 继承者遇险 → 暗中关注但不过度干预
  └── 寿元将尽 → 选择自封或留下传承

GOAP 链（镇压黑暗动乱）：
  1. 感应到禁区方向有异常能量波动
  2. 确认黑暗动乱已经爆发
  3. 以最快速度赶赴战场
  4. 锁定发起者至尊
  5. 全力出手镇压，不计自身消耗
  6. 若自身不支 → 燃烧最后寿元换取一击
  7. 镇压完毕后巡视禁区，威慑剩余至尊
```
