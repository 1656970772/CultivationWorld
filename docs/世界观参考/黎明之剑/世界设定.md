# 第 1 章 世界 World

## 1.1 世界实体

### 1.1.1 宇宙模型

《黎明之剑》的主舞台是**洛伦星球**，一颗围绕巨型气态行星运行的卫星。洛伦星球的宇宙环境具有以下核心特征：

| 天体 | 类型 | 说明 |
|------|------|------|
| **洛伦星球** | 卫星 | 故事主舞台，拥有三块大陆，人类与其他智慧种族的主要栖息地 |
| **恒星"奥"** | 恒星 | 遥远的恒星，对洛伦魔力环境具有长期周期性影响，是魔力潮汐的宇宙级驱动力 |
| **起航者遗迹** | 人造天体群 | 远古外星文明"起航者"留下的轨道设施群，包括空间站"苍穹"、通天巨塔、轨道电梯及监控设施"哨兵系统" |

**宇宙级背景**：洛伦星球并非孤立的奇幻世界。起航者文明曾在此进行大规模改造，留下了遍布星球表面的监控网络。诺依文明（距洛伦约四光年）为应对魔潮而主动寻找盟友，向洛伦提供了大量技术资料。这表明《黎明之剑》的世界观是**科幻与奇幻深度融合**的产物——魔法本质上是宇宙基本力的一种表现形式，神明是信息层面的映射体。

### 1.1.2 大陆结构

洛伦星球的海陆分布以三大主要大陆为核心，辅以若干特殊区域：

| 大陆/区域 | 地理位置 | 主导文明 | 关键特征 |
|-----------|----------|----------|----------|
| **洛伦大陆** | 主大陆 | 人类诸国 | 塞西尔帝国、提丰帝国、白银帝国等人类文明的核心舞台，分布有安苏王国废墟 |
| **卡尔多大陆** | 未详述方位 | 精灵（已衰落） | 精灵的故乡，因未知原因生机断绝，大部分精灵迁徙至白银帝国 |
| **摩尔大陆** | 洛伦以东 | 起航者遗迹 | 建有起航者轨道电梯，是连接洛伦地表与轨道空间的重要枢纽 |
| **塔尔隆德** | 北极地区 | 巨龙文明 | 巨龙国度的"文明备份"所在地，环境严寒，龙族在此完成"成年礼"后浴火重生 |
| **艾欧大陆** | 未详述方位 | 海妖文明 | 海妖（水元素生物文明）的家园，存有起航者殖民船"安塔维恩号" |
| **刚铎废土** | 洛伦大陆区域 | 废墟/畸变体 | 古代魔法帝国刚铎的废墟，遍布魔法畸变体和"蠕行之灾"，是魔潮遗留的灾难现场 |

### 1.1.3 世界分层

洛伦星球并非单一的物质世界，而是由多层叠加的界域构成。从外到内（从有序到混沌）共分四层：

| 层级 | 名称 | 本质 | 居民/现象 | 与上层关系 |
|------|------|------|-----------|------------|
| 第一层 | **浅界（物质界）** | 凡人世界，物理规则主导 | 人类、精灵、巨龙、海妖等一切凡人种族 | 故事主舞台，其他界的"投影源" |
| 第二层 | **暗影界** | 物质界的扭曲倒影 | 暗影生物、暗影住民 | 是物质界的"影子"，空间结构扭曲，可被暗影行者利用进行空间穿梭 |
| 第三层 | **幽影界** | 暗影界的倒影 | 更加混乱的实体 | 暗影界的镜像翻转，规则更加不稳定 |
| 第四层 | **深界** | 信息层面的终极深渊 | 神明亦为其信息映射 | 最深层的界域，所有上层的本质在此还原为纯粹信息 |

**层级交互规则**：
- 上层的变化会在下层产生"投影"或"回响"，下层的信息变动也会向上渗透。
- **深界是理解整个世界观的关键**：神明并非超越世界的存在，而是深界信息的映射体——它们在深界有根源，在浅界有表现。这意味着神明可以被理解、被追踪、甚至被消灭。
- 暗影界与幽影界是**琥珀**等暗影大师的力量来源，她们通过穿梭暗影界实现空间移动。

### 1.1.4 关键自然现象

#### 魔潮（Magic Tide）

魔潮是洛伦世界最核心的周期性灾难，其本质与运作机制如下：

| 属性 | 说明 |
|------|------|
| **本质** | 魔力环境发生周期性剧变，是**深蓝之井爆炸等重大事件在认知层面的"投影"**。它不是单纯的魔力浓度变化，而是整个魔力规则的重新洗牌 |
| **触发机制** | 由恒星"奥"的周期性变化 + 洛伦星球本身魔力循环 + 重大魔法事件（如深蓝之井爆炸）共同触发 |
| **周期** | 长期周期性发生，每一轮魔潮来临前有预兆期 |
| **影响** | 魔力浓度剧烈波动 → 魔法体系崩溃与重建 → 依赖魔法的文明面临灭顶之灾 → 生物变异（畸变体） → 旧魔法帝国的覆灭（如刚铎帝国） |
| **应对方式** | 建造避难所、发展非魔法依赖技术（魔导科技）、提前预警（哨兵系统） |

**深层理解**：魔潮本质上不是"灾难"，而是宇宙对魔法文明过度发展的一种"纠错机制"。当文明对魔力环境的依赖和改造达到临界点，魔潮就会来"重置"一切。这解释了为什么每次魔潮都会摧毁最强大的魔法文明——刚铎帝国就是最典型的例子。

#### 黑阱（Black Trap）

黑阱是与魔潮不同维度的文明约束机制：

| 属性 | 说明 |
|------|------|
| **本质** | 文明发展触发的"**神之枷锁**"。当凡人文明发展到一定阶段时自动触发，导致神明干涉加剧 |
| **触发条件** | 文明的整体认知水平或技术水平突破某个阈值 |
| **影响** | 神明对文明进行主动干涉 → 文明发展停滞甚至毁灭 → 神明通过信仰体系加强对凡人的控制 |
| **与魔潮的关系** | 魔潮是自然/宇宙层级的重置，黑阱是神明层级的压制。两者叠加构成文明发展的"双重天花板" |

#### 哨兵系统（Sentinel System）

哨兵系统是起航者文明留给洛伦的最重要遗产：

| 属性 | 说明 |
|------|------|
| **本质** | 起航者文明布设的全球监控网络，用于**观测神国与魔潮** |
| **组成** | 轨道空间站"苍穹" + 地面监控节点（通天巨塔等） + 数据处理中心 |
| **功能** | 实时监测魔力环境波动、追踪神明活动轨迹、预测魔潮来临时间、向凡人文明提供预警 |
| **产出** | 为凡人文明提供关键的战略预警信息，帮助其规避魔潮和神明干涉的风险，是文明延续的重要保障 |

## 1.2 世界状态

世界状态是一组动态变化的全局参数，决定沙盒世界的运行环境：

| 状态维度 | 描述 | 数值范围 | 影响 |
|----------|------|----------|------|
| **纪元阶段** | 当前所处的历史纪元 | 枚举值：凡人纪元 / 神明降临纪元 / 魔潮前夕 / 魔潮中 / 后魔潮重建纪元 | 决定可用事件池、势力行为倾向 |
| **魔力环境浓度** | 洛伦全球平均魔力密度 | 0-100（0=无魔区，50=正常，100=魔潮峰值） | 影响魔法效果、生物变异率、神明干涉能力 |
| **魔潮周期** | 距离下一轮魔潮的时间 | 时间单位（年/月） | 决定所有势力的紧迫度——越临近魔潮，势力越倾向于合作 |
| **黑阱触发度** | 文明达到触发黑阱的进度 | 0-100（100=触发） | 神明干涉频率与此正相关 |
| **哨兵系统活跃度** | 哨兵系统当前的运作水平 | 0-100（0=完全瘫痪，100=全功率运作） | 影响预警准确率、神明追踪能力 |
| **神国干涉度** | 神明对物质界的整体干预程度 | 0-100 | 影响神术效果、神迹频率、凡人的信仰倾向 |

## 1.3 世界行为/规则

### 1.3.1 魔潮规则

1. **周期性触发**：魔潮按恒星"奥"的运行周期规律性发生，周期长度以千年计。
2. **事件加速**：重大魔法事件（如深蓝之井爆炸）可提前触发或加剧魔潮。
3. **魔力重置**：魔潮期间，旧的魔力使用规则不再有效，法师必须重新学习魔法。
4. **畸变风险**：魔潮高峰期间，生物暴露在高浓度紊乱魔力下会发生不可逆畸变。
5. **避难所机制**：深埋地下或特殊防护的避难所可在魔潮中保护文明火种。

### 1.3.2 黑阱规则

1. **文明触发**：凡人的集体认知/技术突破阈值 → 自动引发神明的注意。
2. **神之枷锁**：神明通过神之眼、神迹、教会等渠道加强对触发文明的干预。
3. **阈值递增**：每次突破黑阱后，下一次触发阈值会提高——文明在"对抗神明"中成长。
4. **不可规避**：只要文明在发展，黑阱就是必然遭遇的关卡。唯一的选择是"直面并突破"。

### 1.3.3 哨兵系统规则

1. **被动监控**：哨兵系统不主动干预，仅提供观测数据和预警。
2. **权限分层**：不同文明/势力对哨兵系统的访问权限不同，取决于其与起航者遗迹的关系。
3. **信息不完整**：哨兵系统的数据需要凡人的解读能力——同样的数据，理解水平不同导致决策质量差异巨大。
4. **维护需求**：哨兵系统是物理设备，需要维护和能量供应，长期荒废会失效。

### 1.3.4 神明干涉规则

1. **信仰绑定**：神明的力量与信徒数量、信仰纯度正相关。失去信徒的神会衰弱甚至消亡。
2. **神国隔离**：神明居住在神国中，不能随意进入物质界。需要通过神之眼、神迹、化身等间接手段干涉。
3. **神性污染**：凡人接触过多神力会产生精神污染，逐渐丧失自我意志。
4. **猎神可能性**：神明并非不朽——战神被联合讨伐即为明证。只要找到神明在深界的"信息根源"，理论上任何神都可被消灭。
5. **神职竞争**：同一领域可以有多个神明竞争神职，最终胜者吸收败者的神职和信徒。

### 1.3.5 深界信息映射规则

1. **万物有源**：浅界一切存在（包括神明）在深界都有对应的"信息根源"。
2. **映射变更**：修改深界的信息映射可以改变浅界的现实——这是极高位存在的能力。
3. **认知影响**：凡人的集体认知和信仰会反向塑造深界的信息结构，这就是神明依赖信仰的深层原因。

## 1.4 世界需求

从沙盒世界设计的角度，世界层级有以下核心需求：

| 需求 | 优先级 | 说明 |
|------|--------|------|
| **制造冲突** | P0 | 魔潮、黑阱、神战是世界级冲突的根本驱动力，推动所有势力做出反应 |
| **推动文明演化** | P0 | 魔潮摧毁旧文明 → 废墟上诞生新文明 → 新文明面对新魔潮，形成螺旋上升的文明演化 |
| **维持平衡** | P1 | 世界需要保持魔力/神力/科技力量的动态平衡，避免单一方垄断导致演化停滞 |
| **避免死局** | P1 | 哨兵系统、起航者遗迹、诺依文明等技术遗产是文明打破魔潮-神明压制循环的关键变量 |
