# 第 4 章 物品 Item

## 4.1 物品静态数据

物品在《黎明之剑》沙盒中具有以下通用静态属性：

| 属性 | 类型 | 说明 |
|------|------|------|
| **物品ID** | 字符串 | 唯一标识符 |
| **物品名称** | 字符串 | 显示名称 |
| **物品类型** | 枚举 | 资源类 / 神之眼 / 哨兵系统 / 魔导装置 / 符文武器护甲 / 神器 / 魔法书 |
| **稀有度** | 枚举 | 普通 / 精良 / 稀有 / 史诗 / 传说 / 神话 |
| **可堆叠** | 布尔 | 是否可堆叠（资源类通常可堆叠，装备类不可） |
| **可交易** | 布尔 | 是否可在势力/NPC间交易 |
| **可消耗** | 布尔 | 使用后是否消失 |
| **耐久度** | 数值 | 装备类物品的耐久度（0=损坏） |
| **绑定状态** | 枚举 | 未绑定 / 灵魂绑定 / 势力绑定 |
| **来源** | 文本 | 获取途径（采集/炼制/交易/掠夺/遗迹探索/神明赐予） |

## 4.2 物品分类体系

### 4.2.1 资源类

资源是沙盒经济系统的基础，支撑所有势力的运转和 NPC 的成长。

#### 魔力（Mana）

| 属性 | 值 |
|------|----|
| **本质** | 宇宙基本力在洛伦的表现形式，由恒星"奥"驱动循环 |
| **获取方式** | 自然吸收（缓慢）、魔力水晶矿采集、魔力节点提取、魔潮期间浓度暴涨 |
| **用途** | 施法燃料、魔导装置能源、符文锻造、修炼辅助 |
| **消耗产出** | 消耗魔力 → 产出法术效果/魔导动力/符文效果 |
| **稀有度** | 普通（基础资源，但高纯度魔力稀有） |
| **特殊规则** | 魔潮期间魔力浓度剧烈波动，旧法术模型可能失效 |

#### 斗气（Fighting Spirit）

| 属性 | 值 |
|------|----|
| **本质** | 通过锻炼精神与意志在体内凝聚的生命能量 |
| **获取方式** | 修炼（骑士训练）、战斗经验积累、特殊药剂辅助 |
| **用途** | 强化肉身、释放战技、形成领域、神性斗气 |
| **消耗产出** | 消耗体力+精神力 → 产出斗气 → 强化战斗力 |
| **稀有度** | 普通（人人可修炼，但高阶斗气稀有） |
| **特殊规则** | 斗气与魔力不兼容，骑士通常无法同时修炼魔法 |

#### 信仰之力（Faith Power）

| 属性 | 值 |
|------|----|
| **本质** | 凡人集体信仰产生的精神能量，神明的主要"食物" |
| **获取方式** | 信徒祈祷、宗教仪式、神迹回应、神之眼收集 |
| **用途** | 神明维持神国、施展神迹、增强神力、神职竞争 |
| **消耗产出** | 消耗信徒信仰 → 产出神力 → 神明干涉能力增强 |
| **稀有度** | 普通（但高纯度信仰之力稀有） |
| **特殊规则** | 信仰之力有"神性污染"风险——凡人接触过多会丧失自我 |

#### 自然之力（Nature Power）

| 属性 | 值 |
|------|----|
| **本质** | 洛伦星球本身的生态能量，由植物、动物、元素循环产生 |
| **获取方式** | 德鲁伊沟通自然之灵、世界之树汲取、自然圣地修炼 |
| **用途** | 德鲁伊施法、自然治疗、天气操控、植物生长加速 |
| **消耗产出** | 消耗自然亲和 → 产出自然法术效果 → 生态恢复 |
| **稀有度** | 精良（需要德鲁伊天赋才能利用） |
| **特殊规则** | 过度抽取自然之力会导致生态崩溃（卡尔多大陆即为前车之鉴） |

#### 龙语（Dragon Tongue）

| 属性 | 值 |
|------|----|
| **本质** | 龙族语言本身即是魔法——每个龙语音节都直接调用世界本源规则 |
| **获取方式** | 龙族传承、龙语学习（非龙族极难掌握）、龙语符文铭刻 |
| **用途** | 龙语魔法施放、符文锻造（龙语符文）、古代知识解读 |
| **消耗产出** | 消耗精神力+龙族血脉 → 产出龙语魔法效果 |
| **稀有度** | 史诗（非龙族几乎无法掌握） |
| **特殊规则** | 龙语魔法威力巨大但学习门槛极高，需要龙族血脉或长期学习 |

#### 符文（Runes）

| 属性 | 值 |
|------|----|
| **本质** | 将魔法规则以图形化方式铭刻于材料上的技术载体 |
| **获取方式** | 符文工匠铭刻、古代遗迹发掘、符文知识传承 |
| **用途** | 制造符文武器/护甲、魔导装置核心组件、阵法构建 |
| **消耗产出** | 消耗材料+魔力+符文知识 → 产出符文装备/装置 |
| **稀有度** | 精良（基础符文）到传说（龙语符文） |
| **特殊规则** | 符文是连接魔法与工程学的桥梁——魔导科技的核心 |

#### 工程材料（Engineering Materials）

| 属性 | 值 |
|------|----|
| **本质** | 用于魔导装置制造和工业生产的物理材料 |
| **获取方式** | 采矿、冶炼、化学合成、贸易进口 |
| **用途** | 魔导车制造、魔能列车轨道、空天要塞建造、符文武器基材 |
| **消耗产出** | 消耗矿石+能源+劳动力 → 产出魔导装置/工业产品 |
| **稀有度** | 普通（基础材料）到稀有（特殊合金/魔法金属） |
| **特殊规则** | 工程材料是塞西尔帝国工业化的物质基础，资源压力直接影响科技发展速度 |

### 4.2.2 道具1：神之眼（Eye of God）

| 属性 | 值 |
|------|----|
| **本质** | 神明留下的"监控/控制终端"，是神明干涉物质界的主要媒介 |
| **外观** | 通常为眼球状或晶体状的神力凝聚体 |
| **功能** | ① 绑定信徒（标记信仰归属） ② 监视特定区域（神明视野延伸） ③ 传递神力（神迹触媒） ④ 精神控制（对非信徒施加影响） |
| **获取方式** | 神明赐予、神迹遗留、神战掉落、遗迹发掘 |
| **消耗产出** | 消耗信仰之力 → 产出监控数据/神力传递/精神控制 |
| **稀有度** | 稀有（普通神之眼）到神话（主神之眼） |
| **风险** | 持有神之眼可能被神明反向追踪、精神污染、强制信仰绑定 |
| **沙盒用途** | 神明干涉度的关键道具——势力持有神之眼越多，神明对该势力的控制力越强 |

### 4.2.3 道具2：哨兵系统（Sentinel System）

| 属性 | 值 |
|------|----|
| **本质** | 起航者文明布设的全球监控网络，由轨道空间站"苍穹"+地面节点+数据处理中心组成 |
| **外观** | 轨道空间站（苍穹）、通天巨塔（地面节点）、数据终端（操作界面） |
| **功能** | ① 实时监测魔力环境波动 ② 追踪神明活动轨迹 ③ 预测魔潮来临时间 ④ 向凡人文明提供预警 ⑤ 记录历史数据（文明兴衰档案） |
| **获取方式** | 起航者遗迹探索、权限继承、技术破解 |
| **消耗产出** | 消耗能源（起航者遗留能源或替代能源） → 产出预警信息/神明追踪数据/历史档案 |
| **稀有度** | 神话（唯一性——全球仅此一套） |
| **风险** | 过度依赖哨兵系统可能导致文明丧失自主判断能力；系统故障或能源耗尽将失去预警能力 |
| **沙盒用途** | 文明生存的关键保障——哨兵系统活跃度直接影响所有势力对魔潮的应对能力 |

### 4.2.4 道具3：魔导装置（Magitech Device）

| 属性 | 值 |
|------|----|
| **本质** | 将魔导理论工业化的产品，是塞西尔帝国的核心竞争力 |
| **分类** | ① 民用：魔导车、魔能列车、魔导照明、魔导通讯器 ② 军用：魔导炮、魔导护盾、空天要塞、魔导机甲 ③ 工业：魔导机床、魔能发电机、符文生产线 |
| **获取方式** | 工程师设计制造、魔导工厂量产、贸易购买 |
| **消耗产出** | 消耗工程材料+魔力+符文+劳动力 → 产出魔导装置 → 生产力/战斗力提升 |
| **稀有度** | 精良（民用）到史诗（军用/空天要塞） |
| **风险** | 魔力过载可能导致爆炸；技术泄露可能被敌对势力复制 |
| **沙盒用途** | 科技水平的直接体现——魔导装置的数量和质量决定势力的综合实力 |

### 4.2.5 道具4：符文武器/护甲（Runic Weapon/Armor）

| 属性 | 值 |
|------|----|
| **本质** | 铭刻了魔法规则的装备，拥有超凡属性 |
| **分类** | ① 符文武器：符文剑、符文弓、符文法杖 ② 符文护甲：符文胸甲、符文盾牌、符文披风 ③ 符文饰品：符文戒指、符文项链 |
| **获取方式** | 符文工匠打造、古代遗迹发掘、战利品缴获、任务奖励 |
| **消耗产出** | 消耗基材+符文+魔力+工匠时间 → 产出符文装备 → 战斗力提升 |
| **稀有度** | 精良（基础符文）到传说（龙语符文装备） |
| **风险** | 符文损坏后修复困难；劣质符文可能反噬使用者 |
| **沙盒用途** | 军队和冒险者的核心装备来源——符文装备的普及程度决定势力的军事力量 |

### 4.2.6 道具5：神器（Divine Artifact）

| 属性 | 值 |
|------|----|
| **本质** | 蕴含强大神力或规则的特殊物品，通常由神明或半神级存在持有 |
| **分类** | ① 神明造物：神明亲自打造的神器（如战神之矛） ② 神战遗留：神明陨落后遗留的神器 ③ 凡人造物：传奇工匠打造的接近神器级别的装备 |
| **获取方式** | 神明赐予、神战掉落、猎神战利品、传奇工匠打造 |
| **消耗产出** | 消耗神力/信仰之力 → 产出神器效果（领域展开/规则扭曲/即死效果等） |
| **稀有度** | 传说（凡人造物）到神话（神明造物） |
| **风险** | 神器可能残留原主人的意志（精神污染）；使用神器可能被神明追踪；凡人使用神器可能承受不住神力反噬 |
| **沙盒用途** | 顶级战力的象征——神器持有者在同阶中具有压倒性优势 |

### 4.2.7 道具6：魔法书（Spellbook）

| 属性 | 值 |
|------|----|
| **本质** | 记录法术模型与魔导知识的载体 |
| **分类** | ① 基础魔法书：记录常见法术模型 ② 高阶魔法书：记录禁忌知识或传奇法术 ③ 魔导手册：记录魔导装置设计图纸 ④ 古代典籍：记录失传的古代魔法知识 |
| **获取方式** | 法师撰写、魔法学院教材、古代遗迹发掘、黑市交易 |
| **消耗产出** | 消耗时间+智力 → 学习魔法书 → 获得新法术/新知识 |
| **稀有度** | 普通（基础魔法书）到传说（禁忌知识典籍） |
| **风险** | 高阶魔法书可能蕴含精神污染（如万物终亡会的禁忌知识）；错误理解法术模型可能导致施法失败或反噬 |
| **沙盒用途** | 法师成长的核心道具——魔法书的质量和数量决定法师的成长速度和上限 |

## 4.4 物品状态

每个物品在沙盒中由以下状态维度动态描述：

| 状态维度 | 定义 | 数值范围 | 影响 |
|----------|------|----------|------|
| **耐久度** | 物品的物理/魔法损耗程度 | 0-100（0=损坏，100=全新） | 耐久度归零则物品失效或损坏 |
| **充能状态** | 魔力/神力/能源的剩余量 | 0-100（0=耗尽，100=满充） | 影响魔导装置/符文装备/神器的使用次数 |
| **绑定状态** | 物品与持有者的绑定关系 | 未绑定/灵魂绑定/势力绑定 | 绑定后不可交易/掉落，灵魂绑定不可解除 |
| **污染度** | 物品被神力/暗影/畸变污染的程度 | 0-100（0=纯净，100=完全污染） | 污染度越高，使用风险越大（精神污染/畸变） |
| **升级潜力** | 物品是否可被进一步强化 | 0-100（0=已达上限，100=巨大潜力） | 决定物品的成长空间 |

## 4.5 物品需求

物品本身没有"需求"，但物品的**获取、使用和维护**会产生需求：

| 需求类型 | 触发条件 | 目标 | 优先级 |
|----------|----------|------|--------|
| **采集资源** | 势力资源压力>60，NPC需要修炼资源 | 获取足够资源 | 0.85 |
| **炼制装备** | NPC修炼进度达到瓶颈，需要更好装备 | 获得更高品质装备 | 0.80 |
| **消耗使用** | 战斗/修炼/建造中需要物品效果 | 物品效果生效 | 0.90 |
| **交易流通** | 势力间资源不均衡，需要互通有无 | 完成交易，双方获益 | 0.70 |
| **掠夺获取** | 敌对势力持有高价值物品 | 夺取物品 | 0.75 |
| **祭炼强化** | 物品有升级潜力，持有者有资源 | 物品品质提升 | 0.65 |
| **开辟新源** | 现有资源枯竭，需要新资源点 | 发现新资源产地 | 0.80 |

## 4.6 物品行为

### 4.6.1 行为类型库

| 行为类型 | 执行条件 | 消耗 | 产出 | 风险 |
|----------|----------|------|------|------|
| **采集** | 有采集工具/技能，资源点可用 | 时间、工具耐久 | 资源（魔力水晶/矿石/药材等） | 资源点枯竭、采集事故 |
| **炼制** | 有配方、材料、炼制工具 | 材料、时间、魔力/斗气 | 成品物品（丹药/装备/装置） | 炼制失败、材料浪费 |
| **消耗** | 物品可用，满足使用条件 | 物品耐久/充能 | 物品效果（治疗/攻击/防护等） | 物品损坏、副作用 |
| **交易** | 双方同意，物品可交易 | 物品所有权 | 等价物品或货币 | 欺诈、价格波动 |
| **掠夺** | 战斗胜利，物品可掉落 | 军队/战力、声望 | 战利品（物品+资源） | 战斗失败、国际谴责 |
| **祭炼** | 物品有升级潜力，有祭炼材料 | 祭炼材料、时间、魔力 | 物品品质提升、新属性解锁 | 祭炼失败、物品损坏 |
| **开辟** | 有勘探能力，未开发区域 | 勘探队、时间、物资 | 新资源点、新遗迹入口 | 勘探失败、遭遇危险 |

### 4.6.2 物品经济流转模型

```
┌──────────────────────────────────────────────────────────────────┐
│                     物品经济流转模型                              │
│                                                                  │
│  资源采集 ──→ 炼制加工 ──→ 成品物品 ──→ 交易/分配 ──→ 消耗使用   │
│     ↑                                          │                │
│     │                                          ↓                │
│  开辟新源 ←── 资源枯竭 ←── 持续消耗 ←── 耐久度归零/充能耗尽      │
│                                                                  │
│  掠夺路径：敌对势力物品 ──→ 战斗 ──→ 战利品 ──→ 己方物品池       │
```
