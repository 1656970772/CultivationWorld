## 3.3 NPC 状态

每个 NPC 由以下九个状态维度动态描述，这些维度会随时间、事件和行为发生变化：

| 状态维度 | 定义 | 数值范围 | 影响 |
|----------|------|----------|------|
| **存活** | 是否存活 | 是/否 | 死亡则所有行为终止，可能触发继承/复仇事件 |
| **年龄** | 生理年龄 | 数值（岁） | 影响修炼速度、寿元消耗、社会地位 |
| **寿元** | 剩余寿命 | 数值（岁） | 为0时自然死亡，可通过突破境界、神器、特殊仪式延长 |
| **伤势** | 当前受伤程度 | 0-100（0=健康，100=濒死） | 影响所有属性发挥，需要治疗或休息恢复 |
| **道基** | 修炼/职业根基 | 文本描述（如"地球科技思维+洛伦斗气"） | 决定成长方向、技能学习限制、突破瓶颈 |
| **心境** | 心理状态 | 文本描述（如"坚定改革者"） | 影响决策质量、突破成功率、人际关系 |
| **修炼进度** | 在当前职业路径上的进展 | 文本描述（如"传奇骑士"） | 决定战斗力、社会地位、可执行任务难度 |
| **因果缠结** | 与其他角色/事件的关联度 | 低/中/高/极多 | 因果越多，越容易被卷入事件，但也可能获得更多机遇 |
| **气运** | 运气/天命值 | 0-100 | 影响奇遇概率、突破成功率、危机存活率 |

## 3.4 NPC 需求

### 3.4.1 需求类型与 GOAP（目标导向行动规划）行为链

NPC 需求驱动其行为。GOAP（Goal-Oriented Action Planning）是一种游戏 AI 常用技术，NPC 会基于当前需求选择最优行动序列。以下是《黎明之剑》NPC 的核心需求类型：

| 需求类型 | 触发条件 | 目标状态 | 优先级权重 |
|----------|----------|----------|------------|
| **生存** | 伤势>70，寿元<10，被追杀 | 伤势<30，寿元>50，安全环境 | 0.95 |
| **修炼突破** | 修炼进度达到瓶颈，资源充足 | 突破到下一境界 | 0.85 |
| **变法改革** | 所属势力需要改革，NPC有政治资本 | 改革成功，势力稳定度提升 | 0.80 |
| **复仇** | 亲友被杀，凶手明确且存活 | 凶手死亡或受惩罚 | 0.75 |
| **保护亲友** | 亲友面临威胁（战争/魔潮/神明） | 亲友安全，威胁解除 | 0.80 |
| **解决宇宙问题** | 魔潮/黑阱/神明威胁文明存续 | 问题解决或缓解 | 0.70 |

### 3.4.2 GOAP 行为链示例：高文·塞西尔

```
当前状态：伤势20，修炼进度传奇骑士，势力稳定度85，魔潮距离3年

需求评估：
1. 解决宇宙问题（魔潮）优先级=0.70×1.25（紧急）×1.0=0.875
2. 变法改革优先级=0.80×1.05×1.0=0.84
3. 修炼突破优先级=0.85×1.0×1.0=0.85

选择最高优先级：解决魔潮（0.875）

GOAP行为链规划：
目标：提升魔潮生存率30%
可用行动：
  A. 建造避难所（消耗：资源压力+20，时间6个月，成功率90%）
  B. 研发魔导防护（消耗：科研资源，时间12个月，成功率60%）
  C. 结盟联合应对（消耗：外交资源，时间3个月，成功率70%）

评估：
  A产出：生存率+30%，但资源压力大增
  B产出：可能失败，但成功后技术可复用
  C产出：生存率+15%，且可后续合作

选择：A（时间紧迫，产出确定）

执行序列：
1. 召集工程师（瑞贝卡）→ 2. 调拨资源 → 3. 监督建造 → 4. 验收启用

状态更新：
  资源压力+20，魔潮生存率+30%，瑞贝卡因果缠结+1
```

## 3.5 NPC 行为

### 3.5.1 行为类型库

| 行为类型 | 执行条件 | 消耗资源 | 预期产出 | 风险 |
|----------|----------|----------|----------|------|
| **修炼** | 有修炼场所、资源充足 | 时间、修炼资源（魔力水晶等） | 修炼进度提升、可能突破 | 走火入魔、资源浪费 |
| **炼丹** | 有丹方、药材、丹炉 | 药材、时间、魔力 | 丹药（恢复/突破/延寿） | 炸炉、丹药毒性 |
| **炼器** | 有材料、锻造工具、符文知识 | 材料、时间、魔力 | 符文武器/护甲/魔导装置 | 失败损毁、材料浪费 |
| **布阵** | 有阵法知识、阵旗/阵基 | 阵旗材料、时间、魔力 | 防御/攻击/辅助阵法 | 阵法反噬、被敌人破解 |
| **征战** | 有军队、敌方明确 | 军队、物资、时间 | 领土扩张、资源掠夺、声望提升 | 战败、军队损失、国内反战 |
| **探索秘境** | 有探险队、秘境坐标 | 探险队、时间、物资 | 古代遗产、稀有资源、修炼机缘 | 秘境陷阱、畸变体、精神污染 |
| **研发科技** | 有实验室、理论基础 | 科研资源、时间、人才 | 新技术、新装置、科技水平提升 | 研发失败、资源浪费、技术泄露 |
| **对抗神明** | 神明干涉度高、有盟友 | 信仰资源、军队、特殊道具 | 神明削弱或死亡、信仰解放 | 神罚、信徒反扑、精神污染 |

### 3.5.2 NPC 行为决策机制

每个 NPC 的行为决策基于以下公式：

```
行为选择得分 = 需求匹配度 × 成功率 × 产出价值 / (消耗资源 × 风险系数)

NPC 会选择得分最高的行为执行。
```

**示例：瑞贝卡决定是否研发新型魔导炮**

```
需求：修炼突破（工程学）权重0.85
可用行为：
  A. 继续改进现有魔导车（成功率90%，产出价值中，消耗低，风险低）
  B. 研发新型魔导炮（成功率60%，产出价值高，消耗高，风险中）

计算：
  A得分 = 0.7×0.9×0.6/(0.3×0.2)=6.3
  B得分 = 0.9×0.6×0.9/(0.7×0.5)=1.39

选择：A（继续改进现有技术）
```

## 3.6 消耗/产出关系

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                            NPC级消耗/产出关系                                 │
├──────────────┬─────────────────────┬─────────────────────────────────────────┤
│  输入（消耗）  │  过程/行为           │  输出（产出）                           │
├──────────────┼─────────────────────┼─────────────────────────────────────────┤
│ 时间          │ 修炼                 │ 修炼进度提升、可能突破境界               │
│ 药材          │ 炼丹                 │ 丹药（恢复伤势/突破瓶颈/延长寿元）       │
│ 锻造材料      │ 炼器                 │ 符文武器/护甲/魔导装置                   │
│ 阵旗材料      │ 布阵                 │ 防御/攻击/辅助阵法                       │
│ 军队          │ 征战                 │ 领土、资源、声望、战利品                 │
│ 探险队        │ 探索秘境             │ 古代遗产、稀有资源、修炼机缘             │
│ 科研资源      │ 研发科技             │ 新技术、新装置、论文发表                 │
│ 信仰资源      │ 对抗神明             │ 神明削弱、信仰解放、神职空缺             │
│ 政治资本      │ 变法改革             │ 势力稳定度提升、科技水平提升             │
│ 人际关系      │ 保护亲友             │ 亲友安全、忠诚度提升、后续支持           │
└──────────────┴─────────────────────┴─────────────────────────────────────────┘
```

**NPC 间资源流动**：
- 高文向瑞贝卡提供：政治支持、资源调拨、工程需求
- 瑞贝卡向高文提供：魔导装置、技术突破、工业产能
- 高文向赫蒂提供：魔法研究支持、信仰自由保障
- 赫蒂向高文提供：魔法理论支持、法术模型
- 琥珀向高文提供：情报、暗影穿梭服务
- 高文向琥珀提供：庇护、任务委托、资源支持
- 罗塞塔向高文提供：军事合作、矿产资源
- 高文向罗塞塔提供：魔导技术、外交支持
- 贝尔塞提娅向高文提供：德鲁伊治疗、自然魔法知识
- 高文向贝尔塞提娅提供：工业产品、军事保护

**NPC 成长循环**：
```
资源获取 → 修炼/研发 → 能力提升 → 执行更高难度任务 → 获取更多资源 → ...
```
这个循环是沙盒中 NPC 实力增长的核心机制，也是玩家（如果存在）与 NPC 互动的主要切入点——玩家可以通过提供资源、任务委托、共同探险等方式介入 NPC 的成长循环，从而影响世界走向。
