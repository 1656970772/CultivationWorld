# 第 2 章 势力 Faction

## 2.1 势力静态数据

《黎明之剑》中的势力按文明类型可分为四大类，每类有独特的组织结构和行为模式：

| 势力类型 | 典型特征 | 核心目标 | 代表势力 |
|----------|----------|----------|----------|
| **人类帝国** | 中央集权、君主制、科技/魔法混合 | 领土扩张、文明延续、对抗魔潮 | 塞西尔帝国、提丰帝国、安苏王国（已灭） |
| **非人文明** | 种族特性鲜明、寿命长、文化独特 | 种族生存、文明备份、应对环境变化 | 白银帝国（精灵）、塔尔隆德（龙族）、海妖、奥古雷部族国（多族） |
| **教会与信仰组织** | 以神明信仰为核心、等级森严 | 传播信仰、对抗异教、获取神力 | 圣光教会、万物终亡会、永眠者教团、战神教会 |
| **星外文明** | 科技远超洛伦、跨星系存在 | 应对宇宙级危机（魔潮）、寻找盟友 | 诺依文明、起航者文明（已消失） |

## 2.3 势力状态

每个势力在沙盒中由以下六个维度动态描述其当前状态：

| 状态维度 | 定义 | 影响范围 | 正常波动范围 |
|----------|------|----------|--------------|
| **稳定度** | 内部政治稳定、社会秩序、民众支持度 | 影响叛乱概率、改革成功率、外交可信度 | 30-90（<30濒临崩溃，>90高度稳定） |
| **资源压力** | 粮食、矿产、魔力水晶、人口等核心资源的紧张程度 | 影响经济发展、军事扩张、科技研发速度 | 0-100（0=资源过剩，100=资源枯竭） |
| **科技/魔法水平** | 综合技术水平（工程学+魔法+组织能力） | 决定生产力、军事实力、魔潮应对能力 | 0-100（0=原始部落，100=星际文明） |
| **神明干涉度** | 神明对该势力的直接或间接干预程度 | 影响神术效果、神迹频率、信仰控制力 | 0-100（0=无神论/神明无视，100=完全被神明控制） |
| **跨界影响力** | 在洛伦大陆及以外地区的影响力 | 决定外交权重、贸易优势、情报获取能力 | 0-100（0=孤立无援，100=全球主导） |
| **声望** | 在其他势力和民众中的声誉 | 影响结盟难度、贸易条件、人才吸引力 | -100到+100（负值=恶名昭著，正值=受人尊敬） |

## 2.4 势力需求

### 2.4.1 需求类型与优先级计算逻辑

每个势力同时存在多种需求，其优先级由以下公式动态计算：

```
需求优先级 = 基础权重 × 紧急系数 × 资源系数

其中：
- 基础权重：由势力类型和核心目标决定的固定权重
- 紧急系数：当前状态与目标状态的差距（差距越大越紧急）
- 资源系数：当前资源压力（压力越大，生存类需求优先级越高）
```

| 需求类型 | 基础权重 | 典型触发条件 | 优先级计算示例（塞西尔帝国） |
|----------|----------|--------------|------------------------------|
| **生存** | 0.9 | 稳定度<40，资源压力>80，外敌入侵 | 稳定度60→目标80，差距20→紧急系数1.2 |
| **扩张** | 0.7 | 稳定度>70，资源压力<50，军力充足 | 当前影响力70→目标90，差距20→紧急系数1.2 |
| **资源获取** | 0.8 | 资源压力>60，科技研发受阻 | 资源压力70→目标40，差距30→紧急系数1.3 |
| **传承延续** | 0.6 | 人口下降，文化传承断裂风险 | 人口增长率-0.5%→目标+1%，差距1.5%→紧急系数1.15 |
| **对抗魔潮** | 0.85 | 魔潮周期临近，哨兵系统预警 | 距离魔潮5年→目标10年缓冲，差距5年→紧急系数1.25 |
| **对抗神明** | 0.75 | 神明干涉度>50，信仰控制加剧 | 神明干涉度30→目标10，差距20→紧急系数1.2 |
| **发展科技** | 0.8 | 科技水平落后于主要对手 | 科技水平80→对手85，差距5→紧急系数1.05 |

## 2.5 势力行为

### 2.5.1 行为类型库

| 行为类型 | 执行条件 | 消耗资源 | 预期产出 | 风险 |
|----------|----------|----------|----------|------|
| **变法改革** | 稳定度>60，领袖支持度高 | 政治资本、短期稳定度下降 | 长期稳定度提升、科技水平提升 | 贵族反抗、内战风险 |
| **军事镇压** | 内部叛乱，稳定度<50 | 军队、财政、民众支持度 | 恢复秩序，稳定度提升 | 民众不满加剧，国际谴责 |
| **结盟** | 有共同敌人（魔潮/神明），声望>50 | 外交资源、部分主权让渡 | 军事互助、资源共享、影响力提升 | 盟友背叛、被拖入无关战争 |
| **征战扩张** | 军力充足，资源压力>60 | 军队、财政、人口 | 领土扩大、资源获取、影响力提升 | 战争消耗、国际孤立、内部反战情绪 |
| **培养天才** | 教育资源充足，稳定度>70 | 教育投入、导师时间 | 传奇级NPC产出、科技突破概率提升 | 天才叛逃、培养失败 |
| **探索遗迹** | 有探险队，科技水平>50 | 探险队、物资、时间 | 起航者技术、古代魔法、资源发现 | 遗迹陷阱、畸变体攻击、精神污染 |
| **建造避难所** | 魔潮预警，资源充足 | 大量资源、工程时间 | 魔潮生存率提升、文明火种保存 | 资源枯竭、建造失败、被其他势力觊觎 |

### 2.5.2 势力行为决策树（以塞西尔帝国为例）

```
当前状态：稳定度85，资源压力60，科技水平90，神明干涉度10，魔潮距离3年

决策流程：
1. 检查最高优先级需求 → 对抗魔潮（优先级=0.85×1.25×1.0=1.06）
2. 评估可行行为：
   - 建造避难所：消耗大（资源压力+20），但产出明确（生存率+30%）
   - 发展科技（魔导防护）：消耗中等，产出不确定（可能失败）
   - 结盟（联合其他势力）：消耗低，产出中等（资源共享）
3. 选择最优行为：建造避难所（虽然资源压力上升，但魔潮迫在眉睫）
4. 执行行为 → 更新状态：资源压力80，魔潮生存率+30%
```
