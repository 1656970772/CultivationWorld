## 世界 World

### 世界静态数据

| 属性 | 说明 |
| --- | --- |
| 世界观名称 | 一念永恒（通天-永恒-星空 三层宇宙） |
| 核心能量 | 灵气（基础）、念力（高阶，贯穿天人以上境界） |
| 层级结构 | 通天世界（下游/中游/上游/生命禁区/蛮荒）→ 永恒大陆（三大皇朝/五大仙域/永恒之花）→ 星空领域（一百零八万大世界） |

| 时间线跨度 | 从凝气到永恒境，跨越通天世界、永恒大陆、星空领域三个时代 |

### 世界实体

#### 1. 通天世界（通天界）

通天世界是故事的主要舞台，由通天海与通天河构成，灵气浓度自中心向边缘递减。世界分为三层，层级越高灵气越稀薄但势力越强。

**a. 下游 — 四大宗门区域**
- **定位**：东脉下游，通天世界灵气最稀薄区域
- **代表宗门**：灵溪宗、血溪宗、玄溪宗、丹溪宗
- **特点**：宗门间为争夺灵脉、洞府等资源竞争激烈，是白小纯的起点
- **关键地标**：灵溪宗南岸（驭物）、灵溪宗北岸（驭兽）、血溪宗（血祖残躯之上）、玄溪宗山门、丹溪宗丹谷

**b. 中游 — 四大学院区域**
- **定位**：位于下游之上，灵气与资源远优于下游
- **代表学院**：空河院、星河院、道河院、极河院
- **特点**：上游大宗门的人才培养基地，下游宗门仰望的上位目标
- **关键地标**：空河院主院、星河院星脉

**c. 上游 — 四大源头宗门区域**
- **定位**：通天世界灵脉与传承的源头，掌控世界命脉
- **代表宗门**：
  - 东脉：星空道极宗（综合实力最强）
  - 南脉：龙腾鬼海宗（驱使灵鬼作战）
  - 西脉：泰斗撼月宗（月之力功法）
  - 北脉：九天云雷宗（雷电攻击）
- **特点**：掌控大量灵脉、传承和秘境

**d. 蛮荒**
- **定位**：环绕通天大陆的未开发地带
- **特点**：生存环境恶劣，栖息着巨人、巨禽等强大蛮荒生物
- **功能**：宗门历练、获取资源的危险区域；部分剧情（如巨鬼城）发生地
- **关键地标**：巨鬼城、巨人领地、上古战场遗迹

**e. 生命禁区**
- **定位**：通天大陆边缘的绝地
- **特点**：灵气紊乱，存在强大禁制，天然屏障阻止蛮荒入侵
- **功能**：顶级强者闭关或封印之地

#### 2. 永恒大陆

通天世界崩溃后，幸存者迁徙到的高阶世界，是太古境及以上强者的舞台。

**a. 三大皇朝**
- **邪皇朝**：占据三块仙域，信奉邪祖（逆凡分身），前期主要敌对势力
- **圣皇朝**：占据两块仙域，信奉圣祖，与邪皇朝长期对峙
- **魁皇朝**：由白小纯以原通天界修士为核心建立，代表魁祖一脉，是第三极势力

**b. 五大仙域**
- **来源**：永恒之花的五片花瓣所化
- **分布**：邪皇朝占三块、圣皇朝占两块
- **功能**：承载永恒大陆的文明与修炼资源

**c. 永恒之花**
- **定位**：永恒大陆的世界本源
- **特性**：传说能吞噬世界寂灭后诞生
- **重要性**：逆凡与白小纯的终极争夺目标；五片花瓣化为五大仙域

#### 3. 星空领域

永恒大陆之上的更高层次世界，由一百零八万大世界构成的庞大星空体系。

- **主宰境**：此层级的最高战力，掌控一方乃至多方可怕的世界意志，可碾压太古境。分为需世界认可的和以自身之道成就的"最强主宰"
- **永恒境**：超越主宰的终极境界，需融合星空一百零八万本源，突破一切封印，达到"罗天"层次，可创造规则，超越时空

### 世界行为/规则

#### 通天河规则
- 通天河贯穿通天世界，是灵气流通的主干道
- 自上游向下游流淌，灵气浓度随流域递减
- 通天河水具有特殊力量，可用于炼器和修炼

#### 生命禁区规则
- 灵气紊乱区域，常规修士无法长驻
- 存在天然或人为的强大禁制
- 阻止蛮荒生物大规模入侵的天然屏障
- 天尊级以上强者可在此闭关或封印

#### 念力规则

| 境界 | 念力运用方式 | 说明 |
| --- | --- | --- |
| 天人 | 融己意于天意 | 天人交感，引天地之力为己用，可初步影响一方天地 |
| 半神 | 自身成为"念根" | 凌驾天意，可调动世界规则 |
| 天尊 | 借众生之念修行 | 凝聚道种，借众生之念修行，是通天世界的巅峰 |
| 太古 | 以自身道念凌驾天地 | 超越天尊，可覆盖世界道念，复活天尊，是世界级主宰 |
| 主宰 | 掌控世界意志 | 星空领域的顶级存在，掌控一方乃至多方可怕的世界意志 |
| 永恒 | 创造规则 | 融合星空一百零八万本源，超越时空 |

#### 永恒之花规则
- 吞噬世界寂灭后方能诞生
- 五片花瓣可化为五大仙域
- 谁能获得永恒之花的认可，即为"永恒之子"
- 是通往永恒境的关键钥匙

#### 星空主宰规则
- 星空领域由一百零八万大世界构成
- 主宰分为两类：获得世界认可的常规主宰，和以自身之道成就的"最强主宰"
- 最强主宰可碾压同境，拥有不依赖世界意志的独立力量

#### 永恒境规则
- 超越主宰的终极境界
- 需融合星空一百零八万本源
- 突破一切封印与限制
- 达到"罗天"层次，可创造规则、超越时空
