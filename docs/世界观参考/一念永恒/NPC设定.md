### NPC 静态数据

#### 角色类型枚举

| 类型 | 说明 | 代表角色 |
| --- | --- | --- |
| 修士 | 常规修炼者，涵盖凝气→元婴各境界 | 李青候、张大胖、鬼牙 |
| 天人 | 天人境修士，初步掌控天地之力 | 各宗门掌教级 |
| 半神 | 半神境强者，成为"念根" | 上游宗门高层 |
| 天尊 | 通天世界巅峰，凝聚道种 | 杜凌菲之父（天尊级） |
| 太古 | 超越天尊，世界级主宰 | 逆凡、白小纯（后期） |
| 主宰 | 星空领域顶级存在 | 各方主宰 |
| 蛮荒生物 | 蛮荒区域的强大生物 | 巨人、巨禽、血祖 |
| 炼丹师 | 专精炼丹的角色 | 白小纯（炼丹天才） |
| 诡异存在 | 非正常生命形态 | 鬼母（公孙婉儿被夺舍后） |
| 皇朝成员 | 永恒大陆皇朝体系角色 | 邪皇朝/圣皇朝/魁皇朝成员 |

**白小纯需求驱动的 GOAP 行为链：**

```
怕死求长生（核心需求）
    │
    ├──→ 炼制长生灵丹（收集灵草→炼丹→服用）
    │
    ├──→ 修炼不死长生功（寻找残卷→修炼→突破）
    │
    ├──→ 寻找龟纹锅炼灵（获取龟纹锅→收集材料→炼灵三十次）
    │
    ├──→ 建立势力保护自己（整合灵溪/血溪/丹溪→逆河宗→魁皇朝）
    │
    ├──→ 争夺永恒之花（进入永恒大陆→对抗邪祖→成为永恒之子）
    │
    └──→ 突破永恒境（融合星空本源→超越时空）
```

### NPC 状态

| 状态维度 | 说明 | 典型变化 |
| --- | --- | --- |
| 存活 | 是否存活 | 李青候（存活贯穿主线） |
| 年龄/寿元 | 当前年龄与剩余寿元 | 凝气→筑基寿元翻倍，元婴千年寿元 |

| 心境 | 当前心魔/心境状态 | 突破瓶颈时心境波动 |
| 修炼进度 | 当前境界 + 距下一境界的进度 | 突破边界条件（丹药/契机/战斗） |
| 叛逃倾向 | 可能脱离当前势力的风险 | 受胁迫或利益诱惑时产生 |

### NPC 需求

#### 白小纯核心需求驱动的 GOAP 行为链

```
怕死求长生
    ├──→ 获得不死长生功 → 修炼残卷 → 寻找完整版 → 突破
    │                                          │
    │                                          └──→ 血祖残躯中获取后续功法
    │
    ├──→ 成为最强炼丹师 → 收集灵草 → 炼制长命丹 → 突破境界
    │                          │
    │                          └──→ 引发丹道异象（获取声望+资源）
    │
    ├──→ 获取龟纹锅 → 炼灵一次 → 炼灵二次 → …… → 炼灵三十次
    │       │              │                     │
    │       └──→ 每次炼灵提升法宝/丹药品质 ───→ 累积质变
    │
    ├──→ 建立个人势力 → 整合灵溪/血溪/丹溪 → 逆河宗 → 攻克中游
    │                                                        │
    │                                                        └──→ 魁皇朝
    │
    └──→ 争夺永恒之花 → 对抗邪祖 → 永恒之子 → 永恒境
```

### NPC 行为

| 行为类型 | 说明 | 实例 |
| --- | --- | --- |
| 修炼 | 日常吸收灵气/念力突破境界 | 所有修士的日常行为 |

| 建立势力 | 组建/整合新势力 | 白小纯整合灵溪/血溪/丹溪为逆河宗 |

### 消耗/产出关系（NPC 层面）

```
以白小纯为例：

  产出                              消耗
  ┌─────────────────┐              ┌─────────────────┐
  │ 特殊丹药（炼丹）  │              │ 灵草/灵石        │
  │ 炼灵物品（炼灵）  │              │ 炼灵材料         │
  │ 势力整合（领导）  │              │ 时间/精力        │
  │ 功法创新（修炼）  │              │ 不死功残卷寻找   │
  │ 声望/信仰（念力） │              │ 寿元（被追杀）   │
  └─────────────────┘              └─────────────────┘

  白小纯的独特循环：
  怕死 → 炼丹/炼灵/修炼 → 变强 → 招惹更强敌人 → 更怕死 → 更拼命变强
```
