## 物品 Item

### 物品静态数据

#### 物品分类体系

```
物品体系
├── 资源类
│   ├── 灵石（下品/中品/上品/极品）
│   ├── 灵脉（小型/中型/大型灵脉）
│   ├── 通天河水
│   ├── 念力（众生之念，天人以上境界核心资源）
│   ├── 灵草（一至九阶）
│   └── 妖兽材料（皮毛/骨骼/妖丹）
│
├── 道具
│   ├── 法宝（按品阶：法器→宝器→灵器→道器→仙器→神器）
│   │   ├── 攻击型
│   │   ├── 防御/辅助型
│   │   └── 特殊型
│   ├── 功法（基础功法/宗门传承/上古功法/自创功法）
│   ├── 丹药（一至九阶 + 特殊丹药）
│   ├── 阵法（护宗/杀阵/封印/传送/特殊）
│   └── 灵兽/战兽（普通→精英→王兽）
```

### 物品状态

| 状态维度 | 说明 | 实例 |
| --- | --- | --- |
| 品阶 | 物品当前品阶（可被炼灵提升） | 丹药经炼灵后从五阶升六阶 |
| 耐久度 | 法宝使用次数/损耗 | 血祖剑使用过多需血气滋养修复 |
| 炼灵次数 | 经过龟纹锅炼灵次数（上限三十次） | 龟纹锅炼灵一次→二次→……→三十次 |
| 绑定状态 | 是否已认主 | 法宝认主后他人无法使用 |
| 灵气含量 | 灵石/灵脉剩余灵气 | 灵脉过度开采后降级 |
| 灵草成熟度 | 灵草生长阶段 | 幼苗→成熟→千年→万年 |
| 灵兽忠诚度 | 灵兽对主人的忠诚 | 铁蛋对白小纯忠诚度极高 |
| 功法完整度 | 残卷/完整版 | 不死长生功初为残卷→后补全 |

### 物品需求

| 需求类型 | 说明 | 优先级 |
| --- | --- | --- |
| 生产/采集 | 灵草需要生长环境、妖兽材料需要猎杀 | 基础需求 |
| 炼制 | 丹药/法宝需要对应品阶的材料 + 炼丹/炼器技能 | 核心需求 |
| 炼灵 | 龟纹锅 + 特定材料 → 可炼灵三十次 | 白小纯专属需求 |
| 消耗 | 修炼消耗灵石、战斗消耗丹药 | 持续需求 |
| 交易 | 灵石作为通用货币，可换取所需物品 | 经济需求 |
| 掠夺 | 战争/打劫获取对手物品 | 冲突需求 |

### 物品行为

| 行为类型 | 说明 | 产出 | 消耗 |
| --- | --- | --- | --- |
| 生产/采集 | 从自然界获取原材料 | 灵草、灵石、妖兽材料 | 时间、人力 |
| 炼丹 | 将灵草炼制成丹药 | 各类功效丹药 | 灵草、丹炉耐久 |
| 炼器 | 打造/强化法宝 | 新法宝 | 妖兽材料、灵石 |
| **炼灵** | **龟纹锅对物品进行炼灵（1~30次）** | **物品品质跃升一档** | **特定材料 + 时间** |
| 消耗 | 使用灵石/丹药 | 修为提升/伤愈 | 灵石/丹药本身 |
| 交易 | 物品交换 | 等价物品/灵石 | 交易物品 |
| 掠夺 | 强行夺取 | 敌方物品 | 战力消耗 |

### 消耗/产出关系（物品层面）

```
物品消耗/产出网络

                        ┌──────────────────┐
                        │    灵脉/自然环境   │
                        │   （万物之源）     │
                        └────┬─────────────┘
                             │
               ┌─────────────┼─────────────┐
               ▼             ▼             ▼
         灵草种植       灵石开采      妖兽猎杀
               │             │             │
               ▼             ▼             ▼
         灵草(产出)    灵石(产出)    妖兽材料(产出)
               │             │             │
               └──────┬──────┘             │
                      ▼                    │
                 炼丹(消耗灵草+灵石)        │
                      │                    │
                      ▼                    │
                 丹药(产出)                │
                      │                    │
               ┌──────┴──────┐             │
               ▼             ▼             │
         修士服用       交易换灵石          │
         (提升境界)     (经济循环)          │
               │             │             │
               ▼             ▼             │
         战力提升 ←── 购买功法/法宝 ←───────┘
               │
               ▼
         获取更多资源（正向循环）


【炼灵特殊循环】（白小纯专属）

  任意物品(输入) + 特定材料 + 龟纹锅
         │
         ▼
  炼灵一次 → 品质+1档 → 可继续炼灵
         │
         ▼
  炼灵二次 → …… → 炼灵三十次(极限)
         │
         ▼
  超品阶物品(输出)
```
