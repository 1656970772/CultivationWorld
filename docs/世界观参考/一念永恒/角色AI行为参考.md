# 《一念永恒》角色AI行为参考

## AI架构映射说明

本文档将角色特性映射到游戏AI四层架构：
- **BT（Behavior Tree）层**：情绪反应与应急抢占，对应角色的即时反应模式
- **Utility层**：长期目标选择与权重评分，对应角色的核心驱动力排序
- **GOAP层**：目标分解为步骤序列，对应角色关键事件的决策链
- **Action Queue**：执行具体动作

---

## 核心角色

### 一、白小纯

#### 性格标签

贪生怕死 / 极度护短 / 机灵狡黠 / 重情重义 / 喜剧外壳下的认真内核

#### BT应急反应

```
[感知：任何形式的威胁/危险]
  → [第一反应：跑路/躲避]（无条件优先）
    ├─ [能跑掉] → [溜之大吉] → [事后合理化：战略性撤退]
    └─ [跑不掉/退无可退]
        → [恐惧阈值突破] → [潜能爆发]
          ├─ [强行提升肉身强度]（铜皮铁骨/不死血）
          ├─ [无厘头但有效的骚操作]（炸丹炉/乱扔丹药/嘴炮）
          └─ [歪打正着] → [意外破局]
[危机解除后]
  → [立刻切换模式：邀功/抱大腿]
    ├─ [对强者] → [献殷勤/表忠心/套近乎]
    └─ [对同伴] → [吹嘘/炫耀/要求认可]
```

**参数配置**：
| 参数 | 值 | 说明 |
|------|-----|------|
| flee_threshold | 0.01 | 极低触发阈值（任何风吹草动都想跑） |
| flee_success_prob | 0.6 | 跑路成功率（不低，因为跑得快） |
| panic_power_multiplier | 2.5 | 恐惧激发下的战力倍率 |
| lucky_break_prob | 0.35 | "歪打正着"事件触发概率 |
| brag_after_crisis | true | 危机解除后必定邀功 |
| flee_to_fight_transition | forced | 跑不掉才打，从不主动选择战斗 |

#### Utility长期目标权重

| 目标 | 权重 | 触发条件 | 说明 |
|------|------|----------|------|
| 怕死→长生 | 1000 | 永久激活 | 一切行为的底层驱动力 |
| 守护身边人 | 800 | 亲友受威胁时 | 侯小小/宋君婉/宗门受威胁时权重飙升 |
| 贪财/资源 | 600 | 看到值钱东西 | 日常行为的主要驱动力 |
| 出风头/被认可 | 500 | 有人围观 | 虚荣心驱动 |
| 偷懒/享乐 | 400 | 无紧急目标 | 能躺着绝不坐着 |

**权重交互修正**：
- 当"守护身边人"和"怕死"冲突时 → 触发`inner_struggle`状态（15%概率恐惧战胜守护欲→后悔→爆发）
- `recognition_seeking`：被认可后，"守护身边人"权重 +100（正向反馈循环）
- `embarrassment_factor`：在美女面前，"出风头"权重 × 1.5

#### GOAP标志事件及决策链

##### 事件1：灵溪宗入门——靠"苟"与"钻营"生存

```
[初始状态：凡人少年，想修仙长生]
  → [子目标：进入修仙门派，获得庇护与资源]
    → [步骤1：寻找最安全、最不容易死的宗门] → 选择灵溪宗
    → [步骤2：讨好长老/师父]
      ├─ 表现乖巧听话（外表）
      ├─ 展示炼丹天赋（真本事）
      └─ 建立"无害"人设
    → [步骤3：日常偷懒但高效完成杂役]
      ├─ 用外物/机关代替体力劳动
      ├─ 发现并利用规则漏洞
      └─ 绝不主动招惹麻烦
    → [步骤4：以炼丹天赋换取保护]
      ├─ 为长老/师兄炼制丹药
      ├─ 建立"有用所以不能被抛弃"的地位
      └─ 丹药意外效果好 → 名声扩大
    → [步骤5：稳步提升修为，同时始终保持跑路准备]
    → [结果：在灵溪宗站稳脚跟，开启修行之路]
```

**GOAP步骤特征**：
- `step_type`：全部为`opportunistic`（机会主义型），无`aggressive`（攻击型）
- `risk_aversion`：per_step（每一步都评估风险，随时可中断）
- `fallback_plan`：每一步都有"如果出事就跑"的备选方案

##### 事件2：血溪宗卧底——一人搅动风云

```
[触发器：接受宗门任务，潜入血溪宗]
  → [子目标：完成卧底任务，同时避免身份暴露]
    → [步骤1：伪装身份，建立新人设]
      ├─ 低调进入血溪宗
      ├─ 观察血溪宗规则与文化
      └─ 设定伪装性格
    → [步骤2：靠"无耻"但有效的方法积累影响力]
      ├─ 种菜（用灵植换取人脉）
      ├─ 收小弟（用丹药/好处笼络人心）
      ├─ 搞出各种"意外"（炼丹爆炸/功法变异）
      └─ 每次意外都歪打正着 → 声望提升
    → [步骤3：多次化解身份暴露危机]
      ├─ 被人怀疑时 → 反向操作（主动靠近怀疑者）
      ├─ 证据出现时 → 制造更大的事件转移注意力
      └─ 身份将暴露时 → 利用"不可或缺"的价值换取宽容
    → [步骤4：最终带走血溪宗核心传承]
      ├─ 在撤离前获取最大收益
      └─ 与血溪宗部分人建立真感情（增加复杂度）
    → [结果：完成任务，带回传承，留下一段传奇]
```

**关键决策点**：
- `DECISION_IDENTITY_CRISIS`：身份将暴露 → 评估：`自身价值 × 对方包容度 > 暴露风险` → 若大于则赌一把留下，否则立刻跑路
- `DECISION_LOOT`：撤离前是否带走传承 → 评估：`(传承价值 × 得手概率) - (被抓风险 × 惩罚强度)` → 净利润为正则执行

##### 事件3：巨鬼城"白大善人"——以送死为名的变强

```
[触发器：被逼婚，需要逃避同时提升实力]
  → [子目标：避免被逼婚，在混乱中变强]
    → [步骤1：假装配合逼婚安排]
      ├─ 表面上接受安排
      ├─ 暗中寻找脱身机会
      └─ 建立"老实人"假象
    → [步骤2：搞出各种"意外"事件]
      ├─ 炼丹时"不小心"炸丹炉 → 波及周围 → 没人敢靠近
      ├─ "误打误撞"改良功法 → 引起高层注意
      ├─ 各种无厘头操作 → 把水搅浑
      └─ 浑水摸鱼，默默吸收资源
    → [步骤3：在混乱中积累实力与声望]
      ├─ 每次意外后实力都诡异提升
      ├─ 逐渐从"麻烦制造者"变成"不可忽视的人物"
      └─ 其他人对他的态度：头疼但无奈→重视→依赖
    → [步骤4：最终成为巨鬼城核心人物]
      ├─ 通过"白大善人"名号建立正面形象
      └─ 完成从"被逼婚的逃兵"到"城内支柱"的转变
    → [结果：逼婚危机解除，实力大幅提升，获得新身份]
```

**行为模式标签**：`浑水摸鱼` `反向操作` `以退为进` `歪打正着`

##### 事件4：对抗通天世界——从逃避到担当

```
[触发器：故乡与亲友面临通天世界的威胁]
  → [子目标：保护故乡与亲人，避免世界毁灭]
    → [阶段1：初期逃避责任]
      ├─ "这么大的事跟我有什么关系"
      ├─ 试图假装看不到危机
      └─ 被亲友逼迫面对现实
    → [阶段2：亲友受威胁后觉醒]
      ├─ 触发"守护身边人"权重飙升
      ├─ 从被动逃避转为主动担当
      └─ 恐惧转化为愤怒与决心
    → [阶段3：整合各方力量]
      ├─ 利用在各宗门/势力积累的人脉
      ├─ 以前建立的"白大善人"等身份发挥作用
      ├─ 以"非典型"方式说服/哄骗各方参战
      └─ 整合资源与战力的效率远超传统方式
    → [阶段4：以"非典型"方式对抗上位世界]
      ├─ 谈判（以长生丹为筹码）
      ├─ 偷袭（不按常理出牌）
      ├─ 心理战（利用对方对"疯子"的忌惮）
      └─ 关键时刻正面硬刚（退无可退时的爆发）
    → [结果：成功保护故乡，成就永恒]
```

**角色成长弧线**：
- `character_arc`：`贪生怕死的少年 → 被迫担当的英雄 → 主动守护的永恒者`
- `turning_point`：亲友受威胁的瞬间，`inner_struggle`状态结束，守护权重首次超越怕死权重

#### 白小纯的独特性分析

| 维度 | 白小纯 | 传统修仙主角 | AI设计启示 |
|------|--------|-------------|------------|
| 驱动力 | 恐惧驱动（怕死→长生） | 变强/复仇/正义驱动 | Utility根节点设为"生存恐惧"而非"变强欲望" |
| 应急模式 | 跑路→退无可退→爆发 | 硬刚/智取/正面对抗 | BT层优先检查逃跑路径，逃不掉才战斗 |
| 成长方式 | 歪打正着、浑水摸鱼 | 苦修/机缘/战斗突破 | 引入`serendipity_system`（意外发现系统） |
| 社交模式 | 讨好/抱大腿/建立"有用性" | 结盟/收小弟/建立势力 | 社交行为以"建立不可替代性"为核心策略 |
| 喜剧性 | 言行不一的搞笑反差 | 严肃/紧张 | AI对话中加入"内心OS"层，与表面言行形成反差 |

**草根崛起逻辑链**：
```
[无背景无靠山]
  → [必须靠自己] 
    → [怕死所以格外小心] 
      → [小心所以活得久] 
        → [活得久所以积累多] 
          → [积累多所以实力强] 
            → [实力强所以不用怕了] 
              └─→ 但习惯性怕死已经刻入骨髓 → 永远保持警惕 → 永远变强
```

### 二、李青候

#### NPC类型：严师型 / 守护者+领路人

#### 性格标签

护短严厉 / 外冷内热 / 极具责任感 / 原则性强 / 包容但有限度

#### BT应急反应

```
[感知：弟子（白小纯）遇险]
  → [第一反应：不惜一切代价庇护]
    ├─ [小危机] → [旁观] → [让弟子自己解决] → [兜底准备就绪]
    ├─ [中危机] → [暗中出手] → [事后严厉训斥] → [但继续守护]
    └─ [大危机] → [正面出手] → [不惜与整个宗门/势力为敌]
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 守护灵溪宗 | 950 | 宗门利益至上 |
| 庇护弟子（白小纯） | 900 | 护短本能，与宗门利益几乎等同 |
| 培养优秀弟子 | 750 | 对弟子寄予厚望 |
| 维护宗门规矩 | 600 | 原则性需求 |

#### 标志事件

- 多次包容白小纯的"胡闹"，每次都在底线边缘踩刹车
- 在宗门危机时挺身而出，作为灵溪宗的支柱力量
- 是白小纯早期最重要的守护者与领路人，建立白小纯对"师父"的信任模型
- 与白小纯的关系动态：`失望→无奈→重新认识→欣慰→骄傲`

#### AI设计要点

- **关系发展曲线**：初始严厉→中期磨合→后期信任与骄傲；对应对话态度逐渐软化
- **行为模式**：对白小纯有专属的`tolerance_bonus`（容忍值加成），其他弟子犯错惩罚 × 1.0，白小纯犯错惩罚 × 0.3
- **关键触发**：听到其他人说白小纯坏话 → 触发护短模式 → 不管对方说的是否属实，先维护再说

### 三、杜凌菲

#### NPC类型：正义型NPC / 傲娇型

#### 性格标签

冷艳高傲 / 内心柔软 / 对正义有执着追求 / 是非分明 / 情感表达笨拙

#### BT应急反应

```
[感知：不公事件/无辜者受害]
  → [第一反应：出手保护无辜者]
    ├─ [评估对方行为是否符合"正义"] → 是 → [帮助对方]
    └─ [评估对方行为是否符合"正义"] → 否 → [保持距离] → [但暗中观察]
[涉及白小纯时]
  → [行为出现内部冲突]
    ├─ 理性判断：这人太无耻 → 应该远离
    └─ 情感反应：但他也不是坏人 → 暗中帮助
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 守护心中正义 | 850 | 核心价值驱动 |
| 追求大道 | 700 | 自身修行 |
| 暗中帮助白小纯 | 550 | 嘴上不承认但权重不低 |

#### 标志事件

- 初期对白小纯印象极差，认定其为"无耻之徒"
- 在相处中逐渐被白小纯的真诚与关键时刻的可靠打动
- 多次在暗中帮助白小纯，但嘴上从不承认
- 情感线复杂——典型的"傲娇"模式：行为与言语相反

#### AI设计要点

- **双层对话系统**：表面对话（冷冰冰的否定） + 内心标记（好感的累积）
- **行为矛盾**：`stated_intent`（声称的意图）≠ `actual_action`（实际行为）
- **好感度阈值事件**：好感度积累到特定阈值时触发特殊剧情（如首次开口承认关心）

### 四、宋君婉

#### NPC类型：霸道女强型 / 情感支撑型

#### 性格标签

霸道女总裁式 / 口是心非 / 对白小纯又爱又恨 / 能力出众 / 情感表达直接但别扭

#### BT应急反应

```
[涉及白小纯时]
  → [情绪优先，理性靠后]
    ├─ [白小纯闯祸] → [暴怒] → [嘴上骂骂咧咧] → [立刻动手收拾烂摊子]
    ├─ [白小纯有危险] → [焦急] → [失去冷静] → [冲动但有效的决策]
    └─ [白小纯邀功] → [嘴上不屑] → [内心认可] → [暗爽]
[不涉及白小纯时]
  → [恢复正常判断力] → [冷静果决的女强者]
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 白小纯相关事务 | 850 | 情感驱动，权重不稳定（波动大） |
| 自身事业/势力 | 700 | 在不涉及白小纯时权重最高 |
| 外在形象（强势） | 600 | 不愿在他人面前示弱 |

#### 标志事件

- 与白小纯从"假夫妻"到真感情的发展
- 多次在白小纯闯祸后为他收拾烂摊子，是白小纯后期重要的情感支撑
- 典型互动模式：白小纯闯祸→宋君婉暴怒→收拾烂摊子→白小纯感动→宋君婉嘴硬→循环

#### AI设计要点

- **情感波动曲线**：对白小纯的好感度不是线性增长，而是锯齿状波动上升
- **行为预测**：白小纯闯祸后，宋君婉有90%概率在3个回合内出现
- **专属互动标签**：`angry_but_helping`（嘴上骂但手上帮）、`denial_mode`（否认关心模式）

## NPC类型化归纳

### 类型1：苟道主角型

**代表角色**：白小纯

| AI参数 | 值 | 说明 |
|--------|-----|------|
| survival_drive_priority | 最高 | 生存恐惧驱动一切行为 |
| flee_first | true | 应急反应永远是先跑 |
| serendipity_factor | 0.35 | 歪打正着事件触发概率 |
| social_strategy | ingratiation | 讨好型社交（抱大腿） |
| growth_style | opportunistic | 机会主义型成长路径 |
| comic_relief | true | 自带搞笑光环 |

**行为特征**：
- BT层以"检测逃生路径"为最高优先级
- Utility层以"活得更久更舒服"为终极目标
- GOAP中所有计划都有"失败即跑"的fallback节点
- 适合作为玩家角色或极具辨识度的核心NPC

**设计建议**：此类角色的AI系统需要独立的`侥幸系统（serendipity_system）`来模拟"歪打正着"，否则行为逻辑会断裂。

**代表角色**：李青候

**设计建议**：此类NPC的对话系统应包含"严厉训斥→内心欣慰"的双层结构，通过行为而非言语传递关心。

### 类型4：搞笑催化剂型

**代表角色**：各路配角中的搞笑担当

| AI参数 | 值 | 说明 |
|--------|-----|------|
| logic_bypass_prob | 0.4 | 行为脱离正常逻辑的概率 |
| plot_acceleration | true | 其行为以意外方式推动剧情 |
| comic_timing | 关键节点 | 在紧张时刻制造笑点 |
| survival_immunity | 部分 | 搞笑角色死亡率低 |

**行为特征**：
- 行为难以用常规Utility系统预测
- 经常在关键时刻以意外方式改变局势
- 作为主角应急反应的"催化剂"——其荒诞行为触发主角的灵感或行动

**设计建议**：此类角色适合使用独立的`random_chaos_engine`，偶尔产出偏离常规AI逻辑的"神来之笔"，但要控制频率防止破坏游戏沉浸感。

**设计建议**：此类角色的Utility系统中应有一个独立的`moral_compass`模块，随剧情推进调整参数。

## 白小纯AI系统的特殊设计建议

### 侥幸系统（Serendipity System）

白小纯的核心特征"歪打正着"需要一个独立的系统来模拟：

```
[白小纯执行某个非战斗行为]
  → 检查行为类型：
    ├─ [炼丹] → serendipity_check(0.25) → 触发则产出意外效果
    ├─ [逃跑] → serendipity_check(0.15) → 触发则意外撞见机缘
    ├─ [讨好] → serendipity_check(0.20) → 触发则被误认为高深莫测
    └─ [闯祸] → serendipity_check(0.30) → 触发则意外解决更大问题
```

`serendipity_check(prob)` = 以概率 prob 触发"意外好运"，效果强度与当前危机严重度正相关。

### 内心OS系统（Inner Monologue）

白小纯的喜剧性来源于外表行为与内心真实想法的反差：

```
[表面行为]               [内心OS]
"长老您放心！"    vs     "千万别让我去危险的地方……"
"我来！"          vs     "我一点都不想来！！"
"不痛不痛……"     vs     "痛死了呜呜谁来救我"
```

建议在AI对话系统中并行维护两套输出：`surface_dialogue`（表面台词）和`inner_monologue`（内心OS），在特定UI位置同时展示以制造喜剧效果。内心OS在恐惧/紧张时激活，在真正愤怒/认真时关闭——关闭内心OS是白小纯"认真了"的信号。
