## 二、势力 Faction

### 2.1 势力静态数据

#### 2.1.1 势力类型枚举

| 类型 | 说明 | 代表 |
|------|------|------|
| **玄门正宗** | 传承有序的正道宗门，占据灵脉福地 | 溟沧派、少清派、玉霄派 |
| **玄门旁支** | 规模较小或传承特殊的玄门 | 南华派、平都教、还真观、骊山派、补天阁 |
| **魔门** | 修炼魔功邪法的宗门 | 魔门六宗 |
| **散修联盟** | 无宗门归属的散修组成的松散组织 | 贞罗盟 |
| **跨界教派** | 跨越界域传播道统的教派 | 演教 |
| **天庭势力** | 涉及帝位之争的高层势力 | 天庭、离忘山 |
| **世家** | 以血脉传承为核心的家族势力 | 陈氏、韩氏、周氏等 |
| **上古遗族** | 神怪血裔、妖魔部族等 | 大鲲、蟒精等 |

### 2.3 势力状态

每个势力的状态是动态变化的，核心状态维度包括：

| 状态维度 | 说明 | 取值范围 |
|----------|------|----------|
| **稳定度** | 内部团结程度和外部威胁程度 | 动荡→稳定→鼎盛 |
| **灵机压力** | 势力所控灵脉的灵机充裕程度 | 枯竭→紧张→充裕 |
| **洞天数量** | 势力拥有的洞天数量 | 0→若干 |
| **高阶修士数量** | 洞天真人及以上修士数量 | 直接影响势力战力 |
| **跨界影响力** | 在其他界域的影响力 | 无→局部→全域 |
| **声望** | 在修真界的名望和号召力 | 默默无闻→威震九洲 |

### 2.4 势力需求

| 需求类型 | 说明 | 优先级计算逻辑 |
|----------|------|----------------|
| **生存** | 确保宗门不被灭门 | 最高优先级，当面临灭门威胁时覆盖所有其他需求 |
| **资源** | 获取灵脉、天材地宝、丹药等 | 灵机压力越大优先级越高 |
| **扩张** | 扩大势力范围和控制区域 | 资源充裕时优先级提升 |
| **传承** | 培养弟子、延续道统 | 高阶修士寿元将尽时优先级最高 |
| **跨界发展** | 向其他界域扩展影响力 | 当前界域资源枯竭时触发 |
| **大道博弈** | 参与真阳层面的大道权柄争夺 | 仅真阳级势力适用 |

### 2.5 势力行为

| 行为 | 说明 | 触发条件 |
|------|------|----------|
| **开山立派** | 创建新宗门 | 高阶散修积累足够资源 |
| **收徒传法** | 招收弟子、传授功法 | 宗门传承需求 |
| **争夺资源** | 与其他势力争夺灵脉、天材地宝 | 灵机压力大 |
| **布设大阵** | 布置护山大阵或攻击阵法 | 防御或进攻需求 |
| **跨界扩张** | 向其他界域扩展势力 | 跨界发展需求 |
| **大道博弈** | 参与真阳层面博弈 | 真阳级势力 |
| **联盟/敌对** | 与其他势力结盟或宣战 | 战略需要 |
| **迁徙** | 举派迁徙至新界域 | 灵机枯竭 |

### 2.6 消耗/产出关系（势力层面）

```
宗门资源池
  ├── 输入
  │   ├── 灵脉开采 → 灵机/灵石
  │   ├── 弟子贡献 → 天材地宝/灵石
  │   ├── 征战掠夺 → 战利品
  │   └── 洞天产出 → 灵药/灵材
  │
  ├── 消耗
  │   ├── 弟子修炼 → 灵机/丹药
  │   ├── 炼丹炼器 → 天材地宝/灵机
  │   ├── 维护大阵 → 灵石/灵机
  │   ├── 开辟洞天 → 大量灵机
  │   └── 跨界行动 → 灵机/资源
  │
  └── 产出
      ├── 高阶修士（洞天真人、凡蜕、真阳）
      ├── 丹药/法宝/阵法
      ├── 洞天福地
      ├── 道统传承
      └── 跨界影响力
```
