## 三、NPC

### 3.1 NPC 静态数据

#### 3.1.1 角色类型枚举

| 类型 | 说明 | 典型境界范围 |
|------|------|-------------|
| **玄门修士** | 玄门正宗或旁支弟子 | 开脉→洞天→凡蜕→真阳 |
| **魔修** | 魔门修士 | 开脉→洞天→凡蜕 |
| **散修** | 无宗门归属的独立修士 | 开脉→元婴 |
| **真阳大能** | 开辟元气大海的宇宙级存在 | 真阳 |
| **炼神** | 触及大道本源的超然存在 | 炼神 |
| **至人** | 大道权柄的掌控者 | 至人 |
| **造化之灵** | 大道本源的映现 | 不可度量 |
| **神怪血裔** | 上古神兽或妖魔后裔 | 各异 |
| **转世者** | 大能转世重修 | 各异 |

### 3.3 NPC 状态

每个NPC具有以下动态状态维度：

| 状态维度 | 说明 | 示例 |
|----------|------|------|
| **存活** | 是否存活 | 存活/重伤濒死/已陨落 |
| **年龄/寿元** | 当前年龄和剩余寿元 | 寿元将尽时触发突破或传承行为 |
| **伤势** | 当前伤势状态 | 无伤/轻伤/重伤/道基受损 |
| **道基** | 修炼根基的稳固程度 | 稳固/受损/破碎 |
| **心境** | 当前心境状态 | 平静/波动/心魔缠身/顿悟 |
| **修炼进度** | 当前境界和突破进度 | 瓶颈期/突破中/刚突破 |
| **因果缠结** | 与他人的因果关联程度 | 因果缠结越深，气运关联越紧密 |
| **气运** | 当前气运强弱 | 气运旺盛时机缘频现，气运低迷时厄运连连 |

### 3.4 NPC 需求

NPC需求采用GOAP（目标导向行动规划）行为链模型：

| 需求类型 | 说明 | GOAP行为链示例 |
|----------|------|---------------|
| **生存** | 保命、疗伤、躲避追杀 | 受伤→寻找疗伤丹→闭关疗伤→恢复 |
| **修炼** | 提升修为、突破境界 | 瓶颈→寻找突破契机→获取丹药/机缘→突破 |
| **突破境界** | 冲击下一境界 | 积累足够→准备渡劫→渡劫→突破/失败 |
| **传承道统** | 收徒传法、延续道统 | 寿元将尽→寻找传人→收徒→传授功法 |
| **复仇** | 向仇敌复仇 | 仇恨→提升实力→寻找仇敌→复仇 |
| **保护亲友** | 保护道侣、弟子、同门 | 威胁出现→评估实力→出手保护/求援 |
| **掌控大道** | 争夺大道权柄（仅高阶NPC） | 真阳→参与大道博弈→争夺权柄→掌控大道 |

### 3.5 NPC 行为

| 行为 | 说明 | 典型触发 |
|------|------|----------|
| **修炼** | 日常修炼提升修为 | 常态行为 |
| **炼丹** | 炼制丹药 | 需要突破/疗伤/交易时 |
| **炼器** | 炼制或祭炼法宝 | 获得新材料/需要提升战力时 |
| **布阵** | 布置阵法 | 防御洞府/困敌/辅助修炼 |
| **征战** | 参与战斗或战争 | 势力冲突/个人恩怨 |
| **探索秘境** | 探索上古遗迹或秘境 | 寻找机缘/天材地宝 |
| **开辟洞天** | 开辟个人洞天 | 达到象相二重 |
| **跨界** | 穿越界域 | 迁徙/追杀/探索 |
| **收徒** | 招收弟子 | 传承需求 |
| **交易** | 与其他修士交换资源 | 获取所需物品 |

### 3.6 消耗/产出关系（NPC层面）

```
NPC个人资源
  ├── 输入
  │   ├── 宗门供给 → 灵机/丹药/功法
  │   ├── 个人采集 → 天材地宝
  │   ├── 征战缴获 → 战利品
  │   └── 机缘获得 → 气运/法宝/功法
  │
  ├── 消耗
  │   ├── 日常修炼 → 灵机/丹药/时间
  │   ├── 突破境界 → 大量灵机/丹药/气运
  │   ├── 炼丹炼器 → 天材地宝/灵机
  │   ├── 战斗 → 法力/丹药/法宝耐久
  │   └── 开辟洞天 → 巨量灵机/心神
  │
  └── 产出
      ├── 修为提升（境界突破）
      ├── 丹药/法宝/阵法
      ├── 洞天
      ├── 弟子（传承）
      └── 影响力/声望
```
