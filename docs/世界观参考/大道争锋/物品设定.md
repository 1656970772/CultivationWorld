## 四、物品 Item

### 4.1 物品静态数据

物品通用属性：
- **名称**：物品名称
- **类型**：所属分类
- **品级**：品质等级
- **来源**：获取方式（采集/炼制/交易/掠夺/传承）
- **效果**：使用效果描述
- **使用条件**：修为/功法/血脉等限制
- **耐久/消耗**：使用次数或消耗量

#### 4.2.2 道具-法宝

法宝等级体系（从低到高，共7级）：

| 等级 | 名称 | 特征 | 实例 |
|------|------|------|------|
| 1 | **法器** | 无灵性，威力依赖修士自身修为 | 普通飞剑、护身符 |
| 2 | **灵器** | 有微弱灵性，可温养提升威力 | 灵性飞剑、灵甲 |
| 3 | **玄器** | 灵智更高，威力远超灵器，有晋升真器的可能 | 高阶玄器飞剑 |
| 4 | **真器** | 化灵为人，有近乎象相真人之威，真灵可转生 | 阴戮刀（残破真宝） |
| 5 | **道器** | 在布须天等造化之地炼制，可对真阳大能构成威胁 | — |
| 6 | **先天至宝** | 造化之地蕴育，有成为炼神之宝的根基 | — |
| 7 | **炼神宝器** | 炼神大能炼制的法宝，威力无穷 | — |

**法宝通用规则**：
- 法宝可被修士祭炼，祭炼程度越高，发挥的威力越大
- 高品级法宝有灵性甚至灵智，可自主择主
- 法宝可吞噬其他法宝或材料进化
- 真器及以上法宝的真灵可转生重修

#### 4.2.4 道具-丹药

| 丹药 | 品级 | 效果 | 适用境界 |
|------|------|------|----------|
| **开脉丹** | 低 | 开脉，提升修炼资质 | 凡人→开脉 |
| **筑元丹** | 低 | 筑元，稳固道基 | 开脉→筑基 |
| **明气丹** | 中低 | 突破明气境，提升法力 | 筑基→明气 |
| **玄光丹** | 中 | 突破玄光境，提升玄光威力 | 明气→玄光 |
| **化丹丹** | 中高 | 凝丹，提升金丹品质 | 玄光→化丹 |
| **元婴丹** | 高 | 突破元婴境，提升元婴威力 | 化丹→元婴 |
| **疗伤丹** | 各异 | 疗伤，恢复伤势 | 通用 |
| **稳固丹** | 中 | 稳固道基，防止走火入魔 | 通用 |

**丹药通用规则**：
- 丹药品质影响效果（下品/中品/上品/极品）
- 同类型丹药有耐药性，连续服用效果递减
- 高阶丹药需高阶炼丹师炼制
- 丹药消耗天材地宝，存在资源约束

#### 4.2.5 道具-阵法

| 类型 | 说明 | 实例 |
|------|------|------|
| **攻击型** | 用于攻击敌人 | 剑阵、火球阵、雷阵 |
| **防御型** | 用于抵御攻击 | 护身罩、防御墙、护山大阵 |
| **辅助型** | 用于提升修炼或战斗效率 | 灵气汇聚阵、修炼加速阵、增幅阵 |
| **特殊型** | 用于特殊目的 | 迷幻阵、隐匿阵、传送阵、困敌阵 |

**阵法通用规则**：
- 布阵需要阵旗/阵盘等载体
- 维持阵法消耗灵石或灵机
- 阵法威力与布阵者修为、阵法品级、能源供给相关
- 高阶阵法可覆盖整个宗门（如山海界禁制大阵）

#### 4.2.6 道具-洞天

| 属性 | 内容 |
|------|------|
| **定义** | 高阶修士开辟的独立小世界 |
| **开辟条件** | 象相二重修士可开辟洞天 |
| **蕴化条件** | 象相三重修士可蕴化元胎，使洞天纳元气 |
| **用途** | 提升修炼速度、储存资源、种植灵药、庇护弟子 |
| **消耗** | 开辟消耗巨量灵机，维持消耗少量灵机 |
| **产出** | 灵药、灵材、修炼加速效果 |
| **战略价值** | 洞天可锁定大量灵机，加速外界灵机衰减，是势力博弈的关键资源 |

#### 4.2.7 道具-真阳大能（作为特殊"物品"或资源）

真阳大能在沙盒系统中可被视为一种特殊的"战略资源"：

| 属性 | 内容 |
|------|------|
| **定义** | 开辟自身元气大海的宇宙级存在 |
| **获取条件** | 凡蜕大修突破至真阳境（极难） |
| **消耗** | 造化精蕴、自身对道法的感悟 |
| **产出** | 开辟部宿（独立宇宙）、影响宇宙生灭、推动大道演化 |
| **战略价值** | 一个真阳大能可改变整个界域的势力格局 |
| **限制** | 跨界需付出极大代价，主要在布须天层面活动 |

### 4.5 物品需求

| 需求类型 | 说明 |

| **采集需求** | 天材地宝需要特定环境和条件才能采集 |

### 4.6 物品行为

| 行为 | 说明 | 消耗 | 产出 |
|------|------|------|------|
| **采集** | 从自然界获取天材地宝 | 时间、探索风险 | 原材料 |
| **炼制** | 将原材料炼制成丹药/法宝 | 天材地宝、灵机、时间 | 丹药/法宝 |
| **消耗** | 使用丹药/灵石等消耗品 | 物品本身 | 修为提升/伤势恢复等 |
| **交易** | 与其他修士交换物品 | 物品/灵石 | 所需物品/灵石 |
| **掠夺** | 通过战斗夺取他人物品 | 战斗风险、法力 | 战利品 |
| **祭炼** | 提升法宝的祭炼度 | 时间、心神、灵机 | 法宝威力提升 |
| **开辟** | 开辟洞天 | 巨量灵机、心神 | 洞天 |
