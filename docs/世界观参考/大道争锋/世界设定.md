## 一、世界 World

### 1.1 世界实体

#### 1.1.1 地理层级总览

| 层级 | 名称 | 说明 |
|------|------|------|
| 第一层 | 九洲 | 最基础的修真世界地理单元，共九座大洲 |
| 第二层 | 大域 | 由若干洲构成的更大区域，是宗门与国家的主要舞台 |
| 第三层 | 界域 | 超越九洲的独立空间，通过天地关等通道连接 |
| 第四层 | 山海界 | 九洲修士集体迁徙后建立的新根据地 |
| 第五层 | 布须天 | 造化之地，大道本源所在，最高层级界域 |

#### 1.1.2 九洲

九洲是《大道争锋》故事展开的基础地理框架：

| 洲名 | 方位 | 特征 |
|------|------|------|
| **东华洲** | 东方 | 故事主舞台，溟沧派、少清派、玉霄派等顶尖玄门所在，灵机最为充沛 |
| **东胜洲** | 东方 | 与东华洲相邻，资源丰富 |
| **东莱洲** | 东方 | 海外洲域 |
| **北溟洲** | 北方 | 苦寒之地，魔门势力盘踞 |
| **南崖洲** | 南方 | 南华派等宗门所在 |
| **中柱洲** | 中央 | 中柱神州，列玄教等势力所在 |
| **西沉洲** | 西方 | 资源相对贫瘠 |
| **西陷洲** | 西方 | 地形险恶 |
| **西绝洲** | 西方 | 极西之地，环境恶劣 |

#### 1.1.3 大域

大域是九洲之上的中间层级，由若干洲组成。例如东华洲本身即为一个大域，中柱神州亦为独立大域。大域之间通过传送阵、虚空通道或天地关相连。

#### 1.1.4 界域

界域是超越九洲的独立空间体系：

| 界域名称 | 说明 |
|----------|------|
| **钧尘界** | 九洲所在的主界域，灵机逐渐衰减 |
| **布须天** | 造化之地，大道本源映现之所，真阳大能争夺的终极目标 |
| **万阙星流** | 星辰界域，玉霄派星光道法的源头 |
| **虚天** | 界域之间的虚空地带，需虚空航行穿越 |
| **山海界** | 九洲修士在大鲲带领下集体迁徙后建立的新根据地，设有禁制大阵，后期剧情主舞台 |

#### 1.1.5 山海界

山海界是九洲修士为应对灵机衰减而集体迁徙建立的新世界。其核心特征：
- 由大鲲承载九洲修士穿越虚空抵达
- 设有覆盖全域的禁制大阵，抵御外敌
- 灵机充沛，为修士提供新的发展空间
- 成为后期剧情的主舞台，各方势力在此博弈

#### 1.1.6 天地关

天地关是连接不同界域的通道/关卡，具有以下特征：
- 是跨界通行的必经之路
- 通常有强大禁制或守护者
- 开启和通过需要特定条件（修为、信物、时机等）
- 是势力争夺的战略要地

### 1.2 世界状态

世界状态是动态变化的，影响所有势力的决策和NPC的行为。

#### 1.2.1 灵机浓度

| 状态等级 | 描述 | 影响 |
|----------|------|------|
| 充沛 | 灵机充足，修炼效率高 | 修士突破概率提升，宗门繁荣 |
| 正常 | 灵机稳定 | 维持正常修炼节奏 |
| 衰减 | 灵机逐渐减少 | 高阶修士争夺资源加剧，低阶修士修炼困难 |
| 枯竭 | 灵机近乎耗尽 | 无法支撑高阶修士，宗门衰落或迁徙 |

**当前状态**：九洲（钧尘界）灵机处于衰减期，山海界灵机充沛。

#### 1.2.2 跨界通道状态

| 通道 | 状态 | 说明 |
|------|------|------|
| 九洲→山海界 | 已开通 | 大鲲承载完成集体迁徙 |
| 钧尘界→布须天 | 争夺中 | 真阳大能博弈的焦点 |
| 各洲之间 | 正常 | 传送阵和虚空通道维持运转 |

#### 1.2.3 大道层级活跃度

| 层级 | 活跃度 | 说明 |
|------|--------|------|
| 人道 | 高 | 九洲修士活动频繁 |
| 仙道/神道 | 中 | 真阳及以上大能活跃于布须天 |
| 造化之灵 | 波动 | 随大道演化而起伏，威胁度递增 |

#### 1.2.4 造化之灵威胁度

造化之灵是大道本源的映现，试图吞噬诸界造化。其威胁度随以下因素变化：
- 真阳大能开辟部宿的规模（越大越吸引造化之灵注意）
- 大道权柄的争夺激烈程度
- 诸界造化的总量

### 1.3 世界行为/规则

#### 1.3.1 灵机衰减规则

灵机是天地间可供修行的基础能量，其衰减遵循以下规则：

1. **开采消耗**：修士修炼、炼丹、炼器、布阵均消耗灵机，高阶修士消耗量远大于低阶修士
2. **自然恢复**：灵脉可缓慢自行恢复灵机，但速度远低于消耗速度
3. **洞天锁定**：高阶修士开辟洞天后，将大量灵机锁定在个人洞天内，加速外界灵机衰减
4. **衰减阈值**：当灵机浓度低于某一阈值时，无法支撑洞天真人的存在，修士必须迁徙或降境
5. **迁徙机制**：灵机枯竭时，修士可选择集体迁徙至灵机充沛的新界域

#### 1.3.2 跨界规则

1. **天地关通行**：需满足特定条件（修为、信物、时机）
2. **虚空航行**：需洞天真人以上修为或借助大鲲等特殊载体
3. **跨界代价**：消耗大量灵机和资源，低阶修士无法独立完成
4. **界域壁垒**：不同界域之间存在天然壁垒，修为越高越难跨界（真阳大能跨界需付出极大代价）
5. **集体迁徙**：可通过大鲲等上古神兽承载大量修士同时跨界

#### 1.3.3 大道层级规则

| 层级 | 代表 | 特征 | 晋升条件 |
|------|------|------|----------|
| **人道** | 九洲修士 | 在资源有限的世界中修行，受灵机约束 | 开脉→筑基→元婴→洞天 |
| **仙道** | 凡蜕/真阳 | 超脱凡俗，掌握大道之力 | 洞天→凡蜕→真阳 |
| **神道** | 炼神/至人 | 触及大道本源，一念生灭 | 真阳→炼神→至人 |
| **大道本源** | 造化之灵/大德 | 大道本身的映现 | 不可晋升，只能争夺权柄 |

#### 1.3.4 造化之灵规则

1. **本质**：大道本源的映现，非生灵非死物
2. **目标**：吞噬诸界造化，回归大道本源
3. **与大德的关系**：与大德争夺大道权柄，互为制衡
4. **触发条件**：当诸界造化总量达到一定程度时，造化之灵苏醒并开始吞噬
5. **对抗方式**：真阳大能联合对抗，或由至人级存在正面博弈

#### 1.3.5 真阳大能规则

1. **元气大海**：真阳大能开辟自身元气大海，一念之间无数界天生灭
2. **部宿开辟**：真阳可在虚空中开辟部宿（独立宇宙），消耗造化精蕴
3. **跨界限制**：真阳大能跨界需付出极大代价，通常只在布须天层面活动
4. **大道博弈**：真阳之间围绕大道权柄展开博弈，胜负影响诸界格局
5. **陨落风险**：真阳大能并非不死不灭，在大道博弈中可能陨落

### 1.4 世界需求

世界本身（作为沙盒系统）具有以下需求，驱动剧情和冲突：

| 需求 | 说明 | 优先级 |
|------|------|--------|
| **制造冲突** | 通过灵机衰减、资源稀缺、势力对立制造矛盾 | 高 |
| **推动演化** | 推动修士突破境界、势力兴衰、大道演化 | 高 |
| **维持平衡** | 防止单一势力或个体过度强大导致系统崩溃 | 中 |
| **避免死局** | 确保总有新的发展空间（如山海界）或新的冲突点 | 中 |

### 1.5 消耗/产出关系（世界层面）

```
灵机（基础资源）
  ├── 消耗：修士修炼、炼丹、炼器、布阵、开辟洞天
  ├── 产出：高阶修士、丹药、法宝、洞天
  └── 衰减后果：修士迁徙、势力重组、新界域开发

造化精蕴（高级资源）
  ├── 消耗：真阳开辟部宿、炼神炼制宝器
  ├── 产出：新宇宙、炼神宝器
  └── 过度消耗后果：吸引造化之灵、大道失衡

气运（无形资源）
  ├── 消耗：修士突破、宗门兴衰、大道博弈
  ├── 产出：机缘、突破契机、势力崛起
  └── 转移规则：可通过因果缠结、道统传承转移
```
