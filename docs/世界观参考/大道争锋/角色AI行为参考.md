# 大道争锋 — 角色AI行为参考

## AI架构映射说明

《大道争锋》是一部以"棋手型主角"为核心的修仙小说，其角色行为模式天然适配AI分层架构：

- **BT层**：张衍的"后手评估 → 阈值触发撤离"是BT抢占式应急反应的标准范例，可作为所有"冷静型NPC"的行为模板。
- **Utility层**：其短期/中期/长期目标的清晰分野，可直接映射为效用权重曲线——前期权重偏向"资源获取"，后期逐渐收敛于"超脱/破局"。
- **GOAP层**：张衍的每一步布局都对应一个可拆解的子目标链（获取身份 → 积累资源 → 整合势力 → 终极破局），天然适合GOAP规划器。

## 核心角色分析

### 1. 张衍

#### 性格标签
极致冷静 / 棋手思维 / 长线布局 / 不依赖情感判断 / 杀伐果断

#### BT层应急反应
| 触发条件 | 抢占行为 | 优先级 |
|---|---|---|
| 遭遇任何危机 | 评估"后手是否够用" | 最高 |
| 胜算低于阈值 | 立刻执行撤离/备用方案 | 次高 |
| 危机解除 | 复盘并补全布局漏洞 | 中 |
| 遭遇侮辱/挑衅（非致命） | 冷静记录，纳入长期博弈 | 低 |

**核心逻辑**：张衍的BT层永不进入"愤怒冲动"分支，所有情绪刺激都被Filter层拦截，转为理性评估。

#### Utility层权重打分

| 周期 | 目标 | 权重(0-1) | 触发条件 |
|---|---|---|---|
| 短期 | 在玄门内部竞争中脱颖而出，获取资源 | 0.9 | 实力低于凡蜕 |
| 中期 | 问鼎大道，超脱九洲世界 | 0.7 | 凡蜕后期 |
| 长期 | 打破末法时代，建立新的修行秩序 | 0.5(渐进至1.0) | 洞天后 |

**权重转换规则**：随着道行突破，短期目标权重逐步"析出"至中长期目标；中期权重达峰后向长期传递。

#### GOAP层决策链

```
[溟沧派入门] → [获取正统玄门身份与庇护]
    ├── 策略：选择齐云天一脉（潜力最大但最凶险）
    ├── 行动：利用对外战争积累功勋
    └── 产出：获得资源倾斜与地位提升

[夺回洞府] → [收回资源，震慑宵小]
    ├── 策略：暗中调查对手弱点
    ├── 行动：邀请同门观礼造势 → 正面击败 → 当众立威
    └── 产出：确立不可侵犯的威信

[布局诸天万界] → [以九洲为棋，博弈天下]
    ├── 策略：感知末法将至
    ├── 行动：扶持代理人 → 布局诸天万界获取资源 → 与各方大能博弈
    └── 产出：九洲成为可控棋盘

[打破末法] → [超脱世界壁垒，建立新秩序]
    ├── 策略：推演破界之法
    ├── 行动：修为至巅峰 → 集中全界之力 → 对抗天道/外界强者
    └── 产出：成功超脱，新秩序建立
```

#### 标志事件

1. **溟沧派入门与初期布局** — 步步为营，借势登高，奠定棋局基础。
2. **夺回洞府清理门户** — 杀鸡儆猴，确立在玄门中的威信。
3. **诸天万界布局** — 感知末法，化被动为主动，将九洲纳为棋盘。
4. **打破末法超脱** — 终极布局，逆天改命，完成从棋子到执棋者的跨越。

## 其他角色

### 秦墨白
**性格标签**：沉稳内敛 / 智谋深远 / 对张衍既竞争又合作

- **BT应急**：遇到复杂局势→优先选择"制衡"而非"彻底消灭"→维持各方平衡
- **Utility**：在玄门中站稳脚跟，实现自身大道
- **决策特征**：多次与张衍既有竞争也有合作，是棋局中重要的"盟友棋子"，其决策常影响九洲局势走向

### 霍轩
**性格标签**：豪爽直率 / 重诺守信 / 实力强横

- **BT应急**：遇不公或朋友危机→直接出手→以力破巧
- **Utility**：追求武道极致，守护对自己重要的人
- **决策特征**：张衍的坚定"执行者"，多次在棋局关键节点以武力破局，是棋手身边不可或缺的"破局者"

## NPC类型化归纳

### 1. 棋手型
- **行为逻辑**：完全理性，以"利益/大道"为唯一判断标准
- **BT映射**：情绪事件一律被Filter拦截，只走"评估 → 计算 → 执行"路径
- **Utility映射**：权重函数高度结构化，长期目标占主导
- **GOAP映射**：每一步都是多步前瞻后的最优子目标
- **实例**：张衍、部分太上长老

### 2. 破局者/武夫型
- **行为逻辑**：重情重义，不喜算计，以力为尊
- **BT映射**：友方受威胁→立即抢占"武力支援"行为
- **Utility映射**：短期目标（守护/复仇）权重大
- **GOAP映射**：目标链短而直接，不绕弯路
- **实例**：霍轩

### 3. 守成者型
- **行为逻辑**：维护现有秩序，反对激进变革
- **BT映射**：秩序受威胁→抢占"镇压/维稳"行为
- **Utility映射**：稳定权重 > 创新权重
- **GOAP映射**：决策偏向保守路径，回避高风险子目标
- **实例**：部分传统派长老

### 4. 殉道者型
- **行为逻辑**：为实现宏大目标甘愿牺牲自我
- **BT映射**：目标受威胁时→抢占"自我牺牲"行为（最高优先级）
- **Utility映射**：道统延续权重 > 个人存活权重
- **GOAP映射**：以"牺牲节点"作为子目标链的终点
- **实例**：为延续道统牺牲的先辈
