# 王林修炼历程 · 时间线事件流

> 原著：《仙逆》 | 主角：王林 | 总计30个事件

### 事件1: 王家屯测资失败，一心向道
【时间/地点】少年时期 / 宣武国王家屯
【经过】王林天资平庸，在测试中未被修真家族看中。他不甘心命运，凭借执拗劲反复恳求父母让他踏上修仙路。
【得失】失去正常少年生活，获得进入修真界的资格（虽被视为废柴）。
【境界】凡人

### 事件2: 艰难入门恒岳派，做杂役弟子
【时间/地点】测试后不久 / 宣武国恒岳派
【经过】王林因资质太差无法成为正式弟子，从杂役做起。暗中观察记名弟子修炼，偷学吐纳之法。
【得失】获得修仙界最底层的立足之地。
【境界】练气初期（未入门）

### 事件3: 偶得天逆珠，开启修炼契机
【时间/地点】恒岳派后山砍柴期间
【经过】在一具枯骨上发现天逆珠，珠子吸收其血液后沉入丹田，构建出修炼空间。
【得失】获得核心金手指天逆珠。
【境界】练气一层

### 事件4: 参与门派试炼，遭遇妖兽危机
【时间/地点】入门数年后 / 恒岳派试炼之地
【经过】凭借微弱修为躲避高阶妖兽，寻找灵草时与同门发生冲突，初显杀伐果断。
【得失】修为突破至练气二层，获得少量低阶灵石。
【境界】练气二层

### 事件5: 恒岳派被灭，死里逃生
【时间/地点】修魔海势力入侵 / 恒岳派
【经过】恒岳派覆灭，王林肉身濒死，天逆珠护住魂魄。在域外战场沉睡多年吸收精气恢复。
【得失】肉身被毁，神魂历经生死磨砺，天逆珠空间初步解锁。
【境界】练气三层（残魂状态）

### 事件6: 重塑肉身于修魔海
【时间/地点】域外战场苏醒后 / 赵国修魔海
【经过】魂魄飘至修魔海，利用阴煞之气和天逆珠滋养，重塑肉体。新肉体资质平庸但神魂异常强大。
【得失】重获肉身，正式踏入修魔者行列。
【境界】练气四层

### 事件7: 加入火焚国修魔小派，隐匿修行
【时间/地点】重塑肉身后 / 火焚国修魔小派
【经过】为获取资源和功法加入修魔小派，隐藏实力暗中掠夺同门灵石，数十年内提升至练气大圆满。
【得失】积累结丹所需灵石和丹药，熟悉修魔者行事风格。
【境界】练气大圆满

### 事件8: 寻找结丹机缘，探寻古修洞府
【时间/地点】结丹前夕 / 火焚国边境古修洞府
【经过】得知古修洞府现世，独自前往探宝。在洞府外以阵法困住竞争对手夺得入场资格。
【得失】获得古修洞府地图残片，确认龙灵草位置。
【境界】练气大圆满

### 事件9: 智取龙灵草，遭遇筑基修士追杀
【时间/地点】寻找灵草 / 火焚国毒龙沼
【经过】在毒龙沼找到龙灵草后被筑基修士盯上。利用毒沼环境和天逆珠力量布下死局，重创对手逃遁。
【得失】获得龙灵草，损失部分法器，对战筑基修士有了经验。
【境界】练气大圆满

### 事件10: 闭关准备结丹，演练凝核法诀
【时间/地点】获得灵草后 / 火焚国僻静山谷
【经过】寻得僻静山谷布置防御阵法闭关三年，炼化龙灵草反复演练凝核法诀。多次险些走火入魔均被天逆珠化解。
【得失】身心调整至最佳，对结丹了然于胸。
【境界】练气大圆满

### 事件11: 引动天劫，九死一生
【时间/地点】出关之日 / 火焚国荒原
【经过】在荒原引动远超普通结丹的天劫，凭借阵法、法器和天逆珠硬抗。肉身几乎崩毁，神魂差点消散。
【得失】肉身重伤，法器尽毁，强行凝聚灵力成核。
【境界】假丹（未稳固）

### 事件12: 凝聚金丹，初窥元婴门槛
【时间/地点】天劫过后 / 疗伤洞府
【经过】在天逆珠空间内疗伤稳固金丹，将天逆珠特性融入金丹使其品质远超同阶。金丹大成后感受元婴门槛。
【得失】正式成为结丹修士，获得数百年寿元。
【境界】金丹初期

### 事件13: 杀巨富报家仇，血染宣武国
【时间/地点】结丹后 / 宣武国王家屯
【经过】得知仇人修真家族依附火焚国巨头，化身魔修潜入一夜灭门。震动宣武国，树敌颇多。
【得失】报得大仇，但暴露行踪被迫远走。
【境界】金丹中期

### 事件14: 前往朱雀星，踏上新征程
【时间/地点】离开赵国后 / 乱星海至朱雀星
【经过】通过乱星海传送阵抵达朱雀星，灵气浓郁强者如云。因口音和功法被轻视，隐忍暗中观察。
【得失】进入更广阔修仙世界。
【境界】金丹中期

### 事件15: 化凡领悟道心，红尘炼心
【时间/地点】朱雀星 / 某凡人城镇
【经过】感悟心境有缺，主动选择化凡在世俗生活数十年。体验生老病死爱恨情仇，道心在红尘中磨砺。
【得失】道心圆满，对因果轮回有了深刻理解。
【境界】金丹后期（道心圆满）

### 事件16: 突破元婴，引动更强天劫
【时间/地点】化凡结束 / 朱雀星荒古禁地
【经过】在荒古禁地闭关冲击元婴，天劫威力更胜结丹时。凝聚元婴时险些魂散，凭借狠劲和机缘成功。
【得失】成为元婴修士，寿元大增，在朱雀星可称一方霸主。
【境界】元婴初期

### 事件17: 获得本命法宝材料，祭炼剑阵
【时间/地点】元婴后 / 古神之地
【经过】得知古神之地有古神血可祭炼本命法宝，深入险地与其他元婴争夺。以元婴之火祭炼剑阵与本命法宝。
【得失】获得强力本命法宝剑阵，战力飙升。
【境界】元婴中期

### 事件18: 婴变过程：凝聚婴变之身
【时间/地点】元婴大圆满后 / 朱雀星某秘境
【经过】在秘境静坐百年感悟日夜星辰变幻，尝试婴变融合元婴与肉身。斩灭心魔，元婴发生质变。
【得失】元婴蜕变，初步掌握天地规则。
【境界】婴变初期

### 事件19: 问鼎天劫，争夺问鼎之资
【时间/地点】婴变大圆满 / 朱雀星核心区域
【经过】收集问鼎之晶，引动问鼎天劫。天劫中现问道幻境，历经无数次轮回后醒悟打破幻境。
【得失】成功问鼎，凝聚问鼎之宝，踏入巅峰行列。
【境界】问鼎初期

### 事件20: 前往仙界遗迹，争夺仙玉
【时间/地点】问鼎后 / 仙界遗迹
【经过】仙界遗迹开启，与其他问鼎修士同入争夺仙玉。遭遇其他修真星强敌，初识仙界与自身差距。
【得失】获得大量仙玉，对仙界有初步认识。
【境界】问鼎中期

### 事件21: 度化天人衰劫，初探轮回
【时间/地点】问鼎后期 / 洞府
【经过】面临第一次天人衰劫，通过天逆珠推演轮回看到前世今生，成功度化衰劫凡根尽去。
【得失】凡根脱去，对轮回法则更深领悟。
【境界】问鼎大圆满

### 事件22: 突破阴虚阳实，掌握完整规则
【时间/地点】度化衰劫后 / 阴虚阳实秘境
【经过】融合因果、轮回理解将规则之力凝实，肉身几度崩溃元神险被撕碎，最终成功凝聚阴神。
【得失】踏入第二步修为，掌握完整规则。
【境界】阴虚（阳实）

### 事件23: 碎涅三境，一路高歌猛进
【时间/地点】阴虚阳实后 / 辗转多个秘境
【经过】先后突破碎涅、天人、虚空三境。获得梦道之力可在梦中影响现实，手段更加诡异。
【得失】修为稳步提升至碎涅巅峰，成为周边星域顶尖强者。
【境界】碎涅巅峰

### 事件24: 领悟因果、生死、轮回本源
【时间/地点】碎涅巅峰 / 游历万界
【经过】观察日月星辰生灭，感悟因果纠缠和生死轮回。将三大真意凝练为本源之力，天逆珠发挥关键作用。
【得失】初步掌握三大本源，拥有踏天资格。
【境界】准踏天（本源境）

### 事件25: 踏天之路，逆转乾坤
【时间/地点】本源大成后 / 踏天阶梯
【经过】对抗整个乾坤意志，选择逆天而行以自身之道镇压天道。一步步走上阶梯承受巨大压力和内心黑暗。
【得失】成功踏天，可制定部分规则。
【境界】踏天境

### 事件26: 创立新秩序，对抗域外强者
【时间/地点】踏天后 / 所属星域
【经过】发现域外闪雷族等威胁，整合星域力量创立新秩序对抗外敌，培养后代和门徒传承自身之道。
【得失】成为一方霸主，保护所属星域。
【境界】踏天巅峰

### 事件27: 融道，成为大道的一部分
【时间/地点】踏天巅峰 / 宇宙最深处
【经过】尝试融道与大道合为一体超脱万物，将本源记忆情感全部融入大道，保留一丝真灵。
【得失】与大道部分融合，获得近乎不朽的能力。
【境界】融道境

### 事件28: 重塑肉身与神魂，找回自我
【时间/地点】融道后 / 天逆珠空间
【经过】发现失去自我并非所愿，利用天逆珠和真灵重新凝聚肉身神魂，找回失落人性。
【得失】找回完整自我，修为更加精深。
【境界】巅峰（接近原著结局）

### 事件29: 终结一切因果，超脱轮回
【时间/地点】最终时期 / 原点终极之地
【经过】了结所有恩怨因果，打开轮回让一切重归秩序。彻底超脱轮回成为不可名状的存在。
【得失】所有因果了结，彻底超脱获得真正自由。
【境界】超脱（原著结局境界）

### 事件30: 化身万物，守护一方
【时间/地点】超脱后 / 守护的世界
【经过】化身万物默默守护珍视的一切，道成为世界规则之一。故事在传说中落幕，成为修真界永恒传奇。
【得失】实现从凡人到巅峰的超越，道韵流传。
【境界】永恒（超脱）
