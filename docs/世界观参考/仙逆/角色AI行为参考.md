# 《仙逆》角色AI行为参考

## AI架构映射说明

本文档将角色特性映射到游戏AI四层架构：
- **BT（Behavior Tree）层**：情绪反应与应急抢占，对应角色的即时反应模式
- **Utility层**：长期目标选择与权重评分，对应角色的核心驱动力排序
- **GOAP层**：目标分解为步骤序列，对应角色关键事件的决策链
- **Action Queue**：执行具体动作

---

## 核心角色

### 一、王林

#### 性格标签

极度理智 / 狠辣果决 / 执着隐忍 / 情感内敛 / 为达目的不择手段

#### BT应急反应

```
[感知威胁] → [评估可逆代价]
    ├─ 代价可接受 → [执行极端手段]（燃血/祭献残魂/自损修为/舍弃肉身）
    └─ 代价不可接受 → [暂避锋芒] → [暗中标记仇恨] → [等待时机]
[恢复安全] → [冷静复盘] → [更新仇恨标记列表] → [纳入长期布局]
```

**参数配置**：
| 参数 | 值 | 说明 |
|------|-----|------|
| threat_threshold | 0.7 | 超过此值触发应急模式 |
| acceptable_cost_ratio | 0.3 | 可接受的自损比例（相对于当前战力） |
| revenge_priority_decay | 0.05/day | 仇恨标记优先级每日衰减系数 |
| feint_retreat_prob | 0.85 | 评估为不利时假装撤退的概率 |

#### Utility长期目标权重

| 目标 | 权重 | 触发条件 | 说明 |
|------|------|----------|------|
| 复活李慕婉 | ∞ | 李慕婉死亡后永久激活 | 覆盖所有其他目标，一旦可执行立刻抢占 |
| 生存需求 | 900 | 生命值 < 30% | 遭遇致命威胁时权重飙升 |
| 复仇欲 | 700 | 仇敌列表非空 | 仇恨标记目标优先级越高，此权重越大 |
| 大道追求 | 600 | 生存与复仇均无紧迫压力 | 基础权重，随修为瓶颈出现而升高 |
| 资源获取 | 450 | 所有高优先目标均无紧迫压力 | 主动寻宝/修炼资源行为驱动 |

**权重计算修正因子**：
- `revenge_proximity_factor`：当仇敌出现在感知范围内，复仇欲权重 × 1.5
- `breakthrough_urgency`：当处于修为瓶颈，大道追求权重 × 2.0
- `resource_scarcity`：当某项关键资源不足，资源获取权重 × 1.8

#### GOAP标志事件及决策链

##### 事件1：化凡悟道

```
[感知：化神瓶颈无法突破]
  → [子目标：感悟生死，突破境界]
    → [步骤1：压制修为至凡人水平]
    → [步骤2：抹去部分记忆（保留核心执念）]
    → [步骤3：入赵国皇族，以凡人身份入世]
    → [步骤4：历经数十年生老病死、悲欢离合]
    → [步骤5：于凡尘中顿悟生死意境]
    → [结果：破境成功，生死意境大成]
```

**GOAP代价评估**：
| 步骤 | 时间成本 | 风险 | 可逆性 |
|------|---------|------|--------|
| 压制修为 | 瞬间 | 极高（此期间脆弱） | 可逆 |
| 抹去记忆 | 瞬间 | 中（可能迷失自我） | 部分可逆 |
| 凡尘入世 | 数十年 | 低（但漫长） | 不可逆（时间） |
| 顿悟破境 | 瞬间 | 低 | 不可逆（已突破） |

##### 事件2：仙玉夺舍危机

```
[触发器：遭遇强敌仙玉夺舍]
  → [子目标：反杀夺舍者，淬炼神魂]
    → [步骤1：伪装放弃抵抗，主动让出肉身控制权]
    → [步骤2：趁敌神识入侵，将其引入自身意境空间]
    → [步骤3：在意境空间中，以主场优势施展生死意境]
    → [步骤4：磨灭敌人神识，吞噬其神魂之力]
    → [步骤5：重铸并强化肉身]
    → [结果：反杀强敌，神魂大幅增强]
```

**关键决策点**：
- `DECISION_POINT`：是否放弃肉身控制权 → 评估：敌方神识强度 vs 自身意境空间强度 → `自身意境强度 × 1.2 > 敌方神识强度` → 执行

##### 事件3：古神之地夺宝

```
[触发器：发现古神之地，多方势力齐聚]
  → [子目标：渔翁得利，获取古神遗物]
    → [步骤1：潜伏于暗处，观察各方势力实力与关系]
    → [步骤2：暗中挑拨，制造多方矛盾]
    → [步骤3：坐山观虎斗，等待各方消耗]
    → [步骤4：在多方两败俱伤时精准出手夺宝]
    → [步骤5：得手后立即远遁，不留痕迹]
    → [结果：以最小代价获得最大收益]
```

**行为模式标签**：`借势` `挑拨` `时机把握` `一击即退`

##### 事件4：复活李慕婉（跨阶段长期目标链）

```
[触发器：李慕婉死亡，激活终极目标]
  → [阶段1子目标：维持李慕婉残魂不散]
    → [步骤：寻找养魂木/天逆珠 → 安置残魂 → 持续温养]
  → [阶段2子目标：收集复活所需天地本源]
    → [步骤：踏遍各界 → 寻找本源之物 → 与各方势力交易/抢夺]
  → [阶段3子目标：感悟轮回规则]
    → [步骤：进入轮回之地 → 观摩轮回运转 → 自行推演轮回之道]
  → [阶段4子目标：与天运子博弈争取时间]
    → [步骤：表面臣服 → 暗中布局 → 关键节点反制]
  → [阶段5子目标：踏天后执行复活]
    → [步骤：踏天成功 → 调用天地之力 → 重聚魂魄 → 重塑肉身]
    → [结果：李慕婉复活，终极执念达成]
```

**跨阶段参数**：
- `progress_persistence`：true（进度跨阶段保留）
- `checkpoint_save`：每个阶段完成后自动存档
- `interrupt_resume`：中断后可从中断阶段恢复

#### 横向对比

| 维度 | 王林（仙逆） | 韩立（凡人修仙传） | 萧炎（斗破苍穹） |
|------|-------------|-------------------|------------------|
| 核心驱动 | 执念（复活爱人） | 生存（苟） | 约定（三年之约） |
| 行事风格 | 极度理智，评估代价后出手 | 谨慎第一，宁可错过 | 热血冲动，先打再说 |

| 成长路径 | 以退为进，借势借力 | 稳扎稳打，步步为营 | 遇强则强，战斗中突破 |
| 对敌态度 | 标记仇恨，择机清算 | 能避则避，避不了再打 | 正面刚，不服就干 |
| AI策略类型 | 棋手型 + 复仇驱动型 | 纯粹生存驱动型 | 热血进取型 |

### 二、李慕婉

#### NPC类型：情感锚点型

#### 性格标签

温柔坚韧 / 深情隐忍 / 外柔内刚 / 自我牺牲倾向

#### BT应急反应

```
[感知：王林遭遇致命危机]
  → [评估自身可替代性]
    ├─ 自身可替代牺牲 → [立即执行牺牲行为]（挡刀/转移伤害/献祭自身）
    └─ 无法替代 → [寻求外援] → [祈祷/呼唤强者]
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 王林安全 | 1000 | 最高优先级，不计代价 |
| 不成为王林拖累 | 800 | 驱动自身努力修炼 |
| 陪伴王林 | 600 | 基础情感需求 |

#### 标志事件

在王林早期多次危机中默默守护，最终为救王林香消玉殒。死亡后转化为王林Utility系统中的∞权重目标——复活李慕婉，成为贯穿全局的最强驱动力。

#### AI设计要点

- **存在时**：作为王林的情感稳定器，降低王林的`revenge_priority_decay`（让王林更"正常"）
- **死亡后**：从NPC转为全局状态标记，持续影响王林所有决策的权重偏移
- **触发条件**：王林遇到与李慕婉相似的女性角色时，Utility层产生`similarity_empathy`修正（权重偏移 +200）

### 三、司徒南

#### NPC类型：过命交情的兄弟型

#### 性格标签

狂放不羁 / 重情重义 / 看似玩世不恭实则极靠谱 / 实力强横

#### BT应急反应

```
[感知：王林（兄弟）遇险]
  → [第一反应：毫不犹豫出手支援]（无论代价多大）
    → [若自身实力不足] → [燃烧修为/开启禁术] → [强行介入]
    → [若距离遥远] → [撕裂空间] → [强行降临]
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 守护兄弟（王林） | 950 | 兄弟义气超越自身安危 |
| 恢复昔日荣光 | 750 | 对过往辉煌的执念 |
| 自身大道 | 600 | 与王林并肩同行 |

#### 标志事件

- 在王林化凡期间默默守护，不出手干预但确保无人干扰
- 多次在王林遭遇无法独自化解的危机时出手相助
- 与王林建立"过命交情"——一种比普通盟友更牢固的社交关系类型

#### AI设计要点

- **关系类型**：`blood_brother`（过命交情）——不受挑拨离间，无背叛可能
- **行为模式**：平时不干预王林成长（保持距离），关键时刻必定出现
- **触发机制**：王林生命值 < 10% 且无自救可能时，司徒南登场概率 → 100%

### 四、天运子

#### NPC类型：神秘算计型

#### 性格标签

神秘莫测 / 算计无漏 / 唯利是图 / 把众生当棋子 / 长线布局

#### BT应急反应

```
[感知：自身布局出现漏洞/算计落空]
  → [评估漏洞对自身本源的影响]
    ├─ 影响可控 → [调整布局] → [舍弃部分棋子] → [维持大局]
    ├─ 影响较大 → [立刻收缩] → [舍弃所有外围棋子] → [保全核心本源]
    └─ 影响致命 → [启动终极后手] → [假死脱身/转世/融合天道]
```

#### Utility权重

| 目标 | 权重 | 说明 |
|------|------|------|
| 保全自身本源 | ∞ | 任何时候的第一优先 |
| 窃取天道本源 | 900 | 终极目标 |
| 操控棋子获利 | 700 | 日常驱动 |
| 消灭威胁（如王林） | 600 → 900 | 随威胁评估动态升高 |

#### 标志事件

布局无数年代，操控多方势力与天骄（包括王林），试图窃取天道本源。在王林成长过程中，既将其视为棋子，又在关键时刻试图扼杀。最终被王林破局反杀。

#### AI设计要点

- **布局层（独立于GOAP的长线系统）**：
  - 天运子始终维护一个`grand_scheme`数据结构，包含多个并行棋子
  - 每个棋子有独立的状态：`培养期 → 利用期 → 可舍弃期`
  - 王林在他的棋盘中经历了全部三个阶段，最终在"可舍弃期"反噬
- **感知系统增强**：天运子对棋子的状态感知范围远大于普通NPC
- **对话风格**：高深莫测，从不把话说满，永远留有解释空间

## NPC类型化归纳

### 类型1：复仇偏执者型

**代表角色**：王林（早期）、以及各类被他标记仇恨的对手

| AI参数 | 值 | 说明 |
|--------|-----|------|
| rationality_threshold | 0.4 | 理性阈值，低于此值进入偏执状态 |
| revenge_weight_base | 800 | 复仇目标基础权重 |
| collateral_acceptable | 0.6 | 可接受的连带伤害比例 |
| forgive_probability | 0.0 | 宽恕概率（永不原谅） |

**行为特征**：
- Utility层始终存在一个高权重的复仇目标
- BT层在遇到仇敌相关刺激时触发应急抢占
- GOAP层会将"强大到足以复仇"作为核心子目标链

**设计建议**：适合作为支线任务驱动者，可与主角建立"共同敌人"的临时同盟。

---

### 类型2：天道意志型

**代表角色**：天道本身、天运子（部分）

| AI参数 | 值 | 说明 |
|--------|-----|------|
| emotion_enabled | false | 无情感模块 |
| rule_adherence | 1.0 | 严格遵守弱肉强食规则 |
| adaptation_speed | 慢 | 行为模式固定，不轻易改变 |
| threat_response | 规则性抹杀 | 对破坏规则者执行清除 |

**行为特征**：
- 不适合作为常规NPC，更适合作为"环境系统"
- 行为完全可预测（按规则行事）
- 作为游戏的压迫力来源，驱动玩家/主角成长

**设计建议**：用全局事件系统实现，而非AI个体。当世界"规则违规度"累积到阈值时触发天劫/天罚。

---

### 类型3：棋子谋略者型

**代表角色**：各路宗主、长老、散修中的精明者

| AI参数 | 值 | 说明 |
|--------|-----|------|
| manipulation_tendency | 0.8 | 操控他人的倾向 |
| loyalty_threshold | 0.2 | 忠诚度阈值，低于此值随时背叛 |
| escape_trigger | 自身利益威胁 | 出现不可控变量时立即抽身 |
| pawn_management | active | 主动维护棋子列表 |

**行为特征**：
- Utility权重以"自身利益最大化"为核心
- BT应急触发条件为"棋子不可控"或"自身受威胁"
- GOAP中频繁出现"挑拨""借势""坐收渔利"子目标

**设计建议**：适合作为中期Boss或关键剧情推动者，其倒戈/背刺可制造剧情转折。

---

### 类型4：情感锚点型

**代表角色**：李慕婉

| AI参数 | 值 | 说明 |
|--------|-----|------|
| anchor_target | 指定角色 | 情感锚定的目标对象 |
| sacrifice_willingness | 1.0 | 愿意为目标牺牲的程度 |
| self_preservation_weight | 200 | 自身生存权重（极低） |
| post_death_effect | 全局权重偏移 | 死亡后对目标角色的影响 |

**行为特征**：
- Utility权重以锚定目标的利益为绝对优先
- BT应急反应中，自身安全评估让位于锚定目标安全
- 死亡后从NPC转为全局状态变量，持续影响锚定目标

**设计建议**：数量极少（全游戏1-2个），其存在/死亡对整个AI世界产生深远影响。
