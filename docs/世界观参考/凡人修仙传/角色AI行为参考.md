# 《凡人修仙传》角色AI行为参考

## 一、AI架构映射说明
本文档按以下四层架构组织每个角色的行为参数：
- **BT层**：情绪反应 / 应急抢占（Root BT → Emergency → Emotion Reaction）
- **Utility层**：人生目标选择（权重打分）
- **GOAP层**：目标分解为步骤链
- **Action Queue**：具体动作序列

## 二、核心角色分析

### 韩立
**性格标签**：极度谨慎（稳健）、隐忍、理智冷酷、利益导向、先评估后行动

**BT层 - 应急反应**：
- 触发条件：感知到危险或未知变故
- 行为序列：隐匿身形/气息 → 冷静观察(收集情报) → 评估实力差距与利弊 → 若胜则设局偷袭 / 若败则果断逃逸(保命优先) → 绝不恋战

**Utility层 - 长期目标权重**：
- 长生（终极权重，∞）
- 提升修为（900）
- 获取保命手段/宝物（850）
- 积攒资源（750）
- 消除隐患/杀人灭口（700）
- 社交/情感（200，除非涉及利益交换）

**GOAP层 - 标志事件决策链**：
1. 血色试炼：隐藏实力扮猪吃虎 → 避开人群中心 → 利用地形和符宝 → 最后关头抢夺战利品 → 远遁
2. 外海猎妖：准备克制道具 → 寻找落单大妖 → 召唤辅助NPC → 雷霆一击 → 绝不深追
3. 坠魔谷：利用阵法知识破解禁制 → 坐山观虎斗 → 关键时刻背后捅刀/顺手牵羊 → 优先保命放弃宝物

**社交Action模式**：
- 陌生人：默认恶意假设，保持距离，随时准备逃跑或反击
- 熟人/合作者：严格利益界限，不出卖但也不深交
- 敌人：必杀之，暂时杀不了则隐忍潜伏，日后必报
- 道侣：有情义也有算计，保护但不完全信任

### 南宫婉
**性格标签**：高冷、端庄、外冷内热、宗门观念重
**BT层**：遇到危机先防御再求援
**Utility层**：宗门利益 > 个人情感（但韩立权重逐渐上升）
**标志事件**：与韩立若即若离，关键时刻成为韩立的软肋或助力

### 大衍神君
**性格标签**：智慧型、偏执（对傀儡术）、亦师亦友、吐槽役
**行为模式**：提供技术咨询，偶尔因性格提激进建议

### 墨大夫
**性格标签**：野心家/阴谋家
**标志事件**：夺舍韩立失败 → 赌徒逻辑：极高风险极高收益

### 银月
**性格标签**：忠诚/分裂（受宝物影响）
**标志事件**：化身器灵辅助韩立，忠诚型随从AI，特定条件下有自主失控风险

### 紫灵
**性格标签**：魅惑/精明/有大毅力
**标志事件**：魔界探险中与韩立互助，亦敌亦友/利益共同体AI

## 三、NPC类型化归纳

### 1. 散修求生型 (Loose Cultivator)
- 示例：低阶散修、黑铁城时期的韩立
- BT层：高敏感度，低荣誉感，容易受惊逃跑
- Utility：生存权重1000 > 杀怪权重500 > 抢劫低阶修士(若实力允许)权重400
- GOAP：独来独往 → 见势不妙使用保命符箓/遁术

### 2. 宗门弟子型 (Sect Disciple)
- 示例：越国七大派弟子
- BT层：有归属感，受伤会求援，抱团行动
- Utility：完成宗门任务(800) > 维护宗门荣誉(500)
- GOAP：组队阵法 → 呼叫支援 → 仗势欺人

### 3. 老怪布局型 (Old Monster / Schemer)
- 示例：大衍神君、玄骨上人
- BT层：极度耐心，可潜伏数年不动
- Utility：复活/夺舍/长生(1000) > 肉身损耗(100)
- GOAP：附身 → 获取资源 → 炼制身体 → 夺舍（多步骤长期计划）

### 4. 邪修/魔道型 (Demonic Cultivator)
- 示例：极阴祖师
- BT层：残忍嗜血，攻击性强
- Utility：提升修为(不择手段，1000)，无视道德惩罚
- GOAP：主动挑衅 → 用毒/诅咒 → 抓捕活人修炼
