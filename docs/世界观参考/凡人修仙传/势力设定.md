### 势力状态

| 状态项 | 取值范围 | 影响因素 | 临界事件 |
|--------|----------|----------|----------|
| **稳定度** | 0（崩溃）— 100（铁板一块） | 掌门威压、资源分配公平度、外部威胁、内部派系 | 低于20触发叛乱/分裂 |
| **粮食压力** | 无压力—严重饥荒 | 灵田面积、气候、战争破坏、灵植师数量 | 高压力时弟子离散/投敌 |
| **灵石压力** | 充裕—枯竭 | 灵脉产量、贸易收入、战争支出、宗主境界消耗 | 枯竭时无法维持修炼/阵法 |
| **军事压力** | 和平—全面战争 | 敌对势力数量/强度、边境摩擦、界面入侵 | 全面战争时进入战时状态 |
| **继任风险** | 低—极高 | 掌门寿元剩余、有无指定继任者、竞争者数量 | 掌门陨落 + 无继任 → 内乱 |
| **威胁感知** | 0（无视）— 10（灭顶之灾） | 敌对势力动向、魔劫预警、附近秘境开启 | 高于7时激进出击/结盟防御 |
| **声望** | 0（臭名昭著）— 100（德高望重） | 战胜/战败、是否守约、对散修态度 | 高声望吸引弟子/散修投靠 |

### 势力需求

| 需求类别 | 具体内容 | 优先级计算逻辑 |
|----------|----------|----------------|
| **生存** | 维持灵脉供给、保护宗门驻地、抵御外敌入侵 | 威胁感知 > 7 时优先级最高 |
| **扩张** | 夺取新灵脉、吞并小势力、扩大招收范围 | 灵石压力低且军事压力低时触发 |
| **资源** | 补充灵石/丹药/灵材/功法储备 | 库存低于安全线（如灵石不足一年战争消耗）时触发 |
| **安全** | 结盟、修建防御阵法、布置哨探 | 周边出现敌对势力集结信号时提升优先级 |
| **复仇** | 为被杀的掌门/长老/弟子复仇 | 触发条件：声望高 + 有足够军事力量 |
| **结盟** | 联姻、签订攻守同盟、交换传承 | 面临共同强敌或需要互补资源时触发 |
| **继任** | 指定接班人、培养下一代、压制竞争者 | 掌门寿元不足百年时优先级急剧上升 |
| **压制叛乱** | 清除内部异己、弹压不满派系 | 稳定度低于30时触发 |
| **声望经营** | 举办论道大会、对外援助、斩妖除魔 | 需要吸引散修或与其他势力竞争时触发 |

**需求权重示例**（以黄枫谷为例，正常时期）：

| 需求 | 权重 | 原因 |
|------|------|------|
| 安全 | 0.30 | 魔道六宗持续施压 |
| 资源 | 0.25 | 灵脉品阶一般，需外部补充 |
| 声望 | 0.15 | 需要吸引散修补给新鲜血液 |
| 继任 | 0.10 | 令狐老祖寿元将近 |
| 扩张 | 0.10 | 被压制，扩张空间有限 |
| 结盟 | 0.10 | 联越抗魔 |

### 势力行为

| 行为 | 描述 | 触发条件 | 消耗 | 产出 |
|------|------|----------|------|------|
| **开山收徒** | 面向散修/凡人招收弟子 | 成员数量不足或需要新鲜血液 | 灵石（考核奖励）、时间 | 新弟子、灵根人才 |
| **征伐** | 主动对敌对势力发起攻击 | 军事压力可控 + 扩张需求高 + 有利窗口 | 海量灵石、丹药、修士生命 | 灵脉、俘虏、资源、声望变化 |
| **结盟** | 与其他势力签订盟约 | 共同威胁感知 > 5 或资源互补需求高 | 外交资源、联姻代价 | 军事同盟、贸易协定、情报共享 |
| **贸易** | 通过商会出售/购买资源 | 灵石过剩或资源短缺 | 灵石/物资 | 所需资源的反向获取 |
| **镇压叛乱** | 清除内部反叛派系 | 稳定度 < 25 | 修士生命（内耗）、灵石 | 稳定度回升、清除潜在威胁 |
| **立储** | 指定下一任掌门/家主 | 掌门寿元不足百年 + 有合适人选 | 培养资源倾斜 | 降低继任风险 |
| **探索秘境** | 组织弟子探索开启的秘境 | 秘境开启 + 军事压力低 | 灵石、丹药、弟子死亡率 | 天材地宝、功法秘术、古宝 |
| **防御** | 加固宗门阵法、增派巡逻 | 威胁感知 > 5 | 灵石（阵法消耗）、人力 | 安全度提升 |
