# 《凡人修仙传》宗门任务体系

> 本文档基于《凡人修仙传》原著及衍生作品整理，为修仙沙盒项目提供可直接复用的宗门任务体系设计参考。

---

## 一、任务类型与难度

### 1.1 任务类型总览

| 大类 | 子类型 | 核心内容 | 典型发布者 |
|------|--------|----------|-----------|
| **日常杂务** | 采集跑腿 | 采集灵药、矿石；送信跑腿、传递物品 | 执事殿 |
| **日常杂务** | 巡逻守卫 | 巡视宗门边界、守卫重要设施 | 执事殿 |
| **日常杂务** | 杂役工作 | 打扫药园、管理灵兽、维护建筑 | 各堂口 |
| **战斗猎杀** | 猎杀妖兽 | 清剿指定区域的妖兽，上缴材料 | 执事殿/坊市 |
| **战斗猎杀** | 清剿据点 | 消灭魔道/邪修据点，铲除威胁 | 执事殿/长老 |
| **战斗猎杀** | 追杀叛徒 | 追杀叛离宗门的修士 | 执法堂 |
| **探索类** | 秘境探索 | 探索古修洞府、秘境遗迹 | 执事殿/长老会 |
| **探索类** | 地图绘制 | 绘制未探索区域的详细地图 | 执事殿 |
| **探索类** | 遗迹发掘 | 发掘远古宗门遗迹 | 长老会/掌门 |
| **委托类** | 炼丹委托 | 替宗门/个人炼制指定丹药 | 炼丹堂/个人 |
| **委托类** | 炼器委托 | 替宗门/个人炼制指定法宝 | 炼器堂/个人 |
| **委托类** | 阵法布置 | 在指定区域布置或修复阵法 | 阵法堂 |
| **护宗类** | 镇守矿脉 | 长期镇守宗门所属灵石矿脉 | 执事殿 |

| **情报类** | 情报收集 | 监视敌对势力、收集情报 | 执法堂/长老 |
| **情报类** | 暗线任务 | 卧底、渗透敌对势力 | 执法堂/掌门 |
| **情报类** | 护送任务 | 护送重要人物或机密物资 | 执事殿 |
| **宗门大比** | 内部比试 | 筑基/结丹/元婴各级别大比 | 长老会 |
| **宗门大比** | 秘境试炼 | 限定修为进入的试炼秘境 | 执事殿/长老 |
| **宗门大比** | 跨服对战 | 与其他宗门/势力的竞技 | 掌门/长老会 |

### 1.2 难度等级体系

| 难度 | 代号 | 推荐修为 | 建议人数 | 风险指数 | 典型任务 |
|------|------|----------|----------|----------|----------|
| **凡级** | F | 炼气期 1-13层 | 单人 | 极低 | 灵药采集、日常跑腿 |
| **黄级** | E | 筑基期 | 单人/2人 | 低 | 猎杀一级妖兽、护送短途 |
| **玄级** | D | 筑基后期/假丹 | 2-5人 | 中低 | 猎杀二级妖兽、情报收集 |
| **地级** | C | 结丹期 | 5-10人小队 | 中 | 秘境探索、清剿据点 |
| **天级** | B | 元婴期 | 10-20人 | 高 | 资源争夺战、大规模战事 |
| **圣级** | A | 化神期 | 不限 | 极高 | 宗门级战争、跨大陆远征 |
| **仙级** | S | 炼虚及以上 | 不限 | 致命 | 界面级冲突、飞升相关 |

### 1.3 难度标识系统

| 标识 | 颜色 | 含义 | 接取限制 |
|------|------|------|----------|
| ☆ | 白 | 新手友好 | 无限制 |
| ★☆ | 绿 | 普通 | 对应修为 |
| ★★☆ | 蓝 | 挑战 | 修为+贡献点要求 |
| ★★★ | 紫 | 危险 | 修为+贡献点+职位 |
| ☠ | 黑 | 致命 | 长老特批 |

## 二、接取条件与奖励机制

### 2.1 接取条件框架

| 条件类型 | 说明 | 示例 |
|----------|------|------|
| **修为要求** | 最低修为境界 | "筑基中期及以上" |
| **贡献点门槛** | 需消耗/冻结贡献点 | "需押金 500 贡献点" |
| **职位要求** | 宗门内部职级 | "内门弟子及以上" |
| **声望门槛** | 宗门或个人声望 | "宗门声望 2000+" |
| **前置任务** | 必须先完成某任务 | "完成《初探黑风岭》后解锁" |
| **限时开放** | 特定时间段可接 | "每月十五日开放" |
| **事件触发** | 特定条件触发 | "兽潮事件期间开放" |
| **队伍要求** | 组队人数限制 | "至少 3 人组队" |
| **职业要求** | 特定副职业 | "炼丹师等级 4 以上" |

### 2.2 奖励体系

#### 2.2.1 基础奖励（所有任务必备）

| 奖励类型 | 说明 | 量级范围 |
|----------|------|----------|
| **灵石** | 通用货币 | F级: 10-100 / A级: 50万-500万 |
| **贡献点** | 宗门内部积分 | F级: 5-20 / A级: 5000-50000 |
| **经验/阅历** | 影响修为提升 | 因难度和完成度浮动 |

#### 2.2.2 额外奖励（部分任务）

| 奖励类型 | 获取方式 | 示例 |
|----------|----------|------|
| **丹药** | 直接发放/丹堂兑换 | 筑基丹、聚灵丹、定颜丹 |
| **功法** | 藏经阁拓印权限 | 地级功法前三层 |
| **法宝/法器** | 炼器堂产出/战利品 | 飞剑、护甲、储物袋 |
| **材料** | 猎杀所得/采集所得 | 妖丹、灵石矿、灵药种子 |
| **洞府** | 灵气浓度更高的修炼府邸 | 乙级洞府使用权（1年） |
| **地位提升** | 贡献点/声望达标 | 外门→内门→真传→长老 |
| **特权重** | 藏经阁高级权限、灵石矿分红 | 年度分红灵石 5000 |
| **人情/关系** | 获得高阶修士的指导/庇护 | "元婴长老指导一次" |

#### 2.2.3 奖励倍率机制

| 条件 | 倍率 | 说明 |
|------|------|------|
| 完美完成 | x1.5 | 超出目标要求 |
| 限时完成 | x1.2 | 在规定时限内提前完成 |
| 首次完成 | x1.3 | 第一次完成该类型任务 |
| 宗门急需 | x2.0 | 当前宗门资源紧缺 |
| 连环任务终章 | x3.0 | 完整完成一个任务链 |

## 三、贡献点与资源体系

### 3.1 贡献点获取途径

| 途径 | 贡献点获取量 | 频率 | 特殊说明 |
|------|-------------|------|----------|
| 日常任务 | 5-30/次 | 每日 | 基础获取方式 |
| 战斗任务 | 50-500/次 | 不定期 | 根据难度和击杀数 |
| 探索任务 | 200-2000/次 | 按月 | 发现新秘境额外奖励 |
| 护宗任务 | 100-1000/月 | 按月结算 | 长期驻守 |
| 情报任务 | 300-3000/次 | 不定期 | 情报质量浮动 |
| 宗门大比 | 500-5000/次 | 按季度 | 名次决定奖励 |
| 捐赠灵石 | 1灵石=1贡献 | 不限 | 每日上限 10000 |
| 捐赠材料 | 按品级浮动 | 不限 | 稀缺材料有额外加权 |
| 炼丹/炼器 | 按产出浮动 | 不限 | 自备材料 vs 宗门材料 |
| 教导弟子 | 按成果浮动 | 按月 | 弟子突破额外奖励 |

### 3.2 贡献点使用体系

| 用途 | 消耗 | 说明 |
|------|------|------|
| **功法兑换** | 100-100000 | 黄级到天级不等 |
| **丹药兑换** | 50-5000 | 各品级丹药 |
| **法宝兑换** | 500-50000 | 从下品法器到上品法宝 |
| **材料兑换** | 10-10000 | 按稀有度定价 |
| **洞府租赁** | 500-50000/年 | 按灵气浓度分级 |
| **藏经阁权限** | 1000-20000 | 解锁更高级别区域 |
| **修炼室使用** | 100-500/天 | 灵气浓度提升修炼效率 |
| **长老指导** | 2000-5000/次 | 高阶修士一对一指导 |
| **宗门拍卖行** | 以物易物 | 特殊物品交易平台 |
| **职位提升** | 按等级提升 | 外门→内门→真传→长老 |
| **保命符** | 5000-20000 | 元婴修士存留的保命手段 |

### 3.3 职位与权限阶梯

| 职位 | 贡献点需求 | 基础修为 | 核心权限 |
|------|-----------|----------|----------|
| **杂役** | 0 | 凡人/炼气初期 | 居住、基本食堂 |
| **外门弟子** | 100 | 炼气中期 | 接取凡级任务、基础功法 |
| **内门弟子** | 1000 | 筑基期 | 接取黄/玄级任务、较好洞府 |
| **核心弟子** | 5000 | 筑基后期 | 地级功法、长老指导 |
| **真传弟子** | 20000 | 结丹期 | 接取地/天级任务、顶级洞府 |
| **执事** | 30000 | 结丹期 | 管理权限、任务发布权 |
| **护法** | 50000 | 元婴期 | 接取天/圣级任务、决策参与 |
| **长老** | 100000 | 元婴后期+ | 宗门决策、资源分配权 |
| **太上长老** | 无上限 | 化神期+ | 终极决策权 |

## 四、典型任务实例模板

### 4.1 任务模板字段定义

以下为沙盒项目可直接复用的任务模板规范：

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `task_id` | string | 唯一任务ID | `TA-001` |
| `task_name` | string | 任务名称 | 《黑风岭猎妖》 |
| `task_type` | enum | 任务类型 | 战斗猎杀 |
| `difficulty` | enum | 难度等级 | 玄级/★★☆ (D) |
| `min_realm` | string | 最低修为要求 | 筑基中期 |
| `recommend_realm` | string | 推荐修为 | 筑基后期 |
| `min_members` | int | 最少人数 | 2 |
| `max_members` | int | 最多人数 | 5 |
| `contribution_cost` | int | 接取消耗贡献点 | 100 |
| `contribution_deposit` | int | 押金贡献点 | 300 |
| `prerequisite_tasks` | list | 前置任务列表 | ["初探黑风岭"] |
| `prerequisite_reputation` | int | 最低声望 | 500 |
| `quest_giver` | string | 发布者 | 执事殿 |
| `availability` | string | 开放时间 | 永久/每月初一 |
| `description` | string | 任务描述（文本） | 详细故事背景 |
| `objectives` | list | 目标列表 | 见下方 |
| `rewards` | object | 奖励 | 见下方 |
| `penalties` | object | 失败惩罚 | 见下方 |
| `time_limit` | string | 时间限制 | 30天 |
| `refresh_interval` | string | 刷新间隔 | 完成后7天刷新 |
| `special_conditions` | list | 特殊条件 | 见下方 |
| `hidden_rewards` | list | 隐藏奖励 | 见下方 |

### 4.5 任务实例四：情报护送类


## 五、任务刷新与调度机制

### 5.1 刷新策略

| 任务类别 | 刷新间隔 | 刷新数量 | 说明 |
|----------|----------|----------|------|
| 日常任务 | 每日 | 5-10个 | 固定池随机抽取 |
| 周常任务 | 每周 | 3-5个 | 难度略高于日常 |
| 月度任务 | 每月 | 2-3个 | 多为探索/委托类 |
| 季度任务 | 每季度 | 1-2个 | 多为秘境/大比 |
| 事件任务 | 触发型 | 不定 | 由世界状态变化触发 |
| 紧急任务 | 即时 | 不定 | 宗门遇袭等突发事件 |
| 连环任务 | 完成型 | 1条 | 完成当前阶段后解锁下一阶段 |

### 5.2 任务池管理

```
总任务池
├── 常规池（60%）：日常、周常、月度任务
├── 事件池（20%）：世界事件触发型任务
├── 紧急池（10%）：宗门危机响应任务
├── 隐藏池（5%）：需特殊条件解锁
└── 特殊池（5%）：宗门大比、秘境探索等
```

## 六、任务系统扩展设计

### 6.1 任务品质与稀有度

| 稀有度 | 颜色 | 出现概率 | 奖励加成 |
|--------|------|----------|----------|
| 普通 | 白色 | 70% | 1.0x |
| 优秀 | 绿色 | 20% | 1.3x |
| 稀有 | 蓝色 | 7% | 1.6x |
| 史诗 | 紫色 | 2.5% | 2.0x |
| 传说 | 金色 | 0.5% | 3.0x |

### 6.2 任务状态机

```
[可接取] → [已接取] → [进行中] → [已完成] → [已领奖]
                ↓            ↓
           [已放弃]      [已失败]
                ↓            ↓
           [冷却中/惩罚期]  [惩罚期]
```

### 6.3 动态难度调整

| 因素 | 影响 |
|------|------|
| 接取人数不足 | 降低推荐人数，降低奖励 |
| 多次失败 | 提升任务奖励（激励完成） |
| 资源过剩 | 降低同类任务刷新率 |
| 宗门战争状态 | 大幅增加战斗类任务 |
