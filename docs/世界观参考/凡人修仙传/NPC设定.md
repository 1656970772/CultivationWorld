### NPC 需求

| 需求类别 | 需求内容 | 优先级 | GOAP 式行为链示例 |
|----------|----------|--------|-------------------|
| **生存** | 避免死亡、伤势恢复、寿元延续 | 最高 | 受重伤 → 寻找高阶疗伤丹药 → 若自己无则去商会购买/求宗门赐药 → 服用后闭关恢复 |
| **修炼** | 突破当前境界 | 高 | 修炼进度100% → 需要破境丹 → 收集丹方 → 收集材料 → 请炼丹师炼制 → 渡劫突破 |
| **疗伤** | 恢复伤势 | 高 | 伤势 > 50 → 识别所需丹药 → 检查背包 → 不足则前往商会/求援 → 服用 → 闭关 |
| **获取资源** | 灵石、丹药、材料、功法 | 中—高 | 缺少关键资源 → 接悬赏任务 → 完成任务获取报酬 → 购买所需资源 |
| **提升地位** | 在势力中晋升 | 中 | 外门弟子 → 积累贡献 → 通过考核 → 晋升内门 → 参加宗门大比 → 获得真传资格 |
| **保护势力** | 宗门/家族存亡关头出力 | 依忠诚度浮动 | 宗门遇袭 → 忠诚 > 70 则参战 → 忠诚 < 30 则考虑逃跑/叛逃 |
| **复仇** | 为亲友/同门报仇 | 依性格浮动 | 重要NPC被杀 → 调查凶手 → 收集情报 → 寻找机会刺杀/借刀杀人 |
| **结交道侣** | 寻找双修伴侣 | 低—中 | 与异性NPC共同经历副本/战斗 → 好感度上升 → 触发结侣事件 |
| **背叛** | 叛逃出当前势力 | 依叛逃倾向浮动 | 叛逃倾向 > 70 → 暗中接触外部势力 → 窃取宗门情报/资源 → 择机叛逃 |
| **继任** | 成为掌门/家主 | 仅特定NPC触发 | 掌门寿元将尽 + 自身为候选人 → 拉拢长老 → 压制竞争对手 → 继位 |
