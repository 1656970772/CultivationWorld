# 门派架构与运行模拟实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**目标:** 建立第一版可真实运行的门派组织、宗门背包、月俸库存压力、统一任务列表、个人悬赏托管、弟子离宗和宗门倒闭闭环。

**架构:** 仍以现有 `FactionEntity` 作为门派主体，不新增独立门派实体；新增 `engine/sect/` 作为门派运行子域。通用任务板属于任务域，放在 `engine/quest/`，由 `QuestBoard`、Repository、StateMachine、VisibilityPolicy、DedupSpecification 和策略注册表组合；门派只作为任务来源与结算策略接入。门派子域采用配置注册表 + 策略规则链：`SectConfigRegistry` 统一读取组织、运行数值和初始化模板，`SectOperationService` 只编排已构建的 `SectOperationContext` 与规则链，月俸、库存压力、离宗、倒闭分别由可注册 `SectOperationRule` 执行。宗门宏观资源继续以 `ResourceRegistry` 管理的 `FactionState` 为单一真相源，丹药、材料、法器等实物保存在 faction inventory，个人悬赏奖励进入独立的悬赏托管仓，避免被势力 state/inventory 同步覆盖。

**技术栈:** JavaScript ES modules、JSON 数据配置、manifest-driven ConfigLoader、现有 WorldEngine / TickManager / EconomicSystem / QuestService / Job-Toil、Node.js 工具验证。

**最后更新:** 2026-06-08

## 文件结构

### 架构与配置化审查结论（2026-06-08）

本计划经多 agent 并行审查后，执行时必须遵守以下硬约束：

1. **禁止把门派运行做成单个上帝服务。** `SectOperationService` 只负责读取上下文、按配置顺序调用规则、汇总结果；月俸、库存压力、离宗、倒闭、后续任命/内部政治都实现为可注册策略规则。
2. **禁止用代码硬编码堂口、任务类型、门派 ID、NPC ID、初始库存、初始道具或阈值。** 这些内容必须来自 JSON 配置：组织定义、运行数值、初始化模板、实体覆盖或任务模板元数据。
3. **初始资源和初始道具全部走配置。** 势力宏观资源使用 `factions.json.resources` 或 `sect-seed-profiles.json.resourceProfiles`；势力实物库存使用 `sect-seed-profiles.json.inventoryProfiles` + `factions.json.inventoryOverrides`；NPC 初始道具使用 `npcs.json.items` 或 `sect-seed-profiles.json.npcStarterKits`；堂口归属使用 `sect-seed-profiles.json.hallAssignmentProfiles` + NPC 显式覆盖。
4. **实体类型必须显式配置。** 核心宗门在 `factions.json` 写 `isSect: true` 与 `sectTemplateId`；功能组织写 `isPublic: true` / `isSect: false`。代码不得再用 `id.startsWith('sect_')` 或“无 subtype 即宗门”推断运行资格。
5. **缺配置是校验失败，不是旧流程回退。** 最新门派运行默认进入正式流程；不得新增“关闭新机制后旧月俸继续跑”的运行时 fallback。测试 fixture 可以构造最小配置，但正式 runtime 不走旧月俸双轨。
6. **配置引用必须校验。** `sect-organization.json`、`sect-operation.json`、`sect-seed-profiles.json` 中引用的 hall、role、rank、item、questTemplate、seed profile 都要在加载验证脚本中校验存在。
7. **任务筛选也走配置。** 任务板与 QuestService 不得用 `qt_slay_monster` 这类硬编码集合判断任务类型，应使用 `quest-templates.json` 的 category/tag/`sectIssuerHints` 或 `sect-operation.json.questSelection`。
8. **通用任务板不得依赖门派领域。** `QuestBoard`、任务仓储、状态机、可见性、去重、任务来源策略和交付处理策略放在 `engine/quest/`；`engine/sect/` 只能注册宗门任务来源、宗门任务结算和个人悬赏处理器。
9. **配置加载必须走 manifest。** 新增运行时 JSON 只登记到 `apps/game/data/config/data-manifest.json`，由 `data-manifest-loader.js` 输出到 `configs`；不得回到 `config-loader.js` 手写文件列表。
10. **宏观资源不得写小白名单。** 宗门灵石、粮食、弟子等 faction state resource 必须经 `ResourceRegistry` 解析；门派实物库存只追加到 faction inventory，不得在 `FactionEntity` 里固定同步 `low_spirit_stone/disciples/food` 三个键。

# ADR-057：门派运行规则链与通用任务板

> 日期：2026-06-08

## 决策

1. 门派仍复用 `FactionEntity`，不新增独立门派实体。
2. 门派运行子域放在 `apps/game/js/engine/sect/`，只处理门派组织、宗门财政、个人悬赏、月俸库存压力、离宗和倒闭。

7. 个人悬赏奖励使用统一经济托管，不进入宗门普通库存，不叠加普通任务模板奖励。
8. 最新门派运行默认进入正式流程，缺配置由 strict validator 报错，不保留旧月俸 fallback。

function clone(value) {
  return value == null ? value : JSON.parse(JSON.stringify(value));
}

export class QuestRepository {
  constructor() {
    this.quests = new Map();
    this.nextId = 1;
  }

  save(input = {}) {
    const id = input.id || `board_quest_${this.nextId++}`;
    const quest = { ...clone(input), id };
    this.quests.set(id, quest);
    return clone(quest);
  }

  byId(id) {
    const quest = this.quests.get(id);
    return quest ? clone(quest) : null;
  }

  update(id, patch = {}) {
    const quest = this.quests.get(id);
    if (!quest) return null;
    Object.assign(quest, clone(patch));
    return clone(quest);
  }

  all() {
    return [...this.quests.values()].map(clone);
  }
}

export class QuestStateMachine {
  constructor({ allowed } = {}) {
    if (!Array.isArray(allowed) || allowed.length === 0) throw new Error('QuestStateMachine 需要配置 allowed states');
    this.allowed = new Set(allowed);
  }

  transition(quest, state, patch = {}) {
    if (!quest || !this.allowed.has(state)) return { success: false, reason: 'quest_state_invalid' };
    return { success: true, patch: { ...patch, state } };
  }
}

export class QuestVisibilityPolicy {
  constructor({ publicBoards } = {}) {
    if (!Array.isArray(publicBoards)) throw new Error('QuestVisibilityPolicy 需要配置 publicBoards');
    this.publicBoards = new Set(publicBoards);
  }

  canSee(quest, { factionId = null, questBoard = null } = {}) {
    if (quest.state !== 'open') return false;
    if (questBoard && quest.questBoard !== questBoard) return false;
    return !factionId || quest.factionId === factionId || this.publicBoards.has(quest.questBoard);
  }
}

export class QuestDedupSpecification {
  hasOpen(repository, dedupeKey) {
    if (!dedupeKey) return false;
    return repository.all().some(q => q.state === 'open' && q.dedupeKey === dedupeKey);
  }
}

export class QuestBoard {
  static fromConfig(config = {}) {
    if (!Array.isArray(config.stateMachine) || !config.visibilityPolicy) {
      throw new Error('QuestBoard.fromConfig 需要 questBoard.stateMachine 与 visibilityPolicy 配置');
    }
    return new QuestBoard({
      repository: new QuestRepository(),
      stateMachine: new QuestStateMachine({ allowed: config.stateMachine }),
      visibilityPolicy: new QuestVisibilityPolicy(config.visibilityPolicy),
      dedupeSpec: new QuestDedupSpecification(config.dedupePolicy),
    });
  }

  constructor({ repository, stateMachine, visibilityPolicy, dedupeSpec } = {}) {
    if (!repository || !stateMachine || !visibilityPolicy || !dedupeSpec) {
      throw new Error('QuestBoard 依赖必须由配置工厂显式注入');
    }
    this.repository = repository;
    this.stateMachine = stateMachine;
    this.visibilityPolicy = visibilityPolicy;
    this.dedupeSpec = dedupeSpec;
  }

  publish(input = {}) {
    const quest = {
      state: 'open',
      createdDay: input.day ?? 0,
      acceptedDay: null,
      completedDay: null,
      cancelledDay: null,
      factionId: input.factionId || null,
      issuerType: input.issuerType || 'faction',
      issuerId: input.issuerId || input.factionId || null,
      issuerName: input.issuerName || input.issuerId || input.factionId || '宗门',
      questBoard: input.questBoard || 'sect',
      questKind: input.questKind || 'sect_task',
      questTemplateId: input.questTemplateId,
      questTypeId: input.questTypeId || input.questTemplateId,
      difficulty: input.difficulty,
      priority: input.priority,
      requiredResourceId: input.requiredResourceId || null,
      dedupeKey: input.dedupeKey || null,
      rewardContribution: input.rewardContribution ?? 0,
      escrowId: input.escrowId || null,
      issuerNpcId: input.issuerNpcId || null,
      acceptedByNpcId: null,
      completedByNpcId: null,
      metadata: clone(input.metadata || {}),
    };
    if (this.dedupeSpec.hasOpen(this.repository, quest.dedupeKey)) {
      return { success: false, reason: 'quest_deduped', dedupeKey: quest.dedupeKey };
    }
    return this.repository.save(quest);
  }

  byId(id) {
    return this.repository.byId(id);
  }

  openFor({ factionId = null, questBoard = null } = {}) {
    return this.repository.all()
      .filter(q => this.visibilityPolicy.canSee(q, { factionId, questBoard }))
      .sort((a, b) => (b.priority || 0) - (a.priority || 0) || a.createdDay - b.createdDay)
      .map(clone);
  }

  hasOpenDemand(dedupeKey) {
    return this.dedupeSpec.hasOpen(this.repository, dedupeKey);
  }

  accept(questId, npc, day = 0) {
    const quest = this.repository.byId(questId);
    if (!quest || quest.state !== 'open') return { success: false, reason: 'quest_not_open' };
    const transition = this.stateMachine.transition(quest, 'accepted', { acceptedDay: day, acceptedByNpcId: npc?.id || null });
    if (!transition.success) return transition;
    return { success: true, quest: this.repository.update(questId, transition.patch) };
  }

  complete(questId, npc, day = 0) {
    const quest = this.repository.byId(questId);
    if (!quest || quest.state !== 'accepted') return { success: false, reason: 'quest_not_accepted' };
    const transition = this.stateMachine.transition(quest, 'completed', { completedDay: day, completedByNpcId: npc?.id || null });
    if (!transition.success) return transition;
    return { success: true, quest: this.repository.update(questId, transition.patch) };
  }

  cancel(questId, day = 0) {
    const quest = this.repository.byId(questId);
    if (!quest || quest.state === 'completed') return { success: false, reason: 'quest_not_cancellable' };
    const transition = this.stateMachine.transition(quest, 'cancelled', { cancelledDay: day });
    if (!transition.success) return transition;
    return { success: true, quest: this.repository.update(questId, transition.patch) };
  }

  snapshot() {
    return this.repository.all();
  }
}

function amount(value) {
  return Math.max(0, Math.floor(Number(value) || 0));
}

export class SectTreasury {
  constructor({ economicSystem = null, config = {} } = {}) {
    this.economicSystem = economicSystem;
    this.config = config;
  }

  settle({ day = 0, scenarioId, type = scenarioId, from, to, asset, source = null } = {}) {
    if (!this.economicSystem) return { success: false, reason: 'economic_system_missing' };
    const normalized = { ...asset, quantity: amount(asset?.quantity) };
    if (normalized.quantity <= 0) return { success: true, quantity: 0 };
    return this.economicSystem.settle({
      day,
      scenarioId,
      type,
      parties: [{ role: 'from', entity: from }, { role: 'to', entity: to }],
      transfers: [{ from, to, asset: normalized }],
      source,
    });
  }

  transferFactionStonesToNpc({ day = 0, faction, npc, quantity, source }) {
    return this.settle({
      day,
      scenarioId: this.config.stipendScenarioId,
      from: faction,
      to: npc,
      asset: { kind: 'faction_state_resource', itemId: this.config.stoneResourceId, quantity },
      source,
    });

import { Inventory } from '../abstract/inventory.js';

export class SectEscrowHolderRepository {
  constructor({ snapshot = [], holderType } = {}) {
    if (!holderType) throw new Error('SectEscrowHolderRepository 需要配置 holderType');
    this.holderType = holderType;
    this.holders = new Map();
    for (const entry of snapshot) this.restore(entry);
  }

  holderFor({ factionId, holderType = this.holderType } = {}) {
    const id = `${holderType}_${factionId}`;
    if (!this.holders.has(id)) {
      this.holders.set(id, {
        id,
        factionId,
        type: holderType,
        name: `${factionId}悬赏托管仓`,
        inventory: new Inventory(),
        state: { get() { return 0; }, set() {} },
      });
    }
    return this.holders.get(id);
  }

  restore(entry = {}) {
    const holder = this.holderFor({ factionId: entry.factionId, holderType: entry.type || this.holderType });
    holder.inventory.loadFrom(entry.inventory || {});
    return holder;
  }

  snapshot() {
    return [...this.holders.values()].map(holder => ({
      id: holder.id,
      factionId: holder.factionId,
      type: holder.type,
      inventory: holder.inventory.toJSON?.() || {},
    }));
  }
}

import { SectTreasury } from './sect-treasury.js';

function rankOrderOf(rankId, ranksData) {
  const rank = ranksData.find(r => r.id === rankId);
  if (!rank) throw new Error(`未知 rankId: ${rankId}`);
  return rank.order;
}

export class SectBountyService {
  constructor({ config, treasuryConfig, ranksData, economicSystem, questBoard, escrowHolders }) {
    if (!config || !treasuryConfig || !ranksData || !escrowHolders) throw new Error('SectBountyService 缺少配置、ranksData 或托管仓');
    this.config = config;
    this.ranksData = ranksData;
    this.economicSystem = economicSystem;
    this.questBoard = questBoard;
    this.treasury = new SectTreasury({ economicSystem, config: treasuryConfig });
    this.escrowHolders = escrowHolders;
  }

  vaultFor(factionId) {
    return this.escrowHolders.holderFor({ factionId });
  }

  canPublish(issuer) {
    const min = this.config.minRankOrder;
    if (!Number.isFinite(min)) throw new Error('personalBounty.minRankOrder 缺失');
    return rankOrderOf(issuer?.state?.get?.('rankId'), this.ranksData) >= min;
  }

  createPersonalBounty({ day = 0, faction, issuer, questTemplateId, difficulty = 1, rewardAssets = [] } = {}) {
    if (!this.canPublish(issuer)) return { success: false, reason: 'rank_too_low' };
    if (!faction || !issuer || !this.questBoard || !this.economicSystem) {
      return { success: false, reason: 'bounty_context_missing' };
    }
    const fee = this.treasury.payBountyFeeToFaction({
      day,
      faction,
      issuer,
      feeItemId: this.config.feeItemId,
      quantity: this.config.feeAmount,
      source: { type: 'personal_bounty_fee', issuerId: issuer.id },
    });
    if (!fee.success) return fee;

    const vault = this.vaultFor(faction.id);
    const escrow = this.economicSystem.openEscrow({
      day,
      purpose: this.config.rewardScenarioId,
      sourceEntity: issuer,
      escrowHolder: vault,
      nominalOwnerId: issuer.id,
      assets: rewardAssets,
      source: { type: 'personal_bounty', issuerId: issuer.id, factionId: faction.id },
    });
    if (!escrow.success) return escrow;

    const quest = this.questBoard.publish({
      day,
      factionId: faction.id,
      issuerType: 'npc',
      issuerId: issuer.id,
      issuerNpcId: issuer.id,
      issuerName: issuer.name || issuer.id,
      questBoard: this.config.defaultQuestBoard,
      questKind: 'personal_bounty',
      questTemplateId,
      difficulty: difficulty || this.config.defaultDifficulty,
      escrowId: escrow.escrowId,
      priority: this.config.defaultPriority,
      metadata: { rewardAssets },
    });
    return { success: true, questId: quest.id, escrowId: escrow.escrowId, quest };
  }

  completePersonalBounty({ day = 0, questId, completer } = {}) {
    const quest = this.questBoard?.byId?.(questId);
    if (!quest || quest.questKind !== 'personal_bounty') return { success: false, reason: 'bounty_missing' };
    const vault = this.vaultFor(quest.factionId);
    const release = this.economicSystem.settleEscrow({
      day,
      escrowId: quest.escrowId,
      holder: vault,
      destination: completer,
      status: 'released',
      source: { type: 'personal_bounty_complete', questId },
    });
    if (!release.success) return release;
    this.questBoard.complete(questId, completer, day);
    return { success: true, questId, escrowId: quest.escrowId };
  }

  cancelPersonalBounty({ day = 0, questId, issuer } = {}) {
    const quest = this.questBoard?.byId?.(questId);
    if (!quest || quest.questKind !== 'personal_bounty') return { success: false, reason: 'bounty_missing' };
    const vault = this.vaultFor(quest.factionId);
    const refund = this.economicSystem.settleEscrow({
      day,
      escrowId: quest.escrowId,
      holder: vault,
      destination: issuer,
      status: 'refunded',
      source: { type: 'personal_bounty_cancel', questId },
    });
    if (!refund.success) return refund;
    this.questBoard.cancel(questId, day);
    return { success: true, questId, escrowId: quest.escrowId };
  }
}

export class MonthlyStipendRule {
  id = 'monthly_stipend';

  run(ctx) {
    const { day, faction, members, config, treasury } = ctx;
    const stipends = config.stipends;
    const roleStones = stipends.roleStones;
    const rankPills = stipends.rankPills;
    const extra = stipends.hallExtraStones;
    const maintenance = config.maintenance;
    if (!roleStones || !rankPills || !extra || !maintenance) throw new Error('sect-operation.stipends/maintenance 配置缺失');
    const territoryCount = number(faction.state.get('territoryCount'));
    let totalStoneDue = number(maintenance.baseStones) + territoryCount * number(maintenance.perTerritoryStones);
    let totalStonePaid = 0;
    let totalPillDue = 0;
    let totalPillPaid = 0;

    for (const npc of members) {
      const role = memberRole(npc);
      const hallExtra = npc.state.get('hallId')
        ? (npc.state.get('isHallChief') === true ? number(extra.chief) : number(extra.member))
        : 0;
      if (!Object.prototype.hasOwnProperty.call(roleStones, role)) throw new Error(`缺少身份月俸配置: ${role}`);
      const stoneDue = number(roleStones[role]) + hallExtra;
      totalStoneDue += stoneDue;
      const stone = treasury.transferFactionStonesToNpc({ day, faction, npc, quantity: stoneDue, source: { type: this.id, factionId: faction.id, npcId: npc.id } });
      if (stone.success) totalStonePaid += stone.record?.assets?.[0]?.asset?.quantity ?? stone.quantity ?? stoneDue;

      const pillRule = rankPills[memberRank(npc)];
      if (pillRule?.itemId && pillRule.quantity > 0) {
        totalPillDue += pillRule.quantity;
        const pill = treasury.transferFactionItemToNpc({ day, faction, npc, itemId: pillRule.itemId, quantity: pillRule.quantity, source: { type: 'sect_pill_stipend', factionId: faction.id, npcId: npc.id } });
        if (pill.success) totalPillPaid += pill.record?.assets?.[0]?.asset?.quantity ?? pillRule.quantity;
      }
    }

    const stoneShort = totalStonePaid < totalStoneDue;
    const pillShort = totalPillPaid < totalPillDue;
    faction.state.set('sectLastMonthlyDay', day);
    faction.state.set('sectLastStoneDue', totalStoneDue);
    faction.state.set('sectLastStonePaid', totalStonePaid);
    faction.state.set('sectLastPillDue', totalPillDue);
    faction.state.set('sectLastPillPaid', totalPillPaid);
    faction.state.set('sectSalaryShortfallStreak', stoneShort ? number(faction.state.get('sectSalaryShortfallStreak')) + 1 : 0);
    faction.state.set('sectPillShortfallStreak', pillShort ? number(faction.state.get('sectPillShortfallStreak')) + 1 : 0);
    if (stoneShort || pillShort) {
      faction.state.set('stability', Math.max(0, number(faction.state.get('stability'), 50) - number(config.decline.stabilityPenaltyPerMonth)));
    }
    return { totalStoneDue, totalStonePaid, totalPillDue, totalPillPaid, stoneShort, pillShort };
  }
}

export class DepartureRule {
  id = 'departure';

  run(ctx) {
    const { day, faction, members, config, rng } = ctx;
    const cfg = config.decline;
    if (!cfg) throw new Error('sect-operation.decline 配置缺失');
    const streak = Math.max(number(faction.state.get('sectSalaryShortfallStreak')), number(faction.state.get('sectPillShortfallStreak')));
    const lowStability = number(faction.state.get('stability'), 50) <= number(cfg.stabilityForDeparture);
    const threshold = number(cfg.shortfallStreakForDeparture);
    const leftNpcIds = [];
    if (streak < threshold && !lowStability) return { leftNpcIds };
    const exempt = new Set(cfg.exemptRoles || []);
    for (const npc of members) {
      if (exempt.has(memberRole(npc))) continue;
      const chance = Math.min(1, number(cfg.leaveChanceBase) + Math.max(0, streak - threshold) * number(cfg.leaveChancePerShortfallStreak));
      const roll = rng?.next?.() ?? Math.random();
      if (roll > chance) continue;
      npc.state.set('factionId', null);
      npc.state.set('hasFaction', false);
      npc.state.set('isWanderer', true);
      npc.state.set('hallId', null);
      npc.state.set('isHallChief', false);
      npc.state.set('sectLeftDay', day);
      npc.state.set('sectLeaveReason', lowStability ? 'low_stability' : 'stipend_shortfall');
      leftNpcIds.push(npc.id);
    }
    if (leftNpcIds.length > 0) addState(faction, 'sectDepartureCount', leftNpcIds.length);
    return { leftNpcIds };
  }
}
