# 十二境界运行时主链重构 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将运行时境界主链从当前低阶链重构为 12 档：凡人、炼气、筑基、金丹、元婴、化神、炼虚、合体、大乘、渡劫、地仙、天仙；同步修炼门槛、寿元、战斗属性表、妖兽阶位、默认战斗开关、测试与文档，并清理“大罗天仙 / 道祖”作为运行时表项的残留。

**Architecture:** 数据优先迁移。`apps/game/data/definitions/ranks.json` 作为运行时境界枚举，`apps/game/data/balance/cultivation.json` 作为修炼数值单一真相源，三张战斗属性表作为同槽位数值映射表。运行时代码只补齐硬编码 rank 顺序和展示映射，不引入新的境界推导逻辑。测试先锁定 12 档链、旧高阶禁用、数值镜像、默认开关与斩妖战斗路径，再改数据和文档。

**Tech Stack:** Node.js ES modules, JSON data configs, PowerShell on Windows, existing `apps/game/tools/test-*.mjs` scripts, long-running simulation via existing game analysis tools.

## Task 1: TDD Guard For The Twelve-Realm Runtime Chain

- [ ] Update `apps/game/tools/test-rank-cleanup.mjs` to assert the exact 12 runtime ranks.
  - Replace the old six-rank `allowedRanks` with `EXPECTED_REALMS.map((realm) => realm.id)`.
  - Assert `ranks.length === 12`.
  - Assert each rank appears at its exact index.
  - Assert each rank has the expected Chinese name and lifespan numbers.
  - Assert `great_luo_heaven_immortal` and `dao_ancestor` are absent from all runtime ranks.
  - Assert `cultivationRequired` and `qiRequired` both match the threshold table.

```js
const bannedRuntimeRanks = ['great_luo_heaven_immortal', 'dao_ancestor'];

for (const bannedRank of bannedRuntimeRanks) {
  if (rankIds.includes(bannedRank)) {
    errors.push(`旧高阶境界不应作为运行时 rank 出现: ${bannedRank}`);
  }
}

if (rankIds.length !== EXPECTED_REALMS.length) {
  errors.push(`运行时 rank 数量应为 ${EXPECTED_REALMS.length}，实际为 ${rankIds.length}`);
}

EXPECTED_REALMS.forEach((realm, index) => {
  const actual = ranks[index];
  if (!actual || actual.id !== realm.id) {
    errors.push(`第 ${index + 1} 档境界应为 ${realm.id}，实际为 ${actual?.id ?? '缺失'}`);
    return;
  }

  if (actual.name !== realm.name) {
    errors.push(`${realm.id} 中文名应为 ${realm.name}，实际为 ${actual.name}`);
  }

  if (actual.cultivationRequired !== realm.threshold || actual.qiRequired !== realm.threshold) {
    errors.push(`${realm.id} cultivationRequired/qiRequired 应同为 ${realm.threshold}`);
  }

  const expectedLife = LIFESPAN_BUCKETS[realm.id];
  for (const field of ['baseLifespan', 'lifespanVariance', 'maxLifespan']) {
    if (actual[field] !== expectedLife[field]) {
      errors.push(`${realm.id}.${field} 应为 ${expectedLife[field]}，实际为 ${actual[field]}`);
    }
  }
});
```
