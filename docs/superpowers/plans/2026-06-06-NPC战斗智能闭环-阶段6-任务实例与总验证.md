## Task 1: 嵌套任务实例

- [ ] **Step 1: 写入 `activeQuestInstance`**

`writeQuestState()` 写：

```js
{
  id,
  templateId,
  type,
  name,
  category,
  difficulty,
  value,
  riskKey,
  riskScore,
  source,
  state: 'accepted',
  target: {
    kind,
    x,
    y,
    monsterIds,
    monsterName,
    monsterGrade,
    requiredKills,
    killedCount
  },
  rewards
}
```

ID 必须确定性生成，使用 `currentDay`、NPC id 和计数，不用 `Date.now()`。

- [ ] **Step 2: 绑定目标同步嵌套字段**

`bindMonsterHuntTarget()` 更新 `activeQuestInstance.target`，同时派生旧字段 `questTargetX/Y/MonsterId`。

- [ ] **Step 3: 多目标击杀进度**

成功击杀后增加 `target.killedCount`。未达 `requiredKills` 时继续任务；达到后才设置 `questComplete=true`。

- [ ] **Step 4: 目标失踪失败**

阶段 3 已处理重定向。若仍找不到目标，写 `activeQuestInstance.state='failed'` 与 `failureReason='target_lost'`。
