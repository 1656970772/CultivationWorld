## 已修复的最终集成缺口

- `RelationshipSystem` 已读取 `relationshipPlatform.projections.legacyEdges`，旧边投影不再依赖代码内映射表。
- 地图编辑器已读取 `apps/editor/data/adapters/map-editor.json`，tile 字段和选项来源不再写死在 UI 代码中。
- `WorldEngine` 与 `WorldContextBuilder` 已把 `monsterResourceRules` 传入运行时上下文，妖兽资源族成为单一配置来源。
- `FactionTradeExecutor` 已改用 `FactionStrategyRegistry` 与 `economicTransactionConfig`，不再写死 `low_spirit_stone/food`。
- `verify-gas-combat.mjs` 已改为 manifest 驱动加载，避免手写配置遗漏 behaviorTrees、aiConfig 等正式数据组。
