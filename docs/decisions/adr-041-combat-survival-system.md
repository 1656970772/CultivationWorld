# ADR-041：战斗与生存系统（HP / 真实扣血 / 锁血 / 遁地符 / 回血丹 / 天材地宝 / 机缘突破）

最后更新：2026-06-03

状态：分阶段实施中。**阶段1（HP+真实扣血+锁血）已实现 2026-06-02**；阶段2（门派庇护+遁地符）待做。

> **2026-06-03 更新（ADR-042）**：阶段1 散落在三处的"伤害/锁血/致死"逻辑（妖兽攻击、风险 hp_damage、PvP killNPCByPvP）已收编为 [ADR-042](adr-042-gameplay-ability-system.md) 的**统一伤害管线 GAS 机制层**。**锁血从此拆为独立的 `effect_lock_hp` 保命 Effect**，由 `ability_lock_hp` 能力授予（可挂在遁地符、特殊体质等多来源），**不区分攻击者（妖兽/PvP/风险通用）**；遁地符额外提供 `ability_escape_talisman` 瞬移逃脱。本 ADR 阶段2B "PvP 致死纳入真实扣血/锁血" 的前置缺口由 ADR-042 的 `combat-pipeline.applyDamage` 一并解决。后续战斗/生存机制以 ADR-042 的 Effect/Ability/Tag 为承载形态。

## 背景

模拟中反复出现：**高天赋低境界 NPC（如天才弟子 npc_999 林天骄）在野外/门派活动时被妖兽"概率秒杀"**，
往往 24~89 天内即殒命，根本来不及展现天赋、修炼出头。根因有两条：

1. **NPC 没有 HP（气血）系统**：NPC state 只有 `injuryLevel`，没有 `hp/maxHp`。
2. **妖兽攻击是"概率直接致死"**：`monster-entity._attack` 用 `roll < winChance × killChanceFactor`
   一击决定生死；`settleRisk` 的行为风险也用 `type:'death'` 概率直接抹杀。低境界 NPC（order=0）
   对任何妖兽都几乎裸奔。

这与世界观参考矛盾——《凡人修仙传》中即便低阶散修野外死亡率高，但顶尖修士靠**保命手段（遁地/瞬移、
"韩跑跑"哲学）、回血/护身丹药、灵宠代死**屡屡死里逃生；修为提升也主要靠**机缘、丹药、秘境、天材地宝**，
而非闷头闭关（见 `docs/世界观参考/凡人修仙传/妖兽与修士关系分析.md`）。

用户明确要求（2026-06-02）：
> 增加遁地符箓，可以贡献兑换，天才自带两个，血量少于5%可以瞬移到某个随机位置；如果一次就死了（比如10%）
> 那么也会锁血到5%。然后增加血量和攻击，现在有的机制都改成真实的攻击和扣血。除了秘境等没做的还用概率触发死亡。
> 然后增加回血的弹药和天材地宝。都是要和境界挂钩。

以及节奏方向：
> 这才合理啊，本来就要靠机缘、丹药、秘境、天材地宝等等（指修为提升不应纯靠闭关，外部资源是主路径）。

## 决策

### 总原则

- **真实攻防扣血取代概率秒杀**：战斗结算改为 `伤害 → 扣 hp → hp<=0 才死`。
  概率直接死亡**仅保留**于尚未做实体战斗的内容（秘境、天劫、机缘事件）。
- **一切与境界挂钩**：HP、攻击、防御、丹药/天材地宝效果、保命阈值都随境界 order 缩放。
- **数据驱动**：全部数值走 `data/balance/combat.json`、`data/items/`、`data/balance/economy.json`，
  代码只读取执行（遵循 AGENTS.md 数据驱动规则）。
- **复用现有框架**：妖兽已是 BT 分档 + hp + grade→order 映射 + 确定性 rng 的良好结构，直接扩展，不重写。

### 一、NPC HP（气血）系统 — 阶段1

NPC state 新增 `hp` / `maxHp`：

```
maxHp = combat.npcHp.baseHp[rankId] × physique.hpBonusMultiplier
```

- `combat.npcHp.baseHp` 按境界递增（凡人最脆、化神极厚）。
- 体质（physique，如先天道体）提供 `hpBonusMultiplier`。
- 初始化时 `hp = maxHp`；**突破成功后** `maxHp` 重算抬高，但**当前 `hp` 不回满**（保持原值夹新上限）。
  突破回满血是少数功法/秘法/体质才有的特殊效果（见 ADR-041/ADR-042 阶段3.5），普遍机制下突破只抬上限。
- 自然恢复：每日（或休整/疗伤时）按 `combat.npcHp.dailyRegenRatio × maxHp` 缓慢回血。

### 二、真实攻防扣血 — 阶段1

妖兽攻击 NPC（`monster-entity._attack` 改造）：

```
攻击力 atk   = 妖兽 power（已有，与 grade 挂钩）
NPC 防御 def = combat.npcCombat.baseDef[rankId]（与境界挂钩）
伤害 dmg     = max(1, atk × (1 - def 减免) × random(0.8,1.2))
NPC.hp      -= dmg
若 hp<=0 → 进入【锁血判定】（见三），否则 NPC 受伤存活
```

- NPC 反击妖兽（扣妖兽 hp）逻辑已存在，保留。
- `killChanceFactor` 概率秒杀路径**移除**（妖兽对 NPC），改为上面真实结算。
- `settleRisk` 中的 `type:'death'`：野外游历/猎妖的"硬概率死"改为"扣 hp"；
  仅"秘境/天劫"等占位事件保留概率死。

### 三、锁血机制 — 阶段1

当一次伤害会致死（`hp - dmg <= 0`）时：

- **若伤害未达"碾压阈值"**：锁血到 `5% × maxHp`，NPC 不死（给保命道具/逃跑机会）。
- **若达到碾压阈值则直接死**：碾压判定 = 妖兽 order 高出 NPC ≥ `combat.lockHp.crushOrderGap`（默认 2 个大境界阶），
  或单击伤害 ≥ `maxHp × combat.lockHp.crushHpMultiple`。碾压级无法锁血（高阶碾压低阶符合世界观）。
- 锁血触发后进入"濒死"标记，配合阶段2遁地符/阶段3回血丹脱险。

### 四、遁地符箓 — 阶段2

- **前置缺口（2026-06-02 真实模拟发现）**：PvP 致死 `killNPCByPvP()` 当前是"直接 alive=false"，
  绕过 HP/锁血（阶段1 只改了妖兽攻击与野外/猎妖风险）。故仇杀/劫掠/夺权场景没有"濒死"时机，
  遁地符即便实现也不会触发。**遁地符在 PvP 生效的前提：先把 PvP 致死纳入真实扣血/锁血**（见阶段2B 待做）。
- 新道具 `item_escape_talisman`（遁地符），与境界挂钩的多个品阶（低阶符对高阶威胁可能失效，待定）。
- 获取：宗门贡献兑换（新增 `act_npc_redeem_escape_talisman`）；**天才 NPC 初始自带 2 个**。
- 触发：`hp < 5% × maxHp`（含锁血后，妖兽/PvP 通用）自动消耗 1 个 → 瞬移到地图随机安全位置 +
  清除妖兽/仇人对其 target/锁定。无符则不触发。

### 五、回血弹药 + 天材地宝 — 阶段3

- 回血丹 `item_heal_pill`：服用回复 hp，**低阶丹对高境界回血递减**（复用 ADR-040 的 rankDecay 机制）。
- AI：`hp` 低于阈值时新增"疗伤/服回血丹"需求与行为，主动回血。
- 天材地宝：稀有道具，回血/淬体/助突破，分品阶与境界挂钩（部分并入阶段4）。

### 六、机缘 / 天材地宝加速突破 — 阶段4

- 游历/秘境/任务有概率获得**机缘**（给大量 insight/qi/progress）与**天材地宝**。
- 高境界突破主要依赖这些外部资源（纯闭关到不了元婴是**有意设计**，见 ADR-040 节奏结论）。
- 让天才靠"闭关打底 + 机缘/丹药/秘境/天材地宝"在几十年内冲到元婴。

## 实施阶段（逐步推进，不一次做完）

1. **阶段1（已实现 2026-06-02）**：NPC HP 系统 + 妖兽真实扣血 + 锁血到5%。
   实测（5 种子真实模拟）：天才存活从 9~89 天提升到 128~170 天，锁血对非碾压伤害生效。
   遗留：凡人 maxHp 低、grade2 妖兽对其仍构成碾压（符合世界观），需阶段2门派庇护+遁地符根治。
   关键根因：妖兽 `_findPrey` 不区分 NPC 是否在势力领地内。
2. **阶段2**：遁地符（兑换 + 天才自带2个 + 血<5%瞬移）。
3. **阶段3**：回血丹 + AI 疗伤行为。
4. **阶段4**：天材地宝 + 机缘加速突破。
5. **阶段5**：跨种子真实模拟统计验证 + 完善本 ADR + 同步 Wiki。

## 验证方式（遵守 AGENTS.md 验证规则）

- **禁止旧摘要回归验证**：不给天才开免疫/锁血特权、不隔离妖兽、不造作弊脚本自证。
- 用正常配置的世界引擎跑**真实长程模拟**（多种子、足够天数、全量 NPC），以统计结果为准：
  关注低境界弟子平均存活天数、天才能否在寿命内推进境界、各境界 NPC 死因分布是否合理。

## 与既有 ADR 的关系

- 与 ADR-040（真气/进度同步 + 聚气丹境界递减 + 秘境改为给 insight）共同构成"修炼-生存"经济闭环。
