# 项目概览

> 最后更新：2026-05-30

## 一句话描述

网页版修仙世界沙盒模拟器。世界按规则自动演化，玩家作为观察者和干预者，通过移动和事件选择影响世界走向。

## 核心体验

打开网页，看到一个活的修仙世界在不断变化——宗门兴衰、势力攻伐、妖潮来袭。你是其中一个行走者，走到哪里就能干预哪里的命运，但你走不到的地方，历史照样发生。

## 核心关键词

- **叙事沙盒** —— 世界自动演化，玩家选择性介入
- **规则驱动** —— 纯规则系统，不依赖 AI API
- **有限视角** —— 神识范围 + 信息传播，玩家不是全知的
- **蝴蝶效应** —— 玩家的每一个选择都可能改变世界走向

## 当前实现范围（2026-05-30）

> 下列为当前代码与数据的真实状态。早期「第一版 100×100」目标见 `docs/plans/implementation-plan.md`（已标为历史计划）。

- 300×300 固定地图（90,000 格），7 种地形
- 约 18 个势力、150+ NPC（以 `apps/game/data/` 为准）

### 做的

- ~~100×100 固定地图~~ → 已扩展为 300×300
- 20-30 个势力（正邪中立妖凡）
- NPC 领袖性格驱动决策

### 不做的（后续迭代）

- 随机世界生成

- 多人模式
- 策略经营元素
- 音效 / 音乐
- 综合实力值（实力由领地、资源、NPC 等综合体现）
