## 关系解读

### 核心对立轴

```
正派联盟（青云+天剑+玄真）
        ↕ 对立
邪派阵营（血煞+幽冥+毒蝎）

      妖族（万妖山+蛮蛟）
        ↕ 对立
      人族（正邪+凡人）

凡人王朝 ←依附→ 正派联盟
```

### 不稳定因素

1. **天剑宗的冲动** — 可能打破正邪平衡
2. **幽冥教的算计** — 可能背刺血煞门
3. **蝎王的忠诚度** — 随时可能倒戈
4. **妖族内部** — 万妖山与蛮蛟族的关系不稳定
5. **大晋衰退** — 如果大晋崩溃，东海沿岸成真空
6. **药王谷的安全** — 如果有人敢灭药王谷，将引发全局公愤
