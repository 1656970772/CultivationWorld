# 大陆地理与分区

> 最后更新：2026-05-23

## 玄天大陆地图分区

100×100 的地图分为 **5 大区域**，每个区域有自己的地理特征和主要势力。

```
          北 (y=0)
    ┌─────────────────────┐
    │     北荒（妖兽之地）   │  y: 0~19
    │   山脉/森林/沼泽为主   │
    ├─────────────────────┤
    │   中原（正派核心区）    │  y: 20~44
    │   平原/山脉/灵脉为主   │
    ├──────────┬──────────┤
    │ 西域     │   东海沿岸  │  y: 45~69
    │ 沙漠/荒原│   平原/森林  │
    │ (x:0~49) │  (x:50~99) │
    ├──────────┴──────────┤
    │   南疆（蛮荒之地）     │  y: 70~99
    │   沼泽/森林/灵脉为主   │
    └─────────────────────┘
          南 (y=99)
```

## 各区域特征

### 北荒（y: 0~19）
- **地形：** 山脉、森林、沼泽混杂
- **特征：** 妖兽横行，人迹罕至
- **主要势力：** 妖族（万妖山、蛮蛟族）
- **资源：** 灵药丰富，但危险极高

### 中原（y: 20~44）
- **地形：** 广袤平原 + 灵脉山脉
- **特征：** 修仙文明中心，宗门林立
- **主要势力：** 正派三大宗门（青云宗、天剑宗、玄真观）
- **资源：** 灵脉密集，灵石丰富

### 西域（x: 0~49, y: 45~69）
- **地形：** 沙漠、荒原为主，零星绿洲
- **特征：** 环境恶劣，法外之地
- **主要势力：** 邪派（血煞门、幽冥教）、散修
- **资源：** 稀缺，主要靠掠夺

### 东海沿岸（x: 50~99, y: 45~69）
- **地形：** 平原、森林、河流
- **特征：** 凡人王朝核心区域，繁华富庶
- **主要势力：** 大晋王朝、中立宗门
- **资源：** 粮食充足，灵石较少

### 南疆（y: 70~99）
- **地形：** 密林、沼泽、隐秘灵脉
- **特征：** 古老而神秘，上古遗迹密布
- **主要势力：** 妖族分支、神秘宗门（天机阁）
- **资源：** 隐藏的上古灵脉
