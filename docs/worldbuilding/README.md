# 世界观设定

> 最后更新：2026-05-27
> 定位：第一版够用的世界观框架，后续迭代丰富

## 文档结构

```
worldbuilding/
├── README.md              # 本文件：世界观导航
├── continent.md           # 大陆地理与分区
├── factions.md            # 势力/宗门列表与设定
├── npcs.md                # 关键 NPC 设定
├── relations.md           # 势力关系网络
├── history.md             # 世界背景与纪元
└── wiki/
    ├── README.md          # 已敲定设定的 Wiki 导航
    ├── wiki-template.md   # Wiki 条目模板
    └── rules/
        └── natural-death.md # 自然死亡规则
```

## Wiki 维护入口

- `wiki/README.md` —— Wiki 导航、分类约定与维护规则。
- `wiki/wiki-template.md` —— 新增设定条目时复制使用的通用模板。
- `wiki/rules/natural-death.md` —— NPC 自然死亡规则。
