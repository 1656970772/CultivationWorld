# 怀璧其罪与江湖热点规则

> 最后更新：2026-05-30
> 状态：已敲定（基础版）
> 类型：规则
> 关联文档：`docs/decisions/adr-024-info-propagation-opportunity.md`、`docs/decisions/adr-025-item-covet-system.md`、`docs/systems/opportunity-system.md`、`docs/systems/item-covet.md`

## 一句话定义

修士所持功法/法宝/财物价值过高（身家 assetScore 暴露）一旦被人目击，消息便经口耳、坊市、宗门情报网扩散，强者闻讯而来——可能仗着实力与身份直接劫掠（甚至杀人夺宝），也可能因同门情面、恩义、道侣关系而放他一马。

## 世界观依据

- **杀人夺宝**：参考自《凡人修仙传》——韩立早期屡因身怀珍稀（如灵药、瓶罐、储物袋）遭觊觎追杀；散修间为天材地宝、秘境机缘相互厮杀是常态。
- **财不露白**：修仙界共识，珍宝当深藏，露白招祸。
- **情面放过**：原著中同门、受恩者、敬重的长辈往往得以幸免，体现修仙江湖的人情与势力约束。

## 已敲定内容

### 身家（assetScore）的"璧"

- 由灵石 + 可转移物品估值 + 已装备法宝 + 功法品阶共同构成。
- 物品分材料/丹药/法宝，高品阶法宝（古宝/灵宝）价值连城，最招祸。

### 暴露与传播

- 高身家修士被周围目击者看到，按概率暴露，生成"怀璧其罪"消息与"怀璧之人"机会点。
- 消息经天地异象/口耳相传/城镇广播/宗门情报网/商会情报网扩散至远处强者。

### 抢夺 vs 放他一马

- **起贪念**：目标够富、自身实力占优（含安全系数）、贪婪/勇敢性格。
- **放他一马**：同门、对方为受保护职位（掌门/继承人/长老）、对其有恩义、道侣关系、自身正义/外交高 —— 任一或累加达阈值则放过。
- **抢夺**：按战力胜负转移可转移物品、部分灵石、夺取法宝，并有概率杀人夺宝；被抢者结仇，可能引发后续复仇。

## 涌现现象

- 散修怀宝远行如履薄冰；宗门弟子因身份庇护相对安全。
- 一件灵宝出世，消息扩散后引来多方强者争夺，形成江湖热点。
- "得宝—走漏风声—强者来夺—因关系放过 / 抢夺并结仇"的故事链自发涌现。

## 数据来源

- 参考自凡人修仙传（杀人夺宝、秘境争夺、储物袋遭觊觎）。
- 放过判定的身份/恩义约束为基于原著人情逻辑的项目实现。
