# 势力危亡抉择与凝聚力规则

> 最后更新：2026-06-01
> 状态：已敲定（v5 实现）
> 类型：规则
> 关联文档：`docs/decisions/adr-034-population-sustainability.md`、`docs/decisions/adr-035-faction-cohesion-crisis.md`、`docs/worldbuilding/wiki/rules/personality.md`、`docs/data-models/relationship.md`

## 一句话定义

当势力遭受重创（攻战大败、稳定度暴跌、濒临覆灭）时，成员**不会全员死战**——每个幸存成员根据自己的性格、利益、与势力的关系和当前处境，做出一次个体抉择。势力的"凝聚力"是这些个体抉择的统计涌现：忠勇者多则难灭，一盘散沙则树倒猢狲散。

## 设计动机

此前（v4 及以前）攻战只是机械地"随机处死几个成员 + 数值托底"，成员完全被动，没有人的选择。这既不真实，也使势力覆灭无法自然发生（见 ADR-034 未解问题）。真实修仙小说里，宗门危亡时人的反应丰富多样，且由性格与利益驱动。

## 危亡反应谱系（7 类）

| 反应 | 主要触发倾向（性格 / 处境） | 后果 | 世界观来源 |
|------|------------------------------|------|------------|
| **死战** | 高忠诚 + 高勇气；领袖/继承人/核心；与宗门有师徒、同门深关系 | 留守，正常承受被杀风险 | 凡人修仙传"危难挺身"、仙逆"守护型舍命保护" |
| **退避隐忍** | 高谨慎 + 低勇气；自身实力远逊于来犯者 | 暂离战场、降低本轮被杀概率，仍留在门派 | 凡人修仙传"谨慎型实力不足则退避、隐藏实力" |
| **叛投敌方** | 低忠诚 + 高野心 + 存在更强的敌对招揽方 | 改投攻方势力，带走战力 | 大道争锋"利益压力下背叛""弃徒转盟友" |
| **出走散修** | 中等忠诚 + 高谨慎 + 门派已濒危 | 脱离门派沦为散修，门派真实人口流失 | 凡人修仙传"散修在阵营间灵活切换" |
| **被迫效忠** | 低野心 + 无处可去 + 攻方强势 | 留下但忠诚受损（埋下后续隐患） | 大道争锋"精血誓言被迫效忠" |
| **抢先逃命** | 极低忠诚 / 极高谨慎，第一批离场 | 最早脱离，几乎不被杀，沦为散修 | 凡人修仙传"作鸟兽散" |
| **投降归顺** | 群体性：门派整体实力悬殊、领袖陨落或稳定度崩溃时倾向集体归顺 | 整门并入攻方（成员改换门派），原势力覆灭 | 凡人修仙传"弱宗归顺强宗"、大道争锋"宗门吞并" |

## 凝聚力（cohesion）= 涌现量，不是硬编数值

- 势力不持有一个固定的"凝聚力分数"；凝聚力是其成员在危亡抉择中"选择死战/留下"的比例的统计涌现。
- 高忠诚、高勇气成员占比高的宗门 → 大多死战 → 真实活 NPC 不会骤降 → 难以覆灭。
- 成员忠诚普遍低的宗门 → 大量叛投/出走/逃命 → 真实活 NPC 骤降 → 触发 ADR-034 的覆灭动态阈值而灭门。
- 因此"有的势力有凝聚力、有的没有、有的容易覆灭"是个体选择的自然结果，而非数值判断。

## 抉择机制（数据驱动）

- 仅在势力"危机态"触发：攻战大败后、稳定度低于阈值、或本势力真实活 NPC 数已很低。
- 对每个幸存成员计算各反应的权重 `weight = base × Π(性格/关系/处境调制因子)`，按权重随机抽取其抉择（保留随机性，避免完全确定化）。
- 核心保护：领袖、继承人、与宗门强关系（师徒/核心同门）的成员，死战权重被显著放大，避免顶梁柱轻易叛逃。
- 投降归顺为群体判定：当势力整体濒危（领袖陨落 / 稳定度崩溃 / 实力悬殊）时，触发整门归顺，成员批量改投攻方。
- 全部系数、阈值与 `enabled` 开关数据驱动于 `apps/game/data/balance/cohesion.json`，`enabled=false` 时退化为 v4 行为（零漂移可回退）。

## 叙事表现

- 死战：可作为"率众死守""血战不退"的高强度日志。
- 退避/逃命：表现为"避其锋芒""仓皇出逃"。
- 叛投/出走/投降：表现为"临阵倒戈""弃门而走""举宗归降"，并应在关系网/消息中留痕（如对原门派记 `betrayed`/`sect_destroyed` 记忆，对新归属建关系边）。

## 规则边界

- 本规则只处理"势力危亡时的成员去留抉择"，不改变攻战胜负判定与战力公式。
- 不替代自然死亡、复仇 PvP、晋升继任等既有规则，只在攻战重创结算处插入个体抉择。
- 玩家干预（保下某人、阻止叛逃等）留作后续扩展。

## 待扩展

- 被迫效忠者后续的"伺机反叛/复国"叙事。
- 投降归顺后攻方对降众的消化（同化 / 猜忌 / 清洗）。
- 凝聚力受领袖个人魅力、宗门待遇（薪俸/晋升公平度）长期影响。
