# Wiki 条目模板

> 最后更新：YYYY-MM-DD
> 状态：草案 / 已敲定 / 待复核 / 废弃
> 类型：规则 / 人物 / 势力 / 地点 / 物品 / 历史 / 事件
> 关联文档：`docs/...`

## 一句话定义

用一句话说明这个设定是什么，以及它在世界中解决什么问题。

## 已敲定内容

- 记录已经讨论确认的设定。
- 每条尽量保持可验证、可迁移到数据或规则实现。
- 不确定的内容不要混入本节。

## 叙事表现

- 说明玩家或观察者会如何感知这个设定。
- 说明它会带来哪些世界变化、日志描述或剧情后果。

## 规则边界

- 说明该设定会影响哪些对象。
- 说明该设定不会处理哪些情况。
- 说明与其他规则冲突时的优先级。

## 数据与实现提示

- 记录给数据模型、事件系统或模拟逻辑的实现参考。
- 只写方向和约束，不在 Wiki 中粘贴具体代码。

## 待扩展

- 记录尚未敲定但可能需要补充的内容。
